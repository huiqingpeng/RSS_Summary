LucidFlux: Caption-Free Photo-Realistic Image Restoration via a Large-Scale Diffusion Transformer

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了LucidFlux，一种无需图像标题的图像恢复框架，通过适配大规模扩散Transformer（Flux.1）实现照片级真实感恢复。该框架针对现有方法在复杂退化场景下易出现的过度平滑、幻觉和语义漂移问题，设计了轻量级双分支条件器和自适应调制策略，在无需文本提示或VLM标题的情况下实现了结构保持与纹理恢复的平衡。实验表明，LucidFlux在合成与真实场景基准上均优于开源和商业基线方法，验证了"何时、何处、以何种方式条件化"是大型DiT图像恢复的关键设计杠杆。

方法概述 / Method

LucidFlux采用轻量级双分支条件器，分别从退化输入和轻恢复代理图像中提取信号，分别用于锚定几何结构和抑制伪影；同时设计了时间步和层自适应的调制调度策略，在骨干网络层级间路由这些线索以实现由粗到细、上下文感知的更新。此外，通过从代理图像提取SigLIP特征实现无标题语义对齐，并配合可扩展的数据筛选流程获取富含结构信息的监督数据。

实验结果 / Results

LucidFlux在合成基准和真实世界（in-the-wild）测试集上均一致优于强大的开源和商业基线方法；消融研究验证了双分支条件器、自适应调制调度和无标题语义对齐等各个组件的必要性，证明该框架在复杂退化条件下能有效保护全局结构同时恢复精细纹理。

应用价值 / Applications

该研究为实际图像恢复应用提供了高效解决方案，特别适用于需要避免文本提示延迟和不稳定性的场景，如移动设备实时图像增强、历史照片修复、低质量监控图像复原等；其无标题设计降低了对昂贵VLM标注的依赖，提升了大规模部署的可行性和鲁棒性。
