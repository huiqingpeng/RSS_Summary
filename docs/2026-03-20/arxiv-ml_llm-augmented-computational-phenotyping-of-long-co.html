论文摘要 / Abstract

本研究提出了一种名为"Grace Cycle"的LLM增强计算表型分析框架，通过迭代整合假设生成、证据提取和特征优化，从纵向患者数据中识别临床有意义的亚组。研究基于13,511名长新冠参与者识别出三种临床表型（Protected、Responder、Refractory），这些表型在症状严重程度峰值、基线疾病负担和纵向剂量反应模式上表现出显著分离，并在多个独立维度上具有强统计学支持。该框架具有疾病无关性，为从复杂纵向数据中筛选临床可解释的亚表型提供了通用方法。

方法概述 / Method

研究采用"Grace Cycle"迭代框架，将大型语言模型整合到统计基础的分析流程中，实现假设驱动的计算表型分析。该方法通过循环迭代的方式，结合LLM的假设生成能力与统计验证，从纵向电子健康记录数据中提取和优化临床特征，最终识别出稳定且具有临床意义的疾病亚型。

实验结果 / Results

研究从13,511名长新冠患者中识别出三种稳定临床表型：Protected（受保护型）、Responder（应答型）和Refractory（难治型）。这些表型在多个关键维度上呈现显著差异，包括症状严重程度峰值、基线疾病负担水平以及随时间变化的剂量反应模式，且分离结果具有跨维度的强统计学支持。

应用价值 / Applications

该框架可广泛应用于其他慢性疾病的异质性研究，为精准医学和个性化干预提供数据驱动的分型基础。通过将LLM整合到可解释的统计流程中，该方法有助于临床医生理解复杂疾病的不同进展轨迹，从而优化患者分层管理和治疗决策。
