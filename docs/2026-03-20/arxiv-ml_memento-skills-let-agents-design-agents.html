Memento-Skills: Let Agents Design Agents

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了 Memento-Skills，一个通用且可持续学习的 LLM 智能体系统，其核心创新在于作为"设计智能体的智能体"——能够自主构建、适应和改进任务专用智能体。该系统基于记忆增强的强化学习框架，通过状态化提示（stateful prompts）和可复用的结构化技能（以 Markdown 文件存储）实现知识的持续积累与跨任务迁移。实验表明，该系统在 General AI Assistants 基准测试和 Humanity's Last Exam 上分别取得了 26.2% 和 116.2% 的相对准确率提升，且无需更新底层 LLM 参数即可实现持续学习。

方法概述 / Method

Memento-Skills 采用"读写反射学习"（Read–Write Reflective Learning）机制：在读阶段，行为可训练的技能路由器根据当前状态化提示选择最相关的技能；在写阶段，智能体基于新经验更新和扩展技能库。技能以结构化 Markdown 文件形式存储，同时编码行为与上下文信息，形成持久且可进化的外部记忆。整个系统从基础技能（如网络搜索、终端操作）出发，通过迭代生成与精炼技能实现端到端的智能体自主设计。

实验结果 / Results

在 General AI Assistants 基准测试中，Memento-Skills 实现了 26.2% 的相对准确率提升；在更具挑战性的 Humanity's Last Exam 上，取得了高达 116.2% 的相对改进。这些结果表明，通过持续技能进化而非模型参数微调，系统能够在多样化任务上实现显著且可持续的性能增长。

应用价值 / Applications

Memento-Skills 适用于需要快速适应新任务领域的通用 AI 助手场景，如自动化科研、复杂工作流管理和个性化服务定制。其价值在于大幅降低人工设计专用智能体的成本，使单一通用智能体能够自主演化为针对特定任务的专家系统，为构建真正自主、终身学习的 AI 系统提供了可行路径。
