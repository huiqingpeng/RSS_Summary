FUMO: Prior-Modulated Diffusion for Single Image Reflection Removal

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对单张图像反射移除（SIRR）任务中反射强度空间变化大、反射与透射结构高度纠缠的挑战，提出了基于先验调制的扩散模型框架FUMO。该框架从混合图像中直接提取两种先验：强度先验估计空间反射严重程度，高频先验通过多尺度残差聚合捕获细节敏感响应。通过两阶段训练策略（先验门控条件注入的粗粒度阶段和图像空间细粒度精修阶段），实现了对反射主导区域和结构敏感区域的空间可控处理，在标准基准测试和野外挑战性图像上均取得竞争性的定量结果和持续提升的感知质量。

方法概述 / Method

FUMO采用扩散模型结合显式先验引导的技术路线，核心创新在于从输入混合图像中自适应提取两种互补先验信息，并将其作为条件信号调制扩散过程。具体实现上，强度先验和高频先验通过门控机制控制条件残差注入，使模型聚焦于需要重点处理的反射区域；随后通过专门的精修网络在像素空间修正局部错位并锐化细节。

实验结果 / Results

实验在标准SIRR基准数据集和真实场景挑战性图像上开展，结果表明FUMO在定量指标上达到竞争水平，同时在感知质量方面实现一致性的显著提升。论文特别强调了该方法对野外复杂场景（in the wild）的泛化能力，这在现有方法中往往是薄弱环节。

应用价值 / Applications

该研究可广泛应用于需要高质量玻璃/水面反射消除的场景，如自动驾驶中车载摄像头透过挡风玻璃的感知、智能手机摄影后期处理、监控视频中透过橱窗的行人识别，以及增强现实系统中虚实融合的视觉预处理等，为下游视觉任务提供更清晰的输入信号。
