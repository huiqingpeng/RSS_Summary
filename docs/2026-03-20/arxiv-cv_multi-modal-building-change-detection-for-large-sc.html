Multi-Modal Building Change Detection for Large-Scale Small Changes: Benchmark and Baseline

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对光学遥感影像变化检测中易受光照波动、季节变化和地表覆盖材料差异影响的问题，提出利用近红外（NIR）信息作为可见光的互补物理线索，以增强建筑材料和小型结构的判别能力。研究构建了大规模小变化多模态数据集（LSMD），这是一个面向真实场景中小变化的双时相RGB-NIR建筑变化检测基准数据集。在此基础上，提出了多模态光谱互补网络（MSCNet），通过三个关键模块实现有效的跨模态特征融合，实验表明该方法在多种输入配置下均优于现有方法。

方法概述 / Method

论文提出的MSCNet包含三个核心组件：邻域上下文增强模块（NCEM）用于强化局部空间细节，跨模态对齐与交互模块（CAIM）实现RGB与NIR特征的深度交互，以及显著性感知多源精化模块（SMRM）对融合特征进行渐进式优化。该网络架构专门针对多模态数据的固有异质性设计，能够有效整合可见光与近红外光谱的互补信息。

实验结果 / Results

大量实验验证了MSCNet能够有效利用多模态信息，在多种输入配置下均一致性地优于现有方法。该网络在细粒度建筑变化检测任务上展现出优异性能，证明了其在复杂环境中进行多模态变化检测的有效性。

应用价值 / Applications

该研究为城市规划、违建监测、灾后评估等领域提供了高精度的建筑变化检测技术方案。
LSMD数据集为相关研究提供了严格的测试平台，MSCNet的跨模态融合方法可推广至其他需要多源遥感数据协同分析的场景，具有重要的实际应用价值和学术研究意义。
