OFTSR: One-Step Flow for Image Super-Resolution with Tunable Fidelity-Realism Trade-offs

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出OFTSR，一种基于流模型的一步图像超分辨率框架，能够在单次推理中生成高质量图像，并支持灵活调节保真度与真实感的权衡。该方法通过训练条件流模型作为教师模型，并采用特殊的ODE轨迹对齐约束进行知识蒸馏，使学生模型的一步预测与教师模型的中间状态预测保持一致。实验表明，OFTSR在FFHQ、DIV2K和ImageNet数据集上达到了一步超分辨率的最先进性能，同时克服了传统蒸馏方法中固定权衡的局限性。

方法概述 / Method

OFTSR采用两阶段训练策略：首先训练一个多步的条件流模型作为教师模型，然后通过知识蒸馏将其压缩为单步学生模型。核心创新在于蒸馏约束的设计——强制学生模型对相同输入的一步预测落在教师模型的采样ODE轨迹上，具体通过让学生模型的单步预测与教师模型从更近中间状态出发的预测相匹配来实现。

实验结果 / Results

OFTSR在多个基准数据集上验证了其有效性，包括人脸图像（FFHQ 256×256）、自然图像超分辨率（DIV2K）以及大规模图像生成（ImageNet 256×256）。该方法在一步推理设置下达到了最先进的超分辨率性能，同时通过调节采样轨迹上的不同初始状态，实现了保真度与真实感之间的连续可调权衡。

应用价值 / Applications

OFTSR适用于需要实时高分辨率图像生成的场景，如移动设备图像增强、视频流超分辨率、医学影像放大等计算资源受限的应用。其可调权衡特性使其能够根据具体需求灵活优化——优先考虑保真度时可用于科学成像和文档处理，优先考虑真实感时可用于照片修复和艺术内容生成。
