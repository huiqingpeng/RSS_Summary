Pi-transformer: A prior-informed dual-attention model for multivariate time-series anomaly detection

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出Pi-Transformer（先验信息Transformer），一种用于多变量时间序列异常检测的双注意力模型。该模型通过数据驱动的序列注意力与平滑演化的先验注意力两条路径，分别捕获时间上下文和跨通道协调关系，其中先验注意力编码尺度自相似性和相位同步等时间不变量。在五个基准数据集上的实验表明，该模型在F1分数上达到最优或接近最优水平，尤其对时间和相位破坏型异常具有显著优势。

方法概述 / Method

Pi-Transformer采用双注意力架构：一条为标准的数据驱动序列注意力，另一条为先验注意力机制，后者通过正则化约束实现平滑演化并从数据集统计特性中进行轻量蒸馏。训练过程中，重构损失与散度项联合优化，促使两种注意力在保持差异性的同时达成一致性；推理阶段则融合对齐加权重构信号（Energy）与失配信号，形成统一的异常评分。

实验结果 / Results

模型在SMD、MSL、SMAP、SWaT和PSM五个标准基准上取得最优或极具竞争力的F1分数，尤其在检测时间偏移和相位破坏类异常方面表现突出。案例分析显示两条注意力流具有互补行为，且在系统状态转换期间可提供可解释的检测结果。

应用价值 / Applications

该研究适用于工业物联网设备监控、关键基础设施（如水处理、电力系统）安全检测、航天器与地面系统状态监测等复杂多变量系统的实时异常检测场景。先验注意力机制提供的振幅不敏感时间参考增强了模型对噪声和尺度变化的鲁棒性，提升了实际部署中的可靠性与可解释性。
