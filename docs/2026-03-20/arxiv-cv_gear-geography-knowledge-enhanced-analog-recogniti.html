GEAR: Geography-knowledge Enhanced Analog Recognition Framework in Extreme Environments

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了一种地理知识增强的类比识别框架（GEAR），用于在青藏高原（250万平方公里）上高效检索与马里亚纳海沟结构同源的陆地类比区域。该框架通过三阶段流水线（骨架引导筛选、物理感知过滤、图神经网络精细识别）解决了跨域地形相似性检索问题。实验表明，所提出的MSG-Net在F1分数上比最先进基线高出1.38个百分点，且提取的特征与生物数据存在显著相关性，为深海生物采样提供了经济高效的替代方案。

方法概述 / Method

GEAR框架采用三阶段处理流水线：（1）骨架引导筛选与裁剪，基于尺寸和线性形态标准识别候选山谷；（2）物理感知过滤，利用地形波形比较器（TWC）和形态纹理模块（MTM）评估波形与纹理特征；（3）基于图的精细识别，设计形态集成孪生图网络（MSG-Net）进行最终相似性判断。同时构建了面向构造碰撞带的地形相似性专家标注数据集。

实验结果 / Results

实验验证了每个阶段的有效性，MSG-Net在F1分数上超越现有最优基线1.38个百分点。使用MSG-Net提取的特征，研究团队发现了与生物数据的显著相关性，为后续生物学分析提供了证据支持。

应用价值 / Applications

该研究为深海生物采样提供了经济高效的替代方案——通过在青藏高原识别与马里亚纳海沟地质起源和微生物代谢功能相似的陆地类比区域，大幅降低研究成本。成果可应用于极端环境生物勘探、地质类比研究及跨域生态学分析等领域。
