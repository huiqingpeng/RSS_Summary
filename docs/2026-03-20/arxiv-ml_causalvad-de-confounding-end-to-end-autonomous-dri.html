CausalVAD: De-confounding End-to-End Autonomous Driving via Causal Intervention

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了CausalVAD，一种基于因果干预的去混淆端到端自动驾驶训练框架，旨在解决现有规划导向型端到端驾驶模型因学习统计相关性而非真实因果关系而导致的因果混淆问题。核心贡献是设计了稀疏因果干预方案（SCIS），一个轻量级即插即用模块，通过构建潜在驾驶上下文的原型字典并对稀疏向量化查询进行干预，消除混淆变量引起的虚假关联。实验表明，CausalVAD在nuScenes等基准上实现了最先进的规划精度和安全性，并对数据偏差和噪声场景表现出卓越的鲁棒性。

方法概述 / Method

CausalVAD基于后门调整理论，通过SCIS模块实现因果干预：首先构建一个原型字典来表示潜在的驾驶上下文（混淆变量），然后利用该字典对模型的稀疏向量化查询进行干预，从而主动消除表示中的虚假因素。该模块具有轻量级和即插即用的特点，可无缝集成到现有的端到端自动驾驶架构中。

实验结果 / Results

在nuScenes等标准基准上的大量实验表明，CausalVAD达到了最先进的规划精度和安全性指标。该方法在专门设计用于诱导因果混淆的数据偏差场景和噪声场景中，均展现出优于现有方法的鲁棒性，有效验证了因果干预在消除虚假关联方面的有效性。

应用价值 / Applications

CausalVAD可广泛应用于端到端自动驾驶系统的训练与部署，特别适用于需要高可靠性和安全性的复杂真实驾驶场景。该方法通过提升模型对分布外数据和对抗性噪声的鲁棒性，为自动驾驶汽车在长尾场景和边缘情况下的安全决策提供了重要保障，具有显著的产业应用价值。
