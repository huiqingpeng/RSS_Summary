TAU-R1: Visual Language Model for Traffic Anomaly Understanding

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对智能交通系统中的交通异常理解（TAU）任务，提出了首个专门的数据集Roundabout-TAU和视觉语言模型TAU-R1。该数据集包含342个真实环岛视频片段及2000多组问答对，填补了该领域基准数据的空白。
TAU-R1采用双层架构设计，结合分解式问答增强监督微调和TAU-GRPO后训练方法，在异常分类和推理任务上取得了优异性能，同时保持了部署效率。

方法概述 / Method

TAU-R1采用双层视觉语言框架：第一层为轻量级异常分类器，用于粗粒度异常类别识别；第二层为大型异常推理器，生成详细的事件描述摘要。训练过程采用两阶段策略，首先进行分解式问答增强的监督微调（SFT），随后使用基于GRPO的后训练方法（TAU-GRPO），并设计了针对TAU任务的特定奖励函数以优化推理能力。

实验结果 / Results

实验结果表明，TAU-R1在异常分类和推理任务上均展现出强劲性能，同时保持了较高的部署效率。通过两阶段训练策略，模型在任务特定的推理能力上得到显著提升，能够有效处理交通异常理解中的多维度问答任务。

应用价值 / Applications

该研究可直接应用于智能交通系统的实时监控与安全管理，辅助交通管理部门快速识别环岛场景中的交通异常事件并理解其成因。
TAU-R1的高效部署特性使其适合边缘计算场景，为城市交通智能化和自动驾驶安全决策提供技术支持。
