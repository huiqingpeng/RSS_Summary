ALIGN: Adversarial Learning for Generalizable Speech Neuroprosthesis

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了 ALIGN，一种基于多域对抗神经网络的会话不变学习框架，用于解决脑机接口（BCI）中跨会话数据分布偏移导致的性能下降问题。
ALIGN 通过对抗优化训练特征编码器，使其在保留语音解码任务相关信息的同时抑制会话特异性特征，从而实现半监督的跨会话自适应。实验表明，ALIGN 在未见过的会话上具有更稳定的泛化能力，显著降低了音素错误率和词错误率，证明对抗域对齐是缓解会话级分布偏移、实现鲁棒纵向 BCI 解码的有效方法。

方法概述 / Method

ALIGN 采用多任务对抗学习架构，联合训练三个组件：特征编码器、音素分类器和域分类器。编码器与音素分类器协作学习语音解码任务，同时与域分类器进行对抗博弈——编码器试图欺骗域分类器以消除会话特异性特征，而域分类器则努力识别样本来源的会话标签。这种对抗优化迫使编码器提取会话不变的潜在表征。

实验结果 / Results

在皮层内语音解码任务上的评估显示，ALIGN 在泛化到全新会话时表现一致优于基线方法，有效提升了跨会话解码性能。具体而言，ALIGN 在音素错误率（PER）和词错误率（WER）两个关键指标上均实现显著改进，验证了对抗域对齐在缓解电极移位、神经信号变化等跨会话非平稳性问题上的有效性。

应用价值 / Applications

该研究对临床语音神经假体的实际部署具有重要价值，解决了 BCI 系统因需频繁重新校准而难以长期使用的瓶颈。
ALIGN 的半监督自适应特性使其能在新会话无需标注数据的情况下快速适应，为瘫痪患者提供稳定、可持续的辅助沟通工具，推动脑机接口技术从实验室走向日常临床应用。
