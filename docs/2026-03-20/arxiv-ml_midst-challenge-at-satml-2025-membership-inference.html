MIDST Challenge at SaTML 2025: Membership Inference over Diffusion-models-based Synthetic Tabular data

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本论文介绍了在SaTML 2025会议上举办的MIDST挑战赛，该挑战赛首次对扩散模型生成的合成表格数据的隐私保护效果进行了定量评估。研究聚焦于成员推理攻击（MIAs）对单表混合数据类型和多关系互联表格两种场景下的扩散模型的威胁。挑战赛的重要成果是催生了针对这些特定扩散模型的新型黑盒和白盒成员推理攻击方法，为全面评估合成表格数据的隐私效能提供了基准。

方法概述 / Method

MIDST挑战赛设计了多目标模型评估框架，涵盖单表和多关系表格两种异构复杂的表格数据结构。参赛者开发了专门针对扩散模型架构特点的成员推理攻击方法，包括利用模型内部信息的**白盒攻击**和仅依赖查询访问的**黑盒攻击**。评估通过标准化的攻击成功率等指标，量化合成数据对成员推理攻击的抵抗能力。

实验结果 / Results

挑战赛结果表明，现有扩散模型生成的合成表格数据对成员推理攻击的抵抗能力存在显著差异，部分攻击方法能够有效识别训练集中的成员样本。新型针对扩散模型的专用攻击方法（特别是白盒方法）相比传统MIAs展现出更高的攻击成功率，揭示了当前表格扩散模型在隐私保护方面的潜在漏洞。完整结果和基准代码已在GitHub开源发布。

应用价值 / Applications

该研究为数据发布者和隐私从业者提供了评估合成表格数据隐私风险的实用工具和基准，有助于指导扩散模型在医疗、金融等敏感表格数据领域的安全应用。研究成果可直接用于改进差分隐私等防御机制，推动更可靠的隐私保护数据生成技术发展，平衡数据效用与隐私安全之间的权衡。
