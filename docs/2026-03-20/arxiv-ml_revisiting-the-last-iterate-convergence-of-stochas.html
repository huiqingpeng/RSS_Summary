Revisiting the Last-Iterate Convergence of Stochastic Gradient Methods

论文摘要 / Abstract

本文重新审视了随机梯度下降（SGD）算法最后迭代点（last-iterate）的收敛性问题。现有研究在证明最优高概率收敛率时，通常要求紧致域或几乎必然有界噪声等限制性假设。作者首次建立了统一的分析框架，能够同时处理一般域、复合目标函数、非欧几里得范数、Lipschitz条件、光滑性、（强）凸性等多种设置，并在期望和高概率意义下证明收敛率。此外，该研究还将分析扩展到了重尾分布和次Weibull噪声的情形。

方法概述 / Method

论文提出了一种统一的证明技术，通过设计新的势函数（potential function）和递归分析技术，摆脱了对紧致域和有界噪声的依赖。该方法能够灵活适应不同的优化设置，包括非光滑/光滑目标、复合优化问题以及一般的Bregman散度诱导的非欧几里得几何结构。

实验结果 / Results

（注：本文主要为理论工作，未明确报告数值实验结果。）理论结果表明，在去除紧致域和有界噪声假设后，SGD的最后迭代点仍能达到最优收敛率$O(\log(1/\delta)\log T/\sqrt{T})$或$O(\sqrt{\log(1/\delta)/T})$（高概率）以及$O(1/\sqrt{T})$或$O(1/T)$（期望）。这些收敛率在更广泛的优化设置下（包括复合目标和非欧几里得范数）均成立，并且首次覆盖重尾噪声和次Weibull噪声情形。

应用价值 / Applications

该研究为随机梯度方法的理论分析提供了更通用、更少限制的框架，对机器学习中的大规模优化问题具有重要指导意义。实际应用包括深度学习训练（常涉及无界噪声和非紧致参数空间）、分布鲁棒优化、以及使用非欧几里得几何的优化场景（如矩阵流形优化和稀疏学习），使理论保证更贴近实际算法行为。
