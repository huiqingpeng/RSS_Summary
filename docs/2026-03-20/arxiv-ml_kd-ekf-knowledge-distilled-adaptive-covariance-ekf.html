KD-EKF: Knowledge-Distilled Adaptive Covariance EKF for Robust UWB/PDR Indoor Localization

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了一种基于知识蒸馏的自适应协方差扩展卡尔曼滤波器（KD-EKF），用于解决UWB/PDR室内定位中非视距（NLOS）条件导致的测量可靠性下降和传统EKF固定协方差参数难以适应异构环境的问题。该方法通过离线训练的大型教师模型从历史轨迹中学习可靠性线索，并将其蒸馏为轻量级学生模型，实现实时自适应调节测量协方差。实验结果表明，KD-EKF显著降低了定位误差，抑制了LOS/NLOS切换时的误差尖峰，并有效缓解了长期漂移问题。

方法概述 / Method

论文采用知识蒸馏框架构建自适应协方差调节机制：首先利用大型教师模型离线学习UWB/PDR序列的时序特征并生成下一位置预测，然后将该预测行为蒸馏至轻量级学生模型以实现实时部署。学生模型基于预测残差动态调节EKF的测量协方差，无需人工重新调参即可实现环境感知的数据融合。

实验结果 / Results

实验验证了KD-EKF在多样化室内环境中的性能优势：相比固定参数EKF，该方法显著降低了整体定位误差，有效抑制了视距与非视距切换过程中的误差突变，并减轻了PDR固有的长期累积漂移问题，展现出更强的测量鲁棒性和环境适应能力。

应用价值 / Applications

该研究适用于对精度和可靠性要求较高的室内定位场景，如仓库物流机器人导航、医院病人监护、商场人员追踪以及应急响应等复杂室内环境。
KD-EKF的自适应能力消除了人工调参负担，使其能够快速部署于布局多变、NLOS条件不可预测的实际场景中，提升了UWB/PDR融合定位系统的工程实用性和泛化性能。
