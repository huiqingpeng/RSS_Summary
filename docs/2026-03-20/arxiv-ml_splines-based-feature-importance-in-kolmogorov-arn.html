Splines-Based Feature Importance in Kolmogorov-Arnold Networks: A Framework for Supervised Tabular Data Dimensionality Reduction

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文研究了基于Kolmogorov-Arnold网络（KANs）的特征选择方法，利用样条参数化特征变换并自然地暴露逐特征重要性分数。作者提出了四种KAN-based选择标准（系数范数、基于梯度的显著性和剔除分数），并在多个真实与合成数据集上与LASSO、随机森林特征重要性等经典方法进行比较。实验表明，KAN-based选择器在分类和回归任务中通常具有竞争力，有时甚至优于传统基线方法，能够有效捕捉非线性交互并去除冗余特征。

方法概述 / Method

该研究利用Kolmogorov-Arnold Networks（KANs）的样条参数化特性，从网络权重中直接提取特征重要性，设计了四种选择标准：系数范数（衡量样条系数的幅度）、基于梯度的显著性（计算输入对输出的梯度影响）、以及剔除分数（通过移除特征观察性能变化）。这些方法充分利用了KANs结构可解释的优势，无需额外的模型训练即可获取特征重要性。

实验结果 / Results

在三个特征保留水平（20%、40%、60%）下，KAN-based选择器在平均F1和R²分数上总体与经典方法相当或更优：在多分类任务中常能匹配或超越现有方法，有效去除冗余特征并捕捉非线性交互；在回归任务中对噪声和异质数据集表现稳健，接近强集成预测器性能。研究也发现KAN-based方法存在特定失效模式，如ℓ₁准则可能导致过度激进的剪枝，同时稳定性和冗余分析表明该方法能产生可复现的特征子集并避免不必要的相关性膨胀。

应用价值 / Applications

该研究为表格数据的监督式降维提供了一种强大且可解释的替代方案，特别适用于需要理解特征间非线性和多元相关性的场景，如金融风控、医疗诊断和科学研究中的变量筛选。
KAN-based特征选择能够帮助实践者在保持预测性能的同时简化模型、降低数据收集成本，并增强模型的可解释性和可信度，尤其在特征冗余严重或存在复杂交互效应的高维表格数据中具有显著优势。
