Multimodal Machine Learning for Soft High-k Elastomers under Data Scarcity

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本研究针对软质高介电常数（high-k）弹性体开发中数据稀缺的关键瓶颈，构建了一个高质量的丙烯酸酯基介电弹性体数据集，并提出了基于大规模预训练聚合物表征的多模态学习框架。该框架利用从海量聚合物语料库迁移的化学与结构知识，实现了介电和力学性能的准确小样本预测。研究显著加速了数据高效的软质高k介电弹性体发现过程，相关数据和代码已公开发布。

方法概述 / Method

论文采用多模态机器学习框架，整合分子序列、介电性能和力学性能数据；核心创新在于利用大规模预训练的聚合物嵌入表示（pretrained polymer embeddings），将来自广泛聚合物文献的化学结构知识迁移至目标预测任务。该方法特别针对数据稀缺场景设计，通过知识迁移实现高效的少样本学习（few-shot learning）。

实验结果 / Results

基于过去十年实验结果聚合的紧凑高质量数据集，预训练嵌入显著提升了介电常数（k）和杨氏模量（E）的预测精度；模型在极少样本条件下即可实现准确预测，有效解决了软质高k弹性体设计中"高介电常数与低模量难以兼得"的核心挑战，验证了数据高效发现新材料的可行性。

应用价值 / Applications

该研究直接服务于软性和可拉伸电子器件领域，包括传感器、执行器和晶体管等现代人机交互和机器人接口应用；通过大幅降低实验数据需求，加速了高性能介电弹性体的研发周期，为材料基因组工程和AI驱动的新材料设计提供了可推广的方法论范式。
