Improving RCT-Based Treatment Effect Estimation Under Covariate Mismatch via Calibrated Alignment

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出CALM（Calibrated ALignment under covariate Mismatch）框架，用于解决随机对照试验（RCT）与观察性研究（OS）之间因协变量不匹配而导致的异质性治疗效果估计难题。该方法通过学习将两个数据源的异构特征映射到共同表示空间的嵌入，避免了传统的插补策略，同时利用试验数据对OS结果模型进行校准以保持随机化的因果识别。理论分析给出了有限样本风险上界，并在51种模拟设置中验证了神经嵌入变体在非线性场景下的显著优势。

方法概述 / Method

CALM采用双阶段策略：首先学习跨源嵌入对齐，将RCT和OS的异构协变量投影到共享低维空间；随后将OS训练的结果模型迁移至RCT嵌入空间，并使用RCT数据对预测结果进行校准。该方法提供线性和神经两种变体，其中线性变体基于校准机制提供负迁移保护，神经变体通过深度网络捕捉非线性表示。

实验结果 / Results

模拟实验涵盖51种不同设置，结果显示：对于线性条件平均处理效应（CATE），基于校准的方法表现等价；而在22种非线性机制设置中，神经嵌入变体均以大幅优势胜出。理论分析表明，在稀疏线性模型假设下，嵌入方法严格泛化了插补方法，且风险上界可分解为对齐误差、结果模型复杂度和校准复杂度三项。

应用价值 / Applications

该研究为医疗健康、政策评估等领域中RCT样本量不足但存在大规模OS数据的场景提供了实用解决方案，使研究者能够整合多源异构数据提升治疗效果异质性估计的精度。方法特别适用于协变量采集标准不一致的跨机构合作研究，为真实世界证据与临床试验证据的融合建立了理论基础和实践工具。
