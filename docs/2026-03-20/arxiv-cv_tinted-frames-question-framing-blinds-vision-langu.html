Tinted Frames: Question Framing Blinds Vision-Language Models

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文揭示了视觉-语言模型（VLMs）存在"选择性失明"现象：即使面对需要相同视觉推理的任务，模型也会根据问题的语言框架（framing）调节对视觉输入的关注程度。研究发现，受限框架（如多选题、是非题）会显著降低模型对图像上下文的注意力，导致准确性下降和跨框架不一致。基于这一机制洞察，作者提出了一种轻量级的可学习提示词调优方法，有效改善了模型的视觉 grounding 能力和跨框架性能。

方法概述 / Method

作者以视觉注意力作为探针，系统量化了不同问题框架（开放式 vs. 受限式）如何改变模型对图像的关注量和空间分布。基于发现——开放式框架能产生更鲁棒、视觉基础更扎实的注意力模式——他们设计了一种提示词调优方法，通过引入可学习 token 来引导模型恢复这种理想的注意力分配，而无需修改模型架构或进行全量微调。

实验结果 / Results

实验表明，受限框架（多选题/是非题）相比开放式框架，导致模型对图像的整体注意力显著降低，对任务相关区域的关注减少，同时注意力被重新分配到无信息量的 token 上。这种注意力错配是跨框架性能不一致的主要原因。所提出的提示词调优方法成功恢复了视觉基础的注意力模式，在保持轻量化的同时提升了模型在各框架下的表现。

应用价值 / Applications

该研究对需要可靠视觉推理的实际应用具有重要价值，如教育测评中的自动答题系统、医疗影像诊断助手、以及需要处理多样化问题形式的视觉问答平台。提出的轻量级调优方法可便捷部署，帮助现有 VLMs 在保持效率的同时，克服因问题表述方式不同而导致的性能波动，提升系统的鲁棒性和用户信任度。
