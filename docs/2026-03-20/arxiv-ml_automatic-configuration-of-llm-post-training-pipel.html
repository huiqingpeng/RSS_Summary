论文摘要 / Abstract

本文提出了AutoPipe，一个面向LLM后训练流程的预算感知型两阶段自动配置框架，用于解决监督微调和强化学习组合配置中高维异构搜索空间、阶段强耦合及评估成本高昂的挑战。该框架离线学习基于数据集条件的学习排序代理模型以捕获跨数据集偏好，在线则结合贝叶斯优化与高斯过程残差模型处理新数据集的特异性偏差，并通过早期训练信号预测器实现低成本评估。在生物医学推理任务上的实验表明，AutoPipe在仅使用最强在线HPO基线不到10%计算成本的情况下，达到了可比的性能表现。

方法概述 / Method

AutoPipe采用离线-在线两阶段架构：离线阶段从历史运行中学习数据集条件化的学习排序代理模型，编码配置空间中的跨数据集偏好关系；在线阶段利用离线指导引导贝叶斯优化，同时用高斯过程残差代理建模数据集特异性偏差。此外，引入基于早期训练信号的预测器作为低成本代理，替代完整的端到端后训练评估。

实验结果 / Results

在生物医学推理任务上的实验显示，AutoPipe持续优于纯离线基线方法，并与最强在线超参数优化（HPO）基线达到相当性能；关键效率指标表明，该方法将计算成本降低至传统在线方法的10%以下，显著提升了配置搜索的性价比。

应用价值 / Applications

该研究适用于需要在有限计算预算下优化LLM后训练流程的研究团队和工业场景，特别是在医疗、法律等高价值垂直领域，可大幅降低模型调优成本并加速迭代周期；其两阶段设计也为其他多阶段机器学习系统的自动配置提供了可迁移的方法论框架。
