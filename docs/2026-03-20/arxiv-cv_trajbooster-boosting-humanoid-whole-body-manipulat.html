TrajBooster: Boosting Humanoid Whole-Body Manipulation via Trajectory-Centric Learning

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了TrajBooster，一种跨形态框架，利用丰富的轮式人形机器人数据来增强双足人形机器人的视觉-语言-动作（VLA）模型性能。核心创新在于使用末端执行器轨迹作为形态无关的接口，通过轨迹提取、重定向和异构三元组训练，仅需10分钟的目标机器人遥操作数据即可实现高效迁移。在Unitree G1上的部署验证了该方法能完成超越桌面的家庭任务，显著提升鲁棒性和泛化能力，同时降低了对昂贵同形态数据的依赖。

方法概述 / Method

TrajBooster采用三阶段流程：（i）从真实轮式人形机器人中提取6D双臂末端执行器轨迹；（ii）在仿真环境中通过启发式增强的和谐在线DAgger算法训练全身控制器，将低维轨迹参考提升为可行的高维全身动作；（iii）构建异构三元组（源视觉/语言+目标人形兼容动作）对VLA进行后预训练，并结合少量目标域遥操作数据进行微调。

实验结果 / Results

在Unitree G1上的实验表明，该方法实现了蹲姿操作、跨高度操作和协调全身运动等复杂家庭任务，展现出显著改善的鲁棒性和泛化能力。关键性能提升包括：仅需10分钟目标域数据即可有效对齐新机器人动作空间，实现了高效的零样本技能迁移，并增强了对动作空间的理解。

应用价值 / Applications

该研究为双足人形机器人的快速部署提供了实用解决方案，特别适用于家庭服务、仓储物流等需要全身协调操作的场景。通过复用现有轮式人形机器人数据，大幅降低了双足机器人数据采集成本和时间门槛，加速了VLA模型在新形态机器人上的适配与泛化，对推动人形机器人的规模化应用具有重要价值。
