On Minimal Depth in Neural Networks

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了一个几何框架来分析ReLU神经网络的表达能力，通过为凸多面体定义"深度复杂度"来量化其结构复杂性。该框架揭示了循环多面体（cyclic polytopes）的深度随顶点数无界增长，证明输入凸神经网络（ICNNs）无法用固定深度表示所有凸连续分段线性函数，从而建立了ICNNs与标准ReLU网络之间的表达能力分离。同时，作者给出了Arora等人（2018）表达性界限的纯几何证明，表明⌈log₂(n+1)⌉个隐藏层足以表示任何连续分段线性函数。

方法概述 / Method

作者将ReLU网络的表达能力与凸多面体的几何构造相联系，定义多面体的"深度"为递归构造该多面体所需的交替凸包（convex hull）和Minkowski和运算的最小次数。这一几何视角将神经网络的深度问题转化为多面体的组合复杂度分析问题，为推导深度下界提供了严格的数学工具。

实验结果 / Results

研究建立了多面体深度的上下界，并对经典多面体族给出了紧界。核心发现包括：在维度n≥4时，循环多面体的深度随顶点数无界增长，这与一般ReLU网络存在通用深度上界形成鲜明对比；同时确认了⌈log₂(n+1)⌉层足以表达任意n维连续分段线性函数。

应用价值 / Applications

该研究为神经网络架构设计提供了理论指导，特别是在需要凸性约束的场景（如经济学中的效用函数建模、最优传输、强化学习中的值函数逼近）中，明确了ICNNs的表达能力局限。结果提示在应用ICNNs时可能需要随问题规模增加网络深度，而不能依赖固定深度的通用逼近。
