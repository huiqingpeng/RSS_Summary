An Optimised Greedy-Weighted Ensemble Framework for Financial Loan Default Prediction

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本研究针对现代金融数据集中普遍存在的非线性关系、类别不平衡和借款人行为动态变化等挑战，提出了一种优化的贪婪加权集成框架用于贷款违约预测。该框架首先利用粒子群优化算法优化多个机器学习分类器的超参数，然后通过正则化贪婪加权机制组合模型预测，并引入基于神经网络的元学习器捕捉模型输出间的高阶关系。在Lending Club数据集上的实验表明，该框架显著优于单个分类器，其中BlendNet集成模型取得了AUC 0.80、宏平均F1分数0.73和违约召回率0.81的最优性能。

方法概述 / Method

该框架采用三阶段方法：首先使用粒子群优化（PSO）对多个基分类器进行超参数优化；其次通过正则化贪婪加权机制动态分配模型权重，依据各模型的实证预测性能进行自适应组合；最后在堆叠集成中引入神经网络元学习器（BlendNet），以捕获基模型输出之间的复杂非线性交互关系。

实验结果 / Results

BlendNet堆叠集成模型表现最优，达到AUC 0.80、宏平均F1分数0.73和违约召回率0.81；校准分析显示Extra Trees和Gradient Boosting等树型集成模型提供最可靠的概率估计，而堆叠集成在排序能力上更胜一筹；递归特征消除分析识别出循环额度使用率、年收入和债务收入比为最具影响力的违约预测因子。

应用价值 / Applications

该框架为金融机构提供可扩展的数据驱动信用评估方案，支持贷款审批决策、风险监控和动态风险管理；其性能驱动的集成加权机制在提升预测准确性的同时增强了模型可解释性，有助于监管合规和精细化风险控制；该方法可推广至消费信贷、中小企业贷款等多种信贷场景，助力普惠金融的智能化发展。
