Causal Intervention Framework for Variational Auto Encoder Mechanistic Interpretability

arXiv Machine Learning 2026-03-20

论文摘要 / Abstract

本文提出了首个针对变分自编码器（VAE）的因果干预框架，用于实现生成模型的机制可解释性。研究开发了识别和分析VAE中"电路基序"的技术，通过输入操控、潜在空间扰动、激活修补和因果中介分析等多层级干预手段，揭示语义因子如何在网络层中被编码、处理和解耦。实验表明该框架能有效分离功能电路、区分多语义与单语义单元，并量化VAE组件的可解释性，其中FactorVAE在解耦分数（0.084）和因果效应强度（均值4.59）上显著优于标准VAE和Beta-VAE。

方法概述 / Method

论文构建了四层因果干预体系：输入层操控、潜在空间扰动、中间激活修补以及基于因果中介分析的效应分解。通过将计算图映射到语义因子的因果图，研究者设计了因果效应强度、干预特异性和电路模块性三类评估指标，以系统化量化VAE内部组件的功能角色和模块化程度。

实验结果 / Results

在合成数据集和标准解耦基准上的实验验证了框架的有效性：FactorVAE的解耦分数达0.084，显著高于标准VAE（0.064）和Beta-VAE（0.051）；其平均因果效应强度（4.59）同样优于后两者（分别为3.99和3.43）。干预成功分离出功能电路，并清晰区分了多语义神经元（编码多个语义因子）与单语义神经元（专用于单一因子）。

应用价值 / Applications

该框架为生成模型提供了透明的机制分析工具，可指导更可控的VAE架构设计，适用于需要可解释生成建模的场景，如医疗影像合成、科学数据生成和公平性敏感的机器学习系统。量化指标还可用于模型选型与正则化设计，推动可信AI的发展。
