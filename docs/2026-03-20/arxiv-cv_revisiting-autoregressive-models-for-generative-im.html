Revisiting Autoregressive Models for Generative Image Classification

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文重新审视了基于自回归（AR）模型的生成式图像分类方法，发现先前方法依赖固定token顺序的限制会阻碍图像理解。作者利用近期发展的任意顺序AR模型来估计顺序边缘化预测，使AR模型在分类任务上超越了扩散模型分类器，同时效率提升高达25倍，并能与最先进的自监督判别模型竞争。

方法概述 / Method

作者采用任意顺序自回归模型，通过对多个token顺序的预测进行平均来估计顺序边缘化的分类概率，从而克服固定顺序带来的归纳偏置问题，获得更全面的图像理解信号。

实验结果 / Results

该方法在多个图像分类基准上持续优于基于扩散模型的分类器，计算效率提升最高达25倍；与最先进的自监督判别模型相比，取得了具有竞争力的分类性能，这是生成式分类器的重要突破。

应用价值 / Applications

本研究为高效、鲁棒的图像分类提供了新的生成式建模范式，适用于需要同时获得良好生成能力和判别性能的场景，如医学影像分析、自动驾驶感知系统等对模型可解释性和不确定性估计有要求的实际应用。
