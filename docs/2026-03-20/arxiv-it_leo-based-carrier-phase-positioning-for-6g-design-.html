LEO-based Carrier-Phase Positioning for 6G: Design Insights and Comparison with GNSS

论文摘要 / Abstract

本文研究了基于低地球轨道（LEO）卫星的非地面网络（NTN）在5G新空口（NR）中的联合时延-载波相位定位技术，并与全球导航卫星系统（GNSS）进行了以收敛速度为核心的对比分析。研究发现，LEO卫星的快速轨道运动在多历元观测中产生了强烈的时间和几何多样性，显著改善了载波相位模型的条件数，从而大幅加快了整周模糊度的收敛速度。通过提出的双波形设计（宽带PRS用于时延估计、连续窄波用于相位跟踪），LEO定位可在数秒内实现厘米级精度，而GNSS在同等短时观测窗口下仅能达米级精度，证明了LEO蜂窝定位可作为GNSS在高精度定位、导航与授时（PNT）服务中的强有力补充甚至替代方案。

方法概述 / Method

论文提出了一种双波形信号设计方案，将宽带定位参考信号（PRS）用于时延估计与连续窄波载波用于相位跟踪相结合，以解决间歇性PRS传输下的稳健载波相位跟踪问题。研究构建了包含LEO轨道动力学的真实仿真框架，对联合时延-载波相位定位进行性能评估，并与GNSS进行系统对比。

实验结果 / Results

仿真结果表明，基于LEO的联合时延-载波相位定位可实现厘米级定位精度，整周模糊度收敛时间仅为数秒量级；相比之下，GNSS在同等短时观测窗口内受限于几何构型变化缓慢，仅能达成米级精度。
LEO卫星的快速轨道运动带来的时间-几何多样性是加速收敛的关键因素。

应用价值 / Applications

该研究为6G及未来无线网络中的高精度定位、导航与授时（PNT）服务提供了技术基础，特别适用于城市峡谷、室内等GNSS信号受阻场景。
LEO-based蜂窝定位可作为GNSS的互补或替代方案，支撑自动驾驶、精准农业、物联网等需要厘米级实时定位的应用场景。
