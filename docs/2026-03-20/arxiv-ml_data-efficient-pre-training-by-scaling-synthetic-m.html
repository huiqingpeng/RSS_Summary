论文摘要 / Abstract

本文研究了在数据受限场景下如何通过合成数据增强实现更优的损失缩放（loss scaling）。作者发现将网页数据与合成改写数据混合训练可提升验证损失，并提出"合成超文档（megadocs）"方法——将同一文档的多个合成改写拼接成单个长文档或插入推理过程扩展文档——将数据效率从1.48×提升至1.80×。关键发现是超文档方法的优势随合成数据量增加而扩大，为设计受益于更多算力的合成数据算法提供了新思路。

方法概述 / Method

研究采用合成数据增强策略，首先测试简单改写（rephrasing）混合原始网页数据的效果，随后提出两种构建超文档的方法：一是将同一网页文档的多个合成改写拼接成连续长文档，二是通过在文档中插入推理过程（rationales）来扩展文档长度。实验系统优化了混合比例和训练轮次（epoching）策略。

实验结果 / Results

简单改写方法在32次改写/文档时达到1.48×数据效率，且损失和基准准确率随合成数据增加而改善、无过拟合现象。超文档方法表现更优：在相同32代/文档条件下数据效率提升至1.80×，且在长上下文损失上改进尤为显著。更重要的是，超文档相对于简单改写的优势随合成数据规模扩大而持续扩大，展现出更好的损失缩放特性。

应用价值 / Applications

该研究为数据受限场景（如低资源语言、专业领域或隐私敏感数据）的大规模模型预训练提供了高效解决方案。超文档方法特别适用于需要长上下文理解能力的任务（如文档分析、代码生成、科学研究），且其随算力扩展而增强的特性使其在未来计算资源增长时更具价值，可显著降低高质量训练数据的获取成本。
