论文摘要 / Abstract

本文提出了一种基于图像频谱特性设计逐实例噪声调度的原则性方法，用于像素级扩散模型。通过推导最小和最大噪声水平有效性的理论边界，作者设计了"紧致"噪声调度以消除冗余步骤，并在推理阶段实现了条件化采样。实验表明，该方法显著提升了单阶段像素扩散模型的生成质量，尤其在低步数采样场景下效果突出。

方法概述 / Method

论文基于图像的频谱特性（spectral properties）为每个输入实例动态设计噪声调度，而非使用手工预设的固定调度。作者从理论上分析了噪声水平上下界的有效性边界，据此构建去除冗余步骤的紧致噪声调度，并在推理时根据输入条件进行自适应采样。

实验结果 / Results

实验验证了所提频谱引导噪声调度在单阶段像素扩散模型上的有效性，特别是在低步数（low-step）采样条件下，生成质量得到明显改善。该方法消除了不必要的去噪步骤，同时保持了或提升了输出图像的质量。

应用价值 / Applications

该研究为高效图像和视频生成提供了优化方案，尤其适用于需要快速推理的资源受限场景（如实时应用、移动设备部署）。动态噪声调度机制可减少计算开销，同时提升生成质量，对扩散模型的实际部署和商业化应用具有重要价值。
