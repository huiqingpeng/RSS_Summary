Multifidelity Simulation-based Inference for Computationally Expensive Simulators

arXiv Machine Learning 2026-03-20

论文摘要 / Abstract

本文提出了一种多保真度模拟推断方法，用于解决高保真计算模拟器参数推断中的计算瓶颈问题。核心贡献是利用迁移学习将低成本低保真模拟的信息高效迁移到高保真模拟器的神经后验估计中，同时提出了基于获取函数的自适应序贯变体来进一步优化高保真模拟的选择效率。实验表明，该方法在基准测试和神经科学任务上可将高保真模拟需求降低两个数量级，同时保持与现有方法相当的推断性能。

方法概述 / Method

该方法采用迁移学习架构，首先用大量低保真模拟数据预训练神经密度估计网络，再通过少量高保真模拟进行微调；同时设计了适用于摊销式（amortized）和非摊销式推断的统一多保真框架。序贯变体引入基于密度估计器预测不确定性的获取函数，自适应选择信息量最大的高保真参数点进行模拟。

实验结果 / Results

在标准基准测试和实际神经科学任务中，所提方法相比单保真度方法减少了10-100倍的高保真模拟调用次数；多保真度方案在大幅降低计算成本的同时，后验估计精度与仅使用高保真模拟的方法相当，验证了迁移学习策略的有效性。

应用价值 / Applications

该方法适用于任何存在多精度模拟器的科学领域，如计算神经科学、气候建模、计算流体力学等，使得此前因计算成本过高而难以进行贝叶斯推断的复杂高保真模型成为可能；特别有利于参数空间探索、模型校准和不确定性量化等需要大量模拟的任务。
