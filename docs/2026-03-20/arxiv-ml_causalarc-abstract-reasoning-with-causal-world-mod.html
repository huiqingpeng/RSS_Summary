CausalARC: Abstract Reasoning with Causal World Models

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了CausalARC，一个用于评估AI在低数据和分布偏移条件下推理能力的实验性测试平台，其设计灵感来源于抽象推理语料库（ARC）。每个CausalARC任务均从一个完整指定的结构因果模型中采样，通过原则性的数据增强提供观察性、干预性和反事实反馈，以支持少样本上下文学习。作者展示了CausalARC在四种语言模型评估场景中的应用，包括测试时训练的抽象推理、反事实推理、程序合成以及结合逻辑推理的因果发现，结果显示模型间及模型内性能差异显著，表明语言模型推理能力仍有很大提升空间。

方法概述 / Method

CausalARC采用结构因果模型（SCM）作为形式化框架，为每个推理任务构建完全指定的因果世界模型。通过系统性的数据增强技术，生成观察数据、干预数据和反事实数据，形成少样本的上下文学习示例。该测试平台支持多种评估设置，允许研究者在受控的因果环境中检验模型的推理能力。

实验结果 / Results

实验评估了语言模型在四种不同设置下的表现，涵盖抽象推理、反事实推理、程序合成和因果发现任务。研究发现，模型性能在任务内和任务间均存在显著差异，表明当前语言模型在处理需要因果理解和分布外泛化的复杂推理任务时表现不稳定，揭示了现有AI系统在因果推理方面的局限性。

应用价值 / Applications

CausalARC为评估和提升AI系统的因果推理能力提供了标准化基准，特别适用于需要快速适应新问题和有限数据的实际场景，如科学发现、决策支持和自适应学习系统。该测试平台有助于推动更鲁棒、更具可解释性的AI系统开发，为研究人类水平抽象推理的机器学习模型提供重要工具。
