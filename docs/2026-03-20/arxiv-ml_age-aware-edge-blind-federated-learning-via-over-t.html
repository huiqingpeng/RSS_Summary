Age-Aware Edge-Blind Federated Learning via Over-the-Air Aggregation

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文研究了无线衰落信道上的联邦学习（FL）问题，提出了一种高效的年龄感知边缘盲空中聚合（over-the-air aggregation）方法。该方法无需设备端获取信道状态信息（CSI），而是通过参数服务器（PS）的多天线系统和最大比合并（MRC）技术来检测模型更新。为应对正交子载波数量有限导致的延迟问题，作者设计了AgeTop-k选择策略，优先选取幅度最大的条目，再选择等待时间最长的k个坐标，确保所有选中参数能装入单个OFDM符号。

方法概述 / Method

该方法采用边缘盲设计，设备无需CSI即可进行传输，PS端利用多天线阵列和基于估计信道增益之和的最大比合并进行信号检测。核心创新在于AgeTop-k坐标选择机制：结合参数幅度大小和"年龄"（自上次被选中以来的等待时间）进行动态稀疏化，实现单符号传输以降低延迟。

实验结果 / Results

实验表明：增加PS天线数量能显著提升准确率和收敛速度；在信道条件较好时，AgeTop-k策略优于随机选择；最优k值取决于信道质量，噪声较大时较小的k表现更佳。理论分析揭示了天线数量增加带来的增益，以及k值增大虽降低压缩误差但会放大信道噪声影响的权衡关系。

应用价值 / Applications

该研究适用于资源受限的物联网和边缘计算场景，特别是设备计算能力有限、信道估计困难的分布式机器学习应用，如智能传感器网络、移动设备协作训练等，能够在保证隐私的前提下实现低延迟、高能效的联邦学习。
