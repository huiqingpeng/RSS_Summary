论文摘要 / Abstract

本文提出了KAN-AFT，一种基于Kolmogorov-Arnold表示的结构化非参数扩展框架，用于加速失效时间(AFT)模型。该方法用未知光滑函数替代传统AFT模型中的线性预测器，能够捕捉协变量与对数生存时间之间的非线性关系。作者形式化了独立右删失下的非线性AFT估计量，证明所提函数类严格包含经典线性AFT模型作为特例，并通过统一的估计框架兼容多种删失调整损失函数。模拟研究和多组临床数据应用表明，该方法在适当时能恢复线性结构，存在非线性效应时能有效捕捉，且具有竞争力的预测性能和可解释的协变量效应估计。

方法概述 / Method

该研究采用Kolmogorov-Arnold网络(KAN)表示回归函数，将多维协变量映射分解为可学习的单变量基函数组合，实现非参数化建模。估计过程整合了Buckley-James、逆概率删失加权(IPCW)和变换方法等多种删失调整损失函数，形成统一的优化框架。此外，引入结构正则化和剪枝策略促进模型简洁性，并通过符号近似获得学习到的分量函数的解析表达式。

实验结果 / Results

模拟研究表明，KAN-AFT在真实效应为线性时能正确识别并恢复线性结构，在非线性场景下能准确捕捉复杂效应模式，展现了良好的模型适应性。在多个临床数据集上的应用显示，该方法与标准生存模型相比具有竞争力的预测性能，同时提供了透明的协变量效应可视化解释，克服了传统黑箱模型的可解释性局限。

应用价值 / Applications

KAN-AFT适用于具有复杂协变量结构和异质性删失机制的临床生存分析场景，如精准医疗中的患者生存预测和预后因素识别。该方法为医学研究者提供了兼具预测准确性和可解释性的分析工具，其符号近似能力还能生成临床可理解的解析公式，有助于构建基于数据驱动的临床决策支持系统。
