Image2Garment: Simulation-ready Garment Generation from a Single Image

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了Image2Garment，一种从单张图像生成可直接用于物理仿真的服装的新方法，解决了现有方法需要多视角采集或无法预测材料属性的局限。该方法通过微调视觉-语言模型推断材料成分和织物属性，再通过轻量级预测器将这些属性映射到物理参数，无需迭代优化即可生成仿真就绪的服装。实验表明，该方法在材料成分估计和织物属性预测上具有更高精度，并能实现更高保真度的物理仿真。

方法概述 / Method

该方法采用两阶段流程：首先微调视觉-语言模型（VLM）从真实图像中推断材料成分和织物属性（如厚度、弯曲刚度等），然后训练一个轻量级物理参数估计器，将这些语义属性映射到具体的物理仿真参数（如杨氏模量、泊松比等）。整个框架为前馈网络，避免了昂贵的可微分仿真迭代优化。

实验结果 / Results

该方法在材料成分估计和织物属性预测任务上优于现有方法，并通过物理参数估计器实现了更高保真度的服装仿真效果。研究还引入了两个新数据集：FTAG（用于织物属性标注）和T2P（用于材料-物理参数对应关系），以支持模型训练和评估。

应用价值 / Applications

该技术可广泛应用于虚拟试衣、数字时装设计、游戏和影视制作中的服装动画生成，以及电子商务平台的3D服装展示。通过从单张图像快速生成物理准确的仿真服装，大幅降低了3D服装内容创作的技术门槛和成本。
