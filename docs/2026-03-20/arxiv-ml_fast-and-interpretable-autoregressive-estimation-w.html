论文摘要 / Abstract

本文提出了一种将自回归（AR）模型结构嵌入前馈神经网络的参数估计新方法，通过反向传播实现系数估计，同时保持模型的可解释性。在125,000组合成AR(p)时间序列（1≤p≤5）的仿真实验中，该方法在所有序列上均能稳定收敛并恢复模型系数，而传统条件最大似然法（CML）约有55%的情况无法收敛。神经网络方法在收敛时估计精度与CML相当，但在CML失效时仍能提供可靠估计，且计算效率显著提升，中位加速比达12.6倍，最高可达34.2倍。

方法概述 / Method

该研究将自回归结构直接嵌入前馈神经网络，利用梯度下降优化替代传统的最大似然估计，使AR系数估计转化为神经网络的权重学习问题。这种方法保留了AR模型的线性可解释结构，同时借助深度学习框架的高效自动微分能力实现参数优化。

实验结果 / Results

实验基于125,000组短程依赖的合成AR(p)序列（阶数p=1至5），结果表明神经网络方法实现100%收敛率，而CML失败率约55%；两者收敛时相对误差、R²及困惑度/似然度差异可忽略；CML失效情况下神经网络仍输出合理估计；计算速度方面，神经网络方法中位加速12.6倍，高阶模型（p=5）最高加速34.2倍。

应用价值 / Applications

该方法为时间序列分析提供了兼具可解释性与计算效率的实用工具，特别适用于需要快速估计、大规模数据或实时处理的场景，如金融高频数据建模、工业传感器监控及在线预测系统。其稳定收敛特性也使其成为传统MLE方法失效时的可靠替代方案。
