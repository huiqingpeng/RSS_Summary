Size-adaptive Hypothesis Testing for Fairness

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对算法公平性评估中的统计脆弱性问题，提出了一种统一的大小自适应假设检验框架。该框架解决了传统方法忽略抽样误差、对小样本子群和大子群一视同仁的缺陷，特别是在交叉分析（多个敏感属性联合考虑）导致子群粒度变细、数据稀疏的情况下。作者证明了大子群统计差异的中心极限定理，并为小样本子群设计了贝叶斯Dirichlet-多项式估计器，使可信区间能随样本量自然收敛到经典区间。

方法概述 / Method

论文采用双轨方法：对于足够大的子群，基于统计差异的中心极限定理推导解析置信区间和Wald检验，确保第一类错误率控制在显著性水平α；对于小样本的长尾交叉子群，采用完全贝叶斯的Dirichlet-多项式估计器，通过蒙特卡洛方法校准可信区间。两种方法在样本量变化时能够平滑衔接，形成统一的统计决策框架。

实验结果 / Results

作者在基准数据集上验证了该方法的有效性，结果表明该框架能够在不同数据可用性和交叉性程度下提供可解释且统计严谨的公平性检验决策。贝叶斯方法在小样本情况下产生合理的不确定度量化，并随着数据增加自然收敛到频率学派的Wald区间，避免了小样本下置信区间过宽导致无法得出有意义结论的问题。

应用价值 / Applications

该研究对算法审计和合规监管具有重要实践价值，可帮助企业和监管机构在数据稀疏的交叉子群（如"黑人女性""老年亚裔"等细粒度群体）上进行科学的公平性评估，避免因样本量不足而被迫忽略这些群体的困境。该方法适用于招聘、信贷、医疗等高风险决策场景的公平性监测，支持负责任的AI部署和公平性标准的证据化执行。
