The Exponentially Weighted Signature

论文摘要 / Abstract

本文提出了指数加权签名（Exponentially Weighted Signature, EWS），一种对经典路径签名的推广方法。
EWS通过引入一般性有界线性算子（而非仅对角矩阵）来实现跨通道的时间加权耦合，支持振荡、增长和状态依赖等更丰富的记忆动态，同时保持经典签名的代数优势。作者证明EWS是张量代数上线性控制微分方程的唯一解，并统一了状态空间模型与路径的拉普拉斯/傅里叶变换。实验表明EWS在随机微分方程回归任务上显著优于经典签名和指数衰减记忆（EFM）签名。

方法概述 / Method

EWS的核心是将指数衰减记忆（EFM）签名中的对角权重矩阵推广为一般性有界线性算子，从而允许不同通道间的时间权重相互耦合。该方法通过生成元的半群作用进行参数化，使得整个框架可通过梯度下降进行端到端学习，同时保持群结构以实现高效计算。

实验结果 / Results

作者在基于随机微分方程（SDE）的两个回归任务上验证了EWS的表达能力：实验明确展示了EWS相对于经典签名和EFM签名的性能优势，证明了引入跨通道耦合和更丰富记忆动态机制的有效性。

应用价值 / Applications

EWS适用于需要处理多变量时间序列并动态权衡历史信息相关性的场景，如金融时间序列预测、物理系统建模和信号处理。该框架为状态空间模型和频域分析方法提供了统一的数学基础，可集成到现代深度学习流程中进行可学习的时序特征提取。
