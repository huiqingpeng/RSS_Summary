Auditing Preferences for Brands and Cultures in LLMs

arXiv Machine Learning 2026-03-20

论文摘要 / Abstract

本文提出了ChoiceEval框架，用于审计大型语言模型（LLMs）在品牌和文化偏好方面的系统性偏差。研究发现，美国开发的模型（Gemini、GPT）显著偏向美国实体，而中国开发的DeepSeek虽更平衡但仍存在可检测的地理偏好，且这些模式在不同用户画像中持续存在，表明这是系统性效应而非偶然现象。该研究为量化LLM驱动市场中介的系统性风险提供了可复现的方法论基础。

方法概述 / Method

ChoiceEval框架通过两大核心技术解决审计难题：（1）基于心理画像（如预算敏感型、健康导向型）生成多样化、贴近真实场景的用户查询；（2）将开放式文本输出转换为标准化的top-k选择集和量化偏好指标。该框架通过归一化处理实现跨主题、跨画像的可比性分析，构建了可扩展的审计流水线。

实验结果 / Results

研究对Gemini、GPT和DeepSeek三大模型在10个商业与文化主题、超过2000个问题上的测试显示：美国模型表现出明显的"美国实体偏好"，DeepSeek虽相对平衡但仍存在可量化的地理偏向性。关键的是，这些偏好模式在不同用户画像间保持稳定，排除了用户语境作为偏差来源的解释，指向模型本身的系统性特征。

应用价值 / Applications

该框架为研究者、平台和监管机构提供了量化LLM市场中介风险的实用工具，可应用于反垄断审查、算法公平性评估和信息多样性监管。通过将模型行为与现实经济后果（如品牌竞争、文化代表性）直接关联，ChoiceEval支持对AI系统商业影响力的循证治理，助力维护数字市场的公平竞争环境。
