A Synthesizable RTL Implementation of Predictive Coding Networks

arXiv Computer Architecture 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了一种可综合的RTL硬件架构，用于实现预测编码网络（Predictive Coding Networks）。该架构通过局部预测误差动力学实现推理和学习，避免了反向传播所需的全局误差传播和集中式存储。每个神经核心独立维护自身活动、预测误差和突触权重，仅通过硬连线与相邻层通信，支持监督学习和推理的统一神经元钳位原语。

方法概述 / Method

该设计采用离散时间预测编码更新规则，构建基于顺序MAC（乘累加）数据通路和固定有限状态调度的确定性数字架构。系统不执行任务特定的指令序列，而是在固定局部更新规则下演化，通过连接结构、参数和边界条件来施加任务结构。

实验结果 / Results

论文主要贡献在于提出了首个完整的可综合数字底层架构，而非新的学习规则。该RTL实现可直接在硬件上执行预测编码学习动力学，实现了完全分布式、在线的硬件学习系统，克服了反向传播在硬件实现中的关键限制。

应用价值 / Applications

该研究为边缘AI和神经形态计算提供了高效的硬件实现方案，特别适用于需要低功耗、实时在线学习的场景，如嵌入式智能系统、传感器网络和自主机器人平台，有望推动预测编码网络从理论走向实际硬件部署。
