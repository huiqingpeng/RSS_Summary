论文摘要 / Abstract

本文提出了一种利用大规模视频生成模型中隐式空间先验来解决多模态大语言模型（MLLMs）"空间盲"问题的新范式。作者开发了VEGA-3D框架，将预训练视频扩散模型重新定位为"潜在世界模拟器"，通过从中级噪声层提取时空特征并与语义表示进行自适应融合，无需显式3D监督即可增强MLLMs的几何推理能力。实验表明，该方法在3D场景理解、空间推理和具身操作等基准上均优于现有最先进方法，验证了生成先验可作为物理世界理解的可扩展基础。

方法概述 / Method

VEGA-3D采用即插即用的架构设计，核心是将视频扩散模型的中间层噪声特征作为隐式3D结构信息的载体。该方法通过token级自适应门控融合机制，将提取的时空特征与MLLM的语义表示进行动态整合，使模型在保持语义理解能力的同时获得密集几何线索。整个框架无需针对3D任务进行额外训练或显式3D数据监督。

实验结果 / Results

该方法在多个3D场景理解、空间推理和具身操作基准测试中均取得领先性能，显著优于依赖显式3D模态或复杂几何支架的现有基线方法。实验验证了从视频生成模型中提取的隐式空间先验能够有效弥补MLLMs在细粒度几何推理和物理动力学理解方面的不足，且展现出良好的泛化能力。

应用价值 / Applications

该技术可直接应用于机器人具身智能、自动驾驶场景理解、AR/VR空间交互等需要精细3D推理的物理世界任务。由于采用即插即用设计且无需昂贵的3D标注数据，该方法为构建可扩展的物理世界理解系统提供了实用路径，有望降低开发具备空间智能的多模态AI系统的数据门槛和计算成本。
