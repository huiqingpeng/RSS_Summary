How Uncertainty Estimation Scales with Sampling in Reasoning Models

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本研究系统分析了推理语言模型中不确定性估计与并行采样规模的关系，发现自一致性（self-consistency）和言语化置信度（verbalized confidence）两种信号在推理模型中均可扩展，但自一致性的初始区分能力较低且在中等采样规模下落后于言语化置信度。研究提出混合估计器，仅需两个样本即可在平均AUROC上提升高达+12，且优于单独使用任一信号的大规模采样效果，同时发现这些效应具有领域依赖性，数学领域表现最优。

方法概述 / Method

论文采用完全黑盒的并行采样方法，结合言语化置信度和自一致性两种不确定性信号，在三个推理模型和17个涵盖数学、STEM及人文学科的任务上进行大规模评估。研究通过系统变化采样规模，表征两种不确定性信号的缩放行为，并探索信号组合策略以构建混合估计器。

实验结果 / Results

实验表明，大多数不确定性增益来自信号组合：混合估计器仅需两个样本即可实现显著性能提升（AUROC平均+12），且已超越单一信号在更大采样预算下的表现，之后收益递减。数学领域表现出更高的不确定性质量、更强的信号互补性和更快的缩放速度，而STEM和人文学科相对较弱。

应用价值 / Applications

该研究为推理语言模型的安全部署提供了实用指导，特别是在资源受限场景下，通过低成本的双样本混合策略即可实现高效的不确定性量化。研究结果有助于优化模型在数学解题、科学推理等高风险决策任务中的可靠性评估，并提示需针对不同领域定制不确定性估计策略。
