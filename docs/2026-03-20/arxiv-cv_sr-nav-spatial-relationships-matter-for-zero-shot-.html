SR-Nav: Spatial Relationships Matter for Zero-shot Object Goal Navigation

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了SR-Nav框架，用于解决零样本目标物体导航中因视角不佳或语义线索不足导致的导航失败问题。核心洞察是物体与区域间的固有空间关系编码了结构化场景先验，即使部分观测也能推断合理的目标位置。该框架通过动态空间关系图（DSRG）建模观测和经验空间关系，在HM3D数据集上取得了最先进的成功率和导航效率。

方法概述 / Method

SR-Nav包含三个核心组件：首先构建以目标为中心的动态空间关系图（DSRG），利用基础模型编码空间关系并随实时观测动态更新；其次引入关系感知匹配模块，用关系匹配替代简单检测，利用DSRG中的多样化关系验证和纠正错误；最后设计动态关系规划模块，基于当前位置的DSRG动态计算最优路径，缩小规划搜索空间。

实验结果 / Results

在HM3D基准数据集上的实验表明，SR-Nav在成功率和导航效率两方面均达到 state-of-the-art 性能。该方法通过空间关系建模有效提升了感知鲁棒性和规划效率，显著减少了探索冗余。

应用价值 / Applications

该研究可应用于家庭服务机器人、仓储物流机器人等需要在未知环境中自主寻找目标物体的场景。通过利用空间关系先验，机器人能够在视觉信息受限的情况下更高效地完成导航任务，降低对高质量视觉感知的依赖，提升实际部署的可靠性和鲁棒性。
