DyMoE: Dynamic Expert Orchestration with Mixed-Precision Quantization for Efficient MoE Inference on Edge

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出DyMoE，一种面向边缘设备高效推理的动态混合精度量化框架，用于解决MoE模型在资源受限边缘平台上的内存占用过大和I/O开销问题。该框架基于专家重要性高度偏斜且与深度相关的观察，实现了运行时动态量化、关键层语义完整性保护以及I/O预取重叠优化。实验表明，DyMoE在商用边缘硬件上相比最先进的卸载基线，首Token时间（TTFT）降低3.44-22.7倍，每输出Token时间（TPOT）最高加速14.58倍，在保持精度的同时实现了实时边缘推理。

方法概述 / Method

DyMoE采用三项核心技术：（1）重要性感知优先级调度，根据运行时专家激活频率动态分配量化精度；（2）深度自适应调度，对深层关键层保留更高精度以保护语义完整性；（3）前瞻预取机制，通过预测下一层所需专家并与计算重叠来隐藏I/O延迟。该方法将混合精度量化与动态执行调度相结合，突破了传统静态量化方法的刚性延迟-精度权衡。

实验结果 / Results

在商用边缘硬件上的评估显示，DyMoE显著优于现有卸载基线方法：首Token生成时间（TTFT）实现3.44倍至22.7倍的降低，每输出Token时间（TPOT）获得高达14.58倍的加速。这些结果表明该框架成功克服了MoE架构在边缘部署中的内存和I/O瓶颈，在精度损失可忽略的情况下达成了实时推理性能。

应用价值 / Applications

DyMoE使大规模MoE模型能够在智能手机、嵌入式设备等资源受限边缘平台上高效运行，拓展了云端大模型向端侧部署的可能性。其实时推理能力适用于低延迟需求的交互式应用，如边缘端智能助手、实时翻译和自动驾驶感知系统，同时动态量化机制为边缘AI的能效优化提供了可扩展的技术路径。
