Evaluating Model-Free Policy Optimization in Masked-Action Environments via an Exact Blackjack Oracle

论文摘要 / Abstract

本研究利用无限牌靴赌场二十一点（infinite-shoe casino blackjack）作为动态掩码动作环境下离散随机控制的严格可验证基准。作者推导出精确动态规划（DP）预言机，覆盖4,600个规范决策单元，获得真实动作值和最优策略标签。实验比较了三种无模型优化器（REINFORCE、SPSA、CEM），发现REINFORCE样本效率最高，但所有方法均存在显著的单元条件遗憾，表明聚合奖励曲线可能掩盖关键的局部策略失败。

方法概述 / Method

研究构建了基于拉斯维加斯规则集的精确动态规划预言机，生成4,600个决策单元的地面真实动作值和最优策略标签。三种无模型策略优化方法通过模拟交互进行训练：带指数移动平均基线的掩码REINFORCE、同时扰动随机逼近（SPSA）和交叉熵方法（CEM）。此外，研究设计了负对照实验，证明在无记牌情况下最优下注规模坍缩至最小注码。

实验结果 / Results

REINFORCE在10⁶手牌后达到46.37%的动作匹配率和-0.04688的期望价值，优于CEM（39.46%，7.5×10⁶次评估）和SPSA（38.63%，4.8×10⁶次评估）。然而，所有方法均表现出显著的单元条件遗憾，说明即使奖励收敛平滑，策略层面的错误仍持续存在。负对照证实：独立同分布抽牌下，更大下注仅增加波动性和破产风险，不改善期望收益。

应用价值 / Applications

该研究为强化学习算法评估提供了可精确验证的基准环境，强调在动态掩码动作和状态访问稀疏的表格环境中，需要精确预言机和负对照来避免将随机波动误认为算法性能。方法论可推广至其他具有动作空间约束的离散随机控制问题，如金融交易、资源调度和安全关键决策系统，为诊断策略优化的局部失败机制提供框架。
