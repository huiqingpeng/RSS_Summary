Variational Phasor Circuits for Phase-Native Brain-Computer Interface Classification

论文摘要 / Abstract

本文提出了**变分相量电路（Variational Phasor Circuit, VPC）**，一种在连续的 $S^1$ 单位圆流形上运行的确定性经典学习架构。
VPC 受变分量子电路启发，用可训练的相位偏移、局部酉混合和结构化干涉替代密集实值权重矩阵，为空间分布信号的二分类和多分类提供统一方法。在脑机接口基准测试中，VPC 以显著更少的可训练参数达到了与标准欧几里得基线相当的准确率，将单位圆相位干涉定位为密集神经计算的一种数学原理严谨且实用的替代方案。

方法概述 / Method

VPC 核心在于用**相位参数化**替代传统神经网络的实值权重：通过可训练相位偏移、局部酉混合和复空间中的结构化干涉实现信息处理。单个 VPC 块支持紧凑的基于相位的决策边界，而堆叠的 VPC 组合通过块间**拉回归一化（pull-back normalization）**扩展到更深电路，保持相位原生特性。

实验结果 / Results

在合成脑机接口基准测试中，VPC 成功解码困难的心理状态分类任务，取得了**具有竞争力的准确率**，同时可训练参数数量**显著少于标准欧几里得基线**。结果表明相位干涉机制能有效捕捉脑电信号的空间分布特征，实现高效分类。

应用价值 / Applications

VPC 可作为**独立的轻量级分类器**部署于资源受限的脑机接口系统，降低计算和存储开销；同时可作为**前端编码层**与未来混合相量-量子系统结合，为神经形态计算和量子-经典混合智能提供桥梁。其相位原生设计特别适合处理具有周期性或相位敏感特征的神经信号。
