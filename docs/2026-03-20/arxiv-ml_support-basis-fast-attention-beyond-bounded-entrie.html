论文摘要 / Abstract

本文提出了支持基分解（support-basis decomposition）技术，用于在不依赖有界条目假设的情况下实现高效准确的注意力推理和训练。作者发现查询矩阵和键矩阵的条目在实际中呈现次高斯行为，基于此特性对稀疏成分进行精确计算、对稠密成分进行多项式逼近，理论上证明了该算法能达到次二次时间复杂度，同时匹配先前工作的逼近误差。研究还推广到多阈值设置以消除所有分布假设，并为多项式注意力的经验成功提供了首个理论解释。

方法概述 / Method

核心方法是支持基分解技术，利用查询-键矩阵条目的次高斯特性将注意力计算分解为两部分：对稀疏成分进行精确计算，对稠密成分采用多项式逼近。该方法进一步扩展为多阈值设置，通过多个多项式注意力来逼近softmax注意力，无需任何限制性分布假设。

实验结果 / Results

论文从理论上证明了算法在次二次时间复杂度下能达到与先前工作相当的逼近误差，并通过实验验证了其计算效率和下游任务性能。多阈值设置下的多项式注意力能够以显著更小的ℓ_p误差紧密逼近softmax注意力。

应用价值 / Applications

该研究可广泛应用于大规模语言模型的训练和推理加速，突破传统softmax注意力的二次复杂度瓶颈。由于消除了有界条目这一限制性假设，该方法对现代LLM更具实际适用性，同时为多项式注意力等高效注意力变体提供了理论基础，有助于开发更高效的Transformer架构。
