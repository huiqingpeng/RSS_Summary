WeNLEX: Weakly Supervised Natural Language Explanations for Multilabel Chest X-ray Classification

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出WeNLEX，一种用于多标签胸部X光分类的弱监督自然语言解释生成模型。该模型通过将解释生成的图像与原图像在黑盒模型特征空间中进行匹配来确保解释的忠实性（faithfulness），同时通过与少量临床医生标注的解释进行分布对齐来保持解释的合理性（plausibility）。实验表明，WeNLEX仅需每类诊断5个真实解释即可生成忠实且合理的解释，且在模型内训练设置下可将分类AUC提升2.21%，证明增加可解释性训练能够提高下游任务性能。

方法概述 / Method

WeNLEX采用弱监督学习框架，无需大量显式标注的解释数据。核心方法包括：（1）利用图像-文本特征空间匹配机制确保解释忠实反映模型推理过程；（2）通过分布对齐技术将生成解释与少量临床专家标注样本对齐以保证合理性；（3）支持事后解释（post-hoc）和模型内训练（in-model）两种工作模式。

实验结果 / Results

WeNLEX在忠实性、可模拟性、多样性和合理性等多个评估指标上表现优异，仅需每类5个真实解释即可达到与全监督方法相当的效果。在模型内训练设置下，分类AUC相比独立分类器提升2.21%。此外，通过更换解释数据库，WeNLEX可灵活适应不同受众需求，研究还成功训练了面向非医学用户的简化版本。

应用价值 / Applications

该研究具有重要的临床辅助诊断价值，可为放射科医生提供可信赖的AI决策解释，增强人机协作信任度。其弱监督特性大幅降低了解释标注成本，便于在不同医疗机构快速部署。同时，模型可适配专业医生和普通患者等不同受众，在医疗AI可解释性和普惠性方面具有广阔应用前景。
