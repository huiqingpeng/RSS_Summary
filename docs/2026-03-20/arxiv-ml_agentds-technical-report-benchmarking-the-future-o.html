AgentDS Technical Report: Benchmarking the Future of Human-AI Collaboration in Domain-Specific Data Science

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本研究提出了AgentDS，一个用于评估AI代理和人类-AI协作在领域特定数据科学任务中表现的基准测试和竞赛平台。该基准涵盖商业、食品生产、医疗保健、保险、制造业和零售银行六大行业的17项挑战，通过29支队伍、80名参与者的公开竞赛，系统比较了纯AI方法与人类-AI协作方案。研究发现，当前AI代理在领域特定推理方面存在明显不足，纯AI基线表现仅接近或低于竞赛参与者中位数水平，而最强解决方案均来自人类-AI协作，这一发现挑战了AI完全自动化的叙事，强调了人类专业知识在数据科学中的持续重要性。

方法概述 / Method

AgentDS采用竞赛驱动的方法构建基准测试，设计了覆盖六大真实行业场景的17项数据科学挑战任务，涵盖从数据探索到模型部署的完整工作流程。研究通过公开招募29支队伍（共80名参与者），收集人类专家、人类-AI协作团队以及纯AI基线（使用主流LLM和AI代理框架）的解决方案，建立多维度评估体系进行系统性对比分析。

实验结果 / Results

实验结果显示，纯AI基线在所有任务上的表现接近或低于人类参与者中位数，未能达到顶尖人类专家水平；相比之下，采用人类-AI协作策略的团队取得了最优成绩，表明人机协作在当前阶段显著优于单一方案。具体而言，AI代理在处理需要深度领域知识的推理任务时表现尤为薄弱，而人类专家在问题理解、假设生成和结果解释等关键环节仍具不可替代的优势。

应用价值 / Applications

该研究为数据科学领域的AI能力评估提供了标准化基准，可指导企业合理规划人机协作的工作流程设计，避免过度依赖自动化带来的决策风险。研究成果对开发下一代增强型AI系统具有重要启示，提示未来AI设计应聚焦于如何有效整合人类专业知识而非追求完全替代，同时为教育机构和行业培训提供了关于核心人类技能培养的实证依据。
