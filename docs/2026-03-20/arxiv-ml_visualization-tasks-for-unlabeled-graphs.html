Visualization Tasks for Unlabeled Graphs

论文摘要 / Abstract

本文研究了无标签图（节点没有持久或语义化标签的图）的可视化任务，指出现有网络可视化技术虽多，但缺乏针对无标签图任务的系统理解。作者提出了一种数据抽象模型，区分了无标签（Unlabeled）、有标签（Labeled）、带属性（Attributed）和增强型（Augmented）四种语义丰富度递增的图上下文，并据此构建了一个针对无标签图任务的分类体系，最后通过评估6种网络可视化方法展示了该分类体系的描述力和评估力。

方法概述 / Method

作者采用文献收集与筛选的方法，基于提出的数据抽象模型从现有文献中提取适用于无标签图的任务，并构建了一个三维任务分类体系：目标数据（Target）、用户意图（Action）和数据范围（Scope）。该分类体系借鉴并扩展了先前可视化任务框架，通过连接具体实例和真实问题来验证其描述能力，同时设计了初步评估方案对不同可视化编码在各任务上的表现进行对比分析。

实验结果 / Results

研究对6种不同的网络可视化图式（idioms）进行了初步评估，针对每个任务-视觉编码组合，分析了用户所需付出的认知努力、任务成功的可能性，以及这两个因素在小型图和大型图之间的变化差异。评估结果揭示了不同可视化方法在无标签图任务中的相对优劣，为后续技术评估和工具设计提供了参考依据，但具体量化指标未在摘要中详述。

应用价值 / Applications

该研究为无标签图可视化技术的系统评估提供了理论框架和实用工具，可指导网络可视化工具的设计者针对特定任务选择或优化视觉编码方案。其应用场景涵盖社交网络分析、生物网络研究、基础设施网络监控等领域中节点身份未知或标签缺失的情况，帮助研究者和实践者更好地理解和比较不同可视化方法在无标签数据环境下的适用性。
