Learn for Variation: Variationally Guided AAV Trajectory Learning in Differentiable Environments

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对自主飞行器（AAV）在6G物联网网络中的轨迹规划问题，提出了一种名为"Learn for Variation (L4V)"的变分引导轨迹学习框架。传统基于奖励的强化学习方法因稀疏标量奖励无法捕捉序列运动的长期非线性效应而面临信用分配问题和训练不稳定性，L4V通过用密集且解析基础明确的策略梯度替代高方差标量奖励信号来解决这些挑战。该框架将AAV运动学、距离相关信道增益和每用户数据收集进度的耦合演化展开为端到端可微计算图，利用时间反向传播作为离散伴随求解器。大量仿真实验表明，L4V在任务完成时间、平均传输速率和训练成本方面均优于遗传算法、DQN、A2C和DDPG等代表性基线方法。

方法概述 / Method

L4V构建了一个完全可微的环境模型，将AAV动力学、无线信道特性和数据收集状态统一表示为端到端的计算图。通过时间反向传播（BPTT）机制，该方法将累积任务目标对控制动作和策略参数的精确敏感度反向传播，从而获得结构化的策略梯度；这些梯度结合时间平滑正则化和梯度裁剪技术，用于训练确定性神经网络策略。

实验结果 / Results

实验结果表明，L4V在多项关键指标上持续优于传统方法：相比遗传算法、DQN、A2C和DDPG等基线，L4V显著缩短了任务完成时间，提升了平均传输速率，并大幅降低了训练成本。该框架通过密集梯度信号有效缓解了传统强化学习中的高方差问题，展现出更稳定的训练特性和更优的收敛性能。

应用价值 / Applications

该研究可直接应用于6G物联网网络中的移动数据采集场景，如环境监测、灾害救援和智能农业等领域中无人机轨迹优化。
L4V的可微分框架设计为其他具有复杂动态耦合的序列决策问题提供了通用方法论，特别是在需要精确梯度信息和稳定训练的大规模网络优化任务中具有重要参考价值。
