Improving Spatial Allocation for Energy System Coupling with Graph Neural Networks

arXiv Machine Learning 2026-03-20

论文摘要 / Abstract

本文针对能源系统分析中不同空间分辨率模型耦合的挑战，提出了一种基于自监督异构图神经网络的创新方法。该方法将高分辨率地理单元建模为图节点，整合多种地理特征生成具有物理意义的权重，显著增强了传统Voronoi空间分配方法的效果。实验结果表明，该方法在可扩展性、准确性和物理合理性方面均优于传统单属性地理加权方法，同时克服了真实标注数据缺乏的问题。

方法概述 / Method

论文采用自监督异构图神经网络（Heterogeneous Graph Neural Network），将高分辨率地理单元表示为图结构中的节点，通过融合多维度地理特征学习空间分配权重。该方法通过自监督学习范式进行训练，无需依赖准确的地面真实数据，生成的权重用于优化基于聚类的Voronoi图空间分配策略。

实验结果 / Results

实验结果表明，应用图神经网络生成的权重对聚类Voronoi图进行增强后，在空间分配任务中显著提升了可扩展性、准确性和物理合理性，相比仅依赖单一地理邻近属性的传统方法具有更高的精度。该方法有效突破了传统模型仅使用单一地理空间属性的局限性。

应用价值 / Applications

该研究为能源系统建模中的多源异构数据空间耦合问题提供了通用解决方案，可广泛应用于气候变化影响评估、可再生能源潜力分析、区域能源规划等需要整合不同分辨率地理数据的场景。其自监督特性使其特别适用于标注数据稀缺的能源地理信息学领域。
