SOL-ExecBench: Speed-of-Light Benchmarking for Real-World GPU Kernels Against Hardware Limits

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了SOL-ExecBench，一个包含235个CUDA内核优化问题的基准测试集，涵盖124个生产级和新兴AI模型（包括语言、扩散、视觉、音频、视频及混合架构），专门针对NVIDIA Blackwell GPU。该基准测试通过SOLAR工具计算硬件层面的"光速"（Speed-of-Light, SOL）理论上限，而非传统的软件基线对比，从而重新定义GPU内核优化的评估标准。研究还提供了一个沙盒化评估环境，包含GPU时钟锁定、L2缓存清除等机制以防止奖励作弊，最终通过SOL Score量化候选内核接近硬件极限的程度。

方法概述 / Method

论文开发了SOLAR流水线，用于从硬件规格出发解析推导SOL理论性能边界，为每个内核建立固定的硬件效率目标。评估框架采用沙盒化设计，集成GPU时钟锁定、L2缓存清除、隔离子进程执行以及基于静态分析的奖励作弊检测机制，确保对自动化优化代理（agentic optimizers）的鲁棒性评估。

实验结果 / Results

SOL-ExecBench包含235个内核优化问题，覆盖前向和反向传播工作负载，支持BF16、FP8和NVFP4精度格式，并特别包含依赖Blackwell特有能力的内核。该基准测试通过SOL Score指标，将传统"超越可变软件基线"的评估范式转变为"缩小与硬件光速上限差距"的固定目标评估。

应用价值 / Applications

该研究为AI驱动的GPU内核自动生成与优化系统提供了标准化、防作弊的评估基础设施，特别适用于代理型AI（agentic AI）系统的开发与验证。通过建立硬件极限导向的评估标准，SOL-ExecBench有助于推动GPU内核优化从相对性能提升向绝对硬件效率优化的范式转变，对AI模型推理和训练效率的持续提升具有重要工程价值。
