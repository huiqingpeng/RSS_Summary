Towards High-Quality Image Segmentation: Improving Topology Accuracy by Penalizing Neighbor Pixels

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了一种名为 SCNP（Softmax Class Neighbor Penalty）的高效方法，通过惩罚与分类最差的邻居像素之间的 logits 差异，强制模型在优化像素自身预测前先改善其邻居区域的预测质量。该方法在 13 个涵盖不同结构形态和成像模态的数据集上验证有效，可无缝集成到语义分割和实例分割的多种框架及损失函数中，显著提升分割结果的拓扑准确性，且计算开销低、易于部署。

方法概述 / Method

SCNP 的核心思想是在训练过程中对每个像素的 logits 施加一个基于其"最差分类邻居"的惩罚项：若某像素的某个邻居被模型严重误分类（即该邻居对应错误类别的 logit 很高），则惩罚该像素对应正确类别的 logit，迫使模型优先纠正邻居区域的错误预测。该机制无需额外的网络架构修改或复杂的后处理，可直接嵌入现有分割模型的训练流程。

实验结果 / Results

实验表明 SCNP 在 13 个多样化数据集上均有效提升拓扑准确性，涵盖血管、细胞、道路等管状与非管状结构；该方法成功集成到三种主流分割框架（包括语义分割和实例分割）中，并能增强多种现有损失函数（如 Dice、Cross-Entropy）的拓扑保持能力，且计算成本显著低于专门的拓扑损失函数（如基于持久同调的方法）。

应用价值 / Applications

SCNP 可广泛应用于医学图像分析（如血管分割、细胞计数）、遥感图像解译（如道路提取）以及工业检测等需要精确拓扑结构的场景，其即插即用的特性使其能快速部署到现有深度学习工作流中，为下游定量分析（如长度测量、连通组件统计）提供更可靠的分割基础，避免因拓扑错误导致的系统性偏差。
