STEP: Detecting Audio Backdoor Attacks via Stability-based Trigger Exposure Profiling

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了STEP，一种针对音频后门攻击的检测方法，利用后门触发器的双重异常特性：在语义破坏扰动下的异常标签稳定性，以及在语义保持扰动下的异常标签脆弱性。该方法无需重训练，仅需黑盒硬标签访问，在七种后门攻击上平均AUROC达97.92%，显著优于现有基线方法，并能泛化到不同模型架构、语音任务、开集验证场景及物理世界环境。

方法概述 / Method

STEP包含两个互补的扰动分支：语义破坏分支通过时域拉伸、音调变化等操作检测触发器的过度稳定性，语义保持分支通过加性噪声、房间脉冲响应等操作检测触发器的异常脆弱性。两个分支的稳定性特征分别由单类异常检测器评分，并通过无监督加权融合得到最终检测结果。

实验结果 / Results

在七种后门攻击（包括基于变换和语义的触发器）的广泛实验中，STEP平均AUROC为97.92%，EER为4.54%，大幅超越现有最先进基线。该方法成功泛化至不同模型架构（如ECAPA-TDNN、ResNet）、多种语音任务（识别、验证、指令识别）、开集说话人验证场景以及过空传播物理世界攻击。

应用价值 / Applications

STEP适用于安全关键场景中的深度学习语音系统防护，如语音助手、身份验证系统和智能指令控制，能够在部署后无需模型重训练或白盒访问的情况下实时检测后门攻击，为云端API和边缘设备上的语音服务提供实用的安全保障。
