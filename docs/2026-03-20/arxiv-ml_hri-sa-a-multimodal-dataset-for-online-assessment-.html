HRI-SA: A Multimodal Dataset for Online Assessment of Human Situational Awareness during Remote Human-Robot Teaming

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了首个公开的多模态数据集HRI-SA，用于系统评估远程人机协作中操作员的在线情境意识（SA）。该数据集采集了30名参与者在搜救任务中的眼动、瞳孔直径、生物信号、用户交互及机器人数据，并建立了感知型和理解型两种SA延迟的地面真值标注。研究表明，仅用眼动特征即可有效检测感知型SA延迟（召回率88.91%，F1=67.63%），融合上下文数据后性能进一步提升（召回率91.51%，F1=80.38%）。

方法概述 / Method

研究采用真实搜救场景的人机协作实验范式，通过预定义需要操作员及时响应的事件来诱发SA缺口，并以"辅助需求发生至解决"的时间差作为SA延迟的客观测量。数据集整合了五类异构数据流，并采用留一组交叉验证策略评估机器学习模型的在线检测能力。

实验结果 / Results

基于通用眼动特征的基线模型实现了88.91%的召回率和67.63%的F1分数；引入任务上下文特征（如机器人状态、交互事件）后，F1分数显著提升12.75个百分点至80.38%，同时召回率增至91.51%。结果验证了多模态融合对SA在线评估的有效性。

应用价值 / Applications

该数据集为开发实时SA监测与自适应辅助系统提供了基准资源，可应用于远程搜救、无人机操控、太空探索等高风险人机协作场景，支持系统在检测到操作员认知负荷过高或注意力缺失时主动介入，从而提升任务安全性与效率。
