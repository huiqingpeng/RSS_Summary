DeeperBrain: A Neuro-Grounded EEG Foundation Model Towards Universal BCI

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了DeeperBrain，一种神经科学启发的EEG基础模型，旨在解决现有EEG基础模型在冻结探测协议下泛化能力不足的问题。该模型通过整合神经活动的生物物理和动力学原理，设计了体积传导感知的空间编码和神经动力学感知的时间编码架构，并采用掩码EEG重建与神经动力学统计预测的双目标预训练策略。实验表明，DeeperBrain在端到端微调下达到最优性能，同时在严格的冻结探测协议下保持卓越效果，验证了神经科学原理赋予表征内在普适性的关键作用。

方法概述 / Method

DeeperBrain在架构层面引入两项神经科学先验：一是体积传导感知通道编码，利用三维几何建模电极间的空间混合效应；二是神经动力学感知时间编码，采用振荡基函数和指数基函数捕捉神经信号的慢速适应性。预训练阶段采用双目标策略，包括掩码EEG重建（MER）以保证局部信号保真度，以及神经动力学统计预测（NSP）以预测频谱功率、功能连接、跨频率耦合和动态复杂度等可解释的宏观脑状态序参量。

实验结果 / Results

DeeperBrain在端到端微调设置下达到当前最优或极具竞争力的性能水平。关键发现是，该模型在严格的冻结探测协议（frozen-probing）下仍保持卓越效能，显著优于现有方法，证明其学习表征具有内在普适性而非依赖下游任务微调。这一结果验证了神经科学原理嵌入对构建通用BCI基础模型的决定性作用。

应用价值 / Applications

DeeperBrain为通用脑机接口（Universal BCI）提供了可迁移的神经表征基础，可广泛应用于多种下游神经解码任务而无需大量任务特定训练。其神经科学可解释的设计有助于临床神经监测、神经疾病诊断和神经调控等场景，同时冻结探测的高效性使其适用于资源受限的实时BCI系统部署。该模型架起了计算神经科学与实用BCI技术之间的桥梁。
