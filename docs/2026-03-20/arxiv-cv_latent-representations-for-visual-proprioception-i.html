Latent Representations for Visual Proprioception in Inexpensive Robots

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文研究了低成本机器人的视觉本体感知问题，探索了多种潜在表示方法（包括CNN、VAE、ViT以及未校准基准标记袋）在单张外部相机图像中快速回归关节位置的可行性。研究采用针对有限数据优化的微调技术，在廉价的6自由度机器人上评估了可达到的精度，证明了无需昂贵编码器即可实现视觉本体感知的潜力。

方法概述 / Method

论文采用单遍回归架构，从单张RGB图像直接预测机器人关节角度；比较了卷积神经网络、变分自编码器、视觉Transformer以及基于ArUco等基准标记的传统计算机视觉方法作为潜在表示，并结合迁移学习与数据增强等微调策略以适应小数据集场景。

实验结果 / Results

实验在低成本6-DoF机器人上验证了各方法的关节位置估计精度，具体量化指标（如平均关节角度误差）未在摘要中披露；研究表明该方法能够在典型操作距离下替代传统编码器，为无传感器或低成本机器人平台提供了可行的本体感知解决方案。

应用价值 / Applications

该研究使缺乏精密编码器的廉价机器人（如教育机器人、DIY机械臂）能够在非结构化环境中获得本体感知能力，降低硬件成本的同时拓展了视觉伺服、遥操作和无传感器力控制等应用的可行性，对资源受限场景下的机器人普及具有重要推动作用。
