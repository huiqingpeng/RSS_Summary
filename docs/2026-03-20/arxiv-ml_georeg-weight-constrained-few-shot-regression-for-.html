GeoReg: Weight-Constrained Few-Shot Regression for Socio-Economic Estimation using LLM

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了GeoReg模型，用于整合卫星图像和网络地理空间信息等多源数据来估计区域GDP、人口和教育水平等社会经济指标，特别针对数据稀缺的发展中国家。该方法利用大语言模型的先验知识作为"数据工程师"提取信息特征，通过分析特征与目标指标之间的正负相关、混合相关或无关关系，为线性估计器设置定制化的权重约束。在三个不同发展阶段国家的实验表明，该模型在低数据可用性的低收入国家仍能优于基线方法。

方法概述 / Method

GeoReg采用大语言模型提取卫星图像和地理空间数据的上下文特征关系，将特征按相关性分类（正相关、负相关、混合相关、无关），并为每类特征设计特定的权重约束机制引入线性估计器。同时，模型识别有意义的特征交互并进行非线性变换，以捕捉复杂的数据模式。

实验结果 / Results

论文在三个处于不同发展阶段的国家进行了实验验证，结果显示GeoReg在少样本设置下估计社会经济指标的性能显著优于基线方法。该方法尤其适用于数据稀缺的低收入国家，即使在标注数据极其有限的情况下仍能保持较高的估计准确性。

应用价值 / Applications

该研究对发展中国家的政策制定和可持续发展规划具有重要价值，能够在传统统计数据缺失或不完整的地区提供可靠的社会经济指标估计。应用场景包括区域发展规划、扶贫资源分配、教育投资优化等需要精细地理粒度经济数据的政府决策支持系统。
