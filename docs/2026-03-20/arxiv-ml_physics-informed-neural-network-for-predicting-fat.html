论文摘要 / Abstract

本研究提出了一种物理信息神经网络（PINN）框架，用于预测核反应堆中经辐照和未经辐照的奥氏体钢及铁素体/马氏体（F/M）钢的低周疲劳（LCF）寿命。该PINN将疲劳寿命控制物理约束嵌入损失函数，实现了物理一致性的学习，同时提高了预测准确性、可靠性和泛化能力。模型在495个应变控制疲劳数据点上进行训练，与传统机器学习方法（包括随机森林、梯度提升、XGBoost和传统神经网络）相比表现出更优性能。研究通过SHAP分析识别出应变幅值、辐照剂量和测试温度为关键特征，并揭示了奥氏体钢与F/M钢不同的合金特异性退化机制。

方法概述 / Method

研究采用物理信息神经网络（PINN）架构，将疲劳寿命相关的物理约束（如材料退化机制和疲劳损伤规律）显式嵌入神经网络的损失函数中，以引导模型学习物理上一致的映射关系。模型基于495个应变控制疲劳实验数据点进行训练，涵盖辐照与未辐照条件，并采用SHAP（SHapley Additive exPlanations）方法进行特征重要性分析和模型可解释性评估。

实验结果 / Results

PINN在预测精度上显著优于随机森林、梯度提升、XGBoost和传统神经网络等基线方法。
SHAP分析表明应变幅值、辐照剂量和测试温度是影响疲劳寿命的主导特征，均与疲劳寿命呈物理意义上的负相关。单变量和多变量分析揭示了明显的合金特异性退化特征：奥氏体钢在应变幅值、辐照剂量和温度之间存在强烈的非线性耦合，导致复合加载下疲劳退化显著；而F/M钢表现出相对稳定的辐照响应（包括剂量饱和行为），但在超过回火阈值的高温下敏感性增加。

应用价值 / Applications

该PINN框架可作为反应堆相关疲劳评估的可靠且可解释的工具，为先进核能应用（如第四代反应堆、聚变堆等）中的材料性能评价提供支持。研究成果有助于优化核结构材料的服役寿命预测、制定更科学的检修维护策略，并为抗辐照疲劳材料的设计开发提供理论指导，从而提升核设施的安全性和经济性。
