SenseShift6D: Multimodal RGB-D Benchmarking for Robust 6D Pose Estimation across Environment and Sensor Variations

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了SenseShift6D，首个系统研究RGB-D传感器参数（曝光、增益、深度模式）和环境光照变化对6D位姿估计影响的基准数据集。该数据集包含19.8万张RGB图像和2万张深度图像，涵盖1,380种独特的传感器-光照组合。实验发现，即使是最先进的预训练模型和实例级模型，对这些环境变化和传感器设置也表现出显著的敏感性，揭示了现有方法在真实部署中的鲁棒性缺陷。

方法概述 / Method

作者通过物理采集方式构建了大规模多模态数据集，针对6个常见家用物体，系统扫描了13种RGB曝光、9种RGB增益、自动曝光、4种深度捕获模式和5种光照水平。该数据集采用严格的实验控制设计，确保每个物体位姿对应多种传感器-环境组合，从而支持对位姿估计模型鲁棒性的细粒度分析。

实验结果 / Results

实验表明，现有SOTA预训练和可泛化位姿估计器在不同光照和传感器设置下性能波动显著；令人惊讶的是，即使实例级估计器（在相同物体和背景下训练测试）也对环境和传感器变化表现出明显敏感。测试时多模态传感器选择的理想化控制器可带来高达+16.7个百分点的性能提升，而基于一致性的实际代理方法仅带来边际改进。

应用价值 / Applications

SenseShift6D为6D位姿估计社区提供了必要的鲁棒性评估基准，推动传感器感知和环境感知的自适应方法研究。该工作对机器人抓取、增强现实、自动驾驶等需要可靠视觉感知的关键应用领域具有重要价值，特别是在光照和传感器条件动态变化的实际部署场景中。
