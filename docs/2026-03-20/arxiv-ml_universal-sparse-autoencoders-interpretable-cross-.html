论文摘要 / Abstract

本文提出了通用稀疏自编码器（USAEs），一种用于发现和跨多个预训练深度神经网络对齐可解释概念的新框架。与现有专注于单一模型的概念可解释性方法不同，USAEs通过训练一个单一的过完备稀疏自编码器，同时学习一个通用概念空间，能够重构和解释多个模型的内部激活。该方法能够捕捉不同任务、架构和数据集之间的共同变化因素，发现从低级特征（如颜色、纹理）到高级结构（如部件、物体）的语义一致且重要的通用概念。

方法概述 / Method

USAEs的核心方法是训练一个单一的过完备稀疏自编码器（SAE），该编码器能够接收来自任意模型的激活输入，并将其解码为考虑中任意其他模型的近似激活。通过优化共享目标函数，学习到的字典能够捕获跨不同模型的共同变化因素——即通用概念，实现跨模型概念对齐。

实验结果 / Results

实验表明，USAEs能够在多种视觉模型中发现语义连贯且重要的通用概念，涵盖从低级视觉特征到高级语义结构的多个层次。该方法还展示了协调激活最大化等新应用，为多模型AI系统的深入分析提供了新途径，验证了其在跨模型可解释性分析中的有效性。

应用价值 / Applications

USAEs为可解释的跨模型分析提供了强大工具，可应用于多模型AI系统的协调激活最大化、模型间概念比较与对齐、以及深入理解不同神经网络架构学习到的共同表示模式。该研究为开发更透明、可解释的多模型机器学习系统奠定了基础，有助于提升AI系统的可审计性和可信度。
