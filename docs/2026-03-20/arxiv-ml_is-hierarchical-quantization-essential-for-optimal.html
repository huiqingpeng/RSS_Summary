Is Hierarchical Quantization Essential for Optimal Reconstruction?

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文质疑了层级量化（hierarchical quantization）在向量量化变分自编码器（VQ-VAE）中对于实现最优重建的必要性。作者指出，尽管VQ-VAE2等层级模型被认为通过分离全局和局部特征而获得更优的重建性能，但高层级信息完全来源于低层级，理论上不应携带额外的重建内容。通过匹配表示预算并采用轻量级干预措施（如数据初始化、周期性重置非活跃码本向量及系统调参）来缓解码本崩溃（codebook collapse），研究表明单层级VQ-VAE能够达到与层级变体相当的重建保真度，挑战了层级量化固有更优的传统假设。

方法概述 / Method

本研究采用对照实验设计，在ImageNet高分辨率图像上对比了双层VQ-VAE与容量匹配的单层级模型。关键方法创新包括：引入数据驱动的码本初始化、周期性重置非活跃码本向量，以及系统性的码本超参数调优策略，以解决单层级模型中常见的码本利用不足和崩溃问题。

实验结果 / Results

实验证实，不充分的码本利用会限制单层级VQ-VAE的性能，而过高维度的嵌入会 destabilize 量化并加剧码本崩溃。当表示预算匹配且码本崩溃得到有效缓解时，单层级VQ-VAE在重建保真度上可与层级变体相媲美。这一发现表明，先前观察到的层级模型优势可能源于未控制的码本利用率和容量差异，而非层级结构本身。

应用价值 / Applications

该研究对神经图像压缩和生成式流水线等领域具有重要启示：简化的单层级架构可降低模型复杂度与计算开销，同时保持相当的重建质量。此外，研究提出的码本优化技术可直接应用于现有VQ-VAE系统，提升码本利用效率并减少训练不稳定性，为资源受限场景下的高效模型部署提供可行方案。
