Adapting Methods for Domain-Specific Japanese Small LMs: Scale, Architecture, and Quantization

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本研究提出了一套系统化的方法论，用于构建领域特定的日语小型语言模型，采用QLoRA微调技术。研究通过三阶段实验回答了三个核心问题：最优训练规模（确定4,000样本为最佳）、基础模型选择（经日语持续预训练的Llama-3模型优于多语言模型）以及架构感知量化策略（Llama-3架构在Q4_K_M量化下表现稳定，而GQA架构严重退化）。最终推荐的Swallow-8B Q4_K_M模型在消费级硬件上实现了高效部署，为低资源技术领域的日语专业模型开发提供了可操作的指导。

方法概述 / Method

研究采用QLoRA（量化低秩适配）微调技术，分三阶段进行系统实验：第一阶段通过1,000-5,000样本的规模学习实验确定最优训练数据量；第二阶段对比四种日语LLM的微调性能；第三阶段评估不同架构在Q4_K_M量化下的表现差异。实验以测试集负对数似然（NLL）为主要评估指标，并结合领域特定评分进行综合判断。

实验结果 / Results

训练规模实验显示，4,000样本时测试集NLL达到最小值1.127，5,000样本后出现过拟合。模型对比表明，经日语持续预训练的Llama-3模型（Swallow-8B、ELYZA-JP-8B）显著优于多语言模型Qwen2.5-7B。量化实验发现，Llama-3架构在Q4_K_M量化下保持稳定，而GQA架构的Qwen2.5性能下降0.280分。最终Swallow-8B Q4_K_M达到2.830/3分，推理速度8.9秒/问题，模型大小仅4.9GB。

应用价值 / Applications

该方法论可推广至低资源技术领域的日语专业模型开发，为法律、医疗、金融等垂直领域的小型语言模型构建提供标准化流程。研究成果特别适用于需要在消费级硬件（如个人电脑、边缘设备）上部署高效日语AI助手的场景，平衡了模型性能与计算资源限制，降低了企业开发定制化日语AI解决方案的门槛。
