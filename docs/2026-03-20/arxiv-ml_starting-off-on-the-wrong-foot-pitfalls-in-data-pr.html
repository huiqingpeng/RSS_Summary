论文摘要 / Abstract

本文揭示了传统数据准备方法（如随机训练-测试划分）在处理高度不平衡的保险损失数据时会导致不可靠和不稳定的结果。为此，作者提出了一个新的数据准备框架，整合了支持点（support points）代表性数据分割和Chatterjee相关系数非参数特征筛选两种统计技术，并将其嵌入到InsurAutoML流程中。实验结果表明，该统计严谨的数据准备方法显著提升了模型稳健性和可解释性，同时大幅降低了计算资源需求。

方法概述 / Method

该研究采用支持点方法进行代表性数据分割，以确保各分区间的分布一致性，避免传统随机划分在不平衡数据中的缺陷；同时利用Chatterjee相关系数进行初始特征筛选，以非参数方式捕捉特征相关性和依赖结构。这两种技术被整合到一个统一框架中，并融入缺失值处理机制，形成完整的InsurAutoML自动化机器学习流程。

实验结果 / Results

研究通过模拟数据集和学术文献中常用的真实数据集对所提方法进行了评估。结果表明，相比传统数据准备流程，该框架在各类保险损失建模任务中显著提高了模型稳健性和可解释性，同时大幅减少了计算资源消耗，验证了统计严谨的数据准备方法对下游建模性能的关键影响。

应用价值 / Applications

该研究为高风险的保险应用场景提供了关键的方法论升级，特别适用于处理高度不平衡的保险损失数据建模。所提出的框架可直接集成到保险精算和风险管理实践中，帮助从业者避免因数据准备不当导致的统计失效问题，提升模型可靠性并降低计算成本。
