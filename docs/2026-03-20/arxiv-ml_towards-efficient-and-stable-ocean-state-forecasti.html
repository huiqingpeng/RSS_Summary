Towards Efficient and Stable Ocean State Forecasting: A Continuous-Time Koopman Approach

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文研究了连续时间Koopman自编码器（CT-KAE）作为双层准地转（QG）系统中长期海洋状态预测的轻量级代理模型。通过将非线性动力学投影到由线性常微分方程控制的潜空间，该模型实现了结构化且可解释的时间演化，并通过矩阵指数公式实现了时间分辨率不变的预测。在2083天的滚动预测中，CT-KAE表现出有界的误差增长和稳定的大尺度统计特性，而自回归Transformer基线则显示出逐渐放大的误差和能量漂移。

方法概述 / Method

CT-KAE采用编码器-解码器架构，将高维非线性海洋动力学映射到低维潜空间，其中演化由线性常微分方程描述，并通过矩阵指数进行时间推进。该方法利用Koopman算子理论将非线性系统转化为线性框架，同时保持连续时间特性，允许任意时间步长的预测而无需重新训练。

实验结果 / Results

在2083天的长期预测中，CT-KAE相比自回归Transformer展现出显著更低的误差累积和能量漂移，保持了稳定的整体能量谱、涡度拟能演化和自相关结构。尽管细尺度湍流结构部分耗散，模型推理速度比传统数值求解器快数个数量级，实现了高效与稳定性的平衡。

应用价值 / Applications

该研究为物理-机器学习气候模型提供了有前景的骨干架构，特别适用于需要长期稳定预测的海洋和大气建模场景。连续时间Koopman代理模型的高效推理能力使其适合实时预测和参数扫描等计算密集型应用，同时其稳定性特征对气候模拟中的能量守恒至关重要。
