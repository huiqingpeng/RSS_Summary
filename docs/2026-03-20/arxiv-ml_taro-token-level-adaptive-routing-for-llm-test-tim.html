TARo: Token-level Adaptive Routing for LLM Test-time Alignment

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了Token-level Adaptive Routing（TARo），一种全新的测试时对齐方法，用于在不进行昂贵后训练的情况下引导冻结的大语言模型进行结构化推理。该方法通过训练基于步骤数学轨迹的奖励模型来捕捉细粒度逻辑一致性信号，并引入可学习的token级路由器自动控制奖励模型对基础模型的指导强度。实验表明，TARo在推理任务上较基础模型提升高达22.4%，较现有token级测试时对齐方法提升8.4%，并能泛化到分布外临床推理和指令遵循任务，同时支持从小模型到大模型的零样本迁移。

方法概述 / Method

TARo的核心架构包含两个组件：首先在逐步数学推理轨迹上训练奖励模型，以捕获细粒度的逻辑一致性信号；其次引入一个可学习的token级路由器，该路由器根据当前生成状态动态决定是否以及如何将奖励模型的反馈注入到基础模型的解码过程中。这种自适应路由机制使得模型能够在推理的不同阶段灵活地利用外部奖励信号，而非对所有token施加均匀的指导。

实验结果 / Results

在数学推理基准测试中，TARo相比基础模型实现最高22.4%的性能提升，相比现有token级测试时对齐方法（如DPO、PPO的测试时变体）提升8.4%。该方法还展现出强大的跨领域泛化能力：在分布外的临床推理数据集MedXpertQA和指令遵循基准AlpacaEval上均取得显著提升。此外，TARo训练的小模型路由器可直接应用于更大规模的基础模型（如从7B迁移到70B）而无需重新训练。

应用价值 / Applications

TARo为部署高性能推理LLM提供了一种轻量级、计算高效的替代方案，避免了昂贵的强化学习后训练，适用于资源受限或需要快速部署的场景。其跨领域泛化能力和跨规模迁移特性使其特别适合医疗诊断辅助、复杂指令遵循等需要可靠推理能力的专业领域应用，同时为测试时对齐从偏好优化向鲁棒推理的范式扩展开辟了新方向。
