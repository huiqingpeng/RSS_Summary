Modeling Inverse Ellipsometry Problem via Flow Matching with a Large-Scale Dataset

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

逆椭偏仪问题（从测量的相位差Δ和振幅比Ψ重建光学常数和薄膜厚度）本质上是一个病态问题。本研究构建了包含超过800万高精度样本、涵盖98种薄膜材料和5种基底的EllipBench大规模基准数据集，系统评估了传统机器学习、深度神经网络和物理信息神经网络等方法，发现现有范式均难以完全解决该任务。为此，作者提出解耦条件流匹配（DCFM）框架，通过将几何膜厚解耦作为物理条件引导连续向量场建模波长相关光学常数的逆概率分布，有效缓解了内在物理歧义性。

方法概述 / Method

论文提出Decoupled Conditional Flow Matching（DCFM）框架，将逆椭偏仪问题从确定性点对点回归重新表述为概率分布建模问题。该方法将几何膜厚显式解耦并作为鲁棒物理条件，结合梯度分离策略和基于物理的约束，引导连续向量场学习光学常数的条件概率分布，而非单一确定值。

实验结果 / Results

基于EllipBench数据集的系统评估表明，传统机器学习方法、深度神经网络和物理信息神经网络均无法充分解决逆椭偏仪的病态特性。
DCFM框架通过显式建模概率分布和解耦物理条件，在重建准确性和鲁棒性方面显著优于现有方法，有效缓解了逆问题中的内在歧义性。

应用价值 / Applications

该研究为薄膜材料表征提供了一种高效、自动化的光学常数与厚度重建方案，可替代传统依赖专家经验的缓慢迭代拟合方法，在半导体制造、光学涂层设计和材料科学等领域具有重要应用价值。
EllipBench数据集和DCFM框架也为相关机器学习研究提供了基准平台和可扩展的方法论基础。
