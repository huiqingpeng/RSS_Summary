Towards Differentiating Between Failures and Domain Shifts in Industrial Data Streams

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对工业数据流中的异常检测问题，提出了一种能够区分系统故障与正常域偏移（domain shift）的新方法。传统异常检测方法常将系统正常演化（如新产品加工启动）误判为故障，而该研究通过结合改进的Page-Hinkley变点检测器、监督式域适应算法和可解释人工智能（XAI）组件，实现了对数据分布变化、异常检测以及故障与域偏移的区分。该方法在钢铁厂真实数据流上进行了验证，展现了在实际工业环境中的实用性和鲁棒性。

方法概述 / Method

该方法采用三组件架构：首先使用改进的Page-Hinkley变点检测器识别数据分布变化；其次结合监督式域适应算法进行快速在线异常检测；最后引入可解释人工智能（XAI）组件辅助人工操作员最终判定变化性质。这种组合实现了从检测到解释的全流程自动化决策支持。

实验结果 / Results

论文在钢铁厂真实数据流上进行了实验验证，证明了该方法能够有效识别数据分布变化并区分故障与正常域偏移。虽然摘要未提供具体量化指标，但实验表明该方法具备在线实时处理能力和实际工业部署的可行性，可显著减少因误判正常工艺变化而导致的无效停机。

应用价值 / Applications

该研究对智能制造和工业4.0具有重要价值，特别适用于钢铁、化工等流程工业的预测性维护系统。通过减少误报率，可避免不必要的生产中断和经济损失，同时XAI组件增强了人机协作的可信度，使操作员能够理解并信任自动化决策，提升整体运维效率。
