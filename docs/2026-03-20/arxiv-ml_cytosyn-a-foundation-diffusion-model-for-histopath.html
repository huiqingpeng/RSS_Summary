CytoSyn: a Foundation Diffusion Model for Histopathology -- Tech Report

论文摘要 / Abstract

本文提出了 CytoSyn，一种专为计算病理学设计的基础潜在扩散模型，能够生成高度真实且多样化的组织病理学 H&E 染色图像。该模型基于超过 10,000 张 TCGA 全切片图像（涵盖 32 种癌症类型）进行训练，并通过方法改进、训练数据扩展和采样策略优化推出了 CytoSyn-v2 版本。尽管仅使用肿瘤切片训练，CytoSyn 在生成炎症性肠病图像时仍保持最先进的性能，且研究揭示了扩散模型和评估指标对预处理细节（如 JPEG 压缩）的高度敏感性。

方法概述 / Method

CytoSyn 采用潜在扩散模型架构，在压缩的潜在空间中进行去噪扩散过程以生成高分辨率组织病理学图像。研究团队系统探索了训练数据规模扩展、采样策略优化以及切片级过拟合问题，并针对预处理流程中的 JPEG 压缩等细节进行了深度分析，最终开发出改进版本 CytoSyn-v2。

实验结果 / Results

CytoSyn 在广泛的基准测试中展现出最先进的图像生成质量，与同期最先进的 PixCell 模型进行了深度对比。实验发现扩散模型和性能指标对预处理特定细节（如 JPEG 压缩）具有强烈敏感性。值得注意的是，模型仅在肿瘤数据上训练，却能泛化到炎症性肠病图像生成任务，显示出良好的跨域迁移能力。

应用价值 / Applications

CytoSyn 可应用于虚拟染色、数据增强、罕见病例模拟等超出传统特征提取器能力的生成任务，为计算病理学研究和临床工具开发提供支持。该模型及其权重、训练验证数据集和合成图像样本均已公开发布，有助于推动病理学 AI 社区的进一步研究与应用。
