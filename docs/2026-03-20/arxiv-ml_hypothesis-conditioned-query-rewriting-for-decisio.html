Hypothesis-Conditioned Query Rewriting for Decision-Useful Retrieval

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出假设条件查询重写（HCQR），一种无需训练的检索前框架，将RAG从主题导向检索重新定向为证据导向检索。
HCQR通过从输入问题和候选选项中推导轻量级工作假设，并重写为三个针对性查询来寻求支持假设、区分竞争替代方案和验证问题关键线索的证据。在MedQA和MMLU-Med数据集上的实验表明，HCQR相比简单RAG平均准确率分别提升5.9和3.6个百分点，显著优于单查询RAG及重排序/过滤基线方法。

方法概述 / Method

HCQR首先基于输入问题和候选答案选项生成一个工作假设，然后将原始查询重写为三个互补的检索查询：支持性查询（寻找证实假设的证据）、区分性查询（寻找区分假设与替代方案的证据）和验证性查询（验证问题中的显著线索）。该方法无需额外训练，可直接集成到现有RAG流程中。

实验结果 / Results

在医学问答基准MedQA和MMLU-Med上的评估显示，HCQR相比Simple RAG分别获得5.9和3.6个百分点的平均准确率提升，且持续优于单查询RAG以及重排序和过滤等基线方法。结果表明，证据导向的多查询检索策略能有效捕获决策相关的判别性信息。

应用价值 / Applications

HCQR适用于需要从多个竞争选项中进行选择的决策型检索场景，如医学诊断辅助、法律案例分析和教育测评系统等。该方法通过提升检索证据的决策相关性，帮助大语言模型做出更准确、可解释的选择，特别适用于高风险专业领域的人机协作决策支持。
