Complementary Text-Guided Attention for Zero-Shot Adversarial Robustness

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对预训练视觉-语言模型（如 CLIP）易受对抗样本攻击的问题，提出了一种基于文本引导注意力的零样本对抗鲁棒性框架。研究发现对抗扰动会导致文本引导注意力的偏移，据此设计了 TGA-ZSR 方法，包含局部注意力细化和全局注意力约束两个模块。为进一步解决模型偶尔关注无关特征的问题，作者提出了互补文本引导注意力方法 Comp-TGA，通过类别提示和反类别提示的双重注意力机制捕获更全面的前景表征，在 16 个数据集上分别实现 9.58% 和 11.95% 的零样本鲁棒准确率提升。

方法概述 / Method

TGA-ZSR 框架包含两个核心模块：局部注意力细化模块（LARM）和全局注意力约束模块（GACM）。
GACM 利用干净样本从目标模型和原始模型获取文本引导注意力，以兼顾干净样本性能与整体鲁棒性。
Comp-TGA 在此基础上引入互补注意力机制，同时采用类别提示引导的前景注意力和非类别提示驱动的反向注意力，通过两者融合获得更准确完整的前景表征。

实验结果 / Results

在 16 个数据集上的实验表明，TGA-ZSR 相比当前最优方法提升 9.58% 的零样本鲁棒准确率，而 Comp-TGA 进一步提升至 11.95%。该方法在保持 CLIP 模型泛化能力的同时显著增强了对抗鲁棒性，有效缓解了注意力偏移导致的性能下降问题。

应用价值 / Applications

该研究可直接应用于需要零样本识别能力的实际场景，如开放集识别、图像分类的对抗防御等，为部署在真实环境中的视觉-语言模型提供安全保障。方法无需针对特定类别重新训练，适用于快速适应新领域且对安全性要求较高的应用，如内容审核、自动驾驶视觉感知等。
