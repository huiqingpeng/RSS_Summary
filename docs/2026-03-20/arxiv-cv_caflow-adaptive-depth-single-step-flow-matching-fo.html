CAFlow: Adaptive-Depth Single-Step Flow Matching for Efficient Histopathology Super-Resolution

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出CAFlow，一种自适应深度的单步流匹配框架，用于高效组织病理学超分辨率重建。该方法通过将图像块路由至最浅的满足重建质量要求的网络出口，实现计算资源的动态分配。核心创新包括：在像素重排空间进行流匹配以降低16倍空间计算量、将50%训练时间专门用于精确的t=0样本生成，以及设计轻量级出口分类器（仅6K参数）实现33%计算节省。实验表明，该方法在多器官病理图像4倍超分辨率任务上达到31.72 dB PSNR，同时单GPU训练时间低于5小时，可将全切片图像推理从分钟级缩短至秒级。

方法概述 / Method

CAFlow采用流匹配生成模型，在像素反洗牌（pixel-unshuffled）重排空间进行训练与推理，将高分辨率空间计算转移至通道维度。网络主干FlowResNet（190万参数）融合卷积与窗口自注意力模块，设置四个早期出口（3.1-13.3 GFLOPs）。训练策略上，将50%迭代专门用于噪声水平t=0的精确样本生成，以保障单步推理质量。轻量级出口分类器基于低层特征动态选择推理深度。

实验结果 / Results

自适应路由策略在4倍超分辨率任务上取得31.72 dB PSNR，相比完整深度仅损失0.12 dB，但计算量显著降低；最浅出口即可超越双三次插值1.9 dB，且计算量仅为SwinIR-light的36%。8倍超分辨率任务中，该方法超越所有同计算量基线，并与参数量大得多的SwinIR-Medium相当。跨器官泛化测试显示，在未见过的结肠组织上仅损失0.02 dB。下游细胞核分割实验验证了临床相关结构的保持能力。

应用价值 / Applications

该研究针对数字病理学中千兆像素全切片图像的超分辨率需求，解决了生成式方法计算开销过大、难以临床部署的瓶颈。自适应深度推理机制使计算资源根据图像复杂度动态分配，在保证诊断相关结构完整性的前提下，将推理速度提升数个数量级。方法可广泛应用于病理图像增强、远程病理诊断、以及基于低分辨率扫描的高效存储与传输系统，为计算病理学的规模化应用提供可行方案。
