论文摘要 / Abstract

本文研究了在差分隐私约束下的多矩阵博弈（polymatrix games）均衡求解问题。作者首先证明了在两种严格设定下（基于欧氏距离的均衡逼近保证，以及敌手可访问所有通信信道）不存在能同时实现高精度均衡和趋零隐私预算的算法。随后，在更现实的设定（敌手仅能访问有限数量的信道）下，提出了一种分布式算法，能够随着玩家数量增加同时实现趋零的Nash间隙（即可利用性）和趋零的隐私预算，这是均衡计算领域的首次突破。

方法概述 / Method

论文采用理论分析与算法设计相结合的方法：首先通过硬度结果（hardness results）建立不可能性定理，刻画差分隐私在博弈中的基本理论极限；然后利用多矩阵游戏的结构特性，设计了一种新型分布式算法，在受限敌手模型下实现隐私与效用保证的协同优化。

实验结果 / Results

作者提供了数值实验结果以验证所提算法的有效性，表明该算法能够在玩家规模增大时同时保证均衡质量（Nash gap）和隐私预算均趋于零，突破了先前工作无法兼顾高精度与低隐私预算的局限。

应用价值 / Applications

该研究对隐私保护的多方博弈决策具有重要价值，可应用于分布式机器学习中的联邦学习（各参与方策略交互）、隐私保护的市场机制设计、以及涉及敏感信息的竞争性资源分配等场景，为大规模隐私保护博弈系统提供了可扩展的理论基础与算法工具。
