SLEA-RL: Step-Level Experience Augmented Reinforcement Learning for Multi-Turn Agentic Training

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了SLEA-RL（Step-Level Experience-Augmented Reinforcement Learning），一种用于多轮智能体训练的新型强化学习框架。该框架针对现有经验增强方法仅在初始任务描述时进行一次静态检索、无法适应多轮环境中每步观测变化的缺陷，实现了基于当前观测的每一步决策时的动态经验检索。
SLEA-RL通过步级观测聚类、自进化经验库和步级信用分配策略优化三个核心组件，使经验库与策略同步进化。在长程多轮智能体基准测试上，该方法显著优于多种强化学习基线。

方法概述 / Method

SLEA-RL包含三个关键组件：（i）步级观测聚类，将结构等价的环境状态分组以实现高效的簇索引检索；（ii）自进化经验库，通过基于得分的准入机制和速率限制的提取策略来蒸馏成功经验和失败模式；（iii）策略优化模块，采用步级信用分配进行细粒度的多轮回合优势估计。经验库通过语义分析而非梯度更新与策略协同进化。

实验结果 / Results

论文在长程多轮智能体基准测试上验证了SLEA-RL的有效性，结果表明该方法在性能上显著优于多种强化学习基线。具体量化指标未在摘要中详细披露，但实验证明了步级动态检索机制相比静态检索的优势。

应用价值 / Applications

该研究适用于需要多轮交互的LLM智能体场景，如复杂工具使用任务、长程决策规划等。
SLEA-RL通过动态利用历史经验提升了样本效率和决策质量，为构建能够持续学习、自我改进的自主智能体系统提供了可行方案，在客户服务、科研辅助、自动化工作流等领域具有潜在应用价值。
