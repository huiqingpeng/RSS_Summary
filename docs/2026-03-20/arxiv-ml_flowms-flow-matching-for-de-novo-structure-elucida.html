FlowMS: Flow Matching for De Novo Structure Elucidation from Mass Spectra

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了FlowMS，首个基于离散流匹配的质谱条件去 novo分子生成框架，用于解决质谱数据中的分子结构解析难题。
FlowMS通过在概率空间中进行迭代精化生成分子图，同时强制执行化学式约束并基于预训练化学式Transformer编码器的谱图嵌入进行条件生成。该方法在NPLIB1基准测试的6项指标中有5项达到最优性能，包括9.15%的top-1准确率（较DiffMS提升9.7%）和7.96的top-10 MCES（较MS-BART提升4.2%），证明了离散流匹配在质谱结构解析中的有效性。

方法概述 / Method

FlowMS采用离散流匹配（discrete flow matching）作为核心生成范式，通过在概率空间中迭代精化分子图结构，而非传统的自回归或扩散方法。模型引入预训练的化学式Transformer编码器提取质谱嵌入特征，并在生成过程中显式强制执行化学式约束以确保生成分子的化学合理性。

实验结果 / Results

在NPLIB1基准数据集上，FlowMS在6项评估指标中的5项达到最优：top-1准确率为9.15%，相比DiffMS实现9.7%的相对提升；top-10 MCES为7.96，较MS-BART提升4.2%。可视化分析表明，FlowMS生成的分子结构与真实结构具有高度相似性，结构合理性显著优于现有方法。

应用价值 / Applications

FlowMS可广泛应用于代谢组学中的未知化合物鉴定、天然产物发现以及药物研发中的先导化合物结构解析。其高效的离散流匹配框架克服了传统扩散模型的计算瓶颈，为大规模质谱数据库的自动化结构注释提供了可行方案，有望加速复杂生物样本中未知代谢物的识别与表征。
