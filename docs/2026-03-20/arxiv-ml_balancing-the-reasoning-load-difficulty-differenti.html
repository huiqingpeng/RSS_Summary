Balancing the Reasoning Load: Difficulty-Differentiated Policy Optimization with Length Redistribution for Efficient and Robust Reinforcement Learning

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文针对大型推理模型（LRMs）存在的"过度思考"（生成冗长冗余答案）和"过度自信"（对难题生成过短错误答案）两大问题，提出了难度差异化策略优化算法（DDPO）。该算法基于任务难度进行分离优化：对简单任务缩短输出长度同时保持准确率，对复杂任务扩展探索空间以提升性能。理论分析揭示了最大化期望准确率需要长度分布接近最优长度且尽可能集中，据此提出以难度级别平均长度作为优化参考。实验表明DDPO相比GRPO平均答案长度减少12%的同时准确率提升1.85%，实现了准确率与长度的更优平衡。

方法概述 / Method

DDPO采用难度感知的强化学习策略，根据模型对问题的响应特征区分简单与复杂任务，并分别应用不同的优化目标：简单任务通过长度惩罚机制抑制冗余生成，复杂任务则放宽长度约束以允许充分推理。该方法从理论上推导了期望准确率最大化的条件，建立了以难度级别平均长度为核心的长度优化框架。

实验结果 / Results

在领域内和领域外多个基准测试上的广泛实验验证了DDPO的优越性和有效性。与GRPO基线相比，DDPO在多个基准上平均答案长度降低12%，准确率提升1.85%，显著改善了推理效率与质量的权衡关系。

应用价值 / Applications

DDPO可广泛应用于需要高效推理的大规模语言模型部署场景，如实时对话系统、资源受限的边缘计算环境以及成本敏感的API服务。通过减少不必要的推理长度同时保证复杂问题的求解能力，该研究为构建更经济、更可靠的推理模型提供了实用解决方案，对推动大型推理模型的实际产业化具有重要意义。
