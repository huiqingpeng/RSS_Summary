Embedding Physical Reasoning into Diffusion-Based Shadow Generation

arXiv Computer Vision 2026-03-20

论文摘要 / Abstract

本文提出了一种将物理推理嵌入扩散模型的阴影生成方法，解决了现有纯图像空间方法因缺乏显式物理约束而导致的阴影错位或不合理问题。该方法通过恢复场景几何结构和估计主光源方向来生成基于物理的粗阴影估计，并引入光照和阴影线索的置信度分数来调节生成过程。在DESOBAV2数据集上的实验表明，该方法在阴影真实感和定位精度上均优于现有最优方法，阴影区域RMSE降低23%，BER降低30%。

方法概述 / Method

该方法首先利用合成图像和物体掩码恢复近似场景几何，并估计主导光源方向，通过几何推理生成基于物理的粗阴影估计作为空间锚点；随后预测光照和阴影线索的置信度分数，将阴影掩码、光源方向及其置信度作为条件输入，引导扩散模型将粗估计细化为真实阴影。

实验结果 / Results

在DESOBAV2基准数据集上的定量评估显示，相比现有最优方法，本文方法在阴影区域RMSE指标上降低23%，在阴影区域BER指标上降低30%，显著提升了阴影的真实感和空间定位准确性。

应用价值 / Applications

该技术可应用于增强现实、虚拟现实和图像编辑等领域，实现插入物体时自动生成符合物理规律的真实阴影，提升合成图像的视觉一致性和沉浸感，对影视后期制作、游戏开发和电商产品展示等场景具有重要实用价值。
