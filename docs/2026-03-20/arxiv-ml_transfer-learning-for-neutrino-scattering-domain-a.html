Transfer Learning for Neutrino Scattering: Domain Adaptation with GANs

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本研究将迁移学习（Transfer Learning, TL）应用于中微子散射领域，利用生成对抗网络（GAN）从合成的中微子-碳非弹性散射数据中提取物理信息，并将其外推至中微子-氩和反中微子-碳等相关过程。研究发现迁移学习不仅能复现轻子运动学的关键特征（包括准弹性和Δ共振峰），而且在有限统计量（10,000至100,000个事件）下仍保持高精度，显著优于从头训练的生成模型。该工作为构建下一代中微子散射事件生成器提供了高效框架，尤其在实验数据稀缺的场景下具有重要价值。

方法概述 / Method

论文采用基于生成对抗网络的迁移学习方法，首先在源域（中微子-碳散射）上训练GAN，然后通过域适应技术将学习到的轻子-原子核动力学知识迁移到目标域（中微子-氩或反中微子-碳相互作用）。研究还评估了当训练数据来自不同中微子-原子核相互作用模型时迁移学习的有效性，以验证模型的跨模型泛化能力。

实验结果 / Results

实验结果表明，迁移学习成功复现了轻子运动学的关键特征，包括准弹性峰和Δ共振峰；在使用10,000和100,000个事件的数据集时，迁移学习在有限统计量下仍保持高准确度，性能显著优于从头训练的生成模型；此外，即使训练数据来自不同的中微子-原子核相互作用模型，迁移学习依然有效，证明了其跨模型迁移的鲁棒性。

应用价值 / Applications

该研究为中微子物理实验提供了高效的事件生成工具，可显著降低对大规模模拟数据的依赖，尤其适用于实验数据稀缺或计算资源受限的场景；迁移学习框架可加速下一代中微子散射事件生成器的开发，对深地下中微子实验（DUNE）等大型实验的物理分析和探测器设计具有重要指导意义；该方法还可推广至其他核物理和粒子物理领域，促进跨过程、跨靶材的物理知识共享与模型构建。
