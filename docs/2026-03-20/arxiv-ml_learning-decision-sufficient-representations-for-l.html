论文摘要 / Abstract

本文研究了如何为具有未知成本向量的线性规划问题构建压缩数据集，使其足以恢复最优决策。作者证明了计算最小充分决策数据集（SDD）的内在维度d*是NP-hard的，并提出了"逐点充分性"的松弛概念，设计了多项式时间切割平面算法。在数据驱动场景下，作者提出了累积算法，实现了与d*而非环境维度d相关的分布无关PAC保证，并将结果应用于上下文线性优化，获得了更优的泛化界。

方法概述 / Method

作者首先建立了计算决策相关维度d*的NP-hard性和判断全局充分性的coNP-hard性理论结果。为克服计算困难，引入了逐点充分性概念——仅要求对单个成本向量充分，并在非退化条件下设计了多项式时间的切割平面算法。进一步提出了累积算法，通过聚合跨样本的决策相关方向，构建大小不超过d*的稳定压缩方案。

实验结果 / Results

理论结果表明：逐点充分性失败概率满足分布无关的PAC保证，以高概率不超过Õ(d*/n)，且该速率在对数因子意义下是紧的。在上下文线性优化应用中，压缩预测器的泛化界为Õ(√(d*/n))，显著优于传统的Õ(√(d/n))，其中d为环境成本维度。

应用价值 / Applications

该研究为大规模线性优化问题提供了数据压缩和降维的理论基础，可应用于需要高效存储和快速决策的运筹优化场景，如供应链优化、资源分配和实时定价系统。上下文线性优化的改进泛化界使得在高维环境下构建更轻量、更具可解释性的预测模型成为可能，对机器学习与优化交叉领域具有重要价值。
