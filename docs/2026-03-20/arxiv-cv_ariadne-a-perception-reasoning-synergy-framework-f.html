ARIADNE: A Perception-Reasoning Synergy Framework for Trustworthy Coronary Angiography Analysis

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了ARIADNE框架，用于解决冠状动脉造影分析中传统像素级损失函数无法保证拓扑约束的问题。该框架采用两阶段设计：感知模块通过直接偏好优化（DPO）微调视觉-语言基础模型Sa2VA，以Betti数作为偏好信号实现拓扑对齐；推理模块将狭窄检测建模为马尔可夫决策过程，并引入显式拒绝机制处理模糊解剖结构。在1,400例临床造影数据及多中心外部验证集上，该方法达到最先进的中心线Dice系数0.838，假阳性降低41%，首次展示了偏好学习在医学影像拓扑对齐中的应用价值。

方法概述 / Method

感知模块以Sa2VA视觉-语言模型为骨干，利用Betti数（刻画拓扑结构的代数不变量）构建偏好对，通过DPO训练使模型输出几何完整的血管树结构而非仅优化像素重叠。推理模块将狭窄定位形式化为强化学习问题，设计具有拒绝动作的MDP，使智能体能够主动跳过分叉、血管交叉等易混淆区域，实现从覆盖率最大化到可靠性优化的策略转变。

实验结果 / Results

在1,400例临床冠状动脉造影数据集上，ARIADNE取得中心线Dice 0.838的最优性能，相比几何基线方法假阳性降低41%。外部验证在ARCADE和XCAD两个多中心基准测试上验证了跨采集协议的泛化能力，证明该方法在保持诊断敏感性的同时有效缓解拓扑违规问题。

应用价值 / Applications

该研究为介入心脏病学工作流程提供了可信赖的自动化造影分析工具，其拓扑感知分割能力可减少血管断裂导致的误判，而带拒绝机制的推理模块增强了临床部署时的可靠性。偏好学习框架可推广至其他需要结构完整性的医学影像任务（如视网膜血管、神经纤维束追踪），为安全关键型医疗AI系统的设计提供了新范式。
