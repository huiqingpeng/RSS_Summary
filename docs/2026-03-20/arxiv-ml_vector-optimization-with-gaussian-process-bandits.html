Vector Optimization with Gaussian Process Bandits

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文研究了基于高斯过程 bandits 的黑箱向量优化问题，其中目标向量之间存在由多面体凸锥描述的不完全序关系。作者提出了 Vector Optimization with Gaussian Process (VOGP) 算法，一种自适应消除算法，通过利用目标函数的平滑性实现样本高效地识别 Pareto 最优解。理论分析建立了基于信息增益和核函数特定的样本复杂度界限，实验表明 VOGP 平均样本复杂度降低约18倍，为结合偏好结构的样本高效多目标黑箱优化开辟了新方向。

方法概述 / Method

VOGP 采用自适应消除策略，利用高斯过程建模目标函数的不确定性，并通过多面体凸锥定义的偏好结构指导采样决策。算法通过信息增益准则主动选择最具信息量的评估点，逐步消除非 Pareto 最优解，同时针对连续设计空间和未知核超参数的情况提供了启发式扩展。

实验结果 / Results

VOGP 在多个真实世界和合成数据集上与最先进的多目标和向量优化算法进行了全面比较，平均样本复杂度降低约18倍。实验验证了理论保证的有效性，并展示了启发式扩展在连续空间和超参数未知场景下的实用性能。

应用价值 / Applications

该研究适用于存在复杂偏好结构的多目标决策场景，如工程设计优化、药物发现、资源分配等黑箱优化问题。
VOGP 在保持理论保证的同时实现了实用效率，为需要在昂贵评估约束下平衡多个冲突目标的实际应用提供了高效的优化工具。
