LiteReality: Graphics-Ready 3D Scene Reconstruction from RGB-D Scans

arXiv Computer Vision 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文提出了LiteReality，一种将室内环境的RGB-D扫描转换为紧凑、逼真且可交互的3D虚拟副本的新颖流程。该系统不仅能重建视觉上逼真的场景，还支持图形管线所需的关键特性，包括物体独立性、关节结构、高质量基于物理的渲染材质以及基于物理的交互。
LiteReality在Scan2CAD基准上实现了无需训练的最先进物体检索性能，并能在严重错位、遮挡和光照不足的情况下将图像外观迁移到3D资产。

方法概述 / Method

LiteReality的核心流程包括：首先通过场景理解将扫描结果解析为连贯的3D布局和结构化场景图中的物体；然后从精选资产数据库中检索视觉上最相似的3D艺术家制作模型进行场景重建；接着通过材质绘制模块恢复高质量的空间变化材质；最后将重建场景集成到支持基本物理属性的仿真引擎中。

实验结果 / Results

LiteReality引入的训练无关物体检索模块在Scan2CAD基准上达到了最先进的相似性性能；其材质绘制模块能够在严重错位、遮挡和不良光照条件下，将任意风格图像的外观迁移到3D资产。该方法在真实扫描和公开数据集上均验证了有效性，生成的场景紧凑、可编辑且与标准图形管线完全兼容。

应用价值 / Applications

该研究适用于AR/VR、游戏、机器人和数字孪生等应用领域。重建的场景具有紧凑性、可编辑性，并与标准图形管线完全兼容，能够为这些领域提供高质量的交互式3D虚拟环境，降低专业内容创作的技术门槛和时间成本。
