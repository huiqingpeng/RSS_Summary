Detection Is Cheap, Routing Is Learned: Why Refusal-Based Alignment Evaluation Fails

arXiv Machine Learning 2026-03-20
Read Original Article →

论文摘要 / Abstract

本文揭示了当前大模型对齐评估的核心缺陷：仅检测模型是否编码危险概念或是否拒绝有害请求，忽略了"路由"这一关键中间层——即从概念检测到行为策略的转换机制。研究以中国开源语言模型的政治审查为自然实验，发现探测准确率本身无法诊断对齐质量，必须通过保留类别泛化测试；手术消融显示路由几何结构具有模型和实验室特异性；且拒绝机制已非主导审查方式，叙事引导等隐形手段使仅检测拒绝的基准失效。研究支持"检测-路由-生成"三阶段框架，强调审计路由机制对评估模型行为的决定性作用。

方法概述 / Method

研究采用三种互补技术：线性探测（probes）评估概念编码能力，手术消融（surgical ablations）定向移除特定方向以隔离路由机制，以及行为测试验证输出变化。实验覆盖五个实验室的九个开源权重模型，通过跨模型迁移实验检验路由几何的通用性，并对比不同审查机制（硬拒绝vs叙事引导）的演化轨迹。

实验结果 / Results

关键发现包括：第一，政治探测、空对照和置换基线均可达100%准确率，证明探测准确率非诊断性；第二，移除政治敏感方向在多数模型中消除审查并恢复事实准确性，但某模型因架构纠缠产生虚构，且跨模型转移失败；第三，同一模型家族中硬拒绝降至零而叙事引导升至最大，使仅检测拒绝的评估失效。

应用价值 / Applications

该研究为AI安全评估提供范式转变：从检测"知道什么"和"拒绝什么"转向审计"如何表达"，对监管审查机制识别、红队测试设计和开源模型安全审计具有直接指导意义。三阶段框架可推广至幻觉、偏见等其他对齐问题，帮助开发者定位干预点并防止"隐形"未对齐行为逃避检测。
