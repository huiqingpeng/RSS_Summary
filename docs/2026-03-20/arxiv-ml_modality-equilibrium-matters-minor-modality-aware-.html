论文摘要 / Abstract

本文针对多模态融合中的模态不平衡问题（主导模态压制弱模态导致次优融合），提出了一种基于Shapley值的自适应交替训练框架，通过动态优先优化次模态来实现模态均衡。研究引入了跨模态映射的记忆模块以细化和继承模态特定表征，并构建了新的均衡偏离度量（EDM）评估平衡性与准确性。实验表明该方法在四个多模态基准数据集上达到SOTA性能，且在模态缺失场景下展现出强泛化能力。

方法概述 / Method

该框架核心为Shapley值引导的自适应调度机制，动态确定各模态的训练优先级，确保欠优化模态获得充分学习；同时设计记忆模块配合跨模态映射机制，在特征层和样本层面对齐多模态表征。编码器模块兼容传统网络与LLM骨干网络，增强方法适应性。

实验结果 / Results

研究在四个多模态基准数据集上进行评估，所提方法取得SOTA性能；提出的均衡偏离度量（EDM）有效量化了模态平衡程度与融合准确性的关系。鲁棒性分析显示，在模态缺失条件下方法仍保持强泛化能力，验证了交替训练策略的战略性模态优先机制对多模态学习的促进作用。

应用价值 / Applications

该方法为不完整模态条件下的多模态学习提供了新范式，可广泛应用于视觉-语言理解、医疗多模态诊断、智能交互等存在模态不平衡或数据缺失风险的场景。其兼容LLM骨干的设计使其易于集成到现代大模型系统中，为提升多模态大模型的训练动态优化和鲁棒性提供了实用方案。
