BUSSARD: Normalizing Flows for Bijective Universal Scene-Specific Anomalous Relationship Detection

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了BUSSARD（双射通用场景特定异常关系检测），一种基于归一化流的模型，用于检测场景图中的异常关系。该方法采用多模态方法，利用语言模型嵌入场景图中的对象和关系标记，以利用现实世界的语义知识。通过归一化流学习从对象-关系-对象三元组到简单基础分布的双射变换，实现基于似然估计的异常检测。实验表明，该方法在SARD数据集上比当前最先进模型的AUROC结果提高约10%，同时速度提升5倍，并在同义词使用方面展现出更强的鲁棒性和通用性。

方法概述 / Method

BUSSARD采用归一化流（Normalizing Flow）作为核心架构，学习对象-关系-对象三元组与简单基础分布（通常为高斯分布）之间的可逆双射变换。该方法结合多模态学习，使用语言模型对场景图中的对象和关系标记进行嵌入，从而整合外部语义知识来增强异常检测能力。

实验结果 / Results

在包含办公室和餐厅场景的SARD数据集上，BUSSARD相比当前最先进模型实现了约10%的AUROC性能提升，同时推理速度提高5倍。消融研究表明，该方法具有卓越的鲁棒性和通用性：在使用同义词的情况下，BUSSARD保持稳定性能，而基线模型出现17.5%的性能偏差。

应用价值 / Applications

该研究可应用于智能监控、机器人场景理解和自动驾驶等领域中的异常行为检测，特别是在需要识别场景中对象间异常空间或语义关系的场景。其高效性和对语义变体（如同义词）的鲁棒性使其适用于实时系统和跨场景部署的实际应用。
