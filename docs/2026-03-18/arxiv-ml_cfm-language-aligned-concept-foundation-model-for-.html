CFM: Language-aligned Concept Foundation Model for Vision

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了CFM（Concept Foundation Model），一种语言对齐的视觉概念基础模型，旨在解决现有视觉基础模型表示不透明、难以解释的问题。
CFM能够提供细粒度、人类可解释且空间定位的概念表示，并与强大的语义表示基础模型结合，为任意下游任务生成解释。通过分析概念的局部共现依赖关系，CFM改进了概念命名并获得了更丰富的解释，在分类、分割和描述生成等任务上取得了与黑盒基础模型相当的性能。

方法概述 / Method

CFM采用语言对齐的方式学习视觉概念，将视觉表示分解为细粒度、空间定位的人类可解释概念。该方法通过分析概念之间的局部共现依赖关系来定义概念关系，从而改进概念命名质量并增强解释能力。
CFM可与现有的语义表示强大的基础模型配对使用，为其提供概念层面的可解释性。

实验结果 / Results

实验表明，CFM在分类、分割和图像描述生成等基准任务上的性能与黑盒基础模型具有竞争力，同时提供了高质量的概念级解释。
CFM生成的概念具有细粒度和空间定位特性，能够有效解释模型在多种视觉任务中的决策过程。

应用价值 / Applications

CFM可应用于需要模型可解释性的高风险视觉决策场景，如医疗影像诊断、自动驾驶感知系统等。该模型为视觉基础模型提供了透明、可审计的决策依据，有助于建立人类对AI系统的信任，并支持模型调试和偏见检测等实际应用需求。
