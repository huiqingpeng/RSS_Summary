IOSVLM: A 3D Vision-Language Model for Unified Dental Diagnosis from Intraoral Scans

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了IOSVLM，首个端到端的3D视觉-语言模型，用于口腔内扫描（IOS）的统一牙科诊断和生成式视觉问答。研究构建了大规模多源IOS诊断数据集IOSVQA，包含19,002个病例和249,055个VQA问答对，覆盖23种口腔疾病。实验表明，直接3D几何建模相比2D或多视图方法，在宏准确率和宏F1分数上分别提升至少9.58%和1.46%，证明了原生3D几何信息对牙科诊断的关键价值。

方法概述 / Method

IOSVLM采用点云表示口腔扫描数据，遵循3D编码器-投影器-大语言模型的架构设计。为弥合无色IOS数据与依赖颜色的3D预训练之间的分布差距，研究提出了几何到色彩代理机制（geometry-to-chromatic proxy）以稳定细粒度几何感知和跨模态对齐，并采用两阶段课程训练策略增强模型鲁棒性。

实验结果 / Results

IOSVLM在统一牙科诊断任务上持续超越强基线模型，宏准确率提升至少9.58%，宏F1分数提升1.46%。该结果验证了直接建模原生3D几何相比基于2D图像或多视图渲染方法的显著优势，尤其在处理多病共存、类别不平衡和细粒度形态歧义等挑战时表现突出。

应用价值 / Applications

该研究可广泛应用于临床牙科诊疗中的自动化病历文档生成、医患沟通辅助以及远程牙科咨询。统一的多疾病诊断能力和视觉问答交互模式，能够提升牙科诊疗效率，降低对专业影像医师的依赖，并支持个性化治疗方案制定与口腔健康教育的智能化发展。
