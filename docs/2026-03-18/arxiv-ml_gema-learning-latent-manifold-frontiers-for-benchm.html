GeMA: Learning Latent Manifold Frontiers for Benchmarking Complex Systems

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出几何流形分析（GeMA），一种用于复杂系统（如铁路网络、可再生能源资产和国民经济）绩效基准测试的潜流形前沿框架。该方法通过生产力流形变分自编码器（ProMan-VAE）将生产集表示为嵌入在联合输入-输出空间中的低维流形边界，而非在观测空间中指定前沿函数。
GeMA能够学习捕捉技术结构和运营效率的潜变量，实现基于流形的效率评估、内生同伴群体聚类、尺度不变基准测试，并通过解码器雅可比矩阵和Lipschitz边界量化效率评分的几何鲁棒性。

方法概述 / Method

GeMA采用分头编码器架构，将输入-输出数据映射到潜空间，其中潜变量分别编码技术结构（z_tech）和运营效率（z_ineff）。通过变分推断学习低维流形表示，利用商构造（quotient construction）分离尺度效应，实现尺度不变的效率比较。效率评分通过计算观测点到学习流形边界的距离获得，并通过解码器雅可比矩阵的局部认证半径评估几何稳定性。

实验结果 / Results

在具有非凸前沿、异质技术和尺度偏差的合成数据上，GeMA展现出对复杂结构的准确恢复能力。四个真实案例研究包括：全球城市轨道交通系统（COMET）、英国铁路运营商（ORR）、国民经济（Penn World Table）和高频风电场数据集。结果表明，当经典假设成立时，GeMA表现与DEA/SFA等传统方法相当；在存在显著异质性、非凸性或规模相关偏差的情境下，GeMA能提供额外洞察。

应用价值 / Applications

GeMA适用于交通规划中的铁路网络绩效评估、能源监管中的可再生能源资产效率分析，以及宏观经济研究中的国家经济生产率比较。该方法突破了传统DEA/SFA对生产集凸性和同质性的限制，为具有复杂技术异质性和规模效应的现实系统提供了更灵活的基准测试工具，支持更精准的政策制定和资源分配决策。
