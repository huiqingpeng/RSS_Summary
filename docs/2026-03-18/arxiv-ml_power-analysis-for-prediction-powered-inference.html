Power Analysis for Prediction-Powered Inference

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对预测驱动推断（Prediction-Powered Inference, PPI）框架下的统计功效分析问题，推导了计算所需标注样本量的闭式公式。研究通过刻画PPI估计量的渐近方差并应用Wald检验反演，解决了"给定高预测能力的AI/ML模型，需要多少标注样本才能达到目标统计功效"这一实际问题。研究发现，相对于经典设计，所需标注样本量的减少程度大致与预测值和真实值之间的R²成正比，并在单细胞转录组学、临床血压测量和皮肤镜成像三个生物医学应用中进行了验证。

方法概述 / Method

本研究基于PPI估计量的渐近正态性，推导了其渐近方差的显式表达式，并通过Wald检验反演得到样本量计算公式。该方法涵盖了两样本比较、2×2表风险测量等常用统计场景，并提供了R包和在线计算器实现。

实验结果 / Results

蒙特卡洛模拟验证了所推导的解析公式的准确性，表明样本量需求随预测模型R²的提高而近似线性下降。在三个真实生物医学应用中，研究展示了该方法如何帮助研究者量化高预测性能AI模型带来的标注成本节约。

应用价值 / Applications

该研究为生物医学研究中AI/ML辅助的数据收集提供了样本量规划工具，特别适用于标注成本高昂的场景（如单细胞测序、医学影像分析）。研究者可通过在线计算器快速评估采用预测模型后的样本量需求，优化研究设计并降低数据收集成本。
