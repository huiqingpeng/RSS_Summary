Fast-HaMeR: Boosting Hand Mesh Reconstruction using Knowledge Distillation

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出Fast-HaMeR，通过知识蒸馏技术加速3D手部网格重建模型。研究将HaMeR模型的ViT-H骨干网络替换为MobileNet、MobileViT、ConvNeXt和ResNet等轻量级替代方案，并评估了输出级、特征级及混合三种知识蒸馏策略。实验表明，仅需原模型35%大小的轻量级骨干网络即可实现1.5倍推理加速，同时保持相似的重建精度（仅0.4mm误差差异），为资源受限设备上的实时应用铺平道路。

方法概述 / Method

本研究采用知识蒸馏框架，将高性能但计算昂贵的HaMeR教师模型（基于ViT-H）的知识迁移至轻量级学生网络。具体而言，系统比较了三种蒸馏策略：输出级蒸馏（对齐最终预测结果）、特征级蒸馏（匹配中间层特征表示）以及两者的混合方案，以确定不同容量学生模型的最优训练策略。

实验结果 / Results

实验结果显示，输出级蒸馏对轻量级学生模型性能提升最为显著，而特征级蒸馏在容量较大的学生模型上效果更佳。采用轻量级骨干的模型在仅保留原模型35%参数量的条件下，推理速度提升1.5倍，重建精度与原始HaMeR相比仅下降0.4mm，实现了效率与精度的有效平衡。

应用价值 / Applications

该技术可直接部署于VR/AR头显、智能手机、嵌入式系统等资源受限设备，支持实时人机交互、机器人操作和医疗健康监测等场景。通过大幅降低计算开销同时保持高精度，Fast-HaMeR使原本难以在边缘设备运行的先进手部重建技术得以实用化，推动了3D视觉技术在消费级硬件上的普及。
