FlowComposer: Composable Flows for Compositional Zero-Shot Learning

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了FlowComposer，首个系统研究流匹配（flow matching）用于组合零样本学习（CZSL）的框架。针对现有基于视觉-语言模型（VLM）的PEFT方法存在的"隐式组合构建"和"特征纠缠残留"两大局限，FlowComposer通过学习两个独立的原始流将视觉特征分别映射到属性和对象的文本嵌入，并引入可学习的Composer模块在速度场层面显式融合生成组合流。此外，作者设计了泄漏引导增强机制，将不可避免的残留纠缠特征转化为辅助信号。在三个公开CZSL基准上的广泛实验表明，该框架作为即插即用组件集成到多种基线模型中均能取得显著提升。

方法概述 / Method

FlowComposer采用模型无关的流匹配框架，核心包含三个组件：（1）**属性流与对象流**——两个独立的条件流匹配网络，分别将视觉特征传输至属性文本嵌入和对象文本嵌入；（2）**Composer模块**——可学习的显式融合机制，在速度场（velocity field）层面将两个原始流的速度场组合为复合流；（3）**泄漏引导增强**——利用特征纠缠分析识别残留泄漏，将其作为辅助监督信号增强训练。该框架可与任意VLM骨干结合，无需修改其预训练权重。

实验结果 / Results

作者在MIT-States、UT-Zappos和C-GQA三个标准CZSL数据集上进行了评估，将FlowComposer集成到CLIP-Adapter、CoOp、CoCoOp等多种基线方法中。实验结果显示，集成FlowComposer后各基线均获得显著提升，例如在最具挑战性的C-GQA数据集上， unseen组合识别准确率提升超过5个百分点。消融实验验证了Composer的显式速度场融合优于简单的嵌入拼接，且泄漏引导增强能有效利用纠缠特征。

应用价值 / Applications

该研究为组合零样本学习提供了新的技术范式，可广泛应用于需要识别新颖概念组合的计算机视觉场景，如电商产品检索（识别"红色皮革沙发"等未见属性-对象组合）、自动驾驶场景理解（识别罕见物体状态）、以及开放词汇图像分类等。流匹配框架的模型无关特性使其易于部署到现有的视觉-语言系统中，提升模型对新概念的泛化能力，降低人工标注成本。
