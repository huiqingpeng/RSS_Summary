SAMSEM -- A Generic and Scalable Approach for IC Metal Line Segmentation

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了SAMSEM，一种基于Meta SAM2模型改进的通用可扩展集成电路金属线分割方法，用于解决全球化硬件供应链中的恶意电路检测问题。该方法通过多尺度分割策略和拓扑损失函数，实现了跨不同技术节点、制造材料、样品制备方法和SEM成像技术的泛化能力。在包含14个IC、48个金属层的超大规模数据集上，SAMSEM在已知IC上达到0.72%的错误率，在未见IC上达到5.53%的错误率，全量微调后错误率降至0.62%，显著推进了制造后IC验证领域的技术前沿。

方法概述 / Method

SAMSEM采用Meta的Segment Anything Model 2 (SAM2)作为基础架构，针对SEM图像的多尺度特性开发了自适应分割策略，能够处理不同尺寸、分辨率和放大倍率的图像。核心创新在于引入拓扑损失函数（topology-based loss），与像素级损失相结合，使模型优化目标从单纯的像素准确性转向电气连通性这一更具物理意义的指标，并通过超参数优化实现跨域泛化。

实验结果 / Results

实验基于48个金属层、14个不同IC的SEM图像数据集（该领域前所未有的规模）展开。在7个IC上微调后，同分布测试错误率为0.72%，跨7个未见IC的泛化错误率为5.53%；在全部14个IC上微调后，整体错误率降至0.62%。结果表明SAMSEM在保持低错误率的同时，展现出优异的跨技术节点、跨制造条件的迁移能力。

应用价值 / Applications

SAMSEM可应用于高安全性场景的硬件供应链保障，特别是加密芯片和关键基础设施IC的制造后验证，帮助检测未授权环境中的恶意植入电路。其通用性和可扩展性大幅降低了对每个目标IC单独调参的需求，为半导体行业的第三方可信验证、逆向工程分析和防伪认证提供了高效自动化工具，具有重要的产业安全价值。
