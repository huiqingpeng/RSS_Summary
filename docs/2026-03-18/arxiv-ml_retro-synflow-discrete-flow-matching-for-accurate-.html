论文摘要 / Abstract

本文提出了RETRO SYNFLOW (RSF)，一种用于单步逆合成预测的离散流匹配框架，通过构建目标产物与反应物之间的马尔可夫桥来解决化学搜索空间的组合爆炸问题。
RSF创新性地引入反应中心识别步骤生成中间体合成子（synthons）作为更有信息量的源分布，并结合Feynman-Kac引导与序贯蒙特卡洛重采样技术提升生成样本的多样性和可行性。实验表明，RSF达到60.0%的top-1准确率，较先前最优方法提升20%，同时FK引导将top-5往返准确率提升19%。

方法概述 / Method

RSF采用离散流匹配框架，首先通过反应中心识别生成合成子作为中间结构，建立从目标产物到反应物的概率桥接路径。推理阶段引入Feynman-Kac引导机制，结合基于序贯蒙特卡洛的重采样策略，利用前向合成模型作为奖励预言机（reward oracle）对高质量生成路径进行定向引导。

实验结果 / Results

RSF在标准基准上取得60.0%的top-1准确率，超越此前无模板最优方法20%；FK引导策略使top-5往返准确率较现有无模板方法提升19%，同时保持具有竞争力的top-k准确率。实验结果验证了反应中心预识别和推理时引导策略对准确性与多样性的双重提升作用。

应用价值 / Applications

该研究为计算机辅助药物发现和有机合成路线设计提供了高精度、高多样性的单步逆合成预测工具，可显著加速化学家设计合成路径的决策过程。
RSF的无模板特性使其适用于新颖化学结构的探索，而推理时的可控引导机制为工业场景中平衡预测准确性与合成可行性提供了实用解决方案。
