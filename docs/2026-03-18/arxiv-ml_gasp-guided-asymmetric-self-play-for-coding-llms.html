GASP: Guided Asymmetric Self-Play For Coding LLMs

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了引导式非对称自博弈（GASP）方法，用于解决现有非对称自博弈方法在训练代码大模型时缺乏目标导向的问题。通过引入"目标标杆问题"（goalpost questions）作为学习锚点，教师模型生成由易到难的变体问题，逐步缩小学生模型与难题之间的差距。该方法在LiveCodeBench上相比无引导的非对称自博弈提升了2.5%的pass@20性能，并成功解决了基线方法无法攻克的目标标杆难题。

方法概述 / Method

GASP的核心机制是利用真实数据中的高难度问题作为目标标杆，教师模型采用"先简化、再复杂化"的两阶段生成策略：首先将目标难题简化为学生可解的变体，然后在此基础上生成更难的变体，形成渐进式课程。这种设计将开放式数据生成与明确的探索目标相结合，避免了传统自博弈中问题难度与信息价值脱节的问题。

实验结果 / Results

在LiveCodeBench基准测试上，GASP相比无引导的非对称自博弈实现了2.5%的pass@20提升；通过教师构建的课程，模型最终解决了所有基线方法均无法攻克的困难目标标杆问题，证明了引导机制在硬探索任务中的有效性。

应用价值 / Applications

该方法适用于代码大模型的后训练阶段，特别是在缺乏高质量人工标注数据时，可实现自动化的课程学习与能力边界扩展；其引导式自博弈框架也可推广至其他需要硬探索的推理任务（如数学证明、科学发现），为构建开放式、目标导向的AI训练系统提供新范式。
