Inference for Deep Neural Network Estimators in Generalized Nonparametric Models

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对深度神经网络（DNN）在广义非参数回归模型（GNRMs）中的统计推断问题展开研究，突破了现有方法依赖估计误差与输入独立这一在GNRMs中常被违反的假设。作者提出了允许误差与输入相关的理论分析框架，证明了在GNRMs下进行有效推断的可行性，并开发了基于U统计量和Hoeffding分解的集成子抽样方法（ESM）来构建可靠的置信区间。该方法实现了模型无关的方差估计，并能处理个体间的异质性，在模拟实验和eICU真实数据集上均验证了有效性。

方法概述 / Method

论文采用深度神经网络作为广义非参数回归模型的估计器，核心创新在于允许估计误差与协变量之间存在依赖性，而非传统方法要求的独立性假设。为实现统计推断，作者设计了集成子抽样方法（ESM），该方法利用U统计量的渐近理论和Hoeffding分解技术，无需指定具体的方差结构即可估计DNN估计量的抽样变异性。

实验结果 / Results

研究在多种非参数回归设定下进行了模拟实验，包括Logistic回归、泊松回归和二项回归模型，结果表明ESM方法能够有效构建覆盖概率良好的置信区间。在eICU大规模重症监护电子健康记录数据的应用中，该方法成功预测了ICU再入院风险，并为个体患者提供了可靠的预测不确定性量化，支持临床决策中的风险评估。

应用价值 / Applications

该研究为深度学习模型在医疗健康等高风险领域的可靠应用提供了统计理论基础，使临床医生能够获得带有置信区间的个体化风险预测，从而做出更明智的决策。方法可推广至任何需要不确定性量化的广义非参数回归场景，如金融风险管理、精准营销和公共卫生监测等领域，弥补了传统DNN"黑箱"预测缺乏统计推断的缺陷。
