Generative Inverse Design with Abstention via Diagonal Flow Matching

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对逆向设计问题提出了一种新的生成式方法——对角流匹配（Diagonal Flow Matching, Diag-CFM）。传统条件流匹配在逆向设计任务中对设计参数与目标性能的任意排序和缩放高度敏感，导致训练不稳定；Diag-CFM通过"零锚定"策略将设计坐标与噪声配对、标签与零配对，使学习问题对坐标置换具有理论不变性。该方法在高达100维的设计空间中将往返精度提升一个数量级，并提出了两种架构内生的不确定性度量（Zero-Deviation和Self-Consistency），实现了多候选选择、不可靠预测弃权以及分布外目标检测三种实用功能。

方法概述 / Method

Diag-CFM的核心创新是采用零锚定策略：在流匹配的训练过程中，将设计参数$x$与噪声向量配对，同时将目标性能$y^*$与零向量配对，形成对角化的条件结构。这一设计消除了标准CFM中对$(x, y^*)$联合表示排序和缩放的依赖性，使模型学习具有置换不变性。基于此架构，作者进一步提出了Zero-Deviation（零偏差）和Self-Consistency（自一致性）两种无需额外计算开销的不确定性量化指标。

实验结果 / Results

在翼型设计、燃气轮机燃烧室以及可扩展维度的解析基准三个任务上，Diag-CFM相比标准CFM和可逆神经网络基线实现了数量级的往返精度提升，且性能优势在设计维度扩展至$P=100$时依然保持。提出的两种不确定性度量在候选选择、预测弃权和分布外检测任务上持续优于基于集成和通用替代方法，展现出稳定可靠的校准性能。

应用价值 / Applications

该研究为工程逆向设计提供了高维、稳定且具备内在不确定性量化的生成式解决方案，适用于航空航天（翼型优化）、能源动力（燃烧系统设计）等需要探索多样化设计方案的领域。不确定性量化能力使系统能够在实际部署中主动拒绝不可靠预测、识别超出训练分布的目标需求，显著提升了AI辅助设计工具的安全性和实用性。
