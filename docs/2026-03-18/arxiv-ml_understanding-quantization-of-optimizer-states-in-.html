论文摘要 / Abstract

本文研究了大型语言模型预训练中优化器状态量化的动态机制，发现低精度指数移动平均（EMA）状态会因量化导致大量名义更新被舍入回同一存储值，从而使状态"陈旧"并减缓自适应速度。作者建立了预测状态停滞概率的简单模型，并解释了为何状态重置在低精度训练中有效——重置可暂时恢复量化EMA的响应性。基于此理论，作者推导出选择最优重置周期的简单方法，实验证明合适的重置调度能在大幅降低优化器状态内存的同时恢复低精度训练损失的性能。

方法概述 / Method

论文通过理论建模和实验验证相结合的方法，首先建立量化EMA状态的单步停滞概率预测模型，刻画停滞随时间累积的动态过程；然后基于此理论推导状态重置周期的选择策略，并在受控模拟和真实LLM预训练中验证该方法的有效性。

实验结果 / Results

实验表明，在受控模拟和实际LLM预训练中，采用理论指导的重置调度策略能够有效恢复因低精度状态存储而损失的训练性能；该方法在显著降低优化器状态内存占用的同时，达到了与全精度训练相当的效果，证明了状态重置在低精度训练中的关键作用及其时序选择的重要性。

应用价值 / Applications

该研究为大规模语言模型预训练中的内存优化提供了理论指导和实用方法，特别适用于需要降低显存占用的分布式训练场景；所提出的状态重置策略可直接集成到现有低精度优化器实现中，帮助研究者和工程师在资源受限环境下高效训练更大规模的模型。
