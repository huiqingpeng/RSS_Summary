Evaluating Causal Discovery Algorithms for Path-Specific Fairness and Utility in Healthcare

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究针对健康数据因果发现中缺乏真实基准的评估难题，通过与专家合作构建代理真实因果图，建立了合成阿尔茨海默病和心力衰竭临床数据的评估基准。研究评估了Peter-Clark、Greedy Equivalence Search和Fast Causal Inference三种算法在结构恢复和路径特定公平性分解方面的表现，发现不同算法在公平性-效用权衡上存在显著差异，强调了在临床部署因果发现方法时需要结合图结构的细粒度公平性评估。

方法概述 / Method

研究采用专家协作方式构建代理真实因果图作为评估基准，并运用路径特定因果效应分解技术，超越传统的复合公平性评分方法，对三种主流因果发现算法（Peter-Clark、GES、FCI）进行综合评估。

实验结果 / Results

在合成数据上，Peter-Clark算法实现了最优的结构恢复性能；在心力衰竭真实数据上，Fast Causal Inference算法获得了最高的效用表现。路径特定效应分析显示，射血分数在真实图中对间接效应贡献了3.37个百分点，这些差异导致各算法的公平性-效用比率产生显著变化。

应用价值 / Applications

该研究为医疗AI系统的公平性评估提供了方法论框架，特别适用于需要理解算法决策背后因果机制的场景，如临床风险预测模型的公平性审计和优化，有助于指导医疗机构在部署因果发现工具时平衡预测准确性与健康公平性。
