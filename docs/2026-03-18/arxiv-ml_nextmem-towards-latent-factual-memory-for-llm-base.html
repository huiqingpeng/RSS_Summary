NextMem: Towards Latent Factual Memory for LLM-based Agents

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了NextMem，一种面向LLM智能体的潜空间事实记忆框架，旨在解决现有文本记忆方法上下文负担重、参数化方法存在灾难性遗忘和高成本的问题。该框架利用自回归自编码器高效构建潜空间记忆并确保准确重建，通过两阶段训练（自回归重建对齐与渐进潜空间替换）和量化技术优化性能与存储效率。实验表明NextMem在检索准确性、鲁棒性和可扩展性方面表现优异。

方法概述 / Method

NextMem采用自回归自编码器架构，将事实信息编码压缩为低维潜空间表示而非原始文本。训练过程分为两个阶段：首先进行自回归重建对齐以确保编码器-解码器的一致性，随后实施渐进潜空间替换策略逐步增强潜空间记忆的独立性和稳定性。此外引入量化机制进一步降低存储开销。

实验结果 / Results

实验显示NextMem在多项评估中取得领先性能，在记忆检索任务上展现出高准确性，同时对输入噪声和分布偏移具有强鲁棒性。该框架支持高效的记忆扩展与更新，避免了传统参数化方法的灾难性遗忘问题，且存储成本显著低于纯文本记忆方案。

应用价值 / Applications

NextMem适用于需要长期事实记忆的大规模LLM智能体系统，如个人助理、知识密集型对话机器人和自主决策代理。其高效的潜空间表示可显著降低推理时的上下文长度，减少API调用成本，同时为边缘设备部署和实时应用提供可行的记忆解决方案。
