Beyond the Embedding Bottleneck: Adaptive Retrieval-Augmented 3D CT Report Generation

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究揭示了3D CT影像报告生成中存在的关键瓶颈：对比学习得到的3D CT嵌入向量虽然包含病理判别信号，但存在严重的维度集中问题（512维中仅2维有效）。基于此发现，作者提出AdaRAG-CT自适应检索增强框架，通过受控检索引入补充文本信息并在生成过程中选择性整合。该模型在CT-RATE基准上取得最优临床效果，Clinical F1从0.420提升至0.480（+6个百分点）。

方法概述 / Method

AdaRAG-CT采用双阶段自适应机制：首先通过受控检索策略从外部知识库获取与当前病例相关的补充文本信息，以补偿视觉表征的信息瓶颈；然后在生成阶段设计选择性整合模块，动态控制检索信息的注入程度，避免噪声干扰。该方法突破了传统静态检索的局限性，实现了视觉-文本信息的自适应融合。

实验结果 / Results

在CT-RATE基准数据集上，AdaRAG-CT达到当前最优的临床效能，Clinical F1指标显著超越此前最佳方法CT-Agent（0.480 vs 0.420）。消融实验验证了检索组件和生成组件对性能提升均有贡献。值得注意的是，单纯扩大语言模型规模无法改善生成质量，进一步证实瓶颈存在于视觉表征而非生成器本身。

应用价值 / Applications

该研究为医学影像智能诊断系统提供了重要的技术改进方向，特别适用于需要处理复杂3D CT体积的临床场景，如肿瘤筛查、多器官病变检测等。
AdaRAG-CT框架可集成至医院PACS系统，辅助放射科医生生成更全面、准确的诊断报告，减少漏诊风险并提升工作效率。其揭示的嵌入瓶颈问题及解决思路对多模态医学AI研究具有广泛借鉴意义。
