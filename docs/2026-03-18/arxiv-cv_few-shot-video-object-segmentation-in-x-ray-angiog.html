Few-Shot Video Object Segmentation in X-Ray Angiography Using Local Matching and Spatio-Temporal Consistency Loss

论文摘要 / Abstract

本文提出了一种用于X射线血管造影视频的小样本视频目标分割（FSVOS）模型，通过局部匹配策略将搜索空间限制在最相关的邻近像素，并采用基于方向的非参数采样机制实现动态变化的采样区域。研究设计了监督时空对比学习方案以增强跨帧特征一致性，同时发布了首个X射线血管造影多目标分割公开基准数据集MOSXAV。在CADICA、XACV和MOSXAV数据集上的实验表明，该方法在分割准确率和泛化能力（包括可见和不可见类别）方面均优于当前最先进的视频分割方法。

方法概述 / Method

该方法采用局部匹配策略替代传统的全局匹配，通过方向视角重新组织局部采样过程，实现无需参数学习的动态采样区域调整。核心创新包括非参数采样机制，避免了标准im2col操作（如空间卷积、深度可分离卷积）和硬件专用CUDA核（如可变形卷积、邻域注意力）的计算开销和跨设备移植性问题；同时引入监督时空对比损失函数，强制约束相邻帧间特征表示的一致性。

实验结果 / Results

在三个X射线血管造影数据集（CADICA、XACV和新发布的MOSXAV）上的广泛实验验证了该方法的有效性，其在分割精度和泛化能力（涵盖已见和未见类别）方面均超越现有最先进的视频分割方法。
MOSXAV数据集提供了详细的手动标注分割真值，为多目标血管结构分割建立了新的评估基准。

应用价值 / Applications

该研究为心血管介入手术中的实时血管结构分析提供了灵活高效的技术方案，可辅助医生进行精确的导管导航和病变评估。由于方法具有跨设备兼容性和无需重新训练的动态适应能力，具备广泛的临床部署潜力，适用于各类X射线血管造影引导的微创介入治疗场景，有望提升手术安全性和效率。
