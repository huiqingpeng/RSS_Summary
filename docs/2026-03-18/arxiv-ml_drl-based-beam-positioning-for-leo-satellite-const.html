DRL-Based Beam Positioning for LEO Satellite Constellations with Weighted Least Squares

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了一种轻量级深度强化学习（DRL）辅助的加权框架，用于低轨（LEO）卫星星座的CSI-free多卫星定位。该方法采用离散动作深度Q网络（DQN）直接从接收的导频测量和几何特征中学习卫星权重，并结合增强的加权最小二乘（WLS）估计器实现物理一致性的定位，同时联合估计接收机时钟偏差。该混合设计在10颗可见卫星的代表性二维场景下实现了亚米级精度（0.395m RMSE），且计算开销低，适用于资源受限的LEO载荷实际部署。

方法概述 / Method

论文采用DRL与WLS相结合的混合架构：DQN作为轻量级策略网络，以离散动作空间输出各可见卫星的权重，输入为导频测量和卫星几何特征；WLS估计器则利用学习到的权重进行位置解算和时钟偏差估计，确保定位结果满足物理约束。该方法避免了传统监督学习对大量标注数据的依赖，通过强化学习的探索-利用机制直接优化定位精度与计算效率的权衡。

实验结果 / Results

在10颗可见卫星的二维LEO场景仿真中，所提方法达到0.395米的均方根误差（RMSE），实现亚米级定位精度；与最优加权最小二乘（OWLS）基准相比，在保持相近精度的同时显著降低了计算复杂度，单次推理时间满足实时性要求。消融实验验证了DRL权重学习对几何精度因子（GDOP）恶化场景的鲁棒性提升。

应用价值 / Applications

该研究适用于星载处理资源受限的LEO导航增强服务，如物联网终端定位、应急通信用户位置获取等场景。其轻量级特性支持直接在卫星载荷上部署，减少对地面站的依赖，为大规模LEO星座提供低功耗、低延迟的自主定位能力，特别适用于无信道状态信息（CSI-free）的广播式导频体制。
