Convergence Bounds for Sequential Monte Carlo on Multimodal Distributions using Soft Decomposition

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文证明了在序贯蒙特卡洛（SMC）算法所得样本的经验测度下，函数f的方差界，其时间复杂度依赖于局部而非全局马尔可夫链混合动态。
SMC是一种MCMC方法，通过从已知分布中抽取粒子，经一系列分布进行重加权和重采样，并在每步应用马尔可夫链进行平滑。本文突破了传统SMC理论依赖全局混合时间假设的局限，首次在多模态真实场景下建立了仅依赖局部MCMC动态的收敛界。

方法概述 / Method

作者采用"软分解"（soft decomposition）技术，将多模态分布分解为局部分量的加权组合，而非硬划分。通过分析SMC算法在序列分布上的传播行为，结合局部马尔可夫链的混合性质，推导出方差上界。该方法避免了全局混合时间的假设，转而利用各模态内部的局部混合特性。

实验结果 / Results

论文建立了显式的方差收敛界，其收敛速率取决于局部混合时间而非全局混合时间，这在多模态设置下可实现指数级改进。理论结果表明，当分布具有良好分离的多个模态时，SMC的样本复杂度与模态数量呈多项式关系，而非指数关系。

应用价值 / Applications

该研究为贝叶斯推断中复杂后验分布（如混合模型、多峰能量景观）的采样提供了理论保障，可应用于统计物理中的自由能计算、机器学习中的潜变量模型推断，以及生物信息学中多稳定态系统的分析，显著扩展了SMC在高维多模态问题中的可适用性。
