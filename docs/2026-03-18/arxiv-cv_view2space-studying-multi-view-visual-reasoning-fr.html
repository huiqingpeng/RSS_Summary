VIEW2SPACE: Studying Multi-View Visual Reasoning from Sparse Observations

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对稀疏多视角视觉推理这一尚未充分研究的领域，提出了VIEW2SPACE基准测试。研究者利用物理仿真引擎构建高精度3D场景，生成数百万个带有精确几何和语义标注的问答对。实验发现当前最先进的视觉-语言模型在此任务上表现接近随机猜测，而作者提出的Grounded Chain-of-Thought with Visual Evidence方法在中等难度任务上显著提升性能，并能泛化到真实世界数据。

方法概述 / Method

研究采用基于物理的仿真引擎生成多样化、高保真的3D场景，确保每个视角具有精确的元数据标注，从而支持可扩展且可迁移至真实场景的数据生成。在此基础上，作者设计了Grounded Chain-of-Thought with Visual Evidence训练方法，通过显式整合视觉证据来引导模型进行空间推理。

实验结果 / Results

综合评估显示，现有SOTA视觉-语言模型和空间模型在稀疏多视角推理任务上表现极差，多数仅略高于随机水平。作者提出的方法在中等难度设置下显著改善性能，并在跨数据集真实世界评估中超越现有方法。扩展性分析表明，几何感知在充足可见性条件下可从规模扩展中受益，但跨稀疏视角的深度组合推理仍是根本性挑战。

应用价值 / Applications

该研究为机器人导航、自动驾驶和增强现实等需要从离散稀疏视角理解三维环境的智能系统提供了重要的基准和训练范式。其仿真到真实的迁移能力降低了大规模多视角标注数据的收集成本，提出的推理方法可应用于需要空间理解和跨视角整合的实际场景。
