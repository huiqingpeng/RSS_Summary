CardioComposer: Leveraging Differentiable Geometry for Compositional Control of Anatomical Diffusion Models

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文提出了CardioComposer框架，用于在保持生成真实性的同时实现对3D心血管解剖结构的几何可控生成。该方法通过可解释的椭球体基元表示解剖子结构的大小、形状和位置等几何属性，并基于体素级几何矩开发可微分测量函数，在扩散模型采样过程中提供基于损失的梯度引导。实验表明，该方法能够以解耦方式约束单个几何属性，并对多个子结构实现组合控制，且适用于包含非凸子结构的多种解剖系统（心脏、血管和骨骼器官）。

方法概述 / Method

CardioComposer采用基于椭球体基元的程序化表示方法，将解剖结构分解为可解释的几何属性（尺寸、形状、位置）。核心创新在于开发了基于体素级几何矩的可微分测量函数，这些函数能够在扩散模型的推理阶段计算几何损失并提供梯度引导，从而实现对生成过程的精确控制而无需重新训练模型。

实验结果 / Results

实验验证了该方法能够以解耦方式独立控制各个几何属性（如单独调整心脏某腔室的大小而不影响其他属性），并支持对多个解剖子结构的组合控制。结果表明该框架具有良好的泛化性，可扩展至心脏、血管系统及具有非凸几何形状的骨骼器官等多种解剖结构，在保持生成样本真实性的同时实现精确的几何约束。

应用价值 / Applications

该研究在临床研究和医疗器械评估领域具有重要应用价值，可生成符合特定几何规范的合成解剖数据用于手术规划模拟、人工心脏瓣膜等植入器械的设计验证与性能测试，以及医学影像算法的训练数据增强。其组合控制能力使研究人员能够系统性地探索解剖变异对临床结果的影响，为个性化医疗和虚拟临床试验提供支持。
