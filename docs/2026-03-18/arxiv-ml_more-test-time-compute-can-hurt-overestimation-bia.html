More Test-Time Compute Can Hurt: Overestimation Bias in LLM Beam Search

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文揭示了大型语言模型束搜索中一个反直觉的现象：增加测试时计算量（扩大束宽）可能损害输出质量。作者基于极值理论分析发现，束选择过程中的噪声评分器会引入系统性高估偏差，该偏差随候选池规模增大而增长，并推导出最大有效束宽 $\hat{k}$ 的理论上限——超过此阈值搜索性能将下降。该临界宽度取决于评分器的信噪比，实验验证了困惑度评分（高噪声，$\hat{k}=1$）与PRM评分（低噪声，$\hat{k}\geq 4$）的显著差异。

方法概述 / Method

作者运用极值理论（Extreme Value Theory）对束搜索中的评分噪声进行数学建模，推导出最大有用束宽 $\hat{k}$ 的闭式解：其随正确路径与错误路径的质量优势 $\Delta$ 与评分噪声 $\sigma$ 之比的平方指数增长。研究在MR-BEN基准（5,975道问题、10个领域）上对比了困惑度引导与PRM（过程奖励模型）引导的束搜索，覆盖三个7B参数模型。

实验结果 / Results

实验表明，相同模型和算法下，不同评分器导致截然不同的最优束宽：高噪声的困惑度评分使 $\hat{k}=1$（任何束宽均无收益），而低噪声的PRM评分可达 $\hat{k}\geq 4$，最高带来8.9个百分点的性能提升。这一结果验证了理论预测——评分器信噪比是束宽选择的核心决定因素。

应用价值 / Applications

该研究为LLM推理系统的束搜索配置提供了理论指导和实用诊断工具：部署前可通过估计评分器的信噪比预判有效束宽范围，避免盲目扩大计算资源导致的性能退化。这对资源受限场景下的推理优化、自适应束宽算法设计，以及评分器改进（如开发更低噪声的过程监督信号）具有直接工程意义。
