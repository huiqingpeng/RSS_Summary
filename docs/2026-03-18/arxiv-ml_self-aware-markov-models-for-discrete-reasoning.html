Self-Aware Markov Models for Discrete Reasoning

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对标准掩码离散扩散模型在推理任务中的局限性，提出了一种自感知马尔可夫模型。该模型通过学习在其自身输出上训练的马尔可夫转移核，实现了token的重新掩码以纠正先前错误，并采用可学习的停止准则替代固定时间调度，使计算量能根据问题难度自适应调整。该方法仅需添加两个轻量级预测头即可复用现有预训练模型，在Sudoku-Extreme数据集上达到95%的有效解率，在Countdown-4任务上平均仅需10步即可正确解决96%的问题。

方法概述 / Method

论文核心方法是训练一个自引用的马尔可夫转移核，该核在模型自身的输出分布上进行学习，允许已生成的token被重新掩码从而实现错误修正。同时引入可学习的停止准则替代传统的固定扩散时间步，使推理过程能够动态终止。整个框架通过仅增加两个轻量级预测头（用于掩码决策和停止判断）实现，可与现有预训练模型无缝集成。

实验结果 / Results

在Sudoku-Extreme数据集上，该方法显著优于其他基于流的方法，有效性达到95%。在Countdown-4算术推理任务中，模型平均仅需10步即可正确解决近96%的问题，且许多简单问题可在2步内完成。实验表明该自适应步长机制能有效匹配问题复杂度，避免固定步长带来的计算浪费或不足。

应用价值 / Applications

该研究为离散推理任务（如数独求解、算术推理等）提供了更高效灵活的生成模型范式，可广泛应用于需要逐步推理的AI系统，如数学问题求解、逻辑谜题生成与验证、代码合成等场景。其轻量级适配特性使其易于部署到现有大语言模型中，提升推理效率的同时降低计算成本，特别适用于资源受限或对响应速度敏感的实际应用。
