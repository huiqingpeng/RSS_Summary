Closed-Loop Action Chunks with Dynamic Corrections for Training-Free Diffusion Policy

arXiv Computer Vision 2026-03-18

论文摘要 / Abstract

本文提出了DCDP（Dynamic Closed-Loop Diffusion Policy），一种无需重新训练即可实现实时闭环动作修正的扩散策略框架。该框架通过整合基于块的动作生成与实时动态校正机制，解决了传统扩散策略在动态环境中响应迟缓、适应性差的问题。实验表明，DCDP在动态PushT仿真中将适应性提升19%，仅增加5%的计算开销，并成功应用于真实机器人操作任务。

方法概述 / Method

DCDP采用模块化设计，包含三个核心组件：自监督动态特征编码器用于提取环境变化信息、交叉注意力融合模块将动态特征与扩散去噪过程结合、以及非对称动作编码-解码器实现动作注入前的实时修正。该框架在推理阶段动态注入环境反馈，无需修改预训练的扩散策略权重即可实现闭环控制。

实验结果 / Results

在动态PushT仿真环境中，DCDP相比基线方法适应性提升19%，计算开销仅增加5%；在真实机器人操作任务中验证了时间一致性与实时响应能力的平衡。其即插即用特性使其能够无缝集成到现有扩散策略系统中，在保持动作块时间连贯性的同时显著增强动态场景适应能力。

应用价值 / Applications

DCDP适用于需要快速适应环境变化的机器人操作场景，如动态抓取、人机协作装配、非结构化环境中的灵巧操作等。其训练无关的修正机制降低了部署成本，模块化设计便于工业集成，为扩散策略在真实动态环境中的实际应用提供了可行路径。
