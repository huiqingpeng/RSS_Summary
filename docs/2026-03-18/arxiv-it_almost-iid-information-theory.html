论文摘要 / Abstract

本文探讨了信息论中"几乎独立同分布（almost-iid）"资源的核心问题：由于实际物理操作无法完全验证系统的独立同分布（iid）特性，基于de Finetti定理的对称性假设只能保证资源是almost-iid的。作者证明了almost-iid态的条件熵在渐近意义下与iid态的条件熵一致，从而回答了almost-iid资源是否与完美iid资源在信息处理任务中同等有效这一关键问题。作为重要应用，该结果表明压缩纠缠（squashed entanglement）对almost-iid态具有鲁棒性。

方法概述 / Method

论文运用de Finetti定理作为理论基础，将操作上有意义的对称性假设与almost-iid态的描述联系起来。通过严格的信息论推导，作者建立了almost-iid态与iid态之间条件熵的渐近等价关系，核心在于证明两者在渐近极限下的差异趋于消失。

实验结果 / Results

主要理论结果是证明了almost-iid态的条件熵渐近等于对应iid态的条件熵。作为直接推论，压缩纠缠这一重要的纠缠度量在almost-iid态上的取值，在渐近意义下与其在iid态上的取值相匹配，表明该纠缠度量对almost-iid偏离具有鲁棒性。

应用价值 / Applications

该研究为量子信息论提供了更坚实的操作基础，使得理论结果能够应用于实际场景中无法严格保证iid条件的物理系统。压缩纠缠的鲁棒性结果对量子密钥分发、量子信道容量分析等依赖纠缠量化的任务具有实际意义，降低了对源完美独立性的苛刻要求。
