HeBA: Heterogeneous Bottleneck Adapters for Robust Vision-Language Models

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了HeBA（异构瓶颈适配器），一种用于视觉-语言模型（如CLIP）下游任务适配的统一架构框架。该框架针对图像和文本模态的不同结构特性（图像的空间局部性与文本的语义密度），引入模态特定的结构归纳偏置，通过三大创新——异构处理、瓶颈正则化和主动梯度初始化——解决了传统"一刀切"适配器架构的局限性。实验表明，HeBA在11个小样本基准上达到了新的最先进水平，展现出卓越的稳定性和准确性。

方法概述 / Method

HeBA的核心架构包含三个关键创新：（1）**异构设计**：视觉token采用2D深度可分离卷积处理以保留空间相关性，文本token则通过密集线性投影捕获语义关系；（2）**瓶颈正则化**：采用D→D/4的压缩瓶颈结构，强制模型学习紧凑鲁棒的特征，起到结构正则化作用；（3）**主动梯度初始化**：摒弃零初始化范式，采用Kaiming初始化策略确保足够的初始梯度流，加速收敛同时保护冻结主干网络的预训练知识。

实验结果 / Results

HeBA在11个小样本学习基准上建立了新的最先进水平，显著优于现有方法。该架构通过模态特定的专业化设计，在保持计算效率的同时实现了更高的稳定性和准确性，证明了结构归纳偏置对于视觉-语言模型适配的重要性。

应用价值 / Applications

HeBA可广泛应用于需要高效适配大规模视觉-语言模型的场景，如小样本图像分类、跨模态检索和下游视觉-语言任务微调。其轻量化适配器设计使其特别适合计算资源受限的部署环境，同时模态感知的架构创新为构建更鲁棒的多模态AI系统提供了可扩展的解决方案。
