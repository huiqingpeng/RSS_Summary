Efficient Continual Learning in Language Models via Thalamically Routed Cortical Columns

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文提出了TRC²（丘脑路由皮层柱）架构，一种专为持续学习设计的解码器模型，通过结合堆叠皮层柱、丘脑调制通路和海马体通路，将快速可塑性与稳定计算分离，使持续适应成为模型本身的固有属性。该设计通过选择性跨柱通信、事件选择性检索和延迟惊喜驱动的写入机制，有效缓解了序列更新中的灾难性遗忘问题。在C4、WikiText-103和GSM8K的任务序列流上，TRC²在任务边界建模质量和累积遗忘控制方面显著优于Transformer、Mamba、MoE和DeepSeek等基线模型，同时保持竞争力的吞吐量和训练成本。

方法概述 / Method

TRC²采用神经科学启发的三层结构设计：堆叠皮层柱作为基础计算单元，丘脑通路实现选择性跨柱通信调制，海马体通路负责事件选择性记忆检索、基于延迟惊喜的记忆写入和重放驱动的巩固。模型引入因果记忆更新方案和在线重放控制器，根据实测遗忘动态调整巩固强度，实现快速可塑性与慢速稳定计算的分离。

实验结果 / Results

在跨越C4、WikiText-103和GSM8K的任务序列语言建模流中，TRC²显著提升了任务边界建模质量，并大幅降低了相对于Transformer、Mamba、MoE和DeepSeek基线的累积遗忘。消融实验表明丘脑和海马体组件是记忆保持增益的核心贡献因素，完整模型在吞吐量和训练成本方面仍保持竞争力。

应用价值 / Applications

该研究为部署在真实环境中的大语言模型提供了内建的持续适应能力，可广泛应用于需要动态适应用户行为变化、任务分布漂移和数据演化的场景，如个性化对话系统、多任务学习平台和长期在线学习服务，避免了传统方法中外置稳定化程序的成本高昂、脆弱性和扩展性难题。
