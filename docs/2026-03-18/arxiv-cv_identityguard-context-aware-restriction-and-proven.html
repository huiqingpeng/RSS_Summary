论文摘要 / Abstract

本文针对个性化文本到图像模型面临的安全挑战，提出了一种上下文感知的安全防护方法。现有全局过滤方法存在"一刀切"问题，为防止滥用而完全擦除概念，导致模型整体效用受损。作者提出的IDENTITYGUARD通过条件限制机制，仅在个性化身份与有害内容结合时进行阻断，并配备概念特定水印实现精准溯源，在防止滥用的同时保留模型效用。

方法概述 / Method

IDENTITYGUARD采用双重技术方案：一是条件性限制机制，根据上下文动态判断个性化身份与潜在有害内容的组合情况；二是概念特定水印技术，为每个个性化概念嵌入可追溯的标识。该方法将安全机制与个性化概念内在绑定，实现精准定向防护而非全局阻断。

实验结果 / Results

实验表明，IDENTITYGUARD能够有效阻止基于个性化身份的恶意内容生成，同时保持模型的正常生成能力和广泛效用。水印机制展现出稳健的溯源性能，可在实际场景中实现精准的责任追踪。

应用价值 / Applications

该研究为个性化AI生成模型（如定制化肖像生成、虚拟形象创作等）提供了负责任的安全部署路径，适用于社交媒体平台、内容创作工具及企业级图像生成服务。其上下文感知的安全理念可推广至其他需要平衡用户定制自由与内容安全的生成式AI应用场景。
