Match4Annotate: Propagating Sparse Video Annotations via Implicit Neural Feature Matching

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了Match4Annotate，一种轻量级框架，用于实现视频内和视频间的点标注与掩膜标注传播。该方法通过在测试时拟合基于SIREN的隐式神经表示到DINOv3特征，生成连续高分辨率的时空特征场，并学习平滑的隐式形变场来引导帧间对应匹配。在三个具有挑战性的临床超声数据集上的评估表明，该方法在跨视频传播方面达到了最先进的性能，同时在视频内传播方面与专用跟踪器保持竞争力。

方法概述 / Method

Match4Annotate采用测试时优化的隐式神经特征匹配策略：首先利用SIREN（正弦激活的隐式神经网络）构建连续的时空特征场，然后学习帧对之间的隐式形变场来实现平滑的对应匹配。该方法统一支持点和掩膜两种标注类型，无需逐视频初始化即可实现跨视频泛化。

实验结果 / Results

在三个临床超声数据集上的实验表明，Match4Annotate在跨视频标注传播任务上显著优于特征匹配和单次分割基线方法，同时与专用视频跟踪器在视频内传播任务上表现相当。这验证了轻量级测试时优化特征匹配管道在可扩展标注工作流中的有效性。

应用价值 / Applications

该研究主要面向医学影像等专家标注成本高昂的专业领域，可大幅减少逐帧视频标注所需的人力投入。其跨视频泛化能力使其能够利用已有标注视频辅助新视频的标注，为构建大规模医学影像数据集提供高效、低成本的解决方案，具有广泛的临床研究和AI辅助诊断应用前景。
