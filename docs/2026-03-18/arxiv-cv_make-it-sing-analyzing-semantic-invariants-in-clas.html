Make it SING: Analyzing Semantic Invariants in Classifiers

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了SING（Semantic Interpretation of the Null-space Geometry）方法，用于分析和解释分类器中的语义不变量。这些不变量存在于分类器的零空间中，导致不同的输入映射到相同的输出，但其语义内容此前难以被人类理解。
SING通过将网络特征映射到多模态视觉语言模型，生成等价图像并为变化分配语义解释，从而揭示了不同模型（如ResNet50和DinoViT）在保持类别语义方面的差异。

方法概述 / Method

SING的核心方法是建立从网络特征到多模态视觉语言模型的映射，从而获取自然语言描述和视觉示例来描述零空间诱导的语义偏移。该方法可应用于单张图像以发现局部不变量，也可应用于图像集合以进行类别和模型层面的统计分析。

实验结果 / Results

研究发现ResNet50会将相关语义属性泄漏到零空间中，而使用自监督DINO预训练的DinoViT在不变量空间中保持类别语义方面表现更优。这些发现通过SING方法生成的语义解释和等价图像得到了验证。

应用价值 / Applications

SING可用于深入理解神经网络的行为特性，帮助评估和改进模型的鲁棒性与可解释性。该方法在模型诊断、安全关键应用中的漏洞分析，以及指导更具语义一致性的模型设计方面具有重要价值。
