GNNVerifier: Graph-based Verifier for LLM Task Planning

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了GNNVerifier，一种基于图神经网络的验证器，用于检测和纠正LLM生成的任务规划中的结构性错误。与现有的基于LLM的验证器不同，该方法将计划表示为带属性的有向图，利用GNN进行结构评估和诊断，生成图级可信度分数以及节点/边级风险分数。实验表明，该方法在多种数据集、主干LLM和规划器上均显著提升了计划质量。

方法概述 / Method

该方法包含四个核心组件：首先将计划编码为节点表示子任务、边表示执行顺序和依赖关系的有向属性图；然后使用GNN执行结构评估，输出图级可接受性分数及细粒度的节点/边风险分数；接着通过从真实计划图构建可控扰动来自动生成带精细标注的训练数据；最后基于GNN反馈引导LLM进行局部编辑以修正缺陷计划。

实验结果 / Results

实验在多个不同数据集、主干LLM和规划器设置下进行，结果表明GNNVerifier在提升计划质量方面取得了显著增益。该方法能够有效检测由结构关系导致的失败，如类型不匹配、中间步骤缺失或依赖关系断裂等问题。

应用价值 / Applications

该研究可应用于自主智能体的任务规划系统，提高LLM生成计划的可靠性和鲁棒性。通过提供可解释的结构化诊断反馈，GNNVerifier有助于构建更可信的自动化决策系统，适用于需要多步骤推理和工具调用的复杂任务场景，如机器人控制、智能助手和科学计算工作流。
