OrthoAI v2: From Single-Agent Segmentation to Dual-Agent Treatment Planning for Clear Aligners

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了OrthoAI v2，这是一个开源的AI辅助隐形正畸治疗规划系统的第二代版本，将原有的单智能体框架扩展为双智能体架构。相较于第一版仅能实现牙齿分割和质心提取的局限，v2版通过引入基于条件热图回归（CHARM）的第二智能体实现直接、无需分割的牙齿标志点检测，并采用六类生物力学综合评分模型替代原有的二分类质量评估，同时新增了基于SLERP插值的多帧治疗模拟器生成时间连贯的6自由度牙齿轨迹。在200例拥挤场景的合成基准测试中，并行集成模式的规划质量评分达到92.8±4.1，较v1版的76.4±8.3提升了21%，且保持了纯CPU部署能力（4.2±0.8秒）。

方法概述 / Method

OrthoAI v2采用双智能体协作架构：第一智能体沿用DGCNN进行牙齿分割，第二智能体基于CHARM方法直接回归牙齿标志点热图，两者通过置信度加权协调器以并行、串行或单智能体三种模式融合；治疗模拟采用SLERP球面线性插值生成F=A×r帧 temporally coherent 的6自由度牙齿运动轨迹，并整合循证分期规则实现ClinCheck 4D可视化。

实验结果 / Results

在合成拥挤场景基准测试中，并行集成模式取得92.8±4.1的规划质量评分，显著优于v1的76.4±8.3（相对提升21%）；系统在标准CPU上运行时间为4.2±0.8秒，保持了完全的可部署性；六类生物力学评分模型（生物力学30%、分期20%、附件15%、邻面去釉10%、咬合10%、可预测性15%）提供了比v1二分类更精细的治疗质量评估。

应用价值 / Applications

该系统可直接集成于隐形正畸临床工作流，为正畸医师提供AI辅助的数字化治疗规划（DTP）工具，实现从分割到标志点检测、质量评估和4D可视化模拟的全流程自动化；开源特性和CPU部署能力使其适用于资源受限的诊所环境，有望降低正畸治疗规划的技术门槛和成本，推动个性化精准正畸的普及。
