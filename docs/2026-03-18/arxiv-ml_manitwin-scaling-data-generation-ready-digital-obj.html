ManiTwin: Scaling Data-Generation-Ready Digital Object Dataset to 100K

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了ManiTwin，一种自动化且高效的流水线，用于生成可直接用于数据生成的数字物体孪生。该方法将单张图像转换为带有语义标注的、可用于仿真的3D资产，并基于此构建了包含10万个高质量标注3D资产的ManiTwin-100K数据集，每个资产配备物理属性、语言描述、功能标注和验证过的操作方案，为可扩展的仿真数据合成和策略学习奠定了基础。

方法概述 / Method

ManiTwin采用自动化流水线实现从单张RGB图像到完整仿真就绪3D资产的转换，包括几何重建、物理属性估计、语义标注生成等模块。该流水线能够自动为资产附加功能部件标注、语言描述以及可行的机器人操作方案，实现端到端的数字孪生生成。

实验结果 / Results

实验表明ManiTwin流水线在资产合成与标注方面具有高效性，生成的ManiTwin-100K数据集在质量与多样性方面表现优异。该数据集成功支持了机器人操作数据生成、随机场景合成以及视觉问答（VQA）数据生成等多项下游任务验证。

应用价值 / Applications

ManiTwin-100K可广泛应用于机器人模仿学习与强化学习的仿真数据生成、大规模场景随机化训练、以及多模态视觉-语言-动作模型训练。该工作显著降低了高质量3D操作资产的获取门槛，加速了基于仿真的机器人学习研究规模化发展。
