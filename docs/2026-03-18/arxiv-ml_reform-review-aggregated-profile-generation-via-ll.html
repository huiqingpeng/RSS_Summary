ReFORM: Review-aggregated Profile Generation via LLM with Multi-Factor Attention for Restaurant Recommendation

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了ReFORM框架，通过利用大语言模型（LLM）从用户评论中提取多因素偏好信息，解决现有LLM增强推荐系统仅依赖项目标题内部知识、忽视用户决策多因素影响的问题。该框架首先生成因素特定的用户和项目画像，再通过多因素注意力机制突出用户决策中最关键的影响因素。在两个不同规模的餐厅数据集上的实验表明，该方法显著优于现有基线模型，并通过深入分析验证了各模块的有效性和个性化来源。

方法概述 / Method

ReFORM框架采用两阶段方法：首先利用LLM从评论文本中生成因素特定的用户画像（用户对项目的偏好）和项目画像（用户对项目的评价），将非结构化的评论信息转化为结构化的多因素表示；随后设计多因素注意力模块，动态识别并加权每个用户决策过程中最具影响力的因素，实现细粒度的个性化建模。

实验结果 / Results

实验在两个餐厅数据集上进行，结果表明ReFORM在推荐准确性和鲁棒性方面均显著优于当前最先进的基线方法；消融实验和深入分析验证了LLM生成画像模块和多因素注意力模块各自的有效性，并揭示了个性化推荐的关键来源在于对用户决策因素的精准捕捉。

应用价值 / Applications

该研究可直接应用于餐厅推荐平台（如Yelp、大众点评等），通过深度挖掘用户评论中的多维度偏好信息，提供更符合用户真实决策需求的个性化推荐；同时，该框架具有通用性，可扩展至其他依赖用户评论的推荐场景（如酒店、电影、电商产品等），提升推荐系统的可解释性和用户满意度。
