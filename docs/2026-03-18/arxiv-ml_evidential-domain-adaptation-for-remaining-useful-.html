论文摘要 / Abstract

本文针对剩余使用寿命（RUL）预测中目标域缺乏标注数据且存在不完全退化轨迹（缺少后期退化阶段）的挑战，提出了一种基于证据学习的域自适应方法EviAdapt。该方法通过基于退化速率的分阶段对齐策略，解决了传统全局对齐导致的源域后期与目标域前期错误匹配问题；同时引入证据不确定性对齐技术，应对相同退化阶段内因工况差异导致的特征分布不一致问题。实验表明，该方法在不完全退化场景下显著优于现有域自适应方法。

方法概述 / Method

EviAdapt采用两阶段技术方案：首先基于退化速率将源域和目标域数据划分为不同的退化阶段，实现阶段级别的精细对齐而非全局对齐；其次利用证据学习（evidential learning）估计各阶段的不确定性分布，并通过不确定性对齐损失函数匹配对应阶段的不确定性特征，从而捕捉工况差异带来的分布变化。

实验结果 / Results

（注：论文摘要中未提供具体实验结果数据，需查看完整论文获取）基于方法描述推断，该研究应在包含不完全退化轨迹的RUL预测数据集上验证，预期结果显示EviAdapt在目标域缺少后期退化数据的场景下，相比传统全局对齐DA方法和标准证据学习方法具有更低的预测误差和更好的泛化性能。

应用价值 / Applications

该研究适用于工业设备健康管理中常见的新设备或早期运行设备RUL预测场景，这些设备往往缺乏完整的退化历史数据。
EviAdapt能够有效迁移已有设备的退化知识，解决实际工程中目标域数据不完整导致的预测难题，提升预测性维护系统的可靠性和实用性。
