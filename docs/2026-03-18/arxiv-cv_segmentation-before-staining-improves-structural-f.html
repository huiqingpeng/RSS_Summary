Segmentation-before-Staining Improves Structural Fidelity in Virtual IHC-to-Multiplex IF Translation

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对虚拟染色技术中核形态结构保真度不足的问题，提出了一种无需监督、与架构无关的条件策略。该方法利用预训练细胞核分割基础模型生成连续细胞概率图作为显式输入先验，并引入方差保持正则化项以维持合成荧光通道中的细胞级异质性。在多种生成架构和两个独立数据集上的对照实验表明，该方法能显著提升核计数保真度和感知质量，且无需针对特定任务调优。

方法概述 / Method

研究核心是将"分割先于染色"（Segmentation-before-Staining）范式引入虚拟IHC-to-mIF转换：首先通过预训练的基础分割模型提取连续细胞概率图（soft prior），替代二值化掩膜以保留梯度级边界信息；同时设计方差保持正则化项匹配局部强度统计，确保合成图像中细胞异质性的真实呈现。该条件策略可即插即用地集成至Pix2Pix、确定性回归U-Net和条件扩散模型等多种生成架构。

实验结果 / Results

在U-Net和ResNet生成器的Pix2Pix、确定性回归U-Net以及条件扩散模型上的控制实验显示，仅添加所提修改即可在核计数保真度和感知质量指标上获得一致提升。连续概率先验相比二值掩膜提供了更丰富的结构约束信号，有效减少了核数量、形状和空间排列的细微失真——这些失真在Ki67增殖指数等临床量化终点中可能导致治疗风险分类的偏移。

应用价值 / Applications

该研究为计算病理学中的虚拟多重免疫荧光合成提供了临床可落地的技术方案，使常规临床广泛可及的明场IHC图像能够转化为具有单细胞精度的mIF表型信息。通过确保核形态结构的保真度，该方法可支撑Ki67增殖指数等关键诊断指标的可靠量化，降低对昂贵试剂、多轮染色流程和专业成像平台的依赖，加速mIF技术的临床普及。
