Stochastic Resetting Accelerates Policy Convergence in Reinforcement Learning

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文研究了随机重置（stochastic resetting）与强化学习的相互作用，发现随机重置能够加速策略收敛，即使它并未减少纯扩散智能体的搜索时间，揭示了一种超越经典首通优化的新机制。在连续控制任务中，随机重置在探索困难、奖励稀疏的情况下能改善深度强化学习性能。与折扣因子不同，重置在保留最优策略的同时，通过截断冗长无信息的轨迹来增强价值传播，从而加速收敛。

方法概述 / Method

研究者在表格型网格环境和连续控制任务中系统评估了随机重置的效果：前者用于理论分析重置与策略学习的交互机制，后者采用基于神经网络值函数近似的深度强化学习设置。通过对比实验，分析了重置与折扣因子的区别，并探究了重置频率对学习动态的影响。

实验结果 / Results

实验表明，随机重置在表格环境中加速策略收敛，且该效应独立于经典的首通时间优化；在连续控制任务中，随机重置在探索困难和稀疏奖励场景下显著提升了深度强化学习性能。重置通过截断长期无信息轨迹、促进价值传播来实现加速，同时不损害最优策略的保持。

应用价值 / Applications

该研究为强化学习提供了一种简单且可调的加速机制，可将统计物理中的经典现象转化为优化原理，特别适用于奖励稀疏、探索困难的复杂决策任务，如机器人控制、导航和游戏AI等领域，有望提升样本效率和训练稳定性。
