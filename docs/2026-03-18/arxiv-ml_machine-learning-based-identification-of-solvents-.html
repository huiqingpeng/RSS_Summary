Machine Learning Based Identification of Solvents from Post-Desiccation Patterns

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本研究提出了一种基于人工神经网络的优化协议，用于从淀粉-液体浆料干燥后的裂纹图案中识别已完全蒸发的溶剂。研究团队通过图像分析技术表征了单组分溶剂（水、乙醇、丙酮）和双组分溶剂（不同浓度的水-乙醇混合物）干燥后形成的图案，并基于九种形态学特征（包括尺寸、形状、几何特征和取向有序性）生成频率直方图作为神经网络的输入数据。实验结果表明，该方法的平均识别准确率达到96%（±1%），其中包含裂纹面积分布的特征组合表现最佳。

方法概述 / Method

研究者采用图像分析技术提取干燥裂纹图案的形态学特征，包括尺寸、形状、几何参数和取向有序性等九个维度，并生成相应的频率直方图。这些直方图被用作不同变体人工神经网络的输入数据，通过系统比较确定最优的特征组合以实现溶剂识别。

实验结果 / Results

该神经网络协议对所有溶剂的综合识别准确率达到96%（±1%）。研究发现，包含裂纹面积分布的特征组合能够获得最高的识别准确率，表明裂纹面积是区分不同溶剂的关键形态学指标。

应用价值 / Applications

该优化协议可推广应用于其他科学和工程领域的图案识别任务，帮助确定最优特征组合。具体应用包括材料科学中的干燥过程监控、法医学中的残留物分析、食品工业中的质量控制，以及任何需要通过表面裂纹图案反推原始液体成分的场景。
