PhysGM: Large Physical Gaussian Model for Feed-Forward 4D Synthesis

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了PhysGM，一种前馈式框架，能够从单张图像联合预测3D高斯表示和物理属性，实现即时物理模拟与高保真4D渲染。该方法通过预训练物理感知重建模型直接推断高斯和物理参数，并采用直接偏好优化（DPO）对模型进行微调，使其与物理合理的参考视频对齐。实验表明，PhysGM可在1分钟内从单张图像生成高保真4D模拟，相比先前方法显著加速，同时提供了包含50K+ 3D资产及物理属性标注的PhysAssets数据集。

方法概述 / Method

PhysGM采用两阶段训练策略：首先预训练一个物理感知重建模型，直接从单张图像同时预测3D高斯参数（位置、颜色、不透明度等）和物理属性（质量、弹性、摩擦系数等）；随后使用直接偏好优化（DPO）对模型进行微调，利用PhysAssets数据集中的物理参考视频作为偏好信号，避免传统Score Distillation Sampling（SDS）优化的高成本和不稳定性。

实验结果 / Results

PhysGM实现了从单张图像到4D物理模拟的前馈推理，整个过程仅需约1分钟，相比依赖逐场景优化的传统方法大幅提速；在渲染质量和物理合理性方面，该方法能够生成高保真、符合物理规律动态效果的新视角合成视频，性能显著优于现有基线方法。

应用价值 / Applications

PhysGM可广泛应用于游戏开发、电影特效、虚拟现实和增强现实中的实时物理内容生成，使创作者能够快速从单张图像生成可交互的动态3D场景；该研究还为机器人仿真、物理教育可视化等领域提供了高效的4D内容合成工具，降低了物理驱动内容创作的技术门槛和时间成本。
