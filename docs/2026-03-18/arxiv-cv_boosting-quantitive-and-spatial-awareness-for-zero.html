Boosting Quantitive and Spatial Awareness for Zero-Shot Object Counting

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对零样本目标计数（ZSOC）任务提出QICA框架，解决了现有方法将计数视为粗粒度检索任务所导致的数量感知不足问题，以及模型微调过程中特征空间扭曲引发的空间不敏感性和泛化能力下降问题。核心创新包括协同提示策略（SPS）、代价聚合解码器（CAD）和多级数量对齐损失（ℒ_MQA），分别用于增强数量感知、保持空间一致性和强化数值约束。实验表明，该方法在FSC-147数据集上取得竞争性性能，并在CARPK和ShanghaiTech-A的零样本跨域评估中展现出优越的泛化能力。

方法概述 / Method

QICA框架采用三大核心组件：协同提示策略（SPS）通过数值条件化提示自适应调整视觉-语言编码器，弥合语义识别与定量推理之间的鸿沟；代价聚合解码器（CAD）直接在视觉-文本相似度图上操作，通过空间聚合细化特征图以避免特征扭曲；多级数量对齐损失（ℒ_MQA）则在网络全流程强制实施数值一致性约束。

实验结果 / Results

在FSC-147基准数据集上，QICA取得了与现有方法竞争的性能表现；更具说服力的是，在CARPK（车载场景）和ShanghaiTech-A（人群场景）的零样本跨域评估中，该方法展现出显著优于现有方法的泛化能力，验证了其在未见领域上的鲁棒迁移特性。

应用价值 / Applications

该研究可广泛应用于需要灵活目标计数的实际场景，如智能交通中的车辆统计、公共安全领域的人群密度监控、以及农业/工业中的自动化盘点等，无需针对特定类别收集训练样本即可实现即开即用的计数功能，大幅降低部署成本并提升系统适应性。
