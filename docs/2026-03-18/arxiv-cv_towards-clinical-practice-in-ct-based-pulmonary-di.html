Towards Clinical Practice in CT-Based Pulmonary Disease Screening: An Efficient and Reliable Framework

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了一个高效可靠的CT肺部疾病筛查框架（ERF），旨在解决深度学习模型处理完整3D CT体积时计算成本过高的问题。该框架包含两个核心创新：基于聚类的子采样方法（CSS）用于高效选择具有代表性和多样性的CT切片子集，以及模糊感知不确定性量化机制（AUQ）用于识别因细微病变和伪影导致的数据模糊性。在包含2,654个CT体积的两个公开数据集上验证，ERF在保持与全体积分析相当诊断性能（超过90%的准确率和召回率）的同时，将处理时间减少超过60%。

方法概述 / Method

CSS方法通过结合高效的k近邻搜索与迭代优化过程，在保持诊断特征完整性的前提下选择紧凑而全面的切片子集，避免了传统子采样技术引入伪影或丢弃关键信息的问题。
AUQ机制利用辅助分类器之间的预测差异构建专门的模糊性评分，通过在训练期间最大化该差异来有效标记模型因视觉噪声或复杂病理而缺乏信心的样本。

实验结果 / Results

ERF在3种肺部疾病的诊断任务上实现了与全体积分析相当的性能，准确率和召回率均超过90%，同时处理时间缩短超过60%。该框架在2,654个CT体积的两个公开数据集上得到验证，证明了其在计算效率和诊断可靠性之间的有效平衡。

应用价值 / Applications

该研究为时间敏感的临床环境中部署快速、准确且可信赖的AI辅助筛查工具提供了重要进展，有助于减轻放射科医生的巨大工作负担。
ERF框架的高效性和可靠性使其特别适用于需要实时或近实时处理的大规模肺部疾病筛查项目，推动深度学习CT分析从研究走向广泛的临床实践。
