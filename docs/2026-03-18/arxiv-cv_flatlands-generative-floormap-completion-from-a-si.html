FlatLands: Generative Floormap Completion From a Single Egocentric View

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了FlatLands，一个用于单视角鸟瞰图（BEV）楼层补全的数据集和基准测试平台。该数据集包含来自17,656个真实室内场景的270,575个观测样本，涵盖对齐的观测图、可见性图、有效性图和真值BEV地图。研究比较了无训练方法、确定性模型、集成模型和随机生成模型，并将该任务实现为端到端的单目RGB到楼层地图的完整流程，为具身导航中的不确定性感知室内建图和生成式补全提供了严格的测试平台。

方法概述 / Method

本研究构建了大规模真实室内场景数据集，并设计了多种方法对比框架，包括训练无关方法、确定性神经网络、模型集成以及随机生成模型。研究最终实现了一个端到端的单目RGB图像到楼层地图的生成式补全流程。

实验结果 / Results

论文通过分布内（in-distribution）和分布外（out-of-distribution）两种评估协议对各类方法进行了系统比较。实验结果表明，随机生成模型在不确定性建模方面具有优势，能够为下游导航任务提供更可靠的地图预测。

应用价值 / Applications

该研究可直接应用于室内机器人导航和增强现实导航系统，通过单张自中心图像生成完整的可通行区域地图。其不确定性感知特性对于安全关键型的具身智能应用（如自动驾驶轮椅、服务机器人）具有重要价值，同时也可扩展到智能家居和建筑信息建模等领域。
