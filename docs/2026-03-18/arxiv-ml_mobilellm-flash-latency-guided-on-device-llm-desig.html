MobileLLM-Flash: Latency-Guided On-Device LLM Design for Industry Scale

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了MobileLLM-Flash，一种面向工业级规模的端侧大语言模型设计方法，通过硬件在环的架构搜索在移动延迟约束下联合优化模型架构和注意力模式。该方法采用注意力跳过机制替代复杂的专用注意力机制，实现了无需自定义内核、兼容标准移动运行时的模型部署。最终得到的350M至1.4B参数模型家族在移动CPU上实现了最高1.8倍的预填充加速和1.6倍的解码加速，同时保持或提升了模型质量。

方法概述 / Method

本研究采用硬件在环的架构搜索方法，将每个候选模型视为预训练骨干网络的剪枝版本以继承权重，从而仅需少量持续预训练即可获得高精度。研究采用分阶段流程：首先学习准确的延迟预测模型，然后搜索延迟与质量的帕累托前沿，联合优化层数、维度等架构参数和注意力模式。

实验结果 / Results

MobileLLM-Flash模型家族（350M/650M/1.4B参数）支持最长8K上下文长度，在移动CPU上相比基线实现1.8倍预填充加速和1.6倍解码加速，同时保持可比或更优的模型质量。通过对帕累托前沿设计选择的分析，研究提炼出可指导端侧大语言模型设计的实用原则。

应用价值 / Applications

该研究为实时AI体验提供了可工业部署的端侧大语言模型解决方案，模型无需自定义内核即可在标准移动运行时（如ExecuTorch）上部署，具有广泛的硬件兼容性。这种方法特别适用于需要近实时响应的移动应用场景，能够最大化用户覆盖范围，为移动设备上的智能助手、实时翻译等应用提供高效基础模型支持。
