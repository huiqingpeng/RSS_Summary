CloSE: A Geometric Shape-Agnostic Cloth State Representation

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了一种新颖的布料变形状态表示方法CloSE（Cloth StatE），用于解决布料操作中的非刚性变形表示难题。该方法首先基于布料边缘段的拓扑指数构建dGLI圆盘表示，然后通过提取跨形状、尺寸和方向一致的关键特征，将其抽象为紧凑连续的圆形表示。实验表明，CloSE能够准确预测多种模拟服装数据集的折叠位置，并在语义标注和高低层规划任务中展现出强大性能。

方法概述 / Method

论文提出两阶段表示方法：首先计算布料边缘段的拓扑指数（dGLI），并将其排列在圆形网格上形成dGLI圆盘热图；随后从该热图中提取与形状无关的判别性特征，压缩为低维的CloSE圆形向量表示。该表示具有几何形状无关性、连续性和紧凑性，可统一描述不同形状布料的变形状态。

实验结果 / Results

CloSE在多个模拟服装数据集上实现了准确的折叠位置预测，验证了其对不同形状、尺寸和方向布料的泛化能力。在语义标注任务中，该表示能有效区分布料的不同折叠状态；在规划任务中，CloSE同时支持高层任务规划（如目标状态设定）和低层动作规划（如抓取点选择），展现出多尺度适用性。

应用价值 / Applications

CloSE可广泛应用于机器人布料操作领域，包括家用服务机器人（如叠衣、整理）、服装制造业自动化（如质检、折叠包装）以及虚拟试衣等场景。其形状无关特性使其能够泛化到训练时未见过的新颖布料形状，降低了实际部署时的数据收集成本，为开发通用布料操作机器人提供了重要的状态感知基础。
