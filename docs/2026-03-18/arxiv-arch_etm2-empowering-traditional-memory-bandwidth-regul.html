论文摘要 / Abstract

本文提出ETM2，一种利用Arm CoreSight架构中嵌入式跟踪宏单元（ETM）实现的新型硬件辅助内存带宽调节器。该方案为实时多核系统提供了一种可移植且有效的内存干扰缓解方案，仅需极少软件干预，兼具MemPol的微秒级精细分辨率和MemGuard等中断方案的可移植性与快速响应特性。在多款64位Arm平台上的评估表明，ETM2能有效执行每核心内存带宽调节，并解锁了此前MemGuard和MemPol无法实现的新型调节选项。

方法概述 / Method

ETM2重新利用ETM的硬件跟踪功能来监控和调节内存访问行为，通过硬件级机制实现带宽限制而无需频繁软件介入。该方法在MemPol的细粒度控制与MemGuard的可移植性之间取得平衡，形成混合式调节架构。

实验结果 / Results

研究在大量64位Arm开发板上验证了方案的可移植性，并在AMD Zynq UltraScale+平台上使用San Diego Vision Benchmark Suite进行对比评估。结果表明ETM2具有良好的可扩展性，能够有效执行每核心内存带宽调节，同时揭示了该方法所带来的设计权衡。
