论文摘要 / Abstract

本文研究了动态系统的轨迹推断问题，即如何在观测到的时间点之间插值路径。针对现有方法仅使用连续几何先验（黎曼度量）的局限，作者提出结合离散有向先验知识（如发育生物学中的谱系树），引入Finsler度量将几何与分类相结合，在合成数据和真实数据的插值任务中取得了更好的性能。

方法概述 / Method

论文采用Finsler几何框架，构建了一种能够同时编码连续空间特征和离散方向性约束的度量方式。该方法将数据相关的空间特征（连续先验）与谱系等离散有向转移信息（离散先验）统一整合到轨迹推断中，突破了传统黎曼几何仅处理连续度量的限制。

实验结果 / Results

实验表明，所提出的Finsler度量方法在合成数据和真实世界数据的插值任务上均优于仅使用连续几何先验的基线方法，验证了结合离散有向先验知识能够有效提升轨迹推断的准确性和合理性。

应用价值 / Applications

该方法可广泛应用于发育生物学中的细胞谱系追踪、时间序列数据分析以及动态系统建模等领域，特别适用于存在已知离散转移结构（如分化路径、状态转换规则）的复杂生物系统，为理解系统动态机制提供更精确的推断工具。
