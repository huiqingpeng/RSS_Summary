Long-Horizon Traffic Forecasting via Incident-Aware Conformal Spatio-Temporal Transformers

论文摘要 / Abstract

本研究针对长时程交通预测中网络条件随机性、事故中断间歇性以及时空依赖关系随时间变化等挑战，提出了一种基于事故感知共形时空Transformer的预测框架。该研究利用俄亥俄州交通部的交通流量数据与事故记录，通过自适应共形预测（ACP）生成具有校准不确定性的多步预测。论文创新性地提出分段变异系数策略，采用对数正态分布建模小时级行程时间变异性，并结合事故严重度信号（包括清障时间、天气、超速、施工区等因素）动态构建邻接矩阵，显著提升了长时程预测精度与预测区间的可靠性。

方法概述 / Method

论文采用时空Transformer（STT）作为核心架构，集成自适应共形预测（ACP）框架以量化预测不确定性。方法核心在于动态图构建：首先使用分段变异系数（CV）策略和对数正态分布刻画小时级行程时间波动，生成逐小时动态邻接矩阵；随后引入事故严重度信号（清障时间、天气条件、超速情况、施工区及道路功能等级）对边权重进行扰动，以捕捉局部中断和峰谷过渡特征。验证阶段通过SUMO仿真在哥伦布市网络进行多小时循环运行，结合蒙特卡洛模拟获取被测车辆的行程时间分布。

实验结果 / Results

实验表明，该方法相较于基线模型在长时程预测精度上实现显著提升，同时生成良好校准的预测区间。通过SUMO仿真的扩展行程测试与蒙特卡洛模拟验证，模型能够有效捕捉事故引发的局部交通扰动及昼夜模式转换，在不确定量化方面展现出优越性能。

应用价值 / Applications

该研究为智能交通系统提供了高可靠性的长时程预测工具，可支持路径规划、出行时间预估及交通管理中心的事故响应决策。动态不确定量化能力使其特别适用于自动驾驶车辆的预期功能安全（SOTIF）验证，以及需要在复杂随机环境下进行风险评估的交通运营优化场景。
