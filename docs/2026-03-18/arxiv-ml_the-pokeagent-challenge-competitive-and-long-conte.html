The PokeAgent Challenge: Competitive and Long-Context Learning at Scale

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了PokeAgent Challenge，一个基于宝可梦多智能体对战系统和角色扮演游戏环境的大规模决策研究基准。该基准通过两个互补赛道解决部分可观测性、博弈论推理和长程规划等前沿AI难题：对战赛道要求在部分可观测条件下进行战略推理和泛化，速通赛道则需要长程规划和序列决策能力。研究发现通用模型（LLM）、专用模型（RL）与精英人类表现之间存在显著差距，且宝可梦对战能力与标准LLM基准几乎正交，为RL和LLM研究提供了未解决的重要挑战。

方法概述 / Method

论文构建了两个标准化评估框架：对战赛道提供2000万+对战轨迹数据集及启发式、强化学习和基于LLM的基线模型；速通赛道则开发了首个RPG速通标准化评估框架，包括开源多智能体编排系统以支持基于harness的LLM方法的模块化、可复现比较。两个赛道均支持自动化评估和实时排行榜系统。

实验结果 / Results

NeurIPS 2025竞赛吸引了100+团队参与，验证了资源质量和研究社区的兴趣。参与者提交方案与基线分析显示，通用LLM、专用RL与顶尖人类玩家之间存在明显性能鸿沟。
BenchPress评估矩阵分析表明，宝可梦对战能力与传统LLM基准测试几乎无关，测量了现有测试套件未能捕捉的能力维度。

应用价值 / Applications

该基准为决策智能研究提供了贴近真实场景的测试平台，可推动强化学习和大语言模型在部分可观测博弈、长程规划等关键方向的发展。其开源框架和实时排行榜系统支持持续迭代研究，适用于通用人工智能能力评估、多智能体系统开发及复杂策略学习环境的研究与教育场景。
