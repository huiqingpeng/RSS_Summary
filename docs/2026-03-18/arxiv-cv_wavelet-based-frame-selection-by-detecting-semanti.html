论文摘要 / Abstract

本文提出了一种基于小波变换的语义边界检测帧选择框架（WFS-SB），用于解决长视频理解中的帧冗余问题。与现有仅关注查询相关性的方法不同，WFS-SB强调捕捉视频叙事结构中的语义转变时刻。该方法利用小波变换的多分辨率分析能力，从嘈杂的查询-帧相似度信号中提取干净的语义变化信号，识别语义边界将视频分割为连贯片段，并采用两阶段策略进行自适应帧预算分配和多样化帧选择。实验表明，WFS-SB在VideoMME、MLVU和LongVideoBench等基准上分别提升5.5%、9.5%和6.2%的准确率，显著优于现有方法。

方法概述 / Method

WFS-SB是一个无需训练的框架，核心创新在于使用小波变换对查询-帧相似度信号进行多尺度分解，从最低频尺度提取语义变化信号并检测其局部极值作为语义边界，从而将视频分割为叙事连贯的片段。在此基础上，框架采用两阶段策略：首先基于复合重要性分数为每个片段自适应分配帧预算，然后在各片段内应用最大边际相关（MMR）方法选择既相关又多样的帧集合。

实验结果 / Results

WFS-SB在多个长视频理解基准上取得显著性能提升：VideoMME上准确率提高5.5%，MLVU上提高9.5%，LongVideoBench上提高6.2%。该方法 consistently 超越现有最先进方法，验证了基于语义边界的帧选择策略相比传统仅依赖查询相关性的方法具有明显优势。

应用价值 / Applications

该研究可直接应用于需要处理长视频内容的大型视觉-语言模型系统，如视频问答、视频摘要和视频检索等场景。通过保留视频叙事结构的完整性，WFS-SB有助于提升模型对复杂长视频内容的理解能力，对于教育视频分析、影视内容理解、监控视频智能分析等实际应用具有重要价值。
