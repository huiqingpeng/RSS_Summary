Analyzing Error Sources in Global Feature Effect Estimation

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究系统分析了全局特征效应估计（如偏依赖PD和累积局部效应ALE）中的误差来源，填补了该领域的关键空白。作者推导了均方误差分解框架，将误差区分为模型偏差、估计偏差、模型方差和估计方差四类，并揭示了训练数据与留出数据选择对估计可靠性的影响。研究发现，尽管留出数据理论上更纯净，但训练数据引入的潜在偏差在实际中可忽略，且更大的样本量优势占主导；交叉验证估计能有效降低模型方差，尤其适用于过拟合模型。

方法概述 / Method

论文采用理论推导与大规模仿真实验相结合的方法：首先建立PD和ALE估计量的均方误差分解数学框架，明确分离四类误差来源及其与模型特性、数据选择和样本量的依赖关系；随后设计涵盖多种数据生成过程、学习器、估计策略（训练数据、验证数据、交叉验证）及样本量的综合模拟研究，以验证理论发现。

实验结果 / Results

实验结果表明：训练数据与留出数据的选择对估计偏差影响微小，样本量的增加带来的收益远超潜在偏差风险；估计方差受特征交互作用和样本量共同影响，ALE对样本量尤为敏感；交叉验证估计策略显著降低模型方差成分，在过拟合模型上效果突出；整体而言，估计方差通常是主导误差来源，而非模型偏差或估计偏差。

应用价值 / Applications

该研究为机器学习模型解释性分析提供了选择估计策略的实用指南：当样本量充足时优先使用训练数据以最大化统计效率，在模型过拟合时采用交叉验证以稳定估计，同时注意ALE在小样本下的高方差风险。这些结论可直接指导金融、医疗、自动驾驶等领域中黑箱模型（如深度学习、梯度提升树）的可解释性实践，提升模型决策的透明度和可信度。
