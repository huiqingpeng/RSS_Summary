pADAM: A Plug-and-Play All-in-One Diffusion Architecture for Multi-Physics Learning

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了pADAM，一种统一的生成式框架，能够学习跨异构偏微分方程（PDE）族的共享概率先验分布。该模型通过学习系统状态和物理参数的联合分布，在单一架构内同时支持正向预测和逆向推断任务，无需重新训练。实验表明，pADAM在从标量扩散到非线性Navier-Stokes方程等多种物理基准上均能实现准确推断，即使在稀疏观测条件下也能结合保形预测提供可靠的、具有覆盖保证的不确定性量化，并能仅从两个稀疏快照中识别控制方程，实现概率模型选择。

方法概述 / Method

pADAM采用基于扩散模型的生成架构，将多物理问题统一建模为系统状态与物理参数的联合概率分布学习问题。该架构通过条件扩散过程实现"即插即用"的灵活推理，可根据不同任务需求（正向模拟、参数反演、模型选择）调整条件输入，而无需修改网络结构或重新训练模型。

实验结果 / Results

pADAM在涵盖标量扩散、对流-扩散、反应-扩散及Navier-Stokes方程的多物理基准测试中表现出色，在稀疏观测场景下仍保持较高的推断精度。结合保形预测方法后，模型能够提供具有理论覆盖保证的不确定性区间。尤为突出的是，该模型仅凭两个时间稀疏快照即可有效识别未知系统的控制物理定律，实现概率性模型选择。

应用价值 / Applications

pADAM为科学计算和工程领域提供了统一的智能推断平台，特别适用于观测数据有限、物理机制不明确或需要同时处理多类PDE的复杂场景，如气候模拟、流体力学反问题和实验设计优化。其内置的不确定性量化和自动模型选择能力，可显著提升科学发现的可信度和效率，降低对专家先验知识的依赖。
