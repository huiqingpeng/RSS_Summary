论文摘要 / Abstract

本文针对函数参数化神经网络架构（如Bag-of-Functions框架）的参数初始化敏感性问题，提出了一种先验信息引导的初始化策略。该方法利用快速傅里叶变换提取数据的主导季节性频谱特征，并结合基于残差的回归方法参数化趋势分量，从而在保持重建保真度的同时显著降低编码器维度。实验表明，嵌入数据驱动先验能加速收敛、减少跨试验性能波动并提升计算效率，实现了更紧凑且可解释的架构设计。

方法概述 / Method

论文采用频谱分析驱动的初始化方法：首先通过快速傅里叶变换（FFT）从数据中自动提取主导季节性周期，以此确定网络深度和初始状态；其次使用残差回归对趋势分量进行参数化。该策略将数据的内在时频结构显式嵌入网络架构，替代传统的数据无关初始化方案，并提供有限样本下的趋势估计理论分析。

实验结果 / Results

在合成数据与真实基准数据集上的广泛实验表明，该方法显著加速了模型收敛速度，降低了不同随机试验间的性能方差，并实现了计算效率的提升。关键发现是：通过结构对齐可在不损害重建质量的前提下大幅缩减编码器维度，使模型在保持紧凑性的同时超越标准初始化基线。

应用价值 / Applications

该研究适用于时间序列预测、信号分解等需要可解释函数参数化的场景，特别是在数据具有明显季节性周期和趋势结构的领域（如金融、能源负荷预测、气候建模等）。其价值在于提供了一种无需修改核心训练流程即可提升模型效率与稳定性的实用方案，为构建轻量级、可解释的深度学习系统开辟了新途径。
