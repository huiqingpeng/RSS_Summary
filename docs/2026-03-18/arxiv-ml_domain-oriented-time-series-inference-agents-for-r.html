Domain-Oriented Time Series Inference Agents for Reasoning and Automated Analysis

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了 TS-Reasoner，首个面向时序分析领域的专业化智能体，用于执行多步时序推理任务。该研究突破了传统时序方法仅关注孤立任务、以及现有大语言模型时序推理局限于单步推断或纯自然语言输出的局限，通过整合大语言模型推理、领域专用计算工具和错误反馈机制，实现了符号推理与精确数值分析相结合的领域感知分析流程。实验表明，该方法在基础时序概念理解（TimeSeriesExam）和复杂多步推理任务上均显著优于通用大语言模型，展现了领域专业化智能体在自动化时序推理中的巨大潜力。

方法概述 / Method

TS-Reasoner 采用"大语言模型+领域工具+错误反馈"的架构设计：以 LLM 作为推理核心，结合预定义的时序分析工具库（如统计检验、预测模型等）执行精确数值计算，并通过迭代式的错误反馈循环实现自我修正。系统支持约束感知的工作流编排，能够根据任务需求动态组合符号推理与数值分析步骤，完成从问题解析、工具调用到结果验证的完整推理链条。

实验结果 / Results

研究从两个维度评估系统性能：在基础时序理解任务上，TS-Reasoner 在 TimeSeriesExam 基准测试中超越通用 LLM；在复杂多步推理方面，作者构建了新的评测数据集以测试组合推理与计算精度，TS-Reasoner 同样表现更优。实验结果验证了领域专业化设计对于提升时序推理准确性的关键作用，尤其在需要多步骤协同和数值精确性的场景下优势更为明显。

应用价值 / Applications

该研究为金融风控、医疗健康监测、工业设备预测性维护、能源负荷预测等需要复杂时序分析的专业领域提供了自动化解决方案。
TS-Reasoner 能够降低领域专家进行深度时序分析的技术门槛，通过可解释的多步推理过程支持决策制定，并可扩展至其他需要结合定性推理与定量计算的科学研究与商业分析场景。
