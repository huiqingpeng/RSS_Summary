When Stability Fails: Hidden Failure Modes Of LLMS in Data-Constrained Scientific Decision-Making

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究揭示了大型语言模型（LLM）在数据受限的科学决策任务中一个被忽视的关键问题：高稳定性并不保证与统计真实值的一致性。作者开发了一个受控行为评估框架，系统分离了稳定性、正确性、提示敏感性和输出有效性四个维度，发现LLM即使在运行间表现出近乎完美的稳定性，仍可能系统性地偏离统计基准，对微小提示变化过度敏感，甚至生成输入中不存在的虚假基因标识符。

方法概述 / Method

研究采用统计基因优先级排序任务作为测试平台，基于差异表达分析构建实验场景，设计包含严格与宽松显著性阈值、边界排名情境及微小措辞变化的提示体系，对多个主流LLM进行四维行为评估（稳定性、正确性、提示敏感性、输出有效性），通过对比LLM输出与统计真实值来量化各类失效模式。

实验结果 / Results

实验发现LLM存在四种隐性失效模式：（1）高稳定性伴随系统性偏离统计真实值；（2）宽松阈值下过度选择基因；（3）对微小提示措辞变化反应剧烈；（4）生成语法合理但输入表中不存在的虚假基因标识符。这些结果表明稳定性指标单独使用会产生误导性安全信号。

应用价值 / Applications

该研究为生物医学信息学、药物基因组学等依赖LLM辅助决策的科学领域提供了关键警示，强调在自动化或半自动化科学工作流中部署LLM时必须实施显式的基准验证和输出有效性检查，而非仅依赖稳定性指标，对提升AI辅助科学研究的可靠性具有重要指导意义。
