When Should a Robot Think? Resource-Aware Reasoning via Reinforcement Learning for Embodied Robotic Decision-Making

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了RARRL（基于强化学习的资源感知推理）框架，用于解决具身机器人系统中大语言模型（LLM）推理带来的计算延迟和资源开销问题。该框架通过学习高层编排策略，使机器人能够自适应地决定何时进行推理、采用何种推理角色以及分配多少计算预算。实验表明，RARRL在提高任务成功率的同时显著降低了执行延迟，并增强了系统鲁棒性，证明了自适应推理控制对于构建可靠高效的具身智能体至关重要。

方法概述 / Method

RARRL采用分层强化学习方法，在机器人的决策层学习一个高层编排策略，而非低层控制策略。该策略以当前观测、执行历史和剩余资源为输入，输出关于是否调用推理、选择推理角色以及分配计算预算的决策，实现资源感知的自适应推理调度。

实验结果 / Results

研究在ALFRED基准数据集的真实延迟 profile 上进行了广泛评估，结果显示RARRL相比固定或启发式推理策略，能够持续提升任务成功率，同时减少执行延迟并增强系统鲁棒性，有效平衡了推理开销与决策质量之间的权衡。

应用价值 / Applications

该研究适用于需要实时决策的具身机器人场景，如家庭服务机器人、工业自动化和自主导航等，能够在资源受限的嵌入式平台上高效部署LLM驱动的智能体。通过动态控制推理时机和计算预算，RARRL有助于提升机器人在实际应用中的响应速度和可靠性，推动LLM在物理世界中的实用化部署。
