How to Utilize Complementary Vision-Text Information for 2D Structure Understanding

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文揭示了纯文本编码器（线性化表格）与纯视觉编码器在处理2D表格结构时存在高度互补性：文本模态保留精确的单元格内容但丢失空间布局信息，视觉模态捕获空间线索但难以精确识别文字。针对直接融合两种模态易产生跨模态干扰的问题，作者提出轻量级架构DiVA-Former，通过视觉token作为动态查询来蒸馏长文本序列，在13个表格基准测试上相比纯文本基线提升23.9%，并持续优于现有单模态和多模态方法。

方法概述 / Method

DiVA-Former采用视觉token作为动态查询（dynamic queries），对冗长的文本序列进行蒸馏压缩，生成可消化的向量表示，从而实现高效的跨模态信息融合。该架构避免了简单的特征拼接，通过查询驱动的注意力机制减少跨模态干扰，实现互补信息的有效提取。

实验结果 / Results

在涵盖13个表格理解任务的广泛基准测试中，DiVA-Former相比纯文本基线取得23.9%的性能提升，并在使用视觉输入、文本输入或两者结合的现有基线方法上均获得一致改进，验证了其在多模态表格结构理解中的优越性。

应用价值 / Applications

该研究可应用于文档智能分析、财务报表自动化处理、科研文献表格抽取等需要精确理解复杂2D表格结构的场景，为大型语言模型处理结构化视觉文档提供了高效的模态融合方案，有望推动多模态文档理解系统的发展。
