Regularized Latent Dynamics Prediction is a Strong Baseline For Behavioral Foundation Models

论文摘要 / Abstract

本文探讨了行为基础模型（BFMs）中复杂表征学习目标的必要性，发现单纯的自监督潜在空间下一状态预测容易导致状态特征趋同、降低特征跨度。作者提出正则化潜在动态预测（RLDP）方法，通过简单的正交正则化保持特征多样性，在零样本强化学习任务上达到或超越现有复杂方法，且在低数据覆盖场景下表现更优。

方法概述 / Method

RLDP 以自监督下一状态预测为基础学习目标，学习状态特征的潜在动态模型；关键创新在于添加正交正则化项，显式约束特征向量间的正交性，防止特征坍缩并维持特征空间的表达能力。该方法无需复杂的辅助任务或对抗训练，实现简单且计算高效。

实验结果 / Results

实验表明，RLDP 在零样本强化学习任务上匹配或超越了现有最先进的复杂表征学习方法；更重要的是，在数据覆盖不足的低覆盖场景中，先前方法性能显著下降，而 RLDP 仍能保持良好表现，展现出更强的鲁棒性和数据效率。

应用价值 / Applications

RLDP 为构建高效的行为基础模型提供了简化且强大的基线方案，适用于机器人控制、游戏AI等需要快速适应新任务的领域；其低数据依赖特性使其在真实世界数据收集成本高昂的场景（如实体机器人）中具有重要实用价值，同时降低了BFM的训练复杂度和门槛。
