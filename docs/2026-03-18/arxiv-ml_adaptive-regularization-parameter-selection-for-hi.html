论文摘要 / Abstract

本文提出了一种结合Tucker分解的变分贝叶斯新方法，用于高效求解高维逆问题。该方法通过Tucker分解将变分推断从高维空间转换到低维核心张量空间，显著降低计算复杂度。核心创新在于引入逐模态精度参数，实现对各向异性结构的自适应正则化，同时能够从数据中估计噪声水平，无需依赖噪声参数的先验知识。实验表明，该方法在2D去模糊、3D热传导和Fredholm积分方程等任务中，相比L曲线准则、GCV、UPRE和DP等传统方法，在PSNR和SSIM等指标上均有显著提升。

方法概述 / Method

该方法采用Tucker低秩分解作为降维工具，将原始高维问题的变分推断转化为核心张量空间中的低维推断问题。关键创新是引入逐模态（per-mode）精度参数，允许不同维度方向具有不同的正则化强度，从而自适应地处理各向异性结构。此外，方法采用数据驱动的噪声水平估计机制，将噪声参数作为隐变量进行联合推断，避免了传统方法需要预设噪声参数的缺点。

实验结果 / Results

实验在2D图像去模糊、3D热传导和Fredholm积分方程三类逆问题上进行验证。该方法可扩展至110,000个变量的规模，在2D去模糊任务中比现有方法提升0.73-2.09 dB，在3D热传导问题中提升达6.75 dB。定量评估指标（PSNR、SSIM）和定性可视化（误差图、精度参数趋势）均优于L曲线准则、广义交叉验证（GCV）、无偏预测风险估计器（UPRE）和差异原理（DP）等传统基准方法。

应用价值 / Applications

该方法为成像、遥感和科学计算中的大规模逆问题提供了可扩展的实用解决方案，特别适用于具有各向异性结构的场景，如方向性图像去模糊。通过自适应正则化和自动噪声估计，降低了方法对先验知识和人工调参的依赖，有望推动贝叶斯方法在实际高维逆问题中的广泛应用。未来在自动化秩选择和理论保证方面的完善将进一步提升其实用性。
