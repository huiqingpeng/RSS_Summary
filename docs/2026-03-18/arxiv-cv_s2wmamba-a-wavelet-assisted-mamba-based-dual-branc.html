S2WMamba: A Wavelet-Assisted Mamba-Based Dual-Branch Network For Pansharpening

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出S2WMamba，一种用于全色锐化（pansharpening）的小波辅助Mamba双分支网络，通过显式解耦模态特定的频率信息来解决空间细节增强与光谱保真度之间的特征纠缠问题。该方法采用局部2D Haar离散小波变换（DWT）提取PAN图像的空间边缘和纹理，同时引入通道级1D Haar DWT将每个像素的光谱视为1D信号以分离共享光谱基与波段特定变化。在WV3、GF2和QB数据集上，S2WMamba匹配或超越了近期强基线方法，PSNR提升最高达0.23 dB，在WV3全分辨率数据上达到0.956的HQNR指标。

方法概述 / Method

S2WMamba采用双分支架构：光谱分支（Spectral branch）将小波提取的空间细节注入多光谱特征，空间分支（Spatial branch）利用DWT1D过程的光谱信息精炼PAN特征。两个分支通过基于Mamba的交叉调制机制进行信息交换，以线性复杂度显式建模解耦子带间的长程依赖关系，实现高效的频率融合。

实验结果 / Results

在WorldView-3（WV3）、Gaofen-2（GF2）和QuickBird（QB）数据集上，S2WMamba在降低分辨率场景下PSNR提升最高达0.23 dB，在全分辨率WV3数据上达到0.956的HQNR（高质量无参考）指标。大量消融实验验证了模态特定DWT放置策略和并行双分支架构的有效性，证明该方法优于FusionMamba、CANNet、U2Net和PanNet等近期强基线。

应用价值 / Applications

该研究可广泛应用于遥感图像处理领域，如卫星影像增强、环境监测、城市规划、农业评估和灾害响应等需要高分辨率多光谱数据的场景。通过显式解耦空间与光谱信息，S2WMamba为需要严格光谱保真度的定量遥感应用提供了可靠的图像融合解决方案，同时线性复杂度的设计使其适用于大规模遥感数据处理。
