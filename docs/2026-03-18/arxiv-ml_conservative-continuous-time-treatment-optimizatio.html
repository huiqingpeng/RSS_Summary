论文摘要 / Abstract

本文提出了一种保守型连续时间随机控制框架，用于从不规则采样的患者轨迹中优化治疗方案。作者将未知的患者动态建模为受控随机微分方程，并将治疗视为连续时间控制变量。为避免模型误差导致的过度外推问题，研究引入基于签名的最大均值差异（MMD）正则化项来惩罚偏离观测轨迹分布的治疗方案，从而最小化真实成本的可计算上界。实验表明，该方法相比非保守基线具有更强的鲁棒性和性能表现。

方法概述 / Method

该方法将患者动态建模为治疗作为连续时间控制的随机微分方程，并采用签名（signature）变换提取路径的时序特征。核心创新在于引入路径空间上的MMD正则化器，通过比较治疗方案诱导的轨迹分布与观测轨迹分布之间的差异，限制模型在分布外区域的过度外推。

实验结果 / Results

在基准数据集上的实验表明，该保守型优化框架相比非保守基线方法展现出更优的鲁棒性和整体性能。具体而言，通过约束轨迹分布偏离，方法有效避免了因模型误差导致的次优或危险的治疗建议，同时在真实成本优化方面取得更好表现。

应用价值 / Applications

该研究对精准医疗和个性化治疗具有重要价值，特别适用于从稀疏、不规则采样的电子健康记录数据中优化长期治疗方案（如慢性病管理或癌症治疗）。方法可推广至任何需要从有限观测中学习连续时间决策策略的领域，如药物剂量调控、康复训练规划等，为临床决策支持系统提供更可靠的安全保障。
