Deep Adaptive Model-Based Design of Experiments

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了一种深度自适应模型实验设计（Deep Adaptive MBDOE）方法，通过将深度自适应设计（DAD）与可微分机理模型相结合，解决了传统自适应MBDOE需要在每步实验后进行昂贵的后验推断和设计优化、无法实时应用的问题。研究扩展了序列对比训练目标以处理冗余参数，并提出了尊重动态系统时间结构的Transformer策略架构。该方法在四个复杂度递增的系统上进行了验证，包括补料分批生物反应器、Haldane生物反应器、两室药代动力学模型和直流电机实时部署。

方法概述 / Method

该方法将深度自适应设计（DAD）与可微分机理模型相结合，将序列实验设计摊销（amortize）为一个离线训练的神经网络策略。针对具有已知控制方程但参数不确定的动态系统，作者扩展了序列对比训练目标以处理冗余参数（nuisance parameters），并设计了基于Transformer的策略架构，该架构能够尊重动态系统的时间结构特性。

实验结果 / Results

研究在四个复杂度递增的系统上验证了该方法：Monod动力学的补料分批生物反应器、具有不确定底物抑制的Haldane生物反应器、具有冗余清除参数的两室药代动力学模型，以及用于实时部署的直流电机系统。实验结果表明该方法能够在不同复杂度的动态系统中实现高效的参数估计，并支持实时应用场景。

应用价值 / Applications

该研究在生物反应器优化、药代动力学参数估计和实时控制系统等领域具有重要应用价值。通过将昂贵的在线计算转移为离线训练，该方法使模型实验设计能够部署于需要实时决策的场景，如工业过程控制和个性化医疗中的剂量优化，显著提升了自适应实验设计的实用性和可扩展性。
