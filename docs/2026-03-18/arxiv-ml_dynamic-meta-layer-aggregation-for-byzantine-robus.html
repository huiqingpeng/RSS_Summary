论文摘要 / Abstract

本文针对联邦学习（FL）中拜占庭攻击（尤其是多标签翻转和未针对性投毒攻击）的防御问题，提出了FedAOT——一种基于元学习启发的自适应聚合框架。
FedAOT通过动态评估客户端更新的可靠性来加权聚合，无需预设阈值或依赖特定的攻击假设。实验表明，该方法在多种数据集和攻击类型上均表现出良好的泛化能力和鲁棒性，同时保持了计算效率。

方法概述 / Method

FedAOT采用元学习（meta-learning）思想构建自适应聚合框架，核心机制是动态元层（dynamic meta-layer）——该层根据历史更新行为实时评估各客户端的可靠性分数，并据此动态调整其在全局聚合中的权重。该方法不依赖预定义的异常检测阈值，而是通过持续学习客户端的行为模式来识别和抑制恶意更新。

实验结果 / Results

实验结果显示，FedAOT在面对多标签翻转、噪声注入、后门攻击及其组合等多种未针对性攻击时，模型准确率显著优于现有防御方法；即使在未见过的攻击场景下，FedAOT仍能保持稳定的鲁棒性能。此外，该方法在计算开销方面具有可扩展性，适用于大规模联邦学习部署。

应用价值 / Applications

FedAOT可广泛应用于医疗、金融、物联网等对隐私和安全要求较高的联邦学习场景，为跨机构协作建模提供可靠的拜占庭容错保障。其无需攻击先验知识的特性使其特别适合攻击类型复杂多变、难以预判的真实环境，为构建可信赖的分布式机器学习系统提供了实用解决方案。
