论文摘要 / Abstract

本文提出了群组条件联邦共形预测（GC-FCP），一种在联邦学习环境下实现不确定性量化的创新协议。该方法能够在保护数据隐私的前提下，为可能重叠的数据群组提供群组条件覆盖保证。通过构建可合并的群组分层核心集，GC-FCP使客户端仅需传输紧凑的加权摘要信息，即可支持服务器端的高效聚合与校准。实验结果表明，该方法在合成数据集和真实数据集上均能达到与中心化校准基线相当的性能。

方法概述 / Method

GC-FCP采用核心集（coreset）技术，将本地校准分数压缩为群组分层的加权摘要，而非传输原始数据。这些核心集具有可合并性，允许服务器直接聚合来自多个客户端的摘要信息，无需反复访问原始校准数据。该方法通过设计特定的加权策略，确保在联邦环境下仍能保持群组条件的覆盖保证。

实验结果 / Results

实验在合成数据集和多个真实世界数据集上进行，验证了GC-FCP的有效性。结果表明，该方法在通信效率显著提升的同时，其覆盖率和预测集大小等关键指标与中心化共形预测基线相当，成功实现了隐私保护与统计保证之间的平衡。

应用价值 / Applications

GC-FCP适用于医疗诊断（多医院协作）、金融风控（跨机构数据）、移动感知等分布式敏感数据场景。其价值在于使组织能够在不共享原始数据的情况下协作构建可信AI系统，特别适用于需要为不同人口统计学或语义群组提供公平、可靠预测区间的应用场景。
