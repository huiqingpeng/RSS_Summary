Enhanced Atrial Fibrillation Prediction in ESUS Patients with Hypergraph-based Pre-training

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究针对不明来源栓塞性卒中（ESUS）患者心房颤动（AF）预测的挑战，提出了一种基于超图的预训练框架。研究团队开发了监督和无监督两种超图预训练策略，在大规模卒中队列（7,780例患者）上预训练患者嵌入模型，以捕获显著特征和高阶交互关系，然后将学习到的嵌入迁移至小规模ESUS队列（510例患者）。实验结果表明，两种预训练方法均优于传统基线模型，在提升预测准确性的同时增强了模型鲁棒性，为卒中后AF风险预测提供了可扩展的高效解决方案。

方法概述 / Method

该研究采用超图神经网络构建患者嵌入模型，利用超边建模患者特征间的高阶关联关系；设计了监督和无监督两种预训练范式，在大规模卒中数据上学习低维、信息丰富的患者表征，再通过迁移学习将预训练嵌入应用于下游ESUS患者的AF预测任务。

实验结果 / Results

基于超图的预训练方法在ESUS患者AF预测任务上显著优于直接使用原始特征训练的传统机器学习模型；预训练嵌入有效降低了高维医疗特征的空间维度，同时保留了临床有意义的信息，使得轻量级模型也能实现有效预测；两种预训练策略均展现出 improved accuracy and robustness。

应用价值 / Applications

该框架可部署于医院卒中中心，辅助临床医生早期识别ESUS患者的心房颤动风险，从而及时启动抗凝治疗以降低复发性卒中和死亡风险；其预训练-迁移学习范式为其他小样本、高维度的医疗预测任务提供了通用方法论，具有良好的临床可扩展性和成本效益。
