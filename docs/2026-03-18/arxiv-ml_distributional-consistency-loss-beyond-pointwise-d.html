Distributional Consistency Loss: Beyond Pointwise Data Terms in Inverse Problems

论文摘要 / Abstract

本文提出了一种名为"分布一致性损失（Distributional Consistency Loss, DC）"的新型数据保真目标函数，用于解决逆问题中的信号恢复挑战。与传统逐点匹配噪声测量数据的方法（如MSE）不同，DC损失在分布层面评估数据保真度，检验观测测量是否与当前估计所隐含的噪声分布统计一致。该损失函数可直接替代标准数据一致性项，兼容现代无监督正则化方法，无需早停即可避免过拟合噪声。实验表明，DC损失在图像去噪和医学图像重建任务中显著提升了性能，为噪声主导的逆问题提供了一种统计基础更扎实、性能更优的替代方案。

方法概述 / Method

DC损失的核心思想是将数据保真度从逐点比较转变为分布级校准：给定当前估计信号，计算其应产生的噪声分布，然后检验实际观测测量是否服从该分布。具体实现上，DC损失通过统计检验（如拟合优度检验）量化观测残差与预期噪声分布之间的一致性，作为优化目标。该方法保持与传统损失函数相同的优化方式（如梯度下降），但天然地对噪声具有鲁棒性，因为它只要求整体分布匹配而非每个数据点精确对应。

实验结果 / Results

在图像去噪任务中，结合深度图像先验（Deep Image Prior）使用DC损失替代MSE，消除了早停需求并实现了更高的峰值信噪比（PSNR）。在医学图像重建任务中，针对泊松噪声数据，DC损失显著减少了高度迭代重建中的伪影，并增强了手工设计正则化器的效果。这些结果表明DC损失在保持计算效率的同时，有效解决了传统方法中噪声过拟合的核心问题。

应用价值 / Applications

DC损失适用于任何已知测量噪声分布且包含大量独立噪声值的实际逆问题场景，包括医学成像（CT、PET、MRI重建）、地球物理反演、天文图像处理和信号恢复等领域。其作为即插即用的替代方案，特别适合无监督学习设置，能够在缺乏配对真值数据的情况下提升重建质量，降低对启发式早停策略的依赖，为临床诊断和科学观测等关键应用提供更可靠、自动化的图像重建流程。
