论文摘要 / Abstract

本文针对三维网格上的机器学习提出了可微分的最直测地线（straightest geodesics）并行计算方法，解决了黎曼算子缺乏闭式解、离散形式不可微以及并行化能力差等关键障碍。作者开发了两种微分方法（基于外在代理函数和测地线有限差分方案），并实现了GPU并行加速。该方法被应用于多种几何学习任务，包括新的测地卷积层、网格流匹配方法以及用于质心Voronoi镶嵌的二阶优化器。

方法概述 / Method

论文基于最直测地线框架计算黎曼曲面上的指数映射，实现了测地线追踪和向量平行传输。作者提出了两种微分策略：一种利用外在代理函数进行近似微分，另一种基于测地线有限差分方案，并开发了完整的GPU并行实现以提升计算效率。

实验结果 / Results

实验验证了所提方法在并行化性能和计算精度上的优势，证明了可微指数映射能够有效改进一般几何上的学习和优化流程。具体应用包括：新提出的测地卷积层、网格流匹配学习方法，以及在质心Voronoi镶嵌任务上应用的二阶优化器，均展示了方法的通用性和有效性。

应用价值 / Applications

该方法为三维网格上的深度学习提供了基础几何工具，可广泛应用于计算机视觉、计算机图形学和几何处理领域，如形状分析、网格生成、表面配准和几何优化等任务。作者还发布了可pip安装的Python库（digeo），便于研究者和工程师快速集成到现有机器学习流程中。
