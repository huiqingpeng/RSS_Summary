Grounding the Score: Explicit Visual Premise Verification for Reliable Vision-Language Process Reward Models

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对视觉-语言过程奖励模型（VL-PRMs）中感知与推理纠缠导致的系统性误判问题，提出了显式视觉前提验证框架（EVPV）。该方法通过让策略模型生成步骤级视觉检查清单，并与从图像独立提取的结构化视觉约束进行匹配，计算出视觉可靠性信号，进而通过可靠性门控机制校准奖励分数。实验表明，EVPV在VisualProcessBench及六个多模态推理基准上显著提升了步骤级验证准确率和Best-of-N重排序性能，且约束损坏实验提供了因果证据支持其有效性。

方法概述 / Method

EVPV的核心是一个轻量级验证接口，包含三个组件：策略模型生成包含视觉事实声明的步骤级检查清单；约束提取器从输入图像独立派生结构化视觉约束；验证模块将清单声明与约束匹配以输出标量可靠性信号。该信号通过可靠性门控机制调节PRM的步级奖励——当可靠性低时衰减视觉依赖步骤的奖励，高时则保留原奖励，从而无需每步工具调用即可解耦感知不确定性与逻辑评估。

实验结果 / Results

在VisualProcessBench和六个多模态推理基准上的实验显示，EVPV持续优于强基线模型，显著提升了Best-of-N重排序准确率。此外，通过对提取的约束注入可控损坏，观察到性能单调下降，这为EVPV的收益源于约束保真度和显式前提验证而非提示效应偶然性提供了因果证据。

应用价值 / Applications

EVPV可广泛应用于需要可靠视觉推理的多模态AI系统，如视觉问答、机器人任务规划和科学图表理解等场景，通过显式验证视觉前提减少幻觉错误并提升推理透明度。其轻量级设计（无需每步工具调用）使其适合部署于资源受限的实时交互系统中，为构建更可信赖的视觉-语言智能体提供了实用解决方案。
