SAC-NeRF: Adaptive Ray Sampling for Neural Radiance Fields via Soft Actor-Critic Reinforcement Learning

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出 SAC-NeRF，一种基于 Soft Actor-Critic (SAC) 强化学习的自适应光线采样框架，用于解决神经辐射场 (NeRF) 中因密集采样导致的计算效率低下问题。该方法将采样过程建模为马尔可夫决策过程，使 RL 智能体能够根据场景特征动态分配采样点。实验表明，SAC-NeRF 在保持渲染质量（PSNR 损失仅 0.3-0.8 dB）的同时，可将采样点数减少 35-48%，证明了数据驱动的采样策略能够发现难以手工设计的有效模式。

方法概述 / Method

论文采用 Soft Actor-Critic 强化学习算法，通过三个关键组件实现自适应采样：(1) 高斯混合分布颜色模型提供不确定性估计以指导采样决策；(2) 多组件奖励函数平衡渲染质量、计算效率和时序一致性；(3) 两阶段训练策略应对环境的非平稳性问题。智能体在每个状态决定下一采样位置，逐步构建稀疏而高效的采样分布。

实验结果 / Results

在 Synthetic-NeRF 和 LLFF 数据集上的实验显示，SAC-NeRF 相比密集采样基线减少 35-48% 的采样点，同时 PSNR 下降控制在 0.3-0.8 dB 范围内。尽管学习到的策略具有场景特异性且 RL 框架增加了系统复杂度，但该方法显著优于简单的启发式采样策略，在效率与质量之间取得了有效平衡。

应用价值 / Applications

该研究适用于需要实时或高效新视角合成的场景，如虚拟现实、增强现实、三维重建和影视渲染等计算资源受限的应用。通过智能降低采样开销，SAC-NeRF 为移动设备上的 NeRF 渲染、大规模场景重建以及需要快速推理的交互式 3D 应用提供了可行路径，推动了神经辐射场技术的实用化部署。
