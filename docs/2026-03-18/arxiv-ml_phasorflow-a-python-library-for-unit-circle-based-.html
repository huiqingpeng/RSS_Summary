论文摘要 / Abstract

本文提出了PhasorFlow，一个开源Python库，引入了一种基于$S^1$单位圆的计算范式。该框架将输入编码为$N$-环面($\mathbb{T}^N$)上的复相量$z = e^{i\theta}$，通过酉波干涉门进行计算，在保持全局范数的同时允许分量漂移至$\mathbb{C}^N$，从而原生利用连续几何梯度进行预测学习。研究验证了该框架在非线性空间分类、时间序列预测、金融波动检测及神经形态任务中的有效性，确立了单位圆计算作为经典神经网络和量子电路的确定性、轻量级且数学原理严谨的替代方案。

方法概述 / Method

PhasorFlow的核心方法包括三个层面：首先形式化相量电路模型（$N$个单位圆线程，$M$个门），构建包含22个门的完整门库，涵盖标准酉、非线性、神经形态和编码操作；其次提出变分相量电路(VPC)，类比于变分量子电路(VQC)，通过优化连续相位参数实现经典机器学习任务；最后引入相量Transformer，用基于DFT的无参数token混合层替代昂贵的$QK^TV$注意力机制。

实验结果 / Results

论文在多个任务上验证了PhasorFlow的有效性，包括非线性空间分类、时间序列预测、金融波动检测以及神经形态任务（如神经绑定和振荡联想记忆）。结果表明，该框架能够在经典硬件上运行，同时共享量子力学的酉基础，实现了与经典神经网络和量子电路相比具有竞争力的性能。
