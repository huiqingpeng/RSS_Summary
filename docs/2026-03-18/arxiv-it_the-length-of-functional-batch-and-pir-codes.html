论文摘要 / Abstract

本研究探讨了在任意有限域上，固定维数和列表大小的功能批处理码（Functional Batch Codes）和私有信息检索码（PIR Codes）的最小长度计算问题。作者将此前针对二元码的若干结果推广到一般有限域情形，并进行了精细化处理。论文提出了新的上下界估计，分析了该参数的渐近行为，并计算了多组参数下的具体数值，同时为非二元域上的功能批处理猜想提供了关于"正确"列表大小的理论见解。

方法概述 / Method

论文采用组合数学与编码理论相结合的方法，通过将二元码的结果推广到任意有限域，建立功能批处理码和PIR码最小长度的理论框架。具体技术包括构造性上界证明、信息论下界推导，以及利用有限域的代数结构进行参数优化分析。

实验结果 / Results

研究获得了功能批处理码和PIR码最小长度的新上下界，刻画了该参数随码长和域大小变化的渐近特性。作者计算了多组具体参数下的最小长度精确值或紧界，并验证了所提界限的紧性，为理解列表大小与码效率之间的权衡关系提供了定量依据。

应用价值 / Applications

该研究对分布式存储系统、隐私保护数据库查询和云计算中的高效数据检索具有重要理论指导意义。功能批处理码和PIR码的优化设计可降低存储冗余度和通信开销，提升大规模数据系统中多用户并行查询和隐私保护访问的性能与安全性。
