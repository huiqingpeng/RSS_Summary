VALD: Multi-Stage Vision Attack Detection for Efficient LVLM Defense

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了VALD（多阶段视觉攻击检测），一种针对大型视觉-语言模型（LVLMs）的高效对抗防御方法。该方法结合图像变换与智能体数据整合技术，通过两阶段检测机制快速过滤干净输入：第一阶段利用内容保持变换评估图像一致性，第二阶段在文本嵌入空间分析差异，仅在必要时调用大型语言模型进行最终裁决。实验表明，该方法在保持最先进准确率的同时显著降低计算开销，即使面对大量对抗样本也能维持最小额外成本。

方法概述 / Method

VALD采用训练无关的三层防御架构：首先通过低成本的内容保持变换（如旋转、缩放）检测图像一致性，快速排除明显干净的输入；对于可疑样本，进一步比较文本嵌入空间的响应差异；最后利用LLM智能体整合多重响应，通过分析相似性与差异性来恢复正确输出。核心创新在于"按需激活"的级联设计，避免对所有输入进行昂贵的重处理。

实验结果 / Results

论文显示VALD在对抗攻击检测上达到当前最优准确率，同时实现显著的效率优势：绝大多数干净图像可跳过高成本处理阶段，即使在高比例对抗样本场景下系统开销仍保持极低水平。这种准确性与效率的权衡优于现有防御方案。

应用价值 / Applications

该方法适用于部署大规模LVLM的实际生产环境，如自动驾驶视觉问答、医疗影像诊断辅助、内容审核等安全敏感场景。其无需额外训练、计算开销可控的特性，使其特别适合资源受限或对实时性要求高的边缘计算部署，为LVLM的可靠商业化应用提供了可扩展的安全保障方案。
