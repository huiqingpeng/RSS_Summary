Dual Consensus: Escaping from Spurious Majority in Unsupervised RLVR via Two-Stage Vote Mechanism

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了双共识强化学习（DCRL），一种无需标签的自监督训练方法，用于解决无监督RLVR（强化学习验证奖励）方法中模型收敛于虚假多数答案的问题。
DCRL通过两阶段共识机制生成更可靠的学习信号：首先让模型作为"锚点"生成主导响应，然后通过临时遗忘过程作为"探索者"生成多样化辅助信号，最终训练目标由两组信号的调和均值确定。该方法无需外部模型或监督，在八个基准测试和多个领域上持续提升了Pass@1性能，同时实现了更稳定的训练动态。

方法概述 / Method

DCRL的核心是一个两阶段投票机制：第一阶段，模型作为锚点生成当前的主导答案分布；第二阶段，通过临时遗忘（unlearning）技术让模型"忘记"已学得的偏见，从而生成多样化的探索性响应。最终的学习信号由锚点信号和探索信号通过调和均值融合得到，有效平衡了共识性与多样性。

实验结果 / Results

在涵盖数学推理、代码生成和通用问答等八个基准测试中，DCRL相比多数投票基线一致性地提升了Pass@1准确率，同时展现出更稳定的训练动态。实验表明该方法成功帮助模型逃离虚假多数陷阱，实现了无需外部标签的持续性能提升。

应用价值 / Applications

DCRL为大语言模型的推理能力增强提供了一条完全自监督、可扩展的技术路径，特别适用于缺乏高质量标注数据的场景。该方法可广泛应用于需要复杂推理能力的领域，如数学问题求解、代码生成和科学问答，降低了对昂贵人工标注或外部验证器的依赖。
