Bridging the Simulation-to-Reality Gap in Electron Microscope Calibration via VAE-EM Estimation

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文提出了一种基于变分自编码器（VAE）和期望最大化（EM）算法的联合估计框架（VAE-EM），用于解决扫描透射电子显微镜（STEM）的参数校准问题。该方法通过VAE从模拟数据中学习图像的低维表示，同时联合估计校准参数到编码表示的映射模型以及最优校准参数，显式地解决了数据驱动方法中模拟数据与真实数据之间的"仿真-现实鸿沟"问题。利用光学系统的已知对称性，作者证明了联合估计问题的全局可识别性，确保存在唯一最优解。实验表明，该方法在实际STEM上比现有方法快2倍且更稳定，估计误差降低50%，同时需要的观测次数更少。

方法概述 / Method

论文采用两阶段方法：首先使用变分自编码器在模拟数据上进行训练，将高维噪声图像压缩为低维潜在表示，而非传统方法提取的标量值；随后通过期望最大化（EM）算法同时估计从校准参数到潜在表示的映射模型和最优校准参数，实现模型参数与校准状态的联合推断。该方法的关键创新在于利用光学系统的对称性约束来保证全局可识别性，从而克服非单射映射导致的解不唯一问题。

实验结果 / Results

在真实STEM系统上的验证实验表明，VAE-EM方法相比现有方法实现了估计误差降低50%（2x reduction），同时具有更快的收敛速度和更高的稳定性。此外，该方法所需的观测次数显著减少，在自动化校准效率和可靠性方面均表现出明显优势，证明了VAE在图像信息压缩方面的有效性。

应用价值 / Applications

该研究为电子显微镜的自动化校准提供了高效可靠的解决方案，可广泛应用于材料科学、纳米技术等领域中依赖高分辨率电子显微表征的科研与工业场景。更重要的是，VAE-EM框架具有通用性，可扩展至其他存在仿真-现实鸿沟的逆问题领域，如机器人控制、医学成像等，特别是在训练数据来自数字孪生且映射关系非单射的情况下，为相关领域的参数估计和系统校准提供了新的方法论范式。
