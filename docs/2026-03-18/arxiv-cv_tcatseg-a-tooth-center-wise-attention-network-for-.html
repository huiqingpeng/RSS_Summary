TCATSeg: A Tooth Center-Wise Attention Network for 3D Dental Model Semantic Segmentation

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了TCATSeg，一种用于3D牙科模型语义分割的新型注意力网络。针对现有方法仅关注局部几何特征而忽略全局上下文信息的问题，该框架通过引入稀疏且具有物理意义的超点（superpoints）来捕获全局语义关系，从而结合局部几何与全局语义信息。研究还构建了一个包含400个牙科模型（含正畸前样本）的新数据集用于评估泛化性能，实验结果表明TCATSeg优于现有最先进方法。

方法概述 / Method

TCATSeg采用以牙齿中心为核心的注意力机制，通过生成稀疏超点来编码全局语义上下文，同时保留局部几何细节。该网络架构实现了局部特征与全局语义信息的有效融合，以解决相邻牙齿形状相似导致的分割困难。

实验结果 / Results

论文在新构建的400例牙科模型数据集上进行了广泛实验，验证了该方法在复杂牙齿排列场景下的分割准确性。
TCATSeg在泛化性能方面表现优异，特别是在正畸前样本上展现出良好的适应性，整体性能超越当前主流方法。

应用价值 / Applications

该研究可直接应用于数字牙科领域的临床诊疗流程，包括正畸治疗方案规划和牙科种植体设计。准确的3D牙齿分割有助于提升数字化口腔诊疗的自动化水平和精度，为计算机辅助牙科手术提供可靠的技术支撑。
