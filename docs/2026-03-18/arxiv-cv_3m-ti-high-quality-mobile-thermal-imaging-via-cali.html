3M-TI: High-Quality Mobile Thermal Imaging via Calibration-free Multi-Camera Cross-Modal Diffusion

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对移动平台热成像传感器因微型化导致的空间分辨率低和纹理细节缺失问题，提出了3M-TI框架——一种无需标定的多相机跨模态扩散模型。该框架通过跨模态自注意力模块（CSM）自适应对齐热成像与RGB特征，在扩散去噪过程中利用生成先验提升热图像的超分辨率、结构保真度和纹理细节。实验表明，3M-TI在视觉质量和量化指标上达到最优，并显著改善了目标检测与分割等下游任务性能。

方法概述 / Method

3M-TI采用扩散模型作为基础架构，将UNet中的原始自注意力层替换为跨模态自注意力模块（CSM），实现热成像与RGB特征在无需显式相机标定条件下的自适应对齐。该设计使网络能够在去噪过程中动态融合双模态信息，充分发挥扩散模型的生成能力进行热图像超分辨率重建。

实验结果 / Results

在真实移动热成像相机和公开基准数据集上的广泛评估显示，3M-TI在视觉质量和量化指标（如PSNR、SSIM等）上均达到当前最优水平。更重要的是，经3M-TI增强后的热图像在目标检测和分割等关键下游任务中带来显著提升，验证了其实际应用价值。

应用价值 / Applications

3M-TI可直接部署于智能手机、无人机、机器人等移动平台的热成像系统，无需繁琐的相机标定流程，降低了工程落地门槛。该技术可广泛应用于夜间监控、搜救行动、工业检测、自动驾驶等需要高质量热成像感知的场景，为构建鲁棒的移动热感知系统提供了有效解决方案。
