Retrieving Counterfactuals Improves Visual In-Context Learning

论文摘要 / Abstract

本文提出了CIRCLES框架，通过属性引导的组合图像检索主动构建包含反事实风格示例的演示集，帮助视觉-语言模型(VLMs)隐式推理属性与结果之间的因果关系，而非依赖表面相关性。在四个多样化数据集上的实验表明，该方法在多种架构上持续优于现有方法，尤其在小型模型和信息稀缺场景下效果显著。
CIRCLES检索的示例更具多样性和因果信息性，为模型如何利用上下文演示进行推理提供了定性洞察。

方法概述 / Method

CIRCLES采用主动式反事实示例检索策略，通过属性引导的组合图像检索(composed image retrieval)构建演示集，而非传统的被动相似度检索。该方法针对性地检索与查询相关但关键属性被系统性改变的反事实风格示例，使VLMs能够区分因果相关特征与虚假相关性，从而实现更稳健的推理。

实验结果 / Results

在四个多样化数据集上的全面实验显示，CIRCLES在多种VLM架构上持续优于现有基线方法，特别是在小规模模型上优势更为明显。该方法在信息稀缺条件下表现出显著的增益，且检索到的示例具有更高的多样性和因果信息量，验证了反事实示例对视觉上下文学习的促进作用。

应用价值 / Applications

该研究可广泛应用于需要细粒度视觉属性理解和因果推理的多模态任务，如医学影像诊断、自动驾驶场景理解等高风险决策领域。通过提升VLMs在有限示例条件下的鲁棒性和可解释性，CIRCLES为资源受限场景下的高效模型适配提供了实用解决方案，有助于减少对大规模标注数据的依赖。
