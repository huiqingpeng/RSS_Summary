RaDAR: Relation-aware Diffusion-Asymmetric Graph Contrastive Learning for Recommendation

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文提出了RaDAR框架，用于解决推荐系统中图对比学习（GCL）的两个核心问题：随机边扰动导致结构信号失真和语义不一致，以及数据稀疏性限制协同信号传播。
RaDAR通过结合图生成模型和关系感知去噪模型两种互补的视图生成机制，实现了在噪声和稀疏条件下的鲁棒推荐。在三个公开基准数据集上的实验表明，RaDAR显著优于现有最先进方法。

方法概述 / Method

RaDAR采用非对称对比学习架构，包含三个关键创新组件：（1）全局负采样的非对称对比学习，在保持语义对齐的同时抑制噪声；（2）扩散引导增强，通过渐进式噪声注入与去噪提升鲁棒性；（3）关系感知边细化，基于潜在节点语义动态调整边权重，以捕获全局结构并精炼噪声边。

实验结果 / Results

在三个公开基准数据集上的大量实验表明，RaDAR持续优于现有最先进方法，尤其在噪声和稀疏数据条件下表现突出。该框架有效缓解了随机边扰动带来的语义不一致问题，并增强了数据稀疏场景下的泛化能力。

应用价值 / Applications

RaDAR可广泛应用于电商、社交媒体和内容平台等推荐场景，特别适用于用户-物品交互数据稀疏或存在噪声的实际环境。该框架通过提升图对比学习的鲁棒性和语义一致性，为构建更精准、更稳定的推荐系统提供了有效解决方案。
