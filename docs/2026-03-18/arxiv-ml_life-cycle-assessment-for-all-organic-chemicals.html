Life cycle assessment for all organic chemicals

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究提出了CRYSTAL（Chemical RetrosYnthesiS for Transparent Assessment of Life-cycles）框架，通过逆向合成和机器学习自动为有机化学品生成一致且透明的生命周期清单（LCI）数据。研究团队利用该框架创建了涵盖超过70,000种有机化学品、110,000多个透明LCI数据集的综合数据库，量化了原料、能源需求及相关辅助材料、生物圈流动和废物流。基于此数据库，研究识别出50个关键环境热点和枢纽化学品，为化学工业的可持续转型提供了系统性的工程和政策干预指导。

方法概述 / Method

CRYSTAL框架整合逆向合成算法与机器学习技术，根据分子结构自动生成门到门（gate-to-gate）的生命周期清单数据。该方法通过计算化学合成路径，结合机器学习预测各合成步骤的能耗与物料投入，实现了对缺乏实测数据化学品的环境影响评估。

实验结果 / Results

研究成功构建了全球最大规模的有机化学品LCI数据库，覆盖70,000余种化学品；识别出50个跨多环境类别的高影响环境热点，并定位了对下游生产最关键的枢纽化学品。该数据库揭示了当前化学工业中大量"未知的未知"（unknown unknowns）可被转化为"已知的未知"（known unknowns），显著提升了数据透明度与可改进性。

应用价值 / Applications

CRYSTAL框架为绿色化学设计、可持续工艺优化和环境政策制定提供数据基础设施，可帮助化学工程师优先改进高影响合成路径，并支持监管机构识别关键化学品进行管控。其模块化透明特性促进学术界与工业界的协作改进，推动化学生命周期评估从数据稀缺向系统化知识管理转型。
