Language as a Wave Phenomenon: Semantic Phase Locking and Interference in Neural Networks

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究提出PRISM——一种复值编码器，通过强制单位范数约束（|z|=1）并用门控谱滤波替代注意力机制，迫使模型仅依赖相位角而非幅度来区分信号与噪声。研究发现语义关系与可测量的相位结构相关：同义词对表现出显著的相位相干性（R=0.198 vs. 0.072, p<0.001），且模型通过层特定的相位旋转解决词汇歧义。相位表示对幅度衰减具有鲁棒性，信号幅度均匀降低时仍保持97%的翻译质量。研究还揭示了需要最小序列长度才能产生支持计算的干涉模式，并提出结合相位编码器与标准注意力的混合架构（Wave-Particle Transformer）在3300万参数下达到Transformer基线水平。

方法概述 / Method

PRISM采用复值神经网络架构，核心创新包括：强制隐藏状态为单位模长（|z|=1），消除幅度信息作为计算资源；用门控谱滤波替代传统自注意力机制，在频域进行相位操作；通过可学习的相位旋转实现层间信息处理。研究还设计了混合架构Wave-Particle Transformer，将PRISM作为编码器与标准Transformer解码器结合。

实验结果 / Results

关键发现包括：同义词对的相位相干性显著高于随机词对（相关系数提升近3倍）；模型在信号幅度衰减至接近零时仍保持97%的翻译BLEU分数；孤立token无法产生相干输出，存在最小序列长度阈值以形成有效干涉模式；Wave-Particle Transformer在33M参数规模下匹配标准Transformer性能，且非嵌入参数更少。

应用价值 / Applications

该研究为开发对信号衰减鲁棒的神经网络提供了新思路，适用于噪声通信、边缘计算等低信噪比环境；相位编码机制可拓展至模型压缩与高效推理场景；混合架构为轻量级语言模型设计提供了新范式，有望在资源受限设备上实现高性能自然语言处理。
