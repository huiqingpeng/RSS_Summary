Learning to Diagnose Privately: DP-Powered LLMs for Radiology Report Classification

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了一种差分隐私（DP）驱动的低秩适配（LoRA）微调框架，用于在保护患者隐私的前提下对大型语言模型（LLM）进行放射学报告多异常分类任务训练。该方法通过整合差分隐私与LoRA技术，有效降低模型记忆训练数据的风险，同时利用大型LLM生成的标签来训练较小的模型以实现高效推理。在MIMIC-CXR和CT-RATE数据集上的实验表明，在中等隐私预算下，该方法可达到0.89的加权F1分数，接近非隐私LoRA（0.90）和全量微调（0.96）的性能，证明强隐私保护仅需适度的性能权衡。

方法概述 / Method

本研究采用差分隐私随机梯度下降（DP-SGD）结合LoRA的参数高效微调策略，在冻结预训练模型主干的同时仅训练低秩适配矩阵，从而在计算效率和隐私保护之间取得平衡。此外，研究还探索了使用大型教师LLM生成软标签来指导小型学生模型训练的知识蒸馏方法，以在保持隐私保证的同时提升推理效率。

实验结果 / Results

在MIMIC-CXR数据集上，DP-LoRA在适度隐私预算（ε=3-8）下取得了0.89的加权F1分数，与无隐私保护的LoRA微调（0.90）差距仅为0.01，相比全量微调（0.96）差距为0.07。实验验证了该框架在不同隐私强度设置下的有效性，表明即使在严格的差分隐私约束下，模型仍能保持较高的临床分类性能。

应用价值 / Applications

该研究为医疗机构在保护患者敏感信息的同时利用LLM自动化处理放射学报告提供了可行方案，特别适用于多医院协作建模场景下需要共享模型但禁止共享原始数据的隐私合规需求。该方法可推广至其他临床文本分类任务，为医疗AI的隐私安全部署和联邦学习应用奠定技术基础。
