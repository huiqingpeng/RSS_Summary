Steering Frozen LLMs: Adaptive Social Alignment via Online Prompt Routing

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对大型语言模型（LLMs）在部署后面临的安全对齐静态性问题，提出了一种无需重新训练的自适应推理时治理框架。作者指出，传统的后训练对齐（如RLHF或DPO）产生静态策略，无法应对不断演化的越狱攻击和多元时变的社会规范。为此，他们设计了共识聚类LinUCB bandit（CCLUB）算法，通过系统提示路由实现自适应社会对齐，并在理论上证明了其具有次线性遗憾保证。实验表明，CCLUB相比强基线实现了10.98%的累积奖励提升和14.42%的平均次优差距降低。

方法概述 / Method

CCLUB采用保守的共识聚类机制，仅在效用相似性图和安全性相似性图的交集内聚合数据，从而防止在语义相近但风险差异较大的上下文之间进行不安全泛化。该方法将在线提示路由建模为上下文bandit问题，结合聚类结构进行高效探索-利用权衡，实现对冻结LLM的推理时行为引导。

实验结果 / Results

实验验证了CCLUB在多个基准上显著优于强基线方法，实现了10.98%的累积奖励提升和14.42%的平均次优差距降低。理论分析表明CCLUB具有次线性遗憾保证，证明了其接近最优的性能表现。

应用价值 / Applications

该研究为部署后的LLM安全治理提供了实用解决方案，特别适用于需要快速响应新型攻击模式、适应多元文化背景或动态监管要求的场景。方法无需昂贵的模型重训练，可广泛应用于在线内容审核、对话系统安全加固以及需要持续对齐演化社会价值观的AI服务中。
