Memorization to Generalization: Emergence of Diffusion Models from Associative Memory

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文从密集联想记忆（DenseAMs）的理论视角重新审视扩散模型，揭示了扩散模型从记忆到泛化的相变过程。研究发现，当训练数据量超过临界容量时，扩散模型会出现"虚假状态"（spurious states）——这在传统联想记忆中被视为干扰记忆检索的负面产物，但在生成模型中却成为生成能力的最初标志。作者系统刻画了这些状态的吸引域、能量景观曲率及其计算特性，并在多种架构和数据集上验证了该现象。

方法概述 / Method

论文将扩散模型的生成过程重新诠释为记忆检索任务，建立其与Hopfield网络及DenseAMs的理论联系。通过分析能量景观中局部极小值的演化行为，识别出数据量驱动的三阶段相变：纯记忆阶段、虚假状态涌现阶段和完全泛化阶段。

实验结果 / Results

实验表明，在多种扩散模型架构和不同规模数据集上均观察到虚假状态的涌现现象。这些状态具有明确定义的吸引域和特定的能量景观曲率特征，其出现标志着模型从简单记忆训练样本向生成新样本能力的转变。

应用价值 / Applications

该研究为理解深度学习生成模型的泛化机制提供了新的理论框架，有助于指导扩散模型的训练策略设计和容量规划。研究成果可用于预测模型何时具备生成能力、诊断训练过程中的相变点，并为开发更高效可控的生成模型提供理论依据。
