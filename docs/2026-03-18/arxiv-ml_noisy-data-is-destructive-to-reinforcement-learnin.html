Noisy Data is Destructive to Reinforcement Learning with Verifiable Rewards

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文揭示了带可验证奖励的强化学习（RLVR）领域的一个重要发现：此前研究表明RLVR算法能从错误标注中有效学习，但这一结论因数据集"污染"（即声称的100%噪声数据实际混有干净数据）而无效。作者通过严格的重新验证流程修正数据集后证明，噪声对RLVR具有破坏性——在真实错误标注上训练的模型在数学推理基准上比干净数据训练的模型性能低8-10%，在Text2SQL任务的真实世界人工标注错误上则低5-12%。

方法概述 / Method

作者构建了严格的重新验证流程（re-verification pipeline）来识别并去除数据集中的"隐性干净数据"，从而获得真正纯净的噪声数据集；在此基础上，系统评估了包括改进版RLVR算法和基础GRPO在内的多种方法在噪声数据与干净数据上的性能对比，并在数学推理和Text2SQL两个领域进行验证。

实验结果 / Results

核心发现包括：现有RLVR算法改进（如针对噪声优化的变体）未能有效缓解噪声影响，其性能与基础GRPO相当；在真实错误标注上训练的模型在数学推理任务上比干净数据模型差8-10%，在真实世界Text2SQL人工标注错误上差5-12%；这些结果在合成噪声和真实噪声场景下均保持一致。

应用价值 / Applications

本研究对大规模语言模型的RLVR训练实践具有重要指导意义：明确否定了"RLVR可自动补偿低质量数据"的观点，强调高质量数据仍是模型性能的关键基础；提醒研究者和工程师在构建训练数据时需进行严格的质量控制，避免隐性数据污染导致的结论偏差；为RLVR算法未来的抗噪声改进方向提供了清晰的基准和动机。
