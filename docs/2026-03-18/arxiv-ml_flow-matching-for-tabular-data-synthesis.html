Flow Matching for Tabular Data Synthesis

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文将流匹配（Flow Matching, FM）技术引入表格数据合成领域，作为扩散模型的替代方案。研究系统比较了流匹配（包括变分流匹配）与当前最先进的扩散方法（TabDDPM和TabSyn），评估了最优传输（OT）和方差保持（VP）概率路径以及确定性与随机采样器的性能。核心发现表明，特别是TabbyFlow方法，流匹配在低函数评估次数（≤100步）下即可超越扩散基线，同时在数据效用与隐私风险之间取得更优平衡。

方法概述 / Method

论文实现了多种流匹配变体用于表格数据合成，包括标准流匹配和变分流匹配（Variational FM），并对比了最优传输（OT）概率路径与方差保持（VP）概率路径两种构造方式。此外，研究还探索了确定性采样器与随机采样器的组合，其中变分流匹配框架支持随机采样，为隐私-效用权衡提供了额外调控手段。

实验结果 / Results

实验表明，TabbyFlow在表格数据合成任务上 consistently 优于TabDDPM和TabSyn等扩散基线方法，且仅需不超过100步采样即可达到优异性能，计算效率显著提升。最优传输（OT）路径是更稳健的默认选择，对早停更具鲁棒性；而方差保持（VP）路径在降低隐私风险方面具有潜力。随机化流在保持边缘分布的同时，部分场景下能生成高效用且披露风险更低的合成数据。

应用价值 / Applications

该研究为隐私保护数据共享提供了更高效、更灵活的合成数据生成工具，适用于医疗健康、金融等敏感数据领域。流匹配方法的低计算成本特性使其适合大规模数据发布和实时应用场景，而随机采样器的引入为组织在数据效用与隐私合规之间进行精细调控提供了新途径。
