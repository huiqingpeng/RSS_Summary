论文摘要 / Abstract

本文揭示了上下文辅助预测（CAF）中一个令人困惑的现象：多模态模型往往未能超越单模态模型。作者假设这一性能差距源于现有数据集中上下文质量不佳，并提出了一种半合成数据增强方法，生成了描述时间动态且可验证地与数值历史互补的上下文。由此构建了包含700万个上下文增强时间序列窗口的CAF-7M数据集，实验证明半合成预训练能有效迁移到真实世界评估，并明确展示了上下文的利用，表明数据集质量而非架构限制才是CAF的主要瓶颈。

方法概述 / Method

作者开发了一种半合成数据增强技术，通过生成既描述时间动态特征又可验证与数值历史互补的高质量上下文，实现了大规模数据集的自动化构建。该方法创建了CAF-7M语料库，包含700万个上下文增强的时间序列窗口，并配有经过严格验证的测试集，用于支持模型的预训练和评估。

实验结果 / Results

实验表明，基于CAF-7M的半合成预训练能够有效迁移到真实世界预测任务，并且模型展现出了明确的上下文利用证据。关键发现是，当提供高质量、可验证互补的上下文时，多模态模型能够成功超越单模态基线，从而证实了数据集质量而非模型架构是此前性能差距的根本原因。

应用价值 / Applications

该研究为时间序列预测领域提供了高质量数据构建的新范式，特别适用于需要整合领域知识和前瞻性信息的金融预测、需求规划、能源负荷预测等场景。
CAF-7M数据集和方法论可推动更可靠的上下文辅助预测系统开发，使AI系统真正发挥超越传统统计方法的潜力。
