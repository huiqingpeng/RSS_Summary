论文摘要 / Abstract

本研究开发了一种三维层析技术，用于精确提取硅/硅锗量子点器件中交换相互作用系数J(V)随栅极电压变化的累积相位φ(V)。研究团队整合了数字全息相移测量技术和最大流/最小割相位解包裹算法(PUMA)，从一系列二维测量中稳健地重建三维相位体积，成功解决了传统方法中余弦反演歧义、相位解包裹噪声敏感等关键挑战。该方法对器件漂移具有鲁棒性，并可用于定位电压空间中最小梯度的π交换脉冲点，为理解器件变异性来源、校准器件模型和系统优化量子比特控制提供了新途径。

方法概述 / Method

论文采用相移数字全息测量技术获取包裹相位，并应用三维最大流/最小割相位解包裹方法(PUMA)在三维电压空间中展开相位，同时结合模型构建与优化技术来精确定位最优控制点。该方法借鉴了光学全息、计算机视觉和图论等多个领域的技术，实现了对交换相互作用累积相位的稳健提取。

实验结果 / Results

研究成功构建了硅/硅锗量子点器件的三维相位体积模型，并通过提高扫描分辨率验证了该方法对器件最小漂移的鲁棒性。基于提取的相位模型，研究团队在电压空间中优化定位了一个最小梯度的π交换脉冲点，证明了该方法在实际量子比特控制中的可行性。

应用价值 / Applications

该技术可为理解导致器件良率变化的物理机制提供详细信息，支持在运行期间将器件模型校准到特定器件以实现更复杂的错误归因分析，并能够系统优化量子比特控制。此外，该方法有望推广应用于其他量子计算平台，为规模化量子处理器的可靠运行和性能优化提供重要工具。
