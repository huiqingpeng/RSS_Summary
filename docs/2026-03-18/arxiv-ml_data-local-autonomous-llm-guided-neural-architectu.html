Data-Local Autonomous LLM-Guided Neural Architecture Search for Multiclass Multimodal Time-Series Classification

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了一种数据本地化的LLM引导神经架构搜索（NAS）框架，用于解决隐私敏感领域（如医疗健康）中多类别多模态时间序列分类的自动化建模难题。该框架允许远程控制器仅通过试验级摘要信息（如管道描述、指标、学习曲线统计和失败日志）指导搜索，而所有训练和评估均在本地执行，确保原始数据不离开本地环境。实验表明，该方法在UEA30公开数据集和SleepEDFx睡眠分期临床数据集上均表现优异，能够在保持数据隐私的同时实现无人值守的架构搜索，显著减少人工干预。

方法概述 / Method

该框架采用"远程控制-本地执行"的分离架构：LLM作为控制器在远程生成候选管道，本地执行器严格按照固定协议完成训练和评估，仅向控制器返回聚合后的试验摘要。搜索空间涵盖逐类别、逐模态的一对多（one-vs-rest）二进制专家网络、轻量级融合MLP，以及联合优化的专家架构和模态特定预处理流程。这种设计确保控制器永远无法访问原始样本或中间特征表示。

实验结果 / Results

在UEA30多变量时间序列分类基准和SleepEDFx睡眠分期任务（含EEG、EOG、EMG等异构临床模态）两个实验设置中，该方法展现出强劲性能：模块化基线模型本身已具备竞争力，而LLM引导的NAS进一步提升了其表现，在大多数基准数据集上达到已发表方法的性能范围。更重要的是，该方法成功实现了完全自动化的架构搜索流程，无需人工介入调参。

应用价值 / Applications

该研究为医疗、金融等严格受隐私法规约束的领域提供了可行的自动化机器学习解决方案，使机构能够在不暴露敏感数据的前提下利用大型语言模型的推理能力进行模型开发。特别适用于医院开发患者特异性模型、多模态生理信号分析等场景，既满足GDPR/HIPAA等合规要求，又降低了深度学习专家的人力成本，推动隐私保护型AI的普及化应用。
