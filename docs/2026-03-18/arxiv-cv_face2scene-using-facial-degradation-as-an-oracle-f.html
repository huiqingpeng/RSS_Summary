Face2Scene: Using Facial Degradation as an Oracle for Diffusion-Based Scene Restoration

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了 Face2Scene，一种两阶段图像修复框架，利用人脸作为感知"先知"来估计退化信息并指导全场景修复。该方法首先从退化图像和身份参考中重建高质量人脸细节，然后从修复前后的人脸对中提取退化编码，转化为多尺度退化感知标记，用于条件化扩散模型完成单步全场景修复。实验表明，该方法在修复效果上显著优于现有最先进方法，有效解决了传统人脸修复方法忽略全身及背景、以及全场景修复方法缺乏退化线索的问题。

方法概述 / Method

Face2Scene 采用两阶段架构：第一阶段使用参考人脸修复模型（Ref-FR）从退化输入和身份参考中重建高质量人脸；第二阶段通过对比修复前后的人脸对，提取包含噪声、模糊、压缩等属性的退化编码，并将其转换为多尺度退化感知标记，作为条件引导扩散模型在单步内完成全场景修复。

实验结果 / Results

论文通过大量实验验证了 Face2Scene 的有效性，结果表明该方法在全场景修复质量上显著优于现有最先进方法，能够同时恢复人脸、身体和背景区域的高质量细节，且有效避免了传统全场景修复方法因退化信息不足导致的欠定预测和视觉伪影问题。

应用价值 / Applications

Face2Scene 可广泛应用于老照片修复、监控视频增强、低质量图像超分辨率等实际场景，特别适用于需要保持人物身份一致性同时恢复完整场景视觉质量的图像修复任务，为数字档案保护、影视后期制作和移动设备摄影增强等领域提供了新的技术解决方案。
