Fast-WAM: Do World Action Models Need Test-time Future Imagination?

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文探讨了世界动作模型（WAMs）在测试时是否需要显式的未来想象（future imagination）。作者发现，WAMs 的主要价值可能在于训练期间的视频建模，而非测试时的未来生成。为此，他们提出了 Fast-WAM，一种在训练时保留视频联合训练、但在测试时跳过未来预测的架构。实验表明，Fast-WAM 在仿真和真实机器人任务上达到了与现有方法相当的性能，同时推理速度提升超过4倍，延迟仅为190毫秒。

方法概述 / Method

Fast-WAM 通过解耦训练时的视频建模与测试时的未来生成，设计了一种无需显式想象即可执行动作的架构。该模型在训练阶段与世界模型联合训练以学习良好的世界表征，但在推理阶段直接基于当前观测预测动作，省去了耗时的视频去噪迭代过程。作者还实例化了多个 Fast-WAM 变体，以控制对比训练时视频建模和测试时未来想象各自的作用。

实验结果 / Results

Fast-WAM 在 LIBERO 和 RoboTwin 仿真基准以及真实世界任务上均取得了与 SOTA 方法相当的性能，且无需具身预训练。关键发现是：移除视频联合训练会导致性能显著下降，而移除测试时未来想象则影响较小。推理延迟从传统 WAMs 的约800毫秒降至190毫秒，实现了实时控制。

应用价值 / Applications

Fast-WAM 为机器人具身控制提供了一种高效实用的替代方案，特别适用于对延迟敏感的实时应用场景，如动态环境中的机器人操作和交互。该研究表明，通过优化训练策略而非增加测试时计算，可以在保持性能的同时大幅提升推理效率，为资源受限的部署环境（如边缘设备）提供了可行路径。
