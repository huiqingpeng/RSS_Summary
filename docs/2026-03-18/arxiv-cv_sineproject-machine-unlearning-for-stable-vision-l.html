SineProject: Machine Unlearning for Stable Vision Language Alignment

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对多模态大语言模型（MLLMs）的机器遗忘需求，发现现有方法会破坏视觉-语言对齐，导致模型同时拒绝有害和良性查询。作者将这一问题追溯到投影器网络的雅可比矩阵在遗忘过程中严重病态化，提出SineProject方法，通过正弦调制的可训练参数增强冻结投影器，显著改善雅可比矩阵的谱条件数。实验表明，该方法在LLaVA v1.5 7B和13B模型上实现了最先进的遗忘-保留权衡，在完全遗忘目标信息的同时减少良性查询拒绝率，且计算开销可忽略。

方法概述 / Method

SineProject的核心是在保持投影器主干冻结的前提下，引入少量正弦调制的可训练参数（形式为θ + α·sin(ω·t)）来增强投影器网络。这种参数化方式能够有效正则化雅可比矩阵的奇异值分布，改善其谱条件数，从而在遗忘优化过程中稳定跨模态嵌入空间，防止视觉-语言对齐的漂移。

实验结果 / Results

在标准安全性和隐私性遗忘基准测试中，SineProject在LLaVA v1.5 7B和13B模型上均取得最优性能：实现了对目标信息的完全遗忘，同时显著降低良性查询的误拒绝率，达到了当前最佳的遗忘-保留权衡。此外，该方法仅需极低的额外计算开销，具有实际部署的可行性。

应用价值 / Applications

该研究适用于需要动态移除敏感信息（如隐私数据、不安全内容）的多模态AI系统，如内容审核平台、企业级视觉问答服务和合规性要求严格的应用场景。
SineProject提供了一种高效、稳定的机器遗忘方案，使MLLMs能够在不重新训练的情况下安全地"遗忘"特定知识，同时保持正常的多模态理解能力。
