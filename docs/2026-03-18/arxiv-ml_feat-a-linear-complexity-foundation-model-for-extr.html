FEAT: A Linear-Complexity Foundation Model for Extremely Large Structured Data

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

FEAT是一种针对超大规模结构化数据的线性复杂度基础模型，解决了现有大型结构化数据模型（LDMs）的三大局限：样本级自注意力的O(N²)复杂度限制、线性序列模型的表征退化问题，以及纯合成预训练与现实分布不匹配的问题。该模型采用多层双轴架构，以自适应融合双Mamba-2（AFBM）处理局部样本依赖，以卷积门控线性注意力（Conv-GLA）捕获全局记忆，实现了线性复杂度的跨样本建模。在11个真实数据集上的实验表明，FEAT在零样本性能上持续超越基线，推理速度提升高达40倍。

方法概述 / Method

FEAT的核心是混合线性编码的双轴架构：AFBM模块通过状态空间模型捕获样本内的局部依赖关系，Conv-GLA模块利用卷积操作扩展感受野以建模跨样本的全局模式。为增强鲁棒性，模型还整合了混合结构因果模型管道和稳定的重构目标函数进行预训练。

实验结果 / Results

在涵盖医疗、金融、电商和科学数据管理的11个真实世界数据集上，FEAT展现出优异的零样本泛化能力， consistently 优于现有基线方法。关键性能指标显示，该模型在保持线性扩展性的同时，推理速度相比传统方法提升最高达40倍，显著降低了大规模结构化数据的处理成本。

应用价值 / Applications

FEAT可广泛应用于需要处理海量结构化记录的行业场景，如医疗电子健康档案分析、金融风控与欺诈检测、电商平台用户行为预测，以及科学研究中的多源异构数据整合。其线性复杂度特性使企业能够以更低计算成本部署大规模AI决策支持系统，特别适合实时性要求高或资源受限的生产环境。
