CD-FKD: Cross-Domain Feature Knowledge Distillation for Robust Single-Domain Generalization in Object Detection

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对单域泛化目标检测问题，提出了一种跨域特征知识蒸馏方法（CD-FKD），通过全局和实例级特征蒸馏增强学生网络的泛化能力。该方法利用降采样和损坏操作生成多样化数据训练学生网络，同时教师网络接收原始源域数据，使学生网络能够有效模仿教师网络的对象中心特征。实验表明，CD-FKD在目标域泛化和源域性能上均优于现有最先进方法，显著提升了模型对域偏移的鲁棒性。

方法概述 / Method

CD-FKD采用双网络架构，教师网络使用原始源域数据，学生网络则使用经过降采样和损坏处理的多样化数据进行训练。通过全局特征蒸馏和实例级特征蒸馏两种机制，引导学生网络学习教师网络提取的对象中心特征，即使面对难以检测的损坏对象也能保持有效特征提取能力。

实验结果 / Results

在具有挑战性的场景上进行的大量实验表明，CD-FKD在目标域泛化和源域性能两方面均超越了当前最先进的方法。该方法有效验证了其在提升目标检测对域偏移（如天气变化、光照条件和场景差异）鲁棒性方面的有效性。

应用价值 / Applications

该研究在实际应用中具有重要价值，特别适用于自动驾驶和智能监控等场景，这些应用场景需要在多样化环境条件下实现鲁棒的目标检测。
CD-FKD能够确保模型在训练时仅接触单一源域数据的情况下，仍能有效应对真实世界中复杂多变的环境挑战。
