Tokenization Tradeoffs in Structured EHR Foundation Models

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究系统探讨了结构化电子健康记录（EHR）基础模型中的分词策略设计选择。作者通过因子设计在儿科EHR数据上预训练Transformer模型，发现联合事件编码（joint event encoding）和位置时间编码（positional time encoding）在74项临床预测任务中的73项和71项上表现最优，同时分别减少了39.5%和9.6%的预训练计算量。该优势源于局部绑定效率的提升，且在外部成人ICU数据中验证了泛化能力，确立了分词作为同时提升EHR基础模型性能和效率的关键杠杆。

方法概述 / Method

研究采用三因素因子设计，系统变化事件编码（联合编码vs分离编码）、时间编码（位置编码vs显式时间嵌入）和工作流标注（有vs无）三种分词维度。在儿科EHR数据上预训练Transformer后，在74项临床预测任务上评估AUROC性能，并通过针对性消融实验解析机制，最后在外部成人ICU队列进行外部验证。

实验结果 / Results

联合事件编码在73/74项任务中优于分离编码，位置时间编码在71/74项任务中优于显式时间嵌入，分别节省39.5%和9.6%的预训练FLOPs。消融分析表明联合编码的优势来自局部绑定效率——将代码-属性对合并为单一token而非分散在多个token中。外部验证显示该优势可泛化至成人ICU数据，但时间和工作流效应具有机构特异性。

应用价值 / Applications

该研究为EHR基础模型的高效设计提供了可操作的指导原则，特别适用于需要处理大规模纵向临床数据的健康医疗AI系统。研究成果可直接指导临床预测模型的预训练策略优化，在保持或提升预测性能的同时显著降低计算成本，对资源受限的医疗场景和跨机构模型迁移具有重要实践意义。
