Beyond Reward Suppression: Reshaping Steganographic Communication Protocols in MARL via Dynamic Representational Circuit Breaking

论文摘要 / Abstract

本文针对去中心化多智能体强化学习（MARL）中的隐写共谋（steganographic collusion）问题，提出了一种新型架构级防御机制——动态表征断路器（DRCB）。现有防御仅作用于行为层或奖励层，无法检测潜在通信信道中的协调行为；DRCB通过在优化底层进行操作，利用VQ-VAE瓶颈将不可观测消息转化为可审计的统计对象，并基于EMA共谋评分触发四级递进干预。实验表明，DRCB将观察者平均准确率从0.858提升至0.938，同时保持联合奖励不变，并揭示了"语义退化"现象和"透明度悖论"。

方法概述 / Method

DRCB基于AI母语（AIM）框架，采用向量量化变分自编码器（VQ-VAE）瓶颈对智能体间通信进行约束和监控。系统实时追踪Jensen-Shannon散度漂移、L2范数码本位移和随机观察者池准确率等信号，计算指数移动平均（EMA）共谋评分；当阈值被突破时，依次触发动态自适应、优势函数梯度惩罚、时序奖励抑制以及码本重排与优化器状态重置的完全底层断路。

实验结果 / Results

在基于MNIST标签的情境囚徒困境实验中，静态监控方法失效（p=0.3517），而DRCB将观察者平均准确率提升9.3%（从0.858至0.938），并将波动性降低43%，同时保持平均联合奖励不变（p=0.854）。对214,298个符号样本的分析证实了"语义退化"现象：高频序列收敛至零熵，从而阻止复杂隐写编码；研究还发现"透明度悖论"——智能体在表层实现确定性的同时，在长尾分布中保留残余容量。

应用价值 / Applications

该任务无关的方法论为自主系统的MICA合规（多智能体内部耦合审计）预部署审计提供了技术路径，适用于需要严格监控智能体间通信的关键安全场景，如金融交易系统、自动驾驶车路协同、军事指挥控制网络等。通过将不可审计的隐式通信转化为可解释的统计对象，DRCB为构建可信赖的多智能体系统奠定了架构基础，同时其揭示的"透明度悖论"也为AI安全治理提供了重要的理论洞察。
