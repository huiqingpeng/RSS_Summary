论文摘要 / Abstract

本文针对大型语言模型（LLM）的链式思维推理在边缘设备部署中的挑战，提出了一种轻量级推理方案。该方法结合LoRA适配器与监督微调实现小模型推理能力，并通过强化学习的预算强制机制显著缩短响应长度。此外，论文引入并行测试时扩展和动态适配器切换机制，在Qwen2.5-7B上的实验表明，该方案能在严格资源约束下实现高效、准确的边缘推理。

方法概述 / Method

论文采用LoRA适配器结合监督微调的方法为小规模LLM赋予推理能力，并通过强化学习实现预算强制（budget forcing）以控制生成长度。针对内存受限的解码过程，利用并行测试时扩展（parallel test-time scaling）提升准确性；同时设计了动态适配器切换机制和KV缓存共享策略，仅在需要时激活推理模块并优化首token延迟。

实验结果 / Results

在Qwen2.5-7B模型上的实验验证了该方法在严格资源约束下的有效性，能够在最小化准确率损失的前提下显著减少响应长度，并通过并行测试时扩展以较小的延迟代价提升推理准确性。研究团队还提供了在移动设备上运行的演示视频，证明了该方案的实用可行性。

应用价值 / Applications

该研究使复杂的LLM推理能力能够在智能手机等移动边缘设备上高效运行，拓展了AI助手、实时问题求解、离线智能应用等场景的可能性。通过降低token生成成本、减少KV缓存占用和优化推理延迟，该方案为资源受限环境下的端侧智能提供了可行的技术路径，具有重要的商业部署价值和普惠AI应用前景。
