FedSDWC: Federated Synergistic Dual-Representation Weak Causal Learning for OOD

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了FedSDWC，一种用于联邦学习（FL）的因果推理方法，通过整合不变特征和变化特征来推断因果语义表示。该方法通过建模不变特征与变化特征之间的弱因果影响，克服了现有不变学习方法在准确捕捉不变特征和直接构建因果表示方面的局限性，显著提升了联邦学习在分布外（OOD）数据上的泛化能力和检测能力。理论分析推导了FedSDWC的泛化误差界，并首次建立了其与客户端先验分布的关系。在多个基准数据集上的实验表明，FedSDWC在处理协变量偏移和语义偏移方面表现优异，在CIFAR-10和CIFAR-100上分别比次优基线FedICON平均提升3.04%和8.11%。

方法概述 / Method

FedSDWC采用双表示学习框架，同时建模不变特征（invariant features）和变化特征（variant features），并通过捕捉二者之间的弱因果影响来推断因果语义表示。该方法在联邦学习架构下实现，允许各客户端在保护数据隐私的前提下协同训练，同时理论分析提供了泛化误差界与客户端数据分布先验之间的数学关系。

实验结果 / Results

在多个基准数据集上的广泛实验验证了FedSDWC的优越性：在CIFAR-10数据集上平均比次优基线FedICON提升3.04%，在CIFAR-100数据集上提升幅度达8.11%。实验结果表明该方法在处理协变量偏移（covariate shift）和语义偏移（semantic shift）两种典型的分布外场景时均具有出色的泛化性能和OOD检测能力。

应用价值 / Applications

FedSDWC适用于对数据隐私要求严格且面临分布偏移挑战的实际场景，如跨医疗机构的联合诊断系统、多边缘设备的协同感知、以及不同地区金融机构的联合风控等。该方法能够在保护本地数据隐私的同时，有效提升模型在非独立同分布（non-IID）环境下的鲁棒性和泛化能力，为联邦学习在实际部署中的可靠性提供了重要保障。
