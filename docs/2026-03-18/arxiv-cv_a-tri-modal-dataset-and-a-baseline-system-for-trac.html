A Tri-Modal Dataset and a Baseline System for Tracking Unmanned Aerial Vehicles

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文提出了首个大规模多模态无人机跟踪基准数据集MM-UAV，包含RGB、红外（IR）和事件信号三种传感模态，涵盖30余种挑战性场景、1,321组同步多模态序列及超过280万帧标注数据。同时，作者设计了一种专门针对无人机跟踪的多模态多目标跟踪框架，通过偏移引导的自适应对齐模块、自适应动态融合模块以及事件增强的关联机制，有效解决了传感器空间失配、多模态信息互补平衡以及复杂场景下身份保持等关键问题。实验结果表明，该框架在各项性能指标上均优于现有最先进方法。

方法概述 / Method

论文提出的多模态多无人机跟踪框架包含三项核心技术：偏移引导的自适应对齐模块用于校正不同传感器间的空间失配；自适应动态融合模块用于动态平衡各模态传递的互补信息；事件增强的关联机制则利用事件模态的运动线索弥补传统外观建模在多目标跟踪中的局限性，实现更可靠的身份维护。

实验结果 / Results

综合实验表明，所提出的框架在MM-UAV数据集上的各项跟踪指标均持续优于现有最先进方法，验证了多模态融合策略在复杂环境（如低光照、背景杂乱、快速运动）下对无人机跟踪鲁棒性的显著提升。

应用价值 / Applications

该研究为低空无人机安防监控提供了关键技术支持，可广泛应用于机场净空保护、重要设施周界防护、城市低空交通管理等领域。公开的数据集和基线系统也将推动多模态目标跟踪领域的学术研究和技术发展。
