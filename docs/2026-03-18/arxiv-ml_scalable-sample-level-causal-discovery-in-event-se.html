Scalable Sample-Level Causal Discovery in Event Sequences via Autoregressive Density Estimation

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文研究了从单条离散事件序列中进行因果发现的问题，这类数据常见于车辆日志、制造系统或患者轨迹等场景。由于单条观测样本缺乏重复性、高维度以及长程时间依赖等特点，该问题极具挑战性。作者提出了TRACE框架，利用自回归模型作为预训练密度估计器来估计条件互信息，从而推断事件类型间的汇总因果图。该框架在词汇表规模上呈线性扩展，支持延迟因果效应，并可在GPU上完全并行化，理论分析证明了其在不完美自回归模型下的可识别性。

方法概述 / Method

TRACE框架将自回归模型重新定位为预训练密度估计器，用于条件互信息（CMI）的估计，进而基于CMI构建事件类型间的因果图。该方法通过自回归密度估计捕捉事件序列中的复杂时序依赖关系，并支持检测具有时间延迟的因果效应。

实验结果 / Results

实验表明TRACE在不同基线方法和变化的词汇表规模下均表现出稳健的性能。特别地，该方法成功应用于包含超过29,100种事件类型的车辆诊断根因分析任务，验证了其在大规模实际场景中的可扩展性和有效性。

应用价值 / Applications

该研究适用于需要从单条长事件序列中挖掘因果关系的领域，如车辆故障诊断、工业制造系统监控和医疗患者轨迹分析等。
TRACE的线性扩展性和GPU并行能力使其能够处理大规模事件词汇表，为复杂系统的实时根因分析和决策支持提供了实用工具。
