A Practical Algorithm for Feature-Rich, Non-Stationary Bandit Problems

arXiv Machine Learning 2026-03-18

论文摘要 / Abstract

本文提出了一种更现实的上下文老虎机问题设定，融合了密集臂特征、非线性奖励函数以及奖励分布随时间变化但相关性保持的动态相关老虎机。为此，作者设计了条件耦合上下文C3汤普森采样算法（C3 Thompson Sampling），该算法在嵌入空间上使用改进的Nadaraya-Watson估计器结合汤普森采样，实现无需重新训练的在线学习。实验表明，C3在四个OpenML表格数据集上平均累积遗憾比次优算法低5.7%，在Microsoft新闻数据集（MIND）上点击率提升12.4%。

方法概述 / Method

C3算法核心在于将改进的Nadaraya-Watson核估计器应用于学习得到的嵌入空间，以捕捉臂特征与奖励之间的非线性关系；同时采用汤普森采样进行探索-利用权衡，支持在线增量更新而无需离线重训练。该方法通过条件耦合机制显式建模臂间相关性，并适应非平稳环境下的奖励分布变化。

实验结果 / Results

在四个OpenML表格数据集的对比实验中，C3实现了5.7%的平均累积遗憾降低；在真实场景的大规模新闻推荐数据集MIND上，C3相较其他基线算法取得了12.4%的点击率相对提升，验证了其在特征丰富、非平稳环境下的优越性能。

应用价值 / Applications

该研究直接适用于个性化推荐系统（如新闻、商品推荐），能够有效处理用户偏好随时间演化的动态场景；同时可扩展至在线广告、临床实验设计等需要平衡探索利用且特征维度高、环境非平稳的决策任务，为工业级部署提供了计算高效的实用解决方案。
