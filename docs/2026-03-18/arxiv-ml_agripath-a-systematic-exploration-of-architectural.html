AgriPath: A Systematic Exploration of Architectural Trade-offs for Crop Disease Classification

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究系统性地比较了三种模型范式（CNN、对比式视觉-语言模型、生成式视觉-语言模型）在细粒度作物病害分类任务中的表现。作者构建了AgriPath-LF16基准数据集，包含11.1万张图像，涵盖16种作物和41种病害，并明确区分实验室与田间图像。实验发现CNN在实验室图像上准确率最高但域迁移性能下降，对比式VLM提供参数高效的跨域鲁棒替代方案，而生成式VLM对分布变化最具韧性但存在自由文本生成的额外失效模式。

方法概述 / Method

研究采用统一协议对三类模型进行训练和评估：CNN（EfficientNet、ResNet等）、对比式VLM（CLIP风格）和生成式VLM（如LLaVA）。通过全数据、仅实验室、仅田间三种训练设置，使用macro-F1和解析成功率（PSR）指标量化性能，特别针对生成模型的可靠性设计了PSR指标以处理自由文本输出的解析问题。

实验结果 / Results

CNN在实验室环境下表现最优，但面对田间图像时性能显著退化；对比式VLM以较少的参数量实现了 competitive 的跨域泛化能力；生成式VLM展现出最强的分布鲁棒性，但其自由文本生成特性引入了新的失败模式（如幻觉或格式不一致）。结果表明单一聚合准确率无法全面反映模型在实际部署中的适用性。

应用价值 / Applications

该研究为农业AI系统的实际部署提供了架构选择的决策依据：实验室环境优先选用CNN，资源受限且需跨域泛化时推荐对比式VLM，而复杂多变的田间场景可考虑生成式VLM（需配合输出验证机制）。
AgriPath-LF16基准填补了现有作物病害数据集缺乏域分离控制的空白，支持更可靠的模型开发与公平比较。
