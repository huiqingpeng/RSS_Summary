Auto Researching, not hyperparameter tuning: Convergence Analysis of 10,000 Experiments

arXiv Machine Learning 2026-03-18
Read Original Article →

论文摘要 / Abstract

本研究通过分析10,469个由Claude Opus和Gemini 2.5 Pro两个LLM智能体在27天内自主设计的机器学习实验，探究LLM智能体是否真正进行架构搜索而非局限于超参数调优。研究发现架构选择解释了94%的性能方差，而超参数变化仅占6%，证实智能体确实发现了人类未提出的新架构（V-JEPA 2视频特征+Zipformer时序编码器，AP达0.9245）。
LLM引导的搜索在50次实验后即显著优于随机搜索（AP 0.985 vs 0.965），且收敛遵循幂律分布，为LLM引导的组合式机器学习实验设计提供了首个大规模实证框架。

方法概述 / Method

研究构建了包含108,000个离散配置单元的组合式配置空间，用于行车记录仪碰撞检测任务，并部署两个LLM智能体自主设计实验。采用方差分析（ANOVA）分解性能方差的来源，量化架构选择与超参数调优的贡献比例；同时引入熵循环和Jensen-Shannon散度刻画多智能体搜索动态与专业化分工。

实验结果 / Results

ANOVA分析显示架构选择对性能方差的解释力高达94%（F=1324，η²=0.94），远超超参数的6%；跨任务验证在第二个碰撞数据集上仍保持75%的架构解释方差，且获胜骨干网络不同，证实架构发现的泛化性。
LLM引导搜索在N=50时AP达0.985，显著优于从零开始的随机搜索（0.965）；修复漏洞后的收敛遵循幂律分布（c=0.11，R²=0.93），低指数反映广泛探索的成本而非低效。

应用价值 / Applications

该研究为自动化机器学习（AutoML）领域提供了从"超参数调优"迈向"自主研究"的范式转变证据，可应用于视频理解、自动驾驶等需要复杂架构设计的任务。研究成果支持开发更具探索性的LLM智能体实验设计系统，减少人工架构工程依赖，并为评估LLM驱动的科学发现能力建立了可量化的实证基准。
