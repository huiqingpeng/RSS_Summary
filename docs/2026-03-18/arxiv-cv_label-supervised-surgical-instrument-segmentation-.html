Label-supervised surgical instrument segmentation using temporal equivariance and semantic continuity

arXiv Computer Vision 2026-03-18
Read Original Article →

论文摘要 / Abstract

本文针对机器人手术视频中仅使用器械存在标签（instrument presence labels）进行弱监督分割的问题，提出了一种利用时间特性的两阶段分割方法。核心贡献包括：引入时间等变性约束（temporal equivariance）增强相邻帧特征的像素级时间一致性；设计类别感知语义连续性约束（class-aware semantic continuity）连接全局与局部区域的时间维度信息；生成时间增强的伪标签以抑制无关区域。在两个手术视频数据集（胆囊切除术基准数据集和真实机器人左外侧肝段切除术数据集）上的实验表明，该方法达到了与全监督方法相当甚至更优的性能。

方法概述 / Method

论文采用两阶段弱监督分割范式：第一阶段通过时间等变性约束强制相邻帧特征在像素级别保持时间一致性，同时利用类别感知语义连续性约束确保全局图像与局部区域在时间维度上的语义连贯性；第二阶段基于连续帧生成时间增强的伪掩膜（pseudo masks），通过聚合多帧信息来过滤背景噪声并精炼目标区域，最终实现像素级分割。

实验结果 / Results

实验在两个数据集上进行验证：公开基准数据集CholecSeg8k（胆囊切除术）和作者新标注的真实机器人左外侧肝段切除术数据集（由3年临床经验的医生双检标注）。结果表明，该方法在仅使用器械存在标签的弱监督设置下，分割性能持续达到或优于先前最先进的弱监督和半监督方法，显著降低了人工像素级标注成本。

应用价值 / Applications

该研究适用于机器人辅助手术中的实时器械跟踪与分割场景，可在仅利用手术视频自带的存在标签（无需昂贵的像素级标注）的情况下实现高质量分割，大幅降低临床数据标注成本。其时间建模策略对其他时序医学影像分析任务（如内镜视频分析、超声序列分割）具有借鉴意义，有助于推动手术自动化和智能辅助系统的临床部署。
