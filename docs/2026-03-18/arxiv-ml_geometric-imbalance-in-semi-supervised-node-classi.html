论文摘要 / Abstract

本文首次正式提出"几何不平衡"（geometric imbalance）概念，用于刻画类别不平衡图数据中消息传递导致少数类节点在黎曼流形嵌入空间中出现几何模糊的现象。作者从理论上严格分析了黎曼流形上的几何不平衡问题，并提出了一个统一框架，通过伪标签对齐、节点重排序和模糊过滤三种机制来缓解该问题。在多个基准数据集上的大量实验表明，该方法在严重类别不平衡场景下显著优于现有方法，为鲁棒的半监督节点分类提供了新的理论见解和实用工具。

方法概述 / Method

论文基于黎曼流形理论对图神经网络中的几何不平衡进行形式化建模，分析类别不平衡如何通过消息传递机制导致少数类节点在嵌入空间中产生几何重叠和模糊。提出的统一缓解框架包含三个核心组件：伪标签对齐（利用高置信度伪标签校准决策边界）、节点重排序（优化消息传递顺序以减少噪声传播）、以及模糊过滤（识别并抑制几何位置模糊的节点）。

实验结果 / Results

实验在多个具有不同类别不平衡程度的图基准数据集上进行，结果表明所提方法在严重不平衡场景（如少数类比例低于10%）下 consistently 优于现有最先进的重采样、重加权和图结构学习方法。具体而言，该方法在宏平均F1分数和少数类召回率上实现了显著提升，且对不同的图神经网络骨干架构（如GCN、GAT）均表现出良好的兼容性和稳定性。

应用价值 / Applications

该研究对实际图数据挖掘应用具有重要价值，特别是在天然存在严重类别不平衡的领域，如金融欺诈检测（欺诈交易极少）、生物信息学（罕见疾病基因识别）、社交媒体分析（虚假信息传播者识别）和网络安全（入侵检测中的新型攻击模式发现）。所提框架可直接集成到现有图神经网络管道中，无需修改网络架构，为工业界部署鲁棒的半监督图学习系统提供了即插即用的解决方案。
