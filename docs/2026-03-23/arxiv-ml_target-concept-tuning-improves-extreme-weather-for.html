Target Concept Tuning Improves Extreme Weather Forecasting

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文提出TaCT（Target Concept Tuning），一种可解释的概念门控微调框架，用于解决深度学习气象预报模型在台风等极端天气事件中表现不佳的问题。该方法通过稀疏自编码器和反事实分析自动发现与失败相关的内部概念，仅在这些概念激活时更新参数，实现对极端事件的针对性改进而保持常规场景性能。实验表明该方法在多个地区的台风预报中取得一致提升，且发现的概念对应具有物理意义的环流模式。

方法概述 / Method

TaCT采用两阶段策略：首先利用稀疏自编码器（SAE）从预训练模型中提取可解释的内部概念表征，然后通过反事实分析识别与极端天气预报失败相关的关键概念；在微调阶段，采用概念门控机制实现选择性参数更新——仅当特定失败相关概念被激活时才调整模型参数，而非均匀地对所有样本进行适应。

实验结果 / Results

实验显示TaCT在不同地区的台风预报任务中均实现一致的性能提升，同时不损害其他气象变量的预报准确度；所识别出的概念对应于具有物理可解释性的环流模式（如特定的气压场配置），揭示了模型的内在偏差，为科学预报任务中的可信适应提供了支持。

应用价值 / Applications

该研究对高影响低概率的极端天气预警具有重要价值，可提升台风路径和强度预报的可靠性，助力防灾减灾决策；其概念驱动的选择性适应框架具有通用性，可拓展至其他稀有事件预测场景（如地震、洪水预报），并为深度学习模型在关键科学领域的可信部署提供方法论支撑。
