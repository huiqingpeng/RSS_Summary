NASimJax: GPU-Accelerated Policy Learning Framework for Penetration Testing

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文提出了NASimJax，一个基于JAX的网络攻击模拟器（NASim）完整重构框架，实现了比原始模拟器高达100倍的环境吞吐率，解决了渗透测试领域强化学习训练速度慢的瓶颈问题。作者将自动化渗透测试形式化为上下文部分可观察马尔可夫决策过程（Contextual POMDP），并设计了结构多样且保证可解的网络生成流程，为研究零样本策略泛化提供了理论基础。通过实验发现，优先级别回放（Prioritized Level Replay）在密集训练分布上优于域随机化，稀疏拓扑训练可形成隐式课程学习以提升分布外泛化能力，而提出的两阶段动作分解（2SAS）方法能有效应对大规模线性增长的动作空间。

方法概述 / Method

NASimJax采用JAX框架对整个训练流程进行GPU加速重构，使环境模拟、策略网络和价值网络计算全部在硬件加速器上运行。研究将渗透测试建模为Contextual POMDP，并开发了网络拓扑生成管道以创建多样化、可控且保证存在解的训练场景。针对动作空间随主机数量线性增长的挑战，提出了两阶段动作分解（2SAS）架构，将动作选择分解为"选择目标主机"和"选择对该主机的操作"两个层级，替代传统的扁平动作掩码方法。

实验结果 / Results

实验在多达40台主机的网络规模上验证了框架的可扩展性，NASimJax相比原始NASim实现了最高100倍的吞吐量提升。结果表明Prioritized Level Replay在处理密集训练分布时显著优于Domain Randomization，尤其在更大规模网络中优势更明显。在泛化能力方面，稀疏拓扑训练产生的隐式课程使策略能够泛化到比训练时更密集的拓扑结构。2SAS在大规模动作空间下大幅优于扁平动作掩码基线，但研究也发现Prioritized Level Replay的回合重置机制与2SAS的信用分配结构之间存在不良交互导致的失效模式。

应用价值 / Applications

NASimJax为基于强化学习的自动化渗透测试提供了一个高速、灵活且逼真的研究平台，使先前因计算成本过高而不可行的大规模网络实验成为可能。该框架可直接用于开发能够零样本泛化到新网络环境的智能渗透测试代理，减少安全专家手动测试的工作量。研究成果对网络安全防御也具有参考价值，可用于评估和增强网络架构对AI驱动攻击的鲁棒性，推动攻防智能化研究的发展。
