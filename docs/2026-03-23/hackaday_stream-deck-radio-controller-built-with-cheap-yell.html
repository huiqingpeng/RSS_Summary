This is a project article about a DIY Stream Deck controller. The author [WhiskeyTangoHotel] built a wireless controller using the inexpensive "Cheap Yellow Display" (a combination of ESP32 microcontroller and 2.8-inch touchscreen LCD) specifically to control RIT/XIT functions on the Flex-6400 software-defined radio.

为什么重要 / Why it matters

这个项目展示了如何用极低成本（约$10-15的模块）替代昂贵的商业Stream Deck（通常$100+），同时实现无线控制和触屏交互。它体现了开源硬件和微控制器普及化带来的"够用就好"工程哲学，对业余无线电爱好者和DIY玩家具有参考价值。
This project demonstrates how to replace expensive commercial Stream Decks (typically $100+) with extremely low-cost components (around $10-15 per module) while achieving wireless control and touchscreen interaction. It embodies the "good enough" engineering philosophy enabled by open-source hardware and microcontroller democratization, offering valuable insights for ham radio enthusiasts and DIY makers.
