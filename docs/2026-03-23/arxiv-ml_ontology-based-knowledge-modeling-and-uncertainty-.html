Ontology-Based Knowledge Modeling and Uncertainty-Aware Outdoor Air Quality Assessment Using Weighted Interval Type-2 Fuzzy Logic

论文摘要 / Abstract

本研究针对传统印度空气质量指数（IND-AQI）采用确定性阈值和聚合规则、无法有效处理类别边界不确定性的问题，提出了一种融合加权区间二型模糊逻辑与语义知识建模的混合本体框架。该框架利用区间二型模糊集建模AQI类别边界附近的不确定性，并通过区间二型模糊层次分析法（IT2-FAHP）确定污染物权重以反映其相对健康影响，同时构建了基于OWL的空气质量本体扩展语义传感器网络（SSN）本体。实验结果表明，该框架相比传统清晰集和一型模糊方法显著提升了AQI分类可靠性和不确定性处理能力，并支持可解释的语义推理与智能决策支持。

方法概述 / Method

论文采用加权区间二型模糊逻辑系统处理AQI评估中的不确定性，其中区间二型模糊集用于捕捉类别边界的不确定性，IT2-FAHP用于动态确定污染物重要性权重。同时，研究开发了基于OWL的空气质量本体，扩展SSN本体以表示污染物、监测站、AQI类别、监管标准及环境治理措施，并通过SWRL规则实现语义推理，利用SPARQL查询进行验证。

实验结果 / Results

基于印度中央污染控制委员会（CPCB）空气质量数据集的实验评估显示，所提框架在AQI分类可靠性和不确定性处理方面均优于传统清晰集方法和一型模糊方法。该框架能够有效处理类别边界处的过渡不确定性，并通过语义推理实现AQI类别、健康风险及缓解措施的智能推断。

应用价值 / Applications

该研究为空气质量监测系统提供了可解释的智能决策支持工具，适用于快速城市化地区的环境管理和公共健康预警。其本体驱动的方法支持监管标准的语义化表达与推理，可整合至智慧城市环境监测平台，为政策制定者和公众提供透明、可靠的空气质量评估与应对建议。
