Adaptive Layerwise Perturbation: Unifying Off-Policy Corrections for LLM RL

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文针对大语言模型强化学习（LLM RL）中的离策略问题（如策略陈旧性和训练-推理不匹配）提出了一种统一解决方案。作者提出**自适应层间扰动（Adaptive Layerwise Perturbation, ALP）**，通过在每层输入隐藏状态中注入可学习的小扰动，有效收紧更新策略与推理策略之间的分布差距，减少重要性比率的厚尾现象。实验表明，ALP不仅提升了单轮数学推理和多轮工具集成推理任务的最终性能，还避免了迭代训练中的KL散度激增问题，同时增强了探索能力。

方法概述 / Method

ALP的核心机制是在策略更新时，向Transformer各层的输入隐藏状态注入可学习的微小扰动，并将扰动后的策略作为重要性比率的分子，与保持不变的推理策略（分母）形成对比。这种表示层级的受控噪声注入防止了更新策略过度偏离推理策略，通过展平策略分布来扩大策略族覆盖范围，从而自然缩小训练-推理差距。

实验结果 / Results

在单轮数学推理（如GSM8K）和多轮工具集成推理任务上，ALP显著提升了最终性能，同时有效抑制了重要性比率的尾部膨胀和KL散度尖峰。消融实验表明，跨所有层的表示级扰动效果最佳，明显优于仅扰动部分层或仅扰动logits的变体方案，验证了层间扰动的必要性。

应用价值 / Applications

ALP可广泛应用于需要高效推理的LLM RL场景，如实时对话系统、工具调用代理和复杂推理任务，在保持训练稳定性的同时提升推理效率。该方法为工业级大模型迭代训练提供了实用的稳定性保障，特别是在资源受限、需要频繁策略更新的生产环境中具有重要价值。
