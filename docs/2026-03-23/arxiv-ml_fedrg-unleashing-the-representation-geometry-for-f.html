FedRG: Unleashing the Representation Geometry for Federated Learning with Noisy Clients

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文针对联邦学习（FL）中分布式场景下噪声标注导致性能下降的问题，提出了一种基于表示几何的新方法FedRG。该方法遵循"表示几何优先"原则，通过自监督学习构建标签无关的球面表示，并迭代拟合球面von Mises-Fisher混合模型来识别噪声样本，同时结合语义-标签软映射机制实现鲁棒的噪声标签修正。大量实验表明，FedRG在数据异构性和多样化噪声客户端场景下显著优于现有最先进方法。

方法概述 / Method

FedRG首先利用自监督学习创建标签无关的球面表示，然后基于已识别的干净样本迭代拟合球面von Mises-Fisher（vMF）混合模型以捕获语义聚类结构。该方法将几何证据与语义-标签软映射机制相结合，计算无标签特征空间与标注标签条件特征空间之间的分布散度，从而鲁棒地识别噪声样本并更新vMF模型，最后通过个性化的噪声吸收矩阵实现鲁棒优化。

实验结果 / Results

实验结果表明，FedRG在多种数据异构性和噪声客户端场景下均显著优于现有最先进方法，有效解决了基于标量损失的噪声样本识别在联邦学习异构场景中可靠性不足的问题，展现出强大的噪声标签识别与修正能力。

应用价值 / Applications

该研究适用于医疗、金融等隐私敏感领域的分布式机器学习场景，能够在保护数据隐私的同时有效处理各参与方标注质量参差不齐的问题。
FedRG为构建鲁棒的联邦学习系统提供了新思路，特别是在标注噪声难以避免的实际应用中具有重要价值。
