论文摘要 / Abstract

本文建立了神经网络学习与抽象代数结构之间的理论联系。研究首先展示了一个极小反例，证明标准神经网络在组合泛化任务上完全失效（0%准确率）；通过引入"三元Gamma半环"逻辑约束，相同架构能学习完美结构化的特征空间，在新组合上达到100%准确率。作者证明该学习到的特征空间构成有限交换三元Γ-半环，其三元运算实现多数投票规则，并揭示了神经网络成功源于对数学"自然"结构的近似、泛化能力来自内化代数公理、逻辑约束引导网络收敛到规范形式这三大结论。

方法概述 / Method

本文采用理论分析与实验验证相结合的方法：首先构造组合泛化任务的极小反例验证标准神经网络的失效，然后引入Ternary Gamma Semiring作为逻辑约束嵌入网络训练，最后运用抽象代数工具（特别是Gokavarapu等人的三元Γ-半环分类理论）对学到的特征空间进行严格的数学刻画和同构分类。

实验结果 / Results

关键实验结果显示：未加约束的标准神经网络在组合泛化任务上准确率为0%，而引入Ternary Gamma Semiring约束后相同架构达到100%准确率；学习到的特征空间被严格证明构成有限交换三元Γ-半环，其三元运算对应多数投票规则，且与Gokavarapu等人枚举中的布尔型三元Γ-半环（|T|=4, |Γ|=1）精确对应，在同构意义下唯一。
