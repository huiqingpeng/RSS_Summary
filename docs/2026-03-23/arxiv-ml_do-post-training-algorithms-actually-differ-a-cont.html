Do Post-Training Algorithms Actually Differ? A Controlled Study Across Model Scales Uncovers Scale-Dependent Ranking Inversions

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本研究通过OXRL统一框架对51种后训练算法进行大规模控制变量比较，首次系统评估了8种主流算法在4个模型规模（0.5B-7B）上的表现。研究发现算法排名随模型规模发生显著反转：1.5B参数时在线强化学习（SGRPO）最优，而7B参数时小规模表现最差的SimPO反而最佳。此外，20种DPO变体均未显著超越基础DPO，且算法效果差异高度依赖任务类型——数学推理任务上的19.3个百分点差距在通用基准上压缩至0.47个百分点。研究最终确立了实践优先级：模型规模（~50pp）>> 训练范式（~10pp）>> 在线vs离线（~9pp）>> 损失函数（~1pp）。

方法概述 / Method

作者开发了OXRL统一框架，在完全相同的计算基础设施下实现51种后训练算法，确保"苹果对苹果"的比较。实验采用2×2析因设计分离模型规模与LoRA正则化的影响，涵盖8种算法、4个模型规模、3个评估领域，并在1.5B规模上进行100次运行（5个随机种子）以检验统计显著性，总计约240次完整训练运行。

实验结果 / Results

核心发现包括：（1）规模依赖的排名反转：GSM8K任务上，1.5B时SGRPO以58.0%±0.57领先，7B时SimPO跃升至85.8%成为最优；（2）损失函数修改无效：经Bonferroni校正后，20种DPO变体均无显著提升，唯一显著异常值SimPO反而更差（-11.5pp, p<10⁻⁴）；（3）任务特异性杠杆效应：算法差距在MATH任务上缩小36倍（至0.54pp），在通用基准上缩小41倍（至0.47pp）。

应用价值 / Applications

本研究为LLM后训练实践提供了明确的决策层级，帮助从业者在有限资源下优先投入模型扩展而非算法调优。
OXRL框架及开源的代码、配置和评估数据可作为动态社区基准，支持新算法的公平比较与复现。研究结果特别警示：小规模实验得出的算法结论可能无法推广到更大模型，需在目标规模上直接验证。
