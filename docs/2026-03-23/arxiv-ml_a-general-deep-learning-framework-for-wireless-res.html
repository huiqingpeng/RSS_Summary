论文摘要 / Abstract

本文针对无线资源分配中离散变量优化的挑战，提出了一种通用的深度学习框架。该框架通过引入支持集（support set）表示离散变量，并将其元素建模为随机变量来学习联合概率分布，通过条件概率的因式分解和顺序学习，有效解决了反向传播中的零梯度问题、复杂离散约束的强制执行难题，以及非相同参数相同决策（non-SPSD）属性的缺失问题。该框架在两个典型的混合离散无线资源分配问题上进行了验证，均显著优于现有基线方法。

方法概述 / Method

论文提出了一种基于概率建模的深度学习框架，将离散变量的支持集元素视为随机变量，通过将联合概率分解为条件概率的乘积进行顺序学习。该框架采用动态上下文嵌入（dynamic context embedding）来捕获不断演化的离散决策，从而在概率分布层面而非硬二进制决策上进行操作，实现对离散变量的可微分优化。

实验结果 / Results

该框架在两个代表性问题上进行了验证：（a）无蜂窝系统中的联合用户关联与波束成形，以及（b）可移动天线辅助系统中的联合天线位置优化与波束成形。仿真结果表明，所提出的深度学习方法在系统性能和计算效率方面均持续优于现有基线方法。

应用价值 / Applications

该研究为5G/6G无线网络中的复杂资源分配问题提供了高效的求解工具，特别适用于需要同时优化离散决策（如用户关联、天线选择/位置）和连续变量（如波束成形权重）的场景。其通用框架可扩展至其他具有混合离散-连续约束的无线通信优化问题，为实时网络优化和智能化资源管理提供了可行的技术路径。
