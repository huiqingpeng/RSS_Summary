GeoLAN: Geometric Learning of Latent Explanatory Directions in Large Language Models

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文提出GeoLAN，一种将token表示视为几何轨迹的训练框架，通过受Kakeya猜想启发的"粘性条件"来增强大型语言模型的可解释性。该框架开发了两种可微正则化器（KT-CW和KT-Attn），以促进表示空间的各向同性和注意力多样性。实验表明，GeoLAN在保持任务准确性的同时，能改善几何指标并减少某些公平性偏见，且在中等规模模型上效果最为显著。

方法概述 / Method

GeoLAN将LLM中的token表示建模为几何轨迹，并引入受Kakeya猜想相关研究启发的几何约束条件。具体实现包括Katz-Tao Convex Wolff (KT-CW)正则化器——用于促进表示空间的各向同性，以及Katz-Tao Attention (KT-Attn)正则化器——用于鼓励注意力头的多样性，两者均可端到端微分并集成到标准训练流程中。

实验结果 / Results

在Gemma-3（1B/4B/12B）和Llama-3-8B上的实验显示，GeoLAN能够在维持下游任务准确率的同时，显著提升几何结构指标（如表示空间的均匀性）并降低特定公平性偏见。研究发现存在规模相关的权衡效应：中等规模模型（约4B-8B参数）从几何感知训练中获益最大，而极小和极大模型则表现出不同的优化动态。

应用价值 / Applications

该研究为提升LLM的机械可解释性提供了新的几何视角，可应用于需要模型透明度的关键场景，如高风险决策支持、算法公平性审计和模型调试。
GeoLAN的训练框架可直接集成到现有预训练或微调流程中，为构建更可解释、更公平的AI系统提供实用工具，特别适用于对偏见敏感的应用领域（如招聘、信贷评估和医疗诊断）。
