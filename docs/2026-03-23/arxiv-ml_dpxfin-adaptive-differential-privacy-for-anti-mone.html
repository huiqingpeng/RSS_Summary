DPxFin: Adaptive Differential Privacy for Anti-Money Laundering Detection via Reputation-Weighted Federated Learning

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文提出了DPxFin，一种面向反洗钱检测的新型联邦学习框架，通过引入声誉引导的自适应差分隐私机制，在保护金融数据隐私的同时提升模型效用。该方法根据客户端本地模型与全局模型的一致性动态计算声誉分数，并据此分配差异化的隐私噪声水平。实验表明，DPxFin在独立同分布（IID）和非独立同分布（non-IID）设置下均优于传统联邦学习和固定噪声差分隐私基线方法，并能有效抵御表格数据泄露攻击。

方法概述 / Method

DPxFin采用声誉加权的自适应差分隐私策略，通过量化客户端模型更新与全局模型的对齐程度来计算声誉分数。基于该声誉机制，框架为高声誉客户端分配较低噪声以放大其可信贡献，同时为低声誉客户端注入更强噪声以抑制潜在风险，从而实现隐私保护与模型性能的动态平衡。

实验结果 / Results

研究在反洗钱数据集上使用多层感知机（MLP）进行验证，结果显示DPxFin在准确性与隐私性的权衡上表现更优，性能提升虽幅度适中但具有一致性。此外，该框架成功抵御了针对表格数据的隐私泄露攻击，证明了其在真实金融场景中的鲁棒性。

应用价值 / Applications

DPxFin适用于银行、支付机构等金融机构的联合反洗钱监测场景，可在不共享原始交易数据的前提下实现跨机构协作建模。该研究为金融隐私计算提供了实用的技术方案，有助于解决数据孤岛与监管合规的双重挑战，推动联邦学习在敏感金融领域的落地应用。
