FedPDPO: Federated Personalized Direct Preference Optimization for Large Language Model Alignment

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文提出了FedPDPO，一种用于大语言模型偏好对齐的联邦个性化直接偏好优化框架。针对联邦学习中偏好数据去中心化、隐私敏感且高度非独立同分布（non-IID）的挑战，该框架通过参数高效的微调架构和个性化训练策略，解决了DPO在联邦环境下性能退化的问题。实验表明，该方法在联邦域内和跨域设置中实现了最高4.80%的平均准确率提升，达到最先进的性能水平。

方法概述 / Method

FedPDPO采用LoRA适配器进行参数高效微调，每个客户端维护冻结的预训练LLM主干和可训练的LoRA适配器以实现通信高效的聚合。为应对非IID异质性，方法设计了三个核心组件：（1）全局共享LoRA适配器结合个性化客户端特定LLM头；（2）带客户端显式奖励头的个性化DPO训练策略以补充隐式奖励；（3）瓶颈适配器用于平衡全局与局部特征。论文还提供了理论分析以建立概率基础和合理性证明。

实验结果 / Results

在多个偏好数据集上的大量实验验证了FedPDPO的有效性，该方法在联邦域内和跨域设置中均达到最先进的性能，平均准确率提升高达4.80%。实验结果证明了该框架在处理非IID偏好数据和提升模型泛化能力方面的显著优势。

应用价值 / Applications

FedPDPO适用于需要在保护用户隐私前提下进行大语言模型个性化对齐的场景，如医疗、金融等敏感领域的对话系统优化。该方法通过联邦学习实现分布式偏好数据的高效利用，同时保持模型个性化能力，为构建隐私保护且用户定制的AI助手提供了可行方案，具有重要的实际部署价值。
