Any-Subgroup Equivariant Networks via Symmetry Breaking

论文摘要 / Abstract

本文提出了Any-Subgroup Equivariant Network (ASEN)，这是一种能够通过调节辅助输入特征同时对多个群保持等变性的单一模型。该方法从完全置换等变的基础模型出发，利用具有特定自同构群的对称破缺输入来获得子群等变性，并通过2-闭包概念将近似对称破缺转化为快速算法。理论证明该网络可模拟等变MLP且保持通用性，实验验证了其在图、图像及序列任务上的优越性。

方法概述 / Method

ASEN的核心方法是以完全置换等变网络为基础架构，通过引入一个对称破缺输入（其自同构群为目标子群）来诱导子群等变性。为解决精确寻找特定自同构群输入的计算困难，作者采用近似对称破缺策略，基于2-闭包理论设计高效算法，使模型能够通过调节这一辅助输入灵活切换不同的等变约束。

实验结果 / Results

实验表明，单一ASEN网络在多个置换子群上的等变性能优于独立的专用等变模型和非等变模型。具体验证包括：图和图像任务中的对称选择、序列任务的多任务学习与迁移学习，证明了该统一架构在处理多样化几何数据时的有效性和灵活性。

应用价值 / Applications

ASEN为构建灵活的多模态基础模型提供了新途径，可处理网格、集合、图等多种几何数据而无需为每种对称性单独设计架构。该研究在需要统一处理异构数据的场景（如跨域迁移学习、多任务学习系统）中具有重要应用价值，有望推动等变神经网络向更通用、可扩展的方向发展。
