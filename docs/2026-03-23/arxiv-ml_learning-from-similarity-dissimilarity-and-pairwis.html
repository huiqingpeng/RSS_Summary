Learning from Similarity/Dissimilarity and Pairwise Comparison

arXiv Machine Learning 2026-03-23

论文摘要 / Abstract

本文针对难以获取实例级显式标签的二分类场景，提出了一种仅依赖相对判断的弱监督学习框架SD-Pcomp。该方法利用相似性/差异性（SD）标签和成对比较（Pcomp）标签这两种二元判断形式的弱标签，开发了两种无偏风险估计器：SD与Pcomp的凸组合估计器，以及通过建模两者关系得到的统一估计器。理论分析和实验结果表明，该方法优于单一弱标签方法，且对标签噪声和类先验估计不确定性具有鲁棒性。

方法概述 / Method

SD-Pcomp框架采用两类二元弱监督信号：SD标签判断两个实例是否属于同一类别，Pcomp标签判断哪个实例更可能属于正类。作者设计了两种无偏风险估计器——简单的凸组合估计器和利用SD与Pcomp标签间统计关系构建的统一估计器，将成对相对判断转化为可训练的分类目标。

实验结果 / Results

实验表明，结合SD和Pcomp两种标签的方法显著优于仅使用单一弱标签的基线方法；所提方法在标签噪声和类先验估计存在偏差的情况下仍保持稳定的分类性能，验证了其对监督信号不确定性的鲁棒性。

应用价值 / Applications

该研究适用于标注成本高昂或难以获取精确概率标签的实际场景，如医学影像诊断、众包标注和内容审核等领域，其中标注者更容易做出"哪个更相似"或"哪个更相关"的相对判断，而非提供精确的置信度分数。
