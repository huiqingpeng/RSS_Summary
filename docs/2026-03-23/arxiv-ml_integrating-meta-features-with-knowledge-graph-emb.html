Integrating Meta-Features with Knowledge Graph Embeddings for Meta-Learning

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文提出了KGmetaSP，一种基于知识图谱嵌入的元学习方法，通过整合现有实验数据来捕捉数据集与机器学习流水线之间的交互关系。该方法构建了统一的知识图谱表示数据集和流水线，并生成支持流水线无关的元模型进行性能估计（PPE）以及基于距离检索的相似性估计（DPSE）。研究基于144,177个OpenML实验构建了大规模基准测试，验证了KGmetaSP在两项核心元学习任务上的有效性，并开源了方法、知识图谱和基准数据集。

方法概述 / Method

KGmetaSP将数据集和机器学习流水线建模为统一知识图谱中的实体，利用图嵌入技术学习低维向量表示。这些嵌入同时编码了数据集元特征和丰富的实验历史信息，支持构建与具体流水线无关的元模型，并可通过向量空间距离度量数据集间的性能相似性。

实验结果 / Results

在包含超过14万条实验记录的大规模基准上，KGmetaSP实现了准确的流水线性能估计，仅需单个流水线无关的元模型即可替代传统方法。在数据集性能相似性估计任务中，KGmetaSP显著优于仅依赖元特征的基线方法，有效捕捉了数据集-流水线交互所揭示的性能模式。

应用价值 / Applications

该研究为AutoML系统提供了更高效的元学习基础，可加速新数据集上的算法选择和超参数优化。通过整合开放实验数据构建统一知识图谱，该方法为研究社区建立了新的元学习参考基准，有助于促进实验知识的复用和协作式机器学习研究。
