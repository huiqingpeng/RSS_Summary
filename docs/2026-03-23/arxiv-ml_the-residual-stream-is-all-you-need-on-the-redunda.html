The Residual Stream Is All You Need: On the Redundancy of the KV Cache in Transformer Inference

arXiv Machine Learning 2026-03-23
Read Original Article →

论文摘要 / Abstract

本文证明Transformer推理中的KV缓存是完全冗余的：每层的键值向量都是残差流的确定性投影，从单个残差向量重新计算可实现比特级精确重建而非近似。研究者在四个架构家族的六个模型（135M至4B参数）上验证了这一理论，发现残差流满足马尔可夫性质且是唯一的信息载体状态。基于此，他们提出了KV-Direct方案，用残差向量检查点替代完整KV缓存，在20轮对话中将峰值内存从103MB降至42MB，同时保持100%的token匹配率。

方法概述 / Method

论文首先通过数学证明和跨层残差补丁实验验证残差流的充分性，即每层KV可由当前层残差向量确定性重构；随后提出KV-Direct推理框架，仅缓存每token的残差向量（Gemma 3-4B上仅5KB），在需要时动态重新计算键值，取代传统的KV缓存存储机制。

实验结果 / Results

在六个模型上的贪婪解码测试显示，完全移除缓存并重新计算可产生与原始模型完全相同的token输出；KV-Direct在20轮对话中峰值内存仅为42MB，而标准缓存增长至103MB以上。与五种驱逐基线方法（H2O、StreamingLLM、SnapKV、TOVA、纯窗口）相比，KV-Direct在所有缓存预算下保持100% token匹配，而基线方法退化至5-28%；操作级延迟分析表明，中等批量下重新计算比读取缓存张量快达5倍。

应用价值 / Applications

该研究为长上下文LLM推理提供了突破性的内存优化路径，使边缘设备部署和无限长度对话成为可能，无需复杂的缓存压缩或驱逐策略；KV-Direct的确定性等价性保证了输出质量零损失，适用于对一致性要求极高的生产环境，同时其计算-内存权衡特性为不同硬件配置提供了灵活的优化空间。
