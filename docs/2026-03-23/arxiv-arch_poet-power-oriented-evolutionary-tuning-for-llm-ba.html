POET: Power-Oriented Evolutionary Tuning for LLM-Based RTL PPA Optimization

论文摘要 / Abstract

本文提出了POET（面向功耗的进化调优框架），用于解决将大语言模型（LLMs）应用于RTL代码PPA（功耗、性能、面积）优化时的两大核心挑战：LLM幻觉导致的功能正确性问题和多目标优化中功耗优先的系统化权衡。
POET通过差分测试生成验证平台和LLM驱动的进化搜索机制，在40个RTL设计基准上实现了100%功能正确性，并在所有设计上取得了最佳功耗优化效果。

方法概述 / Method

POET采用双阶段方法：首先构建基于差分测试的测试台生成流水线，将原始设计作为功能预言机，通过确定性仿真生成黄金参考值，从而将LLM排除在验证过程之外以确保功能正确性；其次设计LLM驱动的进化优化机制，结合非支配排序、功耗优先的层内排序和比例幸存者选择策略，引导搜索向Pareto前沿的低功耗区域收敛。

实验结果 / Results

在RTL-OPT基准的40个多样化RTL设计评估中，POET实现了100%的功能正确率，在所有40个设计上都获得了最优功耗结果，同时在面积和延迟优化方面也取得了具有竞争力的改进。该结果显著优于现有方法，证明了框架在功耗优先的多目标PPA优化中的有效性。

应用价值 / Applications

POET可广泛应用于芯片设计自动化流程中的RTL级功耗优化，特别适用于对功耗敏感的移动设备、物联网边缘节点和数据中心加速器设计场景。该框架消除了人工权重调参的需求，降低了LLM辅助硬件设计的使用门槛，为AI驱动的电子设计自动化（EDA）工具链提供了可靠的功耗优化解决方案。
