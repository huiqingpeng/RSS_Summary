Esparagus Audio Brick ESP32-based DIN-rail 65W Hi-Fi amplifier supports Home Assistant and Squeezelite (Crowdfunding)

CNX Software 2026-03-14
Read Original Article →

是什么 / What it is

Esparagus Audio Brick 是一款基于 ESP32/ESP32-S3 的紧凑型 65W Hi-Fi D 类功放模块，采用 DIN 导轨安装设计，支持 Home Assistant 集成和多种流媒体协议。它由 Sonocotta 开发，目前正在 Crowd Supply 众筹，目标是为家庭音频系统提供开源、可定制的硬件解决方案。
The Esparagus Audio Brick is a compact 65W Hi-Fi Class-D amplifier module based on ESP32/ESP32-S3 with DIN-rail mounting design, supporting Home Assistant integration and multiple streaming protocols. Developed by Sonocotta and currently crowdfunding on Crowd Supply, it aims to provide an open-source, customizable hardware solution for home audio systems.

为什么重要 / Why it matters

这款设备填补了低成本开源硬件与专业级音频性能之间的空白，使 DIY 爱好者和智能家居用户能够以约 60 美元的价格构建多房间同步音频系统或改造老式音箱。其双版本策略（ESP32 专注独立流媒体，ESP32-S3 专注 Home Assistant）灵活覆盖了不同用户群体的需求，体现了开源硬件生态的成熟趋势。
This device bridges the gap between low-cost open-source hardware and professional-grade audio performance, enabling DIY enthusiasts and smart home users to build multi-room synchronized audio systems or retrofit vintage speakers for around $60. Its dual-version strategy (ESP32 for standalone streaming, ESP32-S3 for Home Assistant) flexibly addresses different user needs, reflecting the maturation trend of open-source hardware ecosystems.

我能用什么 / How I can use it

智能家居用户可选择 ESP32-S3 版本通过 ESPHome 集成 Home Assistant，实现语音播报、自动化场景联动和音乐播放；音频发烧友可用 ESP32 版本搭配 Squeezelite/LMS 或 Snapcast 组建低成本无线多房间音响系统。开发者还可利用其开源特性和标准 I²S/GPIO 接口，基于 Arduino 或 ESP-IDF 开发定制化固件，例如专用低音炮 DSP 或 Bi-Amp 分频方案。
Smart home users can choose the ESP32-S3 version to integrate with Home Assistant via ESPHome for voice announcements, automation scene linkage, and music playback; audiophiles can use the ESP32 version with Squeezelite/LMS or Snapcast to build low-cost wireless multi-room audio systems. Developers can also leverage its open-source nature and standard I²S/GPIO interfaces to build custom firmware based on Arduino or ESP-IDF, such as dedicated subwoofer DSP or Bi-Amp crossover solutions.
