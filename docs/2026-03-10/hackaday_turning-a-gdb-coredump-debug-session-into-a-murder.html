是什么 / What it is

这是一篇介绍名为 "Fatal Core Dump" 的开源游戏的文章，该游戏由 Jonathan 开发，将真实的 GDB 调试核心转储（coredump）过程转化为一种侦探解谜游戏体验。游戏背景设定在一颗小行星采矿站的气闸故障事件中，玩家扮演工程师，通过实际的 C 语言调试、汇编分析和 Linux 二进制运行时知识来调查事故原因。
This article introduces "Fatal Core Dump," an open-source game developed by Jonathan that transforms the real process of GDB coredump debugging into a detective mystery experience. Set against the backdrop of an inexplicable airlock failure at an asteroid mining station, players take on the role of an engineer investigating the incident through actual C debugging, assembly analysis, and Linux binary runtime knowledge.

This game bridges the gap between technical education and entertainment, transforming tedious yet critical low-level debugging skills into an immersive learning experience, representing an innovative approach to cultivating system-level programming and cybersecurity talent.
