MOON3.0: Reasoning-aware Multimodal Representation Learning for E-commerce Product Understanding

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出了MOON3.0，首个基于多模态大语言模型（MLLM）的推理感知型产品表征学习模型。针对现有MLLM将产品信息隐式编码为全局嵌入、难以捕捉细粒度属性的局限，作者利用MLLM的推理能力显式建模细粒度产品属性，并解决了长上下文推理中的注意力稀释、监督微调的僵化模仿以及前向传播中细节衰减三大挑战。实验表明，该模型在自建的MBE3.0基准和公开数据集上的多种下游任务中均取得了最先进的零样本性能。

方法概述 / Method

MOON3.0采用三项核心技术：（1）多头模态融合模块，自适应整合原始多模态信号；（2）联合对比学习与强化学习框架，使模型能够自主探索更有效的推理策略，突破传统SFT的模仿限制；（3）细粒度残差增强模块，在网络前向传播过程中逐步保留局部细节信息。此外，研究还发布了大规模多模态电商基准数据集MBE3.0。

实验结果 / Results

MOON3.0在MBE3.0基准和多个公开数据集上展现出卓越的零样本泛化能力，在各类下游产品理解任务中均达到最先进的性能水平。具体而言，该模型无需针对特定任务微调即可超越现有方法，验证了推理感知表征学习在电商产品理解中的有效性。

应用价值 / Applications

该研究可直接应用于电商平台的产品搜索、推荐系统和智能客服等场景，通过细粒度产品属性理解提升用户体验和商业转化率。其零样本学习能力大幅降低了对标注数据的依赖，使模型能够快速适应新品类和新市场，具有较强的实际部署价值和扩展性。
