论文摘要 / Abstract

本文提出了首个差分隐私流形去噪框架，允许用户利用敏感的参考数据集来修正含噪声的非隐私查询点，同时严格保护参考数据的隐私。该框架通过迭代过程实现：在每一轮中差分隐私地估计局部均值和切空间几何结构，并将查询点沿估计的子空间向局部均值投影进行修正。理论分析证明了查询点以高概率收敛到真实流形的非渐近速率，实验展示了在中等隐私预算下仍可实现准确的信号恢复。

方法概述 / Method

方法采用模块化设计，将差分隐私保护的局部几何估计（局部均值和切空间）与查询点更新分离，使用经过校准敏感度的差分隐私机制（如高斯机制）对参考数据进行扰动。通过简单的调度器在迭代轮次和多个查询之间分配隐私预算，并采用高级组合定理进行严格的隐私核算，实现(ε,δ)-差分隐私保证。

实验结果 / Results

模拟实验和案例研究表明，该方法在中等隐私预算下能够实现准确的信号恢复，清晰地展示了效用-隐私权衡曲线。即使加入差分隐私噪声，修正后的查询点仍保留了足够的几何信息以支持下游任务（如嵌入、聚类和可视化），验证了框架在实际部署中的可行性。

应用价值 / Applications

该研究为受监管环境（如医疗、金融等涉及敏感数据的领域）中的流形分析工作流提供了可部署的差分隐私组件，无需重新设计整个隐私系统。方法可广泛应用于需要利用敏感参考数据进行噪声修正的场景，如隐私保护下的单细胞RNA测序数据分析、医学影像去噪以及联邦学习中的数据预处理等。
