Informed Machine Learning with Knowledge Landmarks

arXiv Machine Learning 2026-04-02

论文摘要 / Abstract

本文提出了知识-数据机器学习（KD-ML）的新方向，通过将数值数据与以粒度知识地标形式表达的知识片段相结合，扩展了知情机器学习的框架。作者指出数据具有精确、局部的特性，而知识具有全局、高抽象层次的互补特性，并据此设计了包含数据拟合项和粒度正则化项的增强损失函数。在物理驱动的基准实验上，KD-ML模型一致性地优于纯数据驱动的机器学习模型。

方法概述 / Method

论文将知识表示为信息粒，并组织为输入-输出信息粒的集合（知识地标），构建了一个综合的KD-ML模型设计流程。核心方法是设计了一个原创的增强损失函数L，该函数加性地包含两个部分：基于可用数值数据优化模型的分量，以及作为粒度正则化器、使模型遵循知识地标约束的分量。

实验结果 / Results

实验在两个物理驱动的基准问题上进行，结果表明提出的KD模型一致性地优于纯数据驱动的ML模型。研究还分析了损失函数中的超参数在平衡数据与知识的贡献和指导作用方面的角色，以及数据质量（噪声水平）和知识地标粒度水平对模型性能的关键影响趋势。

应用价值 / Applications

该研究为科学和工程领域中存在先验知识（如物理定律、专家经验）但数据有限或噪声较大的场景提供了新的建模范式，特别适用于物理信息机器学习等相关领域。通过整合数据与知识的互补优势，KD-ML能够构建更鲁棒、可解释且数据效率更高的预测模型。
