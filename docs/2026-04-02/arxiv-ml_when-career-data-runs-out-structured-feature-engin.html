When Career Data Runs Out: Structured Feature Engineering and Signal Limits for Founder Success Prediction

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本研究探讨了基于创始人职业数据预测创业成功所面临的根本性挑战：信号微弱、正标签稀缺（仅9%），且成功与失败的创始人特征高度相似。作者通过结构化特征工程（28个特征）结合确定性规则层和XGBoost模型，在验证集上取得F0.5=0.3030的性能，较零样本LLM基线提升17.7个百分点。关键发现是：即使利用Claude Haiku从文本字段提取特征，LLM特征虽占模型重要性的26.4%，却未带来任何交叉验证信号增益（delta=-0.05pp），原因在于该文本字段实为JSON字段的损失性重编码，而非独立信息源。研究最终表明，当前性能天花板（CV≈0.25, Val≈0.30）反映的是数据集本身的信息局限，而非建模能力不足。

方法概述 / Method

研究采用结构化特征工程方法，直接从原始JSON字段（工作经历、教育背景、退出事件）提取28个结构化特征，并结合确定性规则层与XGBoost提升树桩（boosted stumps）构建预测模型。为对比验证，设计了对照实验：使用Claude Haiku从anonymised_prose文本字段提取9个特征，分别在67%和100%数据集覆盖率下进行测试，以量化LLM衍生特征的真实信息增益。

实验结果 / Results

完整模型在验证集达到F0.5=0.3030、精确率0.3333、召回率0.2222，较零样本LLM基线提升17.7个百分点。
LLM提取的特征虽在特征重要性中占比26.4%，但交叉验证性能反而微降0.05个百分点，表明其未提供增量信号。研究确定当前方法性能天花板约为CV≈0.25、验证集≈0.30，这一上限源于数据集内在信息含量而非模型缺陷。

应用价值 / Applications

本研究作为基准诊断工具，为创业投资领域的预测建模提供了现实性检验，明确指出单纯依赖创始人职业历史数据的预测极限。研究结果指导未来数据收集方向——需纳入超越结构化职业履历的更丰富信息源（如心理特质、社会网络动态、市场时机等），对风险投资机构优化尽职调查流程、改进早期项目筛选机制具有直接参考价值，同时为"AI+风险投资"领域的研究设定了合理的性能预期边界。
