Speeding Up Mixed-Integer Programming Solvers with Sparse Learning for Branching

论文摘要 / Abstract

本文提出了一种基于稀疏学习的分支策略，用于加速混合整数规划（MIP）求解器。与现有的深度学习方法不同，该研究开发了参数极少（不到最先进图神经网络4%）但效果显著的轻量级模型来近似计算代价高昂的强大分支（Strong Branching）分数。实验表明，这些仅使用CPU的模型比默认求解器和GPU加速的GNN更快，且在小数据集上训练即可有效部署，适用于资源受限场景。

方法概述 / Method

该研究采用稀疏学习方法构建可解释的分支决策模型，以替代计算成本高的强分支启发式规则。模型通过特征选择和稀疏正则化技术，从大量候选特征中筛选出最关键的一小部分，从而实现极高的参数效率。这种方法避免了深度神经网络对大规模训练数据和GPU资源的依赖，使模型能够在普通CPU环境下快速训练和推理。

实验结果 / Results

实验在多种问题类别上验证了该方法的有效性：稀疏模型以不到4%的参数量达到了与GNN相当的预测精度；在实际求解速度上，CPU部署的稀疏模型超越了GPU加速的GNN基线以及SCIP求解器的内置分支规则；模型在小型训练集上仍保持稳定性能，显著降低了数据收集和训练成本。

应用价值 / Applications

该研究为工业优化和运筹学领域提供了实用的MIP求解加速方案，特别适用于计算资源受限的生产环境（如边缘设备、云端GPU配额有限或需要快速部署的场景）。轻量级模型的可解释性也有助于分析师理解决策逻辑，增强求解过程的透明度和可信度，可广泛应用于供应链优化、生产调度、资源分配等实际决策问题。
