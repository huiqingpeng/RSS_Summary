ActivityNarrated: An Open-Ended Narrative Paradigm for Wearable Human Activity Understanding

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出ActivityNarrated，一种面向可穿戴人体活动理解的开放式叙事范式，突破了传统闭集分类的局限。研究指出人类活动具有开放性、无脚本性、个性化和组合性特征，应以叙事而非固定类别实例来建模。通过将可穿戴传感器数据与自然语言描述在开放词汇表设置下对齐，该方法实现了对开放式活动叙事的建模，在跨被试评估中取得65.3%的Macro-F1分数，显著优于31-34%的强闭集基线方法。

方法概述 / Method

该方法包含三个核心组件：一是自然主义数据收集与标注流程，结合多位置可穿戴传感与自由形式、时间对齐的行为叙事描述，使活动语义无需预定义词汇表即可涌现；二是基于检索的评估框架，测量传感器数据与语言之间的语义对齐，无需固定类别即可进行原则性评估；三是语言条件学习架构，支持可变长度传感器流和异构传感器位置的传感器到文本推理。

实验结果 / Results

实验表明，固定标签目标训练的模型在现实世界变化下性能急剧下降，而开放词汇表传感器-语言对齐能产生鲁棒且语义基础扎实的表征。学习该对齐后，闭集活动识别成为简单的下游任务。跨被试评估中，该方法Macro-F1达65.3%，而强闭集HAR基线仅为31-34%。

应用价值 / Applications

该研究为现实世界可穿戴人体活动识别提供了实用有效的新基础，可广泛应用于健康监测、老年护理、健身追踪等需要理解复杂、个性化、开放式人类行为的场景，克服了传统预定义活动类别无法覆盖真实生活多样性的瓶颈。
