Thinking Wrong in Silence: Backdoor Attacks on Continuous Latent Reasoning

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文揭示了新一代语言模型中"连续隐状态推理"（continuous latent reasoning）带来的全新安全漏洞。作者提出ThoughtSteer后门攻击方法，仅需在输入层扰动单个嵌入向量，即可通过模型自身的多轮推理放大该扰动，劫持隐状态轨迹以输出攻击者预设的答案，且能绕过所有token级防御机制。该攻击在两种架构（Coconut和SimCoT）、三个推理基准测试及124M至3B参数规模的模型上均达到≥99%的攻击成功率，同时保持接近基线的清洁准确率，并能零样本迁移到新基准（94-100%成功率），甚至经受25轮清洁微调后仍然有效。

方法概述 / Method

ThoughtSteer通过在输入层注入精心设计的单嵌入向量扰动，利用连续推理模型多轮迭代的特性，使微小初始扰动在隐状态空间中被逐步放大为劫持轨迹。攻击针对的是模型完全在不可见的连续隐状态中进行推理的特性，而非传统的token序列，从而规避所有基于token审计的防御机制。

实验结果 / Results

实验显示ThoughtSteer在两种主流连续推理架构上均实现≥99%攻击成功率，清洁准确率几乎无损失；攻击具有强迁移性，无需重新训练即可在held-out基准上达到94-100%成功率；能成功规避五种主动防御机制，并在25轮清洁微调后依然保持有效。机制分析揭示神经坍缩（Neural Collapse）现象使触发表征聚集于紧致几何吸引子，导致后门签名线性可分（探测AUC≥0.999），但 paradoxically 单个隐向量仍编码正确答案而模型却输出错误答案。

应用价值 / Applications

本研究对部署连续隐状态推理模型的安全实践具有重要警示意义，揭示了完全不可审计推理过程的固有脆弱性。同时，该工作为连续推理的机械可解释性研究提供了新视角——后门扰动可作为分析隐状态集体动力学的工具。研究成果可指导未来防御机制的设计，推动开发针对隐状态空间而非token空间的审计与对齐方法。
