HabitatAgent: An End-to-End Multi-Agent System for Housing Consultation

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出了 HabitatAgent，首个面向住房咨询的端到端大语言模型多智能体系统。针对现有住房平台和LLM助手将选房过程简化为排序或推荐所导致的推理不透明、多约束处理脆弱以及事实性保证有限等问题，HabitatAgent 通过四个专业化智能体（记忆、检索、生成、验证）的协作，实现了可审计且可靠的住房咨询服务。在100个真实用户咨询场景（300轮问答对）的端到端正确性评估中，HabitatAgent 达到95%的准确率，显著优于75%的强基线方法。

方法概述 / Method

HabitatAgent 采用四智能体协作架构：记忆智能体通过约束提取、记忆融合和验证门控更新维护多层用户记忆；检索智能体执行混合向量-图检索（GraphRAG）；生成智能体产出带证据引用的推荐与解释；验证智能体实施多层级验证与针对性修复。四个智能体形成闭环工作流，确保咨询过程的可审计性与可靠性。

实验结果 / Results

在包含100个真实用户咨询场景、300轮多轮问答的端到端正确性协议评估中，HabitatAgent 达到95%的准确率。相比之下，强单阶段基线方法（Dense+Rerank）仅取得75%的准确率，表明多智能体架构在复杂住房咨询任务中具有显著优势。

应用价值 / Applications

HabitatAgent 可部署于在线房产平台、智能客服系统及移动购房应用，为高风险的住房决策提供透明、可解释且事实可靠的决策支持。该系统通过结构化记忆管理和多层级验证机制，有效降低用户因信息不透明或约束冲突导致的决策风险，提升购房咨询服务的专业性与用户信任度。
