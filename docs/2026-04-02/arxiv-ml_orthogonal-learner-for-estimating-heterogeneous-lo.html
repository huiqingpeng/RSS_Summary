Orthogonal Learner for Estimating Heterogeneous Long-Term Treatment Effects

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出了LT-O-learners（长期正交学习器），用于估计异质性长期处理效应（HLTE）。该方法通过引入自定义重叠权重来重新设定学习目标，降低处理分配或长期结果观测重叠度低的样本权重，从而解决有限重叠导致的估计不稳定问题。理论证明重设目标后的损失满足Neyman正交性，对干扰项估计误差具有鲁棒性，并在一定条件下可实现准oracle收敛率。这是首个针对HLTE估计中常见低重叠问题具有鲁棒性的正交学习器。

方法概述 / Method

LT-O-learners将短期随机实验数据（𝒟₁）与长期历史观测数据（𝒟₂）相结合，通过设计自定义重叠权重对样本进行重新加权，使学习目标聚焦于高重叠区域。该方法满足Neyman正交性条件，意味着目标函数对干扰参数估计的一阶导数为零，从而保证即使干扰项估计存在误差，HLTE估计仍保持稳定。

实验结果 / Results

作者在合成数据和半合成基准数据上验证了LT-O-learners的理论特性，特别是在低重叠设置下的鲁棒性。实验结果表明，该方法在处理分配或长期结果观测存在有限重叠的子群体中，相比现有方法具有更小的有限样本方差和更稳定的估计性能。

应用价值 / Applications

LT-O-learners可广泛应用于营销、经济学和医学等领域的个性化决策场景，例如评估长期客户价值、政策干预的长期效应或治疗方案的远期预后。该方法模型无关，可与任意机器学习模型结合，为实际中常见的"短期实验+长期观测"数据融合问题提供了可靠且实用的解决方案。
