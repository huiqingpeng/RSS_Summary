GUIDE: Reinforcement Learning for Behavioral Action Support in Type 1 Diabetes

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出了GUIDE，一种基于强化学习（RL）的决策支持框架，用于补充自动胰岛素输送（AID）系统，通过提供行为建议来预防异常血糖事件。该框架生成结构化的行为建议，包括干预类型、幅度和时机，涵盖大剂量胰岛素注射和碳水化合物摄入。实验表明，CQL-BC算法在25名1型糖尿病患者中实现了85.49%的平均血糖达标时间，同时保持低血糖暴露较低，且学习策略与患者实际行为模式具有高度相似性（平均余弦相似度0.87±0.09），证明了该方法在个性化糖尿病管理中的临床可行性和行为合理性。

方法概述 / Method

GUIDE采用离线强化学习与在线强化学习相结合的方法，构建了一个统一的患者特定血糖预测环境。该方法将行为建议定义为结构化动作空间，包括干预类型（如胰岛素注射或碳水化合物摄入）、幅度和时机三个维度，并整合了基于真实世界连续血糖监测数据训练的个性化血糖水平预测器，支持CQL-BC等保守型离线RL算法和PPO等在线RL算法。

实验结果 / Results

在25名1型糖尿病患者的评估中，CQL-BC（保守Q学习结合行为克隆）算法表现最优，平均血糖达标时间（TIR）达到85.49%，同时维持较低的低血糖暴露。行为相似性分析显示，CQL-BC学习策略保留了患者关键的行为模式结构特征，跨受试者平均余弦相似度达0.87±0.09，显著优于其他评估方法，表明该方法在优化血糖控制的同时保持了行为可解释性和患者可接受性。

应用价值 / Applications

GUIDE框架可直接集成到现有AID系统中，为1型糖尿病患者提供实时、个性化的行为决策支持，帮助患者在自动胰岛素调节之外优化生活方式干预（如进餐时机和碳水化合物摄入量）。该方法特别适用于血糖控制未达标的患者群体，有望改善临床结局并减轻患者自我管理负担，为数字疗法和远程糖尿病管理提供可扩展的技术解决方案。
