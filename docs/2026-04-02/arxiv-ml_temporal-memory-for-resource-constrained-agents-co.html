Temporal Memory for Resource-Constrained Agents: Continual Learning via Stochastic Compress-Add-Smooth

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出了一种面向资源受限智能体的持续学习框架，将记忆建模为桥扩散随机过程而非参数向量，通过"压缩-添加-平滑"（CAS）三步递归整合新经验。该框架在固定内存预算下实现无反向传播、无数据存储、无神经网络的计算，时间复杂度仅为O(LKd²)。研究发现遗忘源于有损时间压缩，记忆半衰期与协议段数L呈线性关系，且该框架可提供时间连贯的"电影式"历史回放。

方法概述 / Method

核心方法采用桥扩散过程定义在回放区间[0,1]上的随机过程，终端边缘分布编码当前状态，中间边缘分布编码历史信息。新经验通过Compress-Add-Smooth三步递归整合：压缩历史为高斯混合、添加新经验、平滑处理。具体实现采用固定K个分量的d维高斯混合模型，以L段分段线性协议节点存储状态。

实验结果 / Results

关键理论发现是记忆保留半衰期a₁/₂ ≈ c·L与协议段数L呈线性缩放，比例常数c>1仅取决于动态特性，与高斯混合复杂度K、维度d及目标族几何无关。该常数c具有类似香农信道容量的信息论解释。研究在MNIST潜在空间上直观展示了压缩历史叙事的可视化回放效果。

应用价值 / Applications

该框架适用于控制器资源极度受限的嵌入式硬件场景，如边缘计算设备、微型机器人或物联网传感器，可实现无需神经网络推理的持续学习能力。其完全解析的数学形式为持续学习机制提供了"伊辛模型"式的理论研究平台，便于精确分析遗忘的速率与形式，同时支持可解释的历史回放功能用于智能体决策审计。
