Offline Constrained RLHF with Multiple Preference Oracles

arXiv Machine Learning 2026-04-02

论文摘要 / Abstract

本文研究了基于多偏好预言机的离线约束强化学习人类反馈（RLHF）问题，旨在最大化目标群体效用的同时满足受保护群体的最低福利约束。作者通过最大似然估计从参考策略收集的成对比较中学习预言机特定的奖励函数，并将约束目标转化为KL正则化的拉格朗日形式，其原始最优解为Gibbs策略。研究提出了仅对偶算法，首次为离线约束偏好学习提供了有限样本性能保证，并将理论分析扩展至多约束和一般f-散度正则化情形。

方法概述 / Method

论文采用最大似然估计从离线成对比较数据中学习多个预言机的奖励函数，并通过分析统计不确定性在对偶规划中的传播来处理估计误差。核心方法是将约束优化问题重新表述为KL正则化拉格朗日函数，利用Gibbs策略的闭式解将学习问题简化为凸对偶优化，从而设计出无需显式策略优化的仅对偶学习算法。

实验结果 / Results

论文主要提供理论贡献，建立了高概率约束满足性的有限样本保证，并刻画了统计误差、正则化参数与策略次优性之间的权衡关系。分析表明该对偶算法能够有效控制约束违反风险，且理论框架可自然推广到多约束设置和一般f-散度正则化，保持了算法的计算可处理性。

应用价值 / Applications

该研究适用于需要在性能与安全、公平等社会价值之间进行权衡的关键应用场景，如医疗决策系统（确保不同患者群体获得最低标准护理）、推荐系统（平衡整体用户参与度与少数群体满意度）和自动驾驶（优化效率同时保证行人安全）。多偏好预言机框架特别适合涉及多元利益相关者、存在价值异质性的复杂决策环境。
