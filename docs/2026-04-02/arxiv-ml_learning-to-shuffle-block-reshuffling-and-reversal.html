Learning to Shuffle: Block Reshuffling and Reversal Schemes for Stochastic Optimization

arXiv Machine Learning 2026-04-02

论文摘要 / Abstract

本文提出了一种基于大语言模型（LLM）引导的程序进化框架，用于发现有效的无放回SGD数据重排策略。通过该框架，作者识别出两个关键结构组件：**块重排（block reshuffling）**和**配对反转（paired reversal）**，并分别从理论上证明了块重排能够严格降低前缀梯度方差常数，而配对反转能将阶数敏感性从步长的二次方降至三次方。数值实验验证了所发现算法在凸和非凸基准测试上均优于标准重排方案。

方法概述 / Method

作者采用LLM引导的程序进化框架自动搜索最优的数据重排规则，从发现的实例中抽象出"块重排"和"配对反转"两种基本结构。块重排指在每个epoch内将数据分成块并进行块级别的重排；配对反转则通过对称化epoch映射来抵消依赖于顺序的二阶项。理论分析基于统一的重排框架，将这两种组件分别进行严格的收敛性分析。

实验结果 / Results

理论分析表明，块重排在温和条件下严格优于随机重排，而配对反转将优化过程的阶数敏感性从$O(\eta^2)$降低到$O(\eta^3)$。数值实验在凸和非凸基准测试上验证了所发现算法的有效性，显示出相比标准重排方案（包括随机重排、单次重排和循环梯度）的一致性能提升。

应用价值 / Applications

该研究为随机优化中的数据排序策略设计提供了系统性的理论指导和自动化发现工具，可广泛应用于深度学习模型训练，特别是在需要稳定收敛和优化效率的场景。
LLM引导的程序进化框架也为其他优化算法组件的自动设计提供了可迁移的方法论。
