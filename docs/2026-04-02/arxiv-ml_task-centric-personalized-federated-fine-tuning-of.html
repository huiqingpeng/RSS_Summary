Task-Centric Personalized Federated Fine-Tuning of Language Models

arXiv Machine Learning 2026-04-02
Read Original Article →

论文摘要 / Abstract

本文提出了FedRouter，一种基于聚类的个性化联邦学习方法，通过为每个任务而非每个客户端构建专门模型，解决了传统个性化联邦学习在泛化性和客户端内任务干扰方面的不足。
FedRouter采用双聚类机制（本地聚类将适配器与任务数据样本关联，全局聚类将不同客户端的相似适配器聚合）以及评估路由机制，在任务干扰场景下相对性能提升达6.1%，在泛化评估中相对改进高达136%。

方法概述 / Method

FedRouter使用适配器（adapters）实现模型个性化，通过本地聚类将适配器与特定任务数据样本关联，并通过全局聚类将跨客户端的相似适配器进行聚合，从而构建以任务为中心的个性化模型。此外，该方法还包含一个评估路由机制，用于在测试时将样本路由至最优适配器。

实验结果 / Results

在多任务数据集上的实验表明，FedRouter在面对客户端内任务干扰时性能相对提升6.1%，在处理未见任务的泛化评估中相对改进高达136%，展现出对挑战性场景的强韧性。

应用价值 / Applications

该研究适用于需要在分布式私有数据上微调大语言模型的场景，如企业级多任务NLP服务、跨机构医疗文本分析等，能够在保护数据隐私的同时，有效处理任务异质性和分布变化问题，提升模型的泛化能力和鲁棒性。
