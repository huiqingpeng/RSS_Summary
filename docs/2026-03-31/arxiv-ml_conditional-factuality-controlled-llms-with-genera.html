Conditional Factuality Controlled LLMs with Generalization Certificates via Conformal Sampling

论文摘要 / Abstract

本文提出了**条件事实性控制（Conditional Factuality Control, CFC）**，一种后验保形推理框架，用于在大型语言模型（LLMs）推理时提供具有**条件覆盖保证**的集合值输出，以解决现有方法仅能提供边际保证、使用单一全局阈值导致对困难提示覆盖不足而对简单提示过度覆盖的问题。
CFC通过增强分位数回归学习连续的特征条件接受阈值，并在推理时通过不动点阈值规则部署。理论上证明了CFC在满足交换性假设下具有条件覆盖保证，且在相同目标覆盖率下比边际保形预测具有严格更高的样本效率；进一步提出的CFC-PAC变体基于稳定性边界收缩名义风险水平，给出有限样本泛化证书。实验表明，在合成数据、真实世界推理与问答基准以及Flickr8k视觉语言模型设置上，CFC和CFC-PAC在各难度组均能达到接近目标的覆盖率，同时使用比保形预测和非保形基线更小的预测集合。

方法概述 / Method

CFC采用**增强分位数回归**在潜在"成功"分数上学习连续的特征条件接受阈值，替代传统保形预测中的单一全局阈值；在推理阶段，通过**不动点阈值规则**动态确定每个输入的条件阈值，从而生成大小自适应的预测集合。
CFC-PAC变体进一步引入基于稳定性边界的**PAC风格风险水平收缩机制**，为条件误覆盖率与目标之间的偏差提供有限样本理论保证。

实验结果 / Results

在合成数据、真实世界推理/问答基准（如GSM8K、StrategyQA等）以及Flickr8k视觉语言模型任务上的实验表明，CFC和CFC-PAC能够在不同难度组（difficulty groups）中**一致地达到接近目标的条件覆盖率**，同时生成的预测集合大小显著小于标准保形预测（CP）和非保形基线，实现了更高的样本效率。

应用价值 / Applications

该研究为**高风险LLM部署场景**（如医疗问答、法律咨询、教育辅导等）提供了可靠的事实性控制机制，能够在保证条件覆盖的前提下减少过度保守的冗长输出，提升用户体验和系统可信度；同时，其理论框架可扩展至其他需要**不确定性量化和集合预测**的机器学习应用场景，为多模态模型和复杂推理系统提供可证明的安全保障。
