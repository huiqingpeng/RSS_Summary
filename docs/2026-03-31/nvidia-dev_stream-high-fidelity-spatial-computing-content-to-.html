Stream High-Fidelity Spatial Computing Content to Any Device with NVIDIA CloudXR 6.0

NVIDIA Developer Blog 2026-03-31
Read Original Article →

是什么 / What it is

NVIDIA CloudXR 6.0 是 NVIDIA 于 GTC 2026 发布的通用 OpenXR 流媒体运行时，支持跨头显、操作系统和浏览器的高保真空间计算内容流式传输，并原生集成 Apple Vision Pro 的动态注视点渲染技术。该版本通过云端 RTX GPU 渲染，将原本需要本地高性能硬件的逼真 3D 内容实时传输到任何空间显示设备。
NVIDIA CloudXR 6.0 is a universal OpenXR-based streaming runtime announced at GTC 2026, enabling high-fidelity spatial computing content to be streamed across headsets, operating systems, and browsers with native integration for Apple Vision Pro's dynamic foveated rendering. It leverages cloud-based RTX GPUs to render photorealistic 3D content in real time and stream it to any spatial display, eliminating the need for powerful local hardware. ---

为什么重要 / Why it matters

空间计算正从可视化转向主动协作，但开发者长期面临多平台代码库维护的碎片化困境，而 CloudXR 6.0 通过单一 OpenXR 标准统一了流媒体协议。动态注视点流媒体技术将带宽需求降低 75% 的同时保持 4K 级视觉质量，使 5GHz WiFi 上的 90FPS 无线体验、多用户协作场景和移动设备续航成为现实，显著降低了企业级 XR 部署的门槛。
Spatial computing is shifting from visualization to active collaboration, yet developers have struggled with fragmented multi-platform codebases—CloudXR 6.0 unifies streaming protocols under a single OpenXR standard. Dynamic foveated streaming reduces bandwidth requirements by 75% while maintaining 4K-equivalent visual quality, enabling 90FPS wireless experiences over standard 5GHz WiFi, scalable multi-user collaboration, and extended battery life for mobile devices—significantly lowering barriers to enterprise XR deployment. ---

我能用什么 / How I can use it

开发者可通过 NVIDIA NGC 下载 CloudXR 6.0 Runtime SDK，将 `openxr_cloudxr.json` 注册为 OpenXR 运行时并链接 `cxrServiceAPI.h` 实现服务生命周期管理，或使用 Xcode 模板快速构建 visionOS 原生应用；对于 Web 场景可采用 CloudXR.js 实现零安装访问。建议从 CloudXR LÖVR 示例或 Unreal Engine 专用指南入手，评估现有 OpenXR 应用的云端流式化改造可行性，并针对协作设计、远程培训等场景测试多用户网络承载能力。
Developers can download the CloudXR 6.0 Runtime SDK from NVIDIA NGC, register `openxr_cloudxr.json` as the OpenXR runtime and link `cxrServiceAPI.h` for service lifecycle management, or use the Xcode template for rapid visionOS native app development; for web scenarios, CloudXR.js enables zero-install access. Start with the CloudXR LÖVR sample or Unreal Engine dedicated guide, assess cloud-streaming feasibility for existing OpenXR applications, and test multi-user network capacity for collaborative design or remote training scenarios.
