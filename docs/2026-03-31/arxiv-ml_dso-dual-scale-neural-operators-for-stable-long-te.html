DSO: Dual-Scale Neural Operators for Stable Long-term Fluid Dynamics Forecasting

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文针对神经算子在长期流体动力学预测中的稳定性问题，提出了双尺度神经算子（DSO）。作者识别出现有架构的两种失效模式：局部细节模糊和全局趋势漂移，并指出这些问题的根源在于现有方法对局部和全局信息采用统一处理。
DSO通过深度可分离卷积和MLP-Mixer的显式解耦设计，在湍流基准测试上实现了最先进的精度，预测误差降低超过88%，同时保持长期稳定性。

方法概述 / Method

DSO的核心创新在于将信息处理显式解耦为两个互补模块：深度可分离卷积负责细粒度局部特征提取，MLP-Mixer负责长程全局聚合。这种设计基于物理洞察——邻近扰动主要影响局部涡旋结构，而远距离扰动影响全局运动趋势。

实验结果 / Results

在涡旋动力学的数值实验中，DSO验证了不同尺度扰动的物理影响机制，为设计选择提供了经验支持。在湍流基准测试上的大量实验表明，DSO在保持长期稳定性的同时达到最先进精度，相比现有神经算子预测误差降低超过88%。

应用价值 / Applications

该研究对科学和工程领域的长期流体动力学预测具有重要价值，可应用于天气预报、气候建模、航空航天设计等需要稳定长期仿真的场景。
DSO的双尺度架构为物理信息神经网络的设计提供了新范式，有望推广到其他多尺度物理系统建模任务。
