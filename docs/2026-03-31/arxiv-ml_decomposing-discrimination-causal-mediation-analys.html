Decomposing Discrimination: Causal Mediation Analysis for AI-Driven Credit Decisions

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文针对AI驱动的信贷决策中的公平性评估问题，提出了一种因果中介分析方法，区分了两种因果机制不同的歧视：受保护属性对信贷结果的直接歧视，以及通过合法金融特征传递的结构性不平等。作者基于Pearl的自然直接效应和间接效应框架，解决了处理诱导混杂（treatment-induced confounding）下的识别难题，并在纽约州89,465笔真实房贷数据上验证了方法，发现约77%的种族拒贷差异源于结构性不平等，仅23%为直接歧视的下界估计。

方法概述 / Method

论文采用Pearl的因果中介分析框架，针对受保护属性同时影响金融中介变量和最终决策的处理诱导混杂问题，提出修正序列可忽略性（Modified Sequential Ignorability）假设下的干预直接效应（IDE）和干预间接效应（IIE）识别策略，并证明在单调间接处理响应条件下IDE/IIE为自然效应提供保守边界。方法实现上采用双重稳健增广逆概率加权（AIPW）估计器，结合交叉拟合（cross-fitting）和E值敏感性分析处理残余混杂。

实验结果 / Results

在纽约州2022年89,465笔常规购房抵押贷款申请数据上，研究发现观察到的7.9个百分点种族拒贷差异中，约77%（6.1个百分点）通过金融中介变量传递，归因于结构性不平等；剩余23%（1.8个百分点）构成直接歧视的保守下界。作者同步开源了CausalFair Python工具包，支持资源受限金融机构部署完整分析流程。

应用价值 / Applications

该方法为金融机构和监管机构提供了区分直接歧视与结构性不平等的技术工具，有助于制定更精准的公平性政策——针对直接歧视可采取算法纠偏措施，而对结构性不平等则需社会经济层面的干预。
CausalFair工具包的开源降低了因果公平性分析的门槛，使中小型金融机构也能进行合规审计和歧视归因分析，支持《公平住房法》等反歧视法规的执法实践。
