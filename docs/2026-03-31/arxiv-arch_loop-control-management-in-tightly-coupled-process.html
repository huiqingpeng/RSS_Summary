arXiv Computer Architecture 2026-03-31

论文摘要 / Abstract

本文针对多维循环核在并行循环加速器上控制开销过大的问题，提出了一种从多面体表示中推导并大幅压缩控制条件的方法，可将控制信号减少15至45倍。作者设计了一种轻量级全局控制器架构，使用有界计算单元将条件评估为多面体的并集，仅需单个处理元件相当的硬件资源。在PolyBench基准测试上的评估表明，整个控制流仅占用总阵列资源的不到10%，实现了零开销循环控制。

方法概述 / Method

该方法基于多面体模型对迭代空间进行表示，从中推导控制条件并进行激进式压缩；设计了一种轻量级全局控制器，利用有界计算单元将条件评估为多面体的并集运算；控制信号通过最少数量的延迟元件分布到整个处理器阵列。

实验结果 / Results

在多个基准测试上，控制信号数量实现了15倍至45倍的压缩；全局控制器的硬件开销仅相当于单个处理元件；在PolyBench内核上的评估显示，完整控制流所占用的资源不到处理器阵列总资源的10%。

应用价值 / Applications

该研究可应用于高密度并行加速器（如TCPA）的循环控制优化，显著降低控制开销和硬件资源占用；适用于嵌入式系统和加速器设计领域，为多维循环密集型应用（如信号处理、科学计算）提供高能效的硬件加速方案。
