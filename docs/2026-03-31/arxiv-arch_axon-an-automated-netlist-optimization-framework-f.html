AXON: An Automated Netlist Optimization Framework for High-Speed Adders

arXiv Computer Architecture 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文提出了AXON，一个针对高速加法器的自动化网表优化框架，通过从架构到网表层级的跨层次设计空间探索，结合前缀拓扑搜索与标准单元感知映射，快速收敛至近最优PPA解。作者还提出了一种融合并行前缀与Ling架构的混合超高速加法器以缩短关键路径。在TSMC 28nm工艺库上的实验表明，相比商业综合工具，AXON在延迟、面积-延迟积和能量-延迟积方面分别最高提升10.3%、12.6%和32.1%。

方法概述 / Method

AXON采用层次化方法整合架构级前缀拓扑搜索与网表/单元级感知映射，实现跨抽象层的设计空间探索。该方法特别引入了一种混合架构加法器，将并行前缀结构与Ling架构相结合以优化关键路径延迟。

实验结果 / Results

在TSMC 28nm标准单元库上的评估显示，AXON相比现有商业综合工具可实现最高10.3%的延迟降低、12.6%的面积-延迟积改善，以及32.1%的能量-延迟积优化，证明了其在PPA多目标优化上的显著优势。

应用价值 / Applications

该框架可直接应用于高性能处理器、AI加速器及数据中心芯片等对加法器性能敏感的数字系统设计，帮助设计人员在早期阶段快速获得针对特定PPA目标的优化加法器实现，减少人工迭代与综合调优时间。
