Omni-Modal Dissonance Benchmark: Systematically Breaking Modality Consensus to Probe Robustness and Calibrated Abstention

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文提出了OMD-Bench（全模态不协调基准测试），用于解决现有全模态基准测试中模态贡献测量被混淆的问题。该基准首先让所有模态（视频、音频、文本）呈现一致的锚定信息，再系统性地破坏特定模态以隔离其真实贡献。研究发现，全模态模型在双模态被破坏时过度弃权，但在三模态全被破坏时严重欠弃权，同时保持高置信度（60-100%）；思维链提示虽能改善弃权判断与人类判断的一致性，却加剧了过度自信而非缓解。

方法概述 / Method

OMD-Bench采用"一致性-破坏"范式：以27个可独立通过视频、音频、文本感知的对象或事件为锚点，构建4,080个测试实例，涵盖八种破坏条件（单模态、双模态、三模态破坏的组合）。评估设置包括零样本提示和思维链提示，重点测量模型在跨模态不一致时的鲁棒性和校准弃权能力。

实验结果 / Results

在10个全模态模型上的评估显示：模型表现出系统性的弃权失调——双模态冲突时过度保守，全模态冲突时却未能有效拒绝回答；置信度校准严重失效，即使全部模态被破坏仍维持60-100%的高置信度；思维链提示在提升弃权合理性的同时，意外放大了模型的过度自信倾向。

应用价值 / Applications

OMD-Bench为全模态AI系统提供了诊断工具，可精准识别模型对特定模态的真实依赖程度，评估其在真实世界噪声环境下的鲁棒性。该基准对开发安全关键应用（如自动驾驶、医疗诊断）具有重要价值，帮助研究者设计更好的不确定性量化和可信弃权机制，防止模型在证据冲突时产生危险的高置信度错误。
