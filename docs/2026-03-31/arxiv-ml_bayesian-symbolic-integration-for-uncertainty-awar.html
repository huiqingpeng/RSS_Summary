论文摘要 / Abstract

本文提出了一种松散耦合的神经符号框架，将贝叶斯神经网络（BNN）与符号推理相结合，以解决智能交通系统中停车位预测面临的数据稀疏、噪声和不可预测变化等挑战。该框架通过BNN量化预测不确定性，并利用决策树提取的符号知识进行概率逻辑编程，实现了两种混合策略：低置信度时启用符号推理作为后备机制，以及基于符号约束细化输出类别后重新应用BNN。实验结果表明，两种混合方法均优于纯符号推理，且上下文细化策略在所有预测窗口上持续超越LSTM和BNN基线模型。

方法概述 / Method

论文采用贝叶斯神经网络作为核心预测组件，通过权重分布建模实现不确定性量化；同时利用决策树从数据中提取符号规则，并以概率逻辑编程形式编码领域知识。两种混合策略分别实现不确定性驱动的动态切换（BNN置信度低时 fallback 到符号推理）和符号约束指导的迭代优化（先符号细化再BNN重预测）。

实验结果 / Results

在真实世界停车数据集的全量、稀疏和噪声三种条件下，两种混合策略均显著优于纯符号推理基线；其中上下文细化策略（context-refinement）在所有预测时间窗口上 consistently 超越LSTM网络和标准BNN基线，展现出更强的鲁棒性和预测精度。

应用价值 / Applications

该研究为智能交通系统中的实时停车位预测提供了高可靠性的解决方案，特别适用于数据质量不稳定的城市部署场景；其模块化神经符号架构可推广至其他不确定性敏感的预测任务（如交通流量预测、能源需求预测），为安全关键领域的AI系统提供了可解释且鲁棒的决策支持框架。
