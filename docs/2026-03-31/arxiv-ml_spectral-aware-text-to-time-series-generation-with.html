Spectral-Aware Text-to-Time Series Generation with Billion-Scale Multimodal Meteorological Data

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文提出了一个用于文本引导气象时间序列生成的统一框架，解决了现有方法受限于缺乏大规模、物理基础的多模态数据集以及忽视天气信号频谱-时间结构的架构问题。研究首先构建了十亿规模的气象数据集MeteoCap-3B，通过多智能体协作标注（MACC）流程生成信息密集且物理一致的专家级标注。在此基础上，提出了基于扩散模型的MTransformer，通过频谱提示生成器将文本描述映射到多波段频谱先验，实现精确的语义控制。实验表明该方法在生成质量、跨模态对齐、语义可控性以及数据稀疏和零样本预测任务上均达到最优性能，且可推广至一般时间序列基准。

方法概述 / Method

研究采用多智能体协作标注（MACC）流程构建高质量气象图文数据集，结合大语言模型与领域专家知识确保标注的物理一致性。核心模型MTransformer基于扩散架构，创新性地引入频谱提示生成器（Spectral Prompt Generator），将文本语义映射为多波段频谱先验，并通过频率感知注意力机制指导时间序列生成过程。

实验结果 / Results

在真实世界气象基准测试中，该方法实现了最先进的生成质量，展现出准确的跨模态对齐能力和强大的语义可控性；在数据稀疏和零样本预测设置下，下游预测任务取得显著提升。此外，在通用时间序列基准上的额外实验验证了该框架超出气象领域的良好泛化能力。

应用价值 / Applications

该研究为气象预报、气候模拟等需要直观自然语言控制的复杂大气动力学场景提供了实用工具，支持多尺度天气现象的可解释生成。框架的通用性设计使其可扩展至能源需求预测、金融时间序列分析等其他领域，为文本驱动的时间序列生成任务建立了新的技术范式。
