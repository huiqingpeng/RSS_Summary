Dynamic resource matching in manufacturing using deep reinforcement learning

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文研究了制造业中需求-产能类型资源的动态匹配问题，将其建模为具有大状态空间和动作空间的多期多对多序贯决策过程。针对维度灾难和转移动态难以显式建模的挑战，作者提出了一种基于领域知识的深度强化学习方法。该方法通过引入基于先验策略的领域知识惩罚项和满足供需约束的不可行动作惩罚项，改进了传统Q学习算法，并将其扩展为领域知识引导的深度确定性策略梯度算法（DKDDPG）。实验表明，该方法在中小规模和大规模问题上均优于传统DDPG及其他强化学习算法，实现了更高的奖励和更快的收敛效率。

方法概述 / Method

论文采用无模型深度强化学习方法解决制造资源匹配问题，核心创新在于设计了两个惩罚机制：一是基于先验策略的领域知识惩罚项，用于纠正Q学习中最大算子导致的初始估计偏差；二是不可行性惩罚项，用于处理违反供需约束的无效动作。对于小规模问题，建立了领域知识引导Q学习的收敛性理论保证；对于大规模问题，将改进策略嵌入DDPG框架形成DKDDPG算法。

实验结果 / Results

计算实验涵盖小规模和大规模场景，结果显示DKDDPG在所有测试设置中均稳定优于传统DDPG及其他对比的强化学习算法。具体而言，DKDDPG获得了更高的累积奖励，同时在训练时间和所需回合数上表现出更优的收敛效率，验证了领域知识注入对缓解Q学习初期估计偏差和加速策略学习的有效性。

应用价值 / Applications

该研究可直接应用于制造企业的产能共享与资源调度系统，如云制造平台中的分布式机床、劳动力或产能的动态分配。通过实现需求与产能的高效实时匹配，有助于提升制造资源利用率、降低闲置成本，并为工业4.0环境下的智能决策支持系统提供可扩展的算法基础。
