GSR-GNN: Training Acceleration and Memory-Saving Framework of Deep GNNs on Circuit Graph

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文针对电路图分析中的深度图神经网络（GNN）训练瓶颈问题，提出了一种高效的领域特定训练框架GSR-GNN。研究表明，可训练的深度GNN显著优于浅层架构，但受限于GPU内存和训练成本。
GSR-GNN通过可逆残差模块与分组稀疏非线性算子的结合，实现了数百层GNN的训练，同时降低计算和内存开销，在电路图任务上达到87.2%的峰值内存缩减和30倍以上的训练加速。

方法概述 / Method

GSR-GNN的核心创新包括三方面：采用可逆残差模块避免存储中间激活值，引入分组稀疏非线性算子在压缩节点嵌入的同时保留任务相关信息，以及设计优化的执行流水线消除碎片化的激活存储并减少数据移动。该框架专门针对电路图的领域特性进行定制优化。

实验结果 / Results

在采样电路图上的实验表明，GSR-GNN实现了最高87.2%的峰值内存降低和超过30倍的训练速度提升，同时在基于相关性的质量指标上仅有可忽略的性能退化。这使得深度GNN能够实际应用于大规模电子设计自动化（EDA）工作负载。

应用价值 / Applications

该研究为大规模电路分析和电子设计自动化（EDA）领域提供了实用的深度GNN训练方案，解决了工业级电路图规模下的内存瓶颈和训练效率问题。
GSR-GNN框架使原本因资源限制而不可行的深度模型训练成为可能，有望推动GNN在芯片设计验证、时序分析、功耗估计等关键EDA任务中的广泛应用。
