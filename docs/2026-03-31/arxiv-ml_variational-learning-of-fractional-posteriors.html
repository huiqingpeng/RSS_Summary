论文摘要 / Abstract

本文提出了一种新颖的单参数变分目标函数，该函数对数据证据给出下界估计，并支持近似分数后验（fractional posteriors）的估计。作者将该框架扩展至分层构造和贝叶斯后验，为概率建模提供了通用工具。实验表明，分数后验相比传统变分下界能获得更好的校准性，在变分自编码器（VAE）上可实现更高的证据下界，并使解码器与先验生成更加对齐。

方法概述 / Method

论文提出了一种参数化的变分目标函数，通过引入分数幂次参数来控制后验的"温度"，从而桥接变分推断与分数后验估计。该框架支持层次化扩展，可同时学习近似贝叶斯后验与分数后验，并在两种情形下推导出可解析计算的梯度。

实验结果 / Results

在混合模型上的模拟研究表明，分数后验比传统变分后验具有更好的校准性。应用于VAE时，该方法获得了更高的证据下界（ELBO），并且联合学习的分数后验与贝叶斯后验均表现优异。此外，使用分数后验训练的VAE生成的解码器与先验分布更加对齐，提升了生成质量。

应用价值 / Applications

该研究为概率生成模型（尤其是VAE）提供了更灵活的后验推断工具，可改善模型校准性和生成质量。潜在应用包括更可靠的密度估计、更好的隐空间结构学习，以及需要精确控制后验不确定性的场景（如安全关键决策、科学模拟等）。
