Central-to-Local Adaptive Generative Diffusion Framework for Improving Gene Expression Prediction in Data-Limited Spatial Transcriptomics

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本文提出了一种名为C2L-ST（Central-to-Local Spatial Transcriptomics）的自适应生成扩散框架，旨在解决空间转录组学（ST）中因高成本、低通量和数据共享限制导致的数据稀缺问题。该框架通过预训练全局中心模型学习大规模组织病理学图像的形态学先验知识，再利用少量配对图像-基因位点通过轻量级基因条件调制适应机构特异性局部模型，从而生成真实且分子一致的病理学图像。实验表明，生成的图像在视觉保真度、细胞组成复现和嵌入空间分布方面与真实数据高度一致，且合成数据用于下游训练时可显著提升基因表达预测的准确性和空间连贯性，仅需少量采样位点即可达到与真实数据相当的性能。

方法概述 / Method

C2L-ST采用"中心-局部"两阶段训练策略：首先在大型组织病理学数据集上预训练全局扩散模型以捕获可迁移的形态学表征；随后针对特定机构的有限ST数据，通过基因条件调制模块（gene-conditioned modulation）进行轻量级微调，实现机构特异性的局部适应。该框架利用扩散模型的生成能力，以基因表达为条件合成对应的组织病理学图像块，从而在数据受限场景下实现分子级别的数据增强。

实验结果 / Results

生成的合成图像在多个器官数据集上展现出高视觉真实感和结构保真度，与真实数据的嵌入空间高度重叠，同时保持了数据的多样性。在下游基因表达预测任务中，使用合成数据增强训练集显著提升了预测准确性和空间一致性，其性能可与使用完整真实数据相媲美，而仅需10%-25%的原始采样位点。该方法在跨器官泛化性实验中也表现出良好的域适应能力。

应用价值 / Applications

C2L-ST为空间转录组学研究提供了一种可扩展、数据高效的分子级数据增强方案，特别适用于数据获取受限的临床和研究机构。该框架可广泛应用于整合组织形态学与转录组学的空间生物学研究，如肿瘤微环境分析、器官发育研究和精准医疗中的生物标志物发现，为计算病理学和空间组学模型的训练提供了域自适应且可泛化的技术基础。
