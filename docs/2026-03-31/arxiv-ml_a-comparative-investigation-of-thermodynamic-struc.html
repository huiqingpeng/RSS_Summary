A Comparative Investigation of Thermodynamic Structure-Informed Neural Networks

arXiv Machine Learning 2026-03-31
Read Original Article →

论文摘要 / Abstract

本论文系统比较了不同热力学结构信息神经网络（thermodynamic structure-informed neural networks）的性能，通过将牛顿力学、拉格朗日力学、哈密顿力学等保守系统表述，以及昂萨格变分原理和扩展不可逆热力学等耗散系统表述融入物理信息神经网络（PINNs）。研究发现，基于牛顿残差的PINNs虽能重构系统状态，但无法可靠恢复关键物理和热力学量；而结构保持型表述能显著提升参数识别、热力学一致性和鲁棒性。该研究为热力学一致性模型的原则性设计提供了实用指导。

方法概述 / Method

论文采用系统比较研究方法，将多种热力学理论框架（牛顿力学、拉格朗日力学、哈密顿力学、昂萨格变分原理、扩展不可逆热力学）分别嵌入PINNs的损失函数中，构建不同的结构信息神经网络变体。通过设计涵盖常微分方程和偏微分方程的代表性数值实验，定量评估各表述在准确性、物理一致性、抗噪性和可解释性方面的表现。

实验结果 / Results

实验结果表明，牛顿残差基础的PINNs在状态重构上表现尚可，但在关键物理量和热力学量的恢复上存在显著缺陷；相比之下，基于结构保持表述（如哈密顿力学和昂萨格变分原理）的神经网络在参数识别精度、热力学一致性约束满足度以及对观测噪声的鲁棒性方面均有显著优势。量化指标显示结构保持方法在多任务评估中 consistently 优于传统残差方法。

应用价值 / Applications

该研究为科学机器学习领域提供了模型选择的原则性指导，特别适用于需要严格满足热力学约束的复杂物理系统建模，如能源系统、材料相变、流体力学和生物物理过程等。研究成果有助于推动非平衡热力学结构与物理信息机器学习的深度融合，为开发更具物理可解释性和可靠性的AI for Science模型奠定基础。
