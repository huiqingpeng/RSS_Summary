论文摘要 / Abstract

本文提出了DRIFT（Maximin Learning框架），用于从高维项目级数据中估计稳健的个体化治疗效果（ITE），以解决精准心理健康中多领域异质性症状的挑战。该方法通过广义因子分析学习潜在构念，构建锚定目标的不确定性集合来外推至更广泛的潜在结果超总体，并通过对抗学习优化最坏情况性能。在重度抑郁症随机对照试验（EMBARC）中，DRIFT展现出优越的性能，并能有效泛化至训练时未使用的副作用和自我报告症状等外部多领域结果。

方法概述 / Method

DRIFT采用两阶段方法：首先利用广义因子分析从观测症状中学习低维潜在因子表示，然后构建基于锚定目标的不确定性集合来近似潜在结果的超总体分布。通过在该不确定性集合上求解极大极小优化问题，DRIFT获得了对潜在因子可容许重参数化不变的闭式解，确保了对未观测或代表性不足领域的鲁棒性。

实验结果 / Results

在EMBARC临床试验数据分析中，DRIFT相比现有方法表现出更优的个体化治疗效果估计性能，并成功泛化到训练阶段未纳入的外部多领域结果，包括药物副作用和患者自我报告症状。理论分析证明了DRIFT在识别性和收敛性方面的保证，验证了其统计可靠性。

应用价值 / Applications

该研究为精准心理健康临床决策提供了重要工具，使治疗推荐能够同时考虑多个异质性症状领域而非依赖单一汇总指标。
DRIFT的鲁棒性保证使其特别适用于症状测量不完整或存在未观测临床相关领域的实际医疗场景，有望改善抑郁症等精神疾病的个性化治疗分配和患者预后。
