Project and Generate: Divergence-Free Neural Operators for Incompressible Flows

arXiv Machine Learning 2026-03-27

论文摘要 / Abstract

本文提出了一个统一框架，将不可压缩连续性方程作为硬性内在约束嵌入到确定性和生成式流体动力学模型中。对于确定性模型，作者引入基于Helmholtz-Hodge分解的可微分谱Leray投影，将回归假设空间限制在物理可接受的速度场；对于生成式建模，则通过基于旋度的前推构造无散度高斯参考测度，确保整个概率流保持子空间一致性。实验表明，该方法在2D Navier-Stokes方程上实现了精确的不可压缩性，并显著提升了稳定性和物理一致性。

方法概述 / Method

论文采用两种核心技术：一是可微分谱Leray投影算子，利用Helmholtz-Hodge分解将任意向量场分解为无散度分量和无旋分量，通过投影消除散度分量；二是基于旋度算子的前推映射，将标准高斯噪声转换为无散度随机场，作为扩散模型的参考测度，从根本上保证生成分布的先验兼容性。

实验结果 / Results

在2D Navier-Stokes方程上的实验表明，该方法实现了精确的无散度约束（误差仅来自离散化），相比惩罚方法消除了虚假散度；长期模拟稳定性显著提升，避免了传统方法因散度累积导致的数值崩溃，同时保持了生成样本的物理合理性和分布多样性。

应用价值 / Applications

该研究适用于需要严格满足质量守恒的不可压缩流模拟，如气象预报、海洋环流建模和工业CFD仿真；其生成式建模能力可用于流场不确定性量化和稀有事件采样，为科学机器学习中的物理约束嵌入提供了可推广的范式。
