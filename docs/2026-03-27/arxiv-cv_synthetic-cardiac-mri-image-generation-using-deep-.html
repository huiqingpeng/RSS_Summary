论文摘要 / Abstract

本文综述了基于深度生成模型的心脏MRI（CMRI）合成图像生成技术，系统比较了GANs、VAEs、扩散概率模型和流匹配等方法在解决医学影像数据稀缺、厂商差异和隐私泄露风险方面的进展。研究发现，掩码条件生成、扩散与流匹配模型能够提升解剖结构保真度和边界保持能力，而差分隐私和成员推断攻击等机制可有效保障数据隐私。该综述强调需要建立整合保真度、实用性和隐私评估的综合性框架，以支持可靠的临床应用。

方法概述 / Method

论文综述了多种深度生成模型在CMRI合成中的应用，包括生成对抗网络（GANs）、变分自编码器（VAEs）、扩散概率模型和流匹配技术；其中掩码条件生成通过分割图引导合成以提升结构准确性，厂商风格条件化和强度归一化等预处理步骤用于实现跨域泛化，同时采用成员推断攻击、最近邻分析和差分隐私机制评估和增强隐私保护。

实验结果 / Results

现有研究表明，解剖约束的合成数据能够提升多厂商环境下的分割准确性和鲁棒性；扩散模型和流匹配模型展现出较强的边界保持能力和高效的确定性变换特性；隐私评估方面，差分隐私机制可有效降低模型记忆导致的隐私泄露风险，而下游分割性能是衡量合成数据实用性的常用指标。

应用价值 / Applications

该研究为医学影像数据增强、隐私保护型数据共享和跨厂商模型训练提供了技术路径，可缓解标注数据稀缺问题并降低患者隐私泄露风险；建立的评估框架有助于推动合成CMRI在临床研究、算法开发和多中心协作中的可靠应用，对促进精准医疗和AI辅助诊断具有重要意义。
