DepthCharge: A Domain-Agnostic Framework for Measuring Depth-Dependent Knowledge in Large Language Models

arXiv Machine Learning 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出了DepthCharge，首个用于测量大语言模型（LLMs）深度知识能力的领域无关框架。该框架通过自适应追问、按需事实验证和恒定样本量的生存统计三大创新，解决了现有方法无法评估模型在特定领域细节中持续准确响应能力的问题。实验发现，不同模型-领域组合的期望有效深度（EVD）差异显著（3.45-7.55），且昂贵模型未必具备更深的知识，挑战了传统基准测试的有效性。

方法概述 / Method

DepthCharge采用自适应探测机制，根据模型实际提及的概念生成递进式追问，而非预设问题链；同时整合权威来源的实时事实验证，并运用生存统计方法确保每层深度的评估样本量恒定。该框架无需预构建测试集或领域专家知识，可部署于任何具有公开可验证事实的知识领域。

实验结果 / Results

在医学、宪法、古罗马史和量子计算四个领域的验证显示，标准基准掩盖了模型的深度依赖性能差异：模型排名因领域而异，无单一模型全面领先。成本效益分析进一步揭示，高价模型（如GPT-4）在某些领域的知识深度不及更经济的替代方案，EVD指标为模型选择提供了更精细的决策依据。

应用价值 / Applications

DepthCharge适用于需要深度专业知识的场景，如法律咨询、医疗诊断、学术研究等高风险专业领域，帮助从业者根据具体领域而非综合排名选择最优模型。该框架还可用于模型开发者识别知识盲区、优化训练策略，以及监管机构建立领域特定的AI能力评估标准。
