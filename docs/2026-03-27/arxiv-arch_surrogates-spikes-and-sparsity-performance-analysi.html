Surrogates, Spikes, and Sparsity: Performance Analysis and Characterization of SNN Hyperparameters on Hardware

arXiv Computer Architecture 2026-03-27
Read Original Article →

论文摘要 / Abstract

本研究系统分析了脉冲神经网络（SNN）训练超参数对硬件推理性能的影响，发现标准准确率指标与硬件效率之间存在显著脱节。研究量化了替代梯度函数（如Fast Sigmoid、Spike Rate Escape）和神经元模型（LIF、Lapicque）在三个事件视觉数据集上的敏感性，揭示了通过稀疏性感知的超参数选择可同时提升准确率9.1%并降低延迟2倍以上，为从训练参数预测硬件行为建立了方法论基础。

方法概述 / Method

研究采用工作负载特征分析方法，在自定义的周期精确FPGA-based SNN仪器平台上进行硬件级验证。通过解耦替代梯度函数和神经元模型配置对分类准确率与推理效率的影响，系统评估了不同超参数组合在真实硬件上的延迟表现。

实验结果 / Results

关键发现包括：Fast Sigmoid在DVS-CIFAR10上准确率最高，但Spike Rate Escape在DVS128-Gesture上可减少12.2%的推理延迟且精度损失极小；从LIF神经元切换至Lapicque神经元可降低高达28%的延迟；稀疏性感知的超参数选择相比基线实现准确率提升9.1%、延迟降低超过2倍。标准准确率指标被证明是硬件效率的差预测因子。

应用价值 / Applications

该研究为边缘AI和神经形态计算系统的低功耗推理提供了实用的超参数优化指导，特别适用于事件相机等稀疏视觉传感场景。建立的训练-硬件协同设计方法可指导SNN在FPGA等可定制硬件上的高效部署，弥合理论能效优势与实际硬件性能之间的差距。
