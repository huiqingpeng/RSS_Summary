TIGeR: A Unified Framework for Time, Images and Geo-location Retrieval

arXiv Computer Vision 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出了TIGeR，一个统一的地理-时间感知图像检索框架，用于解决数字取证、城市监控等应用中需要联合推理视觉外观、地理位置和时间的复杂需求。作者形式化定义了"地理-时间感知图像检索"问题，并构建了包含450万训练三元组和8.6万测试三元组的大规模基准数据集。
TIGeR通过多模态Transformer将图像、地理位置和时间映射到统一的地理-时间嵌入空间，在地理定位、拍摄时间预测和地理-时间感知检索三项任务上均显著超越现有方法。

方法概述 / Method

TIGeR采用多模态Transformer架构，支持灵活的输入配置（单模态或多模态查询），能够处理图像、地理坐标和时间戳三种模态信息。该方法将三种模态编码到统一的嵌入空间中，使得模型可以基于场景的"何地何时"身份而非纯粹视觉相似性进行检索，从而更好地保持场景在大幅外观变化下的位置身份一致性。

实验结果 / Results

实验表明TIGeR在各项任务上均显著优于强基线和最先进方法：年度时间预测提升16%，日内时间预测提升8%，地理-时间感知检索召回率提升14%。这些结果凸显了统一地理-时间建模的优势，证明了联合建模三种模态能够有效应对场景外观随时间变化的挑战。

应用价值 / Applications

TIGeR在数字取证（如定位特定时间和地点的监控影像）、城市监测（追踪同一地点不同时期的城市变化）和环境分析（长期生态监测）等领域具有重要应用价值。该框架的灵活查询能力使其能够适应多种实际场景，例如"找到与查询图同地点但特定目标时间拍摄的图像"这类传统视觉检索无法完成的复杂任务。
