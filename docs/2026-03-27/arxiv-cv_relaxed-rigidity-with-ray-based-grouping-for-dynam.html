Relaxed Rigidity with Ray-based Grouping for Dynamic Gaussian Splatting

arXiv Computer Vision 2026-03-27

论文摘要 / Abstract

本文提出了一种用于动态3D高斯溅射的新方法，通过引入视图空间射线分组策略来显式保持高斯点在时间序列中的局部几何结构。该方法将沿同一条射线相交且α混合权重超过阈值的高斯点聚类，并对其施加约束以维持一致的空间分布，从而无需依赖光流或2D轨迹等外部先验即可实现物理上更合理的运动建模。实验表明，该方法在单目视频数据集上显著优于现有方法，实现了更优的时间一致性和重建质量。

方法概述 / Method

核心创新是"松弛刚性"（Relaxed Rigidity）约束机制：基于射线的高斯分组策略在视图空间中将沿同一相机射线相交的高斯点动态聚类，仅考虑对最终渲染贡献显著（α权重超过阈值）的点；随后对这些组施加几何一致性约束，保持其相对空间分布稳定。该方法可直接集成到现有动态高斯溅射基线模型中，作为即插即用的时间正则化模块。

实验结果 / Results

作者在两种不同基线模型上验证了方法的有效性，在具有挑战性的单目视频数据集上进行了大量实验。结果表明，该方法显著提升了现有方法的性能，在保持时间一致性的同时改善了重建质量，且无需依赖外部光流或跟踪信息作为监督信号。

应用价值 / Applications

该方法适用于从单目视频重建动态3D场景的任务，在虚拟现实、增强现实、数字人建模和动态场景理解等领域具有重要价值。由于消除了对外部先验的依赖，该方法降低了动态神经渲染的门槛，使其更易部署于计算资源受限或缺乏预训练模型的实际应用场景。
