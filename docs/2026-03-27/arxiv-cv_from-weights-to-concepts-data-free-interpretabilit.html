From Weights to Concepts: Data-Free Interpretability of CLIP via Singular Vector Decomposition

论文摘要 / Abstract

本文提出了SITH（Transformer头的语义检查），一种完全无需数据、无需训练的CLIP视觉Transformer可解释性框架。该方法通过奇异向量分解直接分析注意力头的权重空间，并引入COMP算法将奇异向量解释为人类可理解的语义概念组合。研究表明，SITH能够实现精确、忠实的头内解释，并支持可解释的权重空间模型编辑，同时揭示微调过程主要是对稳定语义基底的重新加权而非学习全新特征。

方法概述 / Method

SITH框架对每个注意力头的值-输出矩阵进行奇异向量分解（SVD），将权重矩阵分解为具有明确语义意义的正交分量。作者提出COMP（相干正交匹配追踪）算法，通过稀疏、语义相干的可解释概念组合来解释每个奇异向量，从而实现无需任何训练数据或模型激活的纯权重空间分析。

实验结果 / Results

实验验证了SITH在重建保真度和可解释性方面的有效性，证明其能够产生连贯且忠实的头内解释。基于SITH的权重空间模型编辑可以精确放大或抑制特定概念，无需重新训练即可提升下游任务性能。此外，对模型适应过程的分析表明，微调主要重新加权预训练阶段建立的稳定语义基底。

应用价值 / Applications

SITH为大规模部署的视觉-语言模型提供了数据无关的安全审计工具，可在无法访问敏感训练数据的场景下进行模型解释。该方法支持精确的模型编辑和概念操控，适用于需要干预模型行为的安全关键应用，同时为理解模型微调机制提供了新视角，有助于开发更高效的迁移学习策略。
