Towards Safe Learning-Based Non-Linear Model Predictive Control through Recurrent Neural Network Modeling

arXiv Machine Learning 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出Sequential-AMPC，一种基于循环神经网络（RNN）的序列神经策略，通过跨预测时域共享参数来生成MPC候选控制序列，显著减少了专家MPC数据需求和训练成本。该策略配合安全增强的在线评估与回退机制（Safe Sequential-AMPC），在多个基准测试中展现出比朴素前馈策略更高的可行性率和闭环安全性。在高维系统上，该方法以更少的训练轮次实现更优的学习动态和性能，同时保持稳定的验证改进，避免了前馈基线可能出现的停滞问题。

方法概述 / Method

核心方法是采用循环神经网络建模，通过跨预测时域共享参数来生成序列化的MPC候选控制序列，而非独立预测每个时刻的控制量。部署时引入安全增强的在线评估机制，当神经网络生成的候选序列不满足安全约束时，自动回退到可靠的备用控制器，确保闭环系统的安全性保证。

实验结果 / Results

实验表明，Sequential-AMPC相比朴素前馈策略基线，仅需极少量的专家MPC rollout即可训练，且生成的候选控制序列具有更高的可行性率。在高维系统实验中，该方法展现出更优的学习动态，在更少训练轮次内达到更好性能，同时验证损失持续稳定下降，而前馈基线则出现明显的训练停滞现象。

应用价值 / Applications

该研究为嵌入式硬件上的非线性模型预测控制提供了实用解决方案，特别适用于机器人控制、自动驾驶等需要高控制频率且计算资源受限的场景。通过大幅降低专家数据需求和在线计算开销，同时保持形式化的安全保证，该方法有望加速学习型MPC在工业实时控制系统中的实际部署。
