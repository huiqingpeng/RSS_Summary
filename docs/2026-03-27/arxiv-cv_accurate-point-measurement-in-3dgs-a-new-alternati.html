Accurate Point Measurement in 3DGS -- A New Alternative to Traditional Stereoscopic-View Based Measurements

arXiv Computer Vision 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出了一种基于3D高斯泼溅（3DGS）的精确三维点测量新方法，作为传统立体视觉测量的替代方案。该方法利用3DGS生成的高质量视图，允许用户在多个视图中直观选取同名点，通过三角测量实现精确三维测量，无需昂贵的立体工作站或专业操作技能。实验表明，该方法在标准硬件上可达到1-2厘米的RMSE精度，在薄结构和尖锐角点等挑战性场景下显著优于基于网格的直接测量方法。

方法概述 / Method

该方法将3DGS作为新型视图合成器，用户可在渲染的精确源视图和平滑插值中间视图中交互选取同名点，通过多视图三角测量计算三维坐标。研究团队开发了基于Web的概念验证应用，支持多视图交会（超过两个视图）以提高测量精度，突破了传统立体测量仅需双目视图的限制。

实验结果 / Results

在多个无人机航拍数据集上的测试显示，该方法在标准硬件上达到与传统立体测量相当或更高的精度。定量结果表明：良好定义点的RMSE为1-2厘米；在薄结构测量中，网格方法RMSE为0.062米，而该方法达到0.037米；在网格重建失败的尖锐角点处，该方法成功测量所有点，RMSE仅为0.013米。

应用价值 / Applications

该方法可广泛应用于测绘、建筑检测、基础设施监测等领域，显著降低了高精度三维测量的设备和技能门槛。基于Web的实现方案使专业测量工具更加普及化，特别适用于需要快速部署、远程协作或资源受限的场景，为数字孪生、智慧城市等应用提供了经济高效的测量解决方案。
