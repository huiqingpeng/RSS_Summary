Attention-based Pin Site Image Classification in Orthopaedic Patients with External Fixators

arXiv Computer Vision 2026-03-27
Read Original Article →

论文摘要 / Abstract

本研究针对骨科外固定器患者针道感染识别难题，构建了一个针道伤口感染图像数据集，并提出了一种基于注意力机制的深度学习分类方法。该方法将针道图像分为A组（有炎症或感染迹象）和B组（无明显并发症），通过强调相关区域并减少金属针干扰，实现了仅基于视觉感染征象的针道状态判别。模型在仅使用577万参数的情况下取得了0.975的AUC和0.927的F1分数，显著优于基线方法，展现了深度学习在辅助临床针道感染评估中的潜力。

方法概述 / Method

论文提出了一种基于注意力机制的深度学习模型，专门用于处理金属针/皮肤界面的复杂场景；同时引入了高效冗余重建卷积（ERRC）模块，在增强特征图丰富度的同时有效减少模型参数量。该注意力机制能够自动聚焦于感染相关区域，而抑制金属针等无关结构的干扰。

实验结果 / Results

所提模型在针道图像分类任务上表现优异，达到0.975的AUC和0.927的F1分数，显著超越基线对比方法；模型参数量仅为5.77M，实现了高精度与轻量化的有效平衡。这些结果表明该模型能够可靠地基于视觉征象区分感染性与非感染性针道，与临床专业人员的评估标准高度一致。

应用价值 / Applications

该研究可直接应用于骨科外固定器患者的远程监测与早期感染筛查，辅助医护人员及时识别针道感染迹象，减少患者痛苦并降低并发症发生率；轻量化的模型设计便于部署在移动设备或边缘计算平台，支持家庭护理和偏远地区的医疗资源延伸，具有显著的临床转化价值和公共卫生意义。
