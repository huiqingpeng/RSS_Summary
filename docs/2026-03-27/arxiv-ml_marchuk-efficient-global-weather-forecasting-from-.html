Marchuk: Efficient Global Weather Forecasting from Mid-Range to Sub-Seasonal Scales via Flow Matching

arXiv Machine Learning 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出了Marchuk，一种基于生成式潜空间流匹配（flow matching）的全球天气预报模型，可将预测时效从中期（约15天）延伸至次季节尺度（最长30天）。该模型以当前天气图为条件，在学习的潜空间中自回归预测后续天气状态。通过将旋转位置编码（RoPE）替换为可训练位置嵌入并扩展时间上下文窗口，模型显著增强了对长程时间依赖的表征和传播能力。
Marchuk仅用2.76亿参数即可达到16亿参数的LaDCast模型相当的预测性能，同时推理速度大幅提升。

方法概述 / Method

Marchuk采用生成式潜空间流匹配架构，将高维天气数据压缩至低维潜空间进行预测，再通过解码器重建天气图。核心改进包括：用可训练位置嵌入替代RoPE以更好地捕捉时序信息，以及扩展时间上下文窗口以增强长程依赖建模。模型以自回归方式逐日生成未来天气状态，实现长达30天的连续预报。

实验结果 / Results

Marchuk在2.76亿参数规模下取得了与16亿参数LaDCast相当的预测精度，实现了约5.8倍的参数效率提升。同时，该模型具有显著更高的推理速度，在保持预报质量的前提下大幅降低了计算成本。实验验证了模型在中期至次季节预报任务上的有效性，突破了传统数值模式约15天的可预报性极限。

应用价值 / Applications

Marchuk可广泛应用于农业规划、水资源管理、能源调度、灾害预警等对次季节预报有迫切需求的领域，为两周至一个月尺度的决策提供科学依据。其高效轻量的架构设计特别适合在计算资源受限的环境中部署，有望推动全球天气预报技术的普惠化发展。开源发布将促进气象AI社区的技术迭代与协作创新。
