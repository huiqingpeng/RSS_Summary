论文摘要 / Abstract

本文提出了KitchenTwin，一种尺度感知的3D融合框架，用于构建具有精确度量几何和语义基础的厨房数字孪生环境。该方法通过视觉语言模型（VLM）引导的几何锚点机制，解决了基于Transformer的前馈重建方法中固有的尺度模糊和坐标不一致问题，实现了局部物体重建网格与全局点云的可靠配准。实验表明，该方法在真实室内厨房环境中显著提升了跨网络物体对齐和几何一致性，并开源了首个具有度量尺度场景和语义注册物体网格标注的室内数字孪生数据集。

方法概述 / Method

论文核心方法包含VLM引导的几何锚点机制和几何感知配准流水线两部分：前者利用视觉语言模型从图像中恢复真实世界度量尺度，解决坐标系统失配问题；后者通过重力对齐垂直估计、曼哈顿世界结构约束和无碰撞局部优化，显式强制物理合理性以融合不同网络输出。

实验结果 / Results

在真实室内厨房环境中的实验表明，该方法实现了改进的跨网络物体对齐性能和几何一致性，支持下游任务如多基元拟合和度量测量；同时发布的开源数据集包含度量尺度场景和语义注册的物体中心网格标注，为具身智能训练和评估提供了基准。

应用价值 / Applications

该研究为具身智能（Embodied AI）的训练与评估提供了高精度、度量一致的三维数字孪生环境，可广泛应用于机器人操作规划、室内场景理解、AR/VR内容生成及智能厨房自动化等领域；开源数据集填补了室内度量尺度数字孪生数据的空白，有助于推动相关算法的标准化评测。
