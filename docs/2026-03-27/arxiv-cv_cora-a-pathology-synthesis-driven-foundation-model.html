CORA: A Pathology Synthesis Driven Foundation Model for Coronary CT Angiography Analysis and MACE Risk Assessment

arXiv Computer Vision 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出了CORA，一种面向冠状动脉CT血管造影（CCTA）分析的3D视觉基础模型，用于综合心血管风险评估。
CORA采用以病理为中心、合成驱动的自监督学习框架，通过解剖学引导的病变合成引擎显式训练模型检测模拟血管异常，使表征学习偏向临床相关的疾病特征而非主导的背景解剖结构。该模型在12,801例未标注CCTA数据上训练，在九家独立医院的多中心数据集上验证，在斑块特征分析、狭窄检测和冠状动脉分割等任务中 consistently 优于现有最先进的3D视觉基础模型，性能提升高达29%。此外，通过将图像编码器与大型语言模型结合，CORA扩展为多模态框架，显著改善了30天主要不良心血管事件（MACE）风险分层。

方法概述 / Method

CORA的核心创新在于其病理合成驱动的自监督预训练策略：利用解剖学引导的病变合成引擎在健康CCTA数据中生成模拟的冠状动脉斑块和狭窄病变，使模型在预训练阶段即学习识别局部病理特征。该框架采用3D视觉Transformer架构处理体素级CCTA数据，通过对比学习或掩码预测任务让模型区分合成病变与正常组织，从而克服传统掩码图像建模对全局解剖统计的偏向性。

实验结果 / Results

CORA在来自九家独立医院的多中心数据集上进行了全面评估，涵盖诊断任务（斑块表征、狭窄检测）和解剖任务（冠状动脉分割）。与现有最先进的3D视觉基础模型（如MAE、Swin Transformer等）相比，CORA在所有任务上均实现性能提升，最高达29%。在关键的临床应用场景中，CORA与大型语言模型结合形成的多模态框架在30天MACE风险预测任务上表现出显著改善的风险分层能力，证明了其临床转化潜力。

应用价值 / Applications

CORA具有广泛的临床应用价值：可作为统一的基础平台支持冠状动脉疾病的自动化筛查与诊断，减轻放射科医师的阅片负担；其MACE风险预测能力可整合入临床决策支持系统，辅助制定个体化的治疗策略和随访计划；该模型的可扩展架构也支持未来向其他心血管影像模态（如心脏MRI、血管内超声）迁移，为精准心血管医学提供可泛化的AI基础设施。
