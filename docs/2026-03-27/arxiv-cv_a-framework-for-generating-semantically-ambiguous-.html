A Framework for Generating Semantically Ambiguous Images to Probe Human and Machine Perception

arXiv Computer Vision 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文提出了一个基于心理物理学的方法框架，通过在CLIP嵌入空间中插值生成语义模糊的图像连续谱，以探测人类和机器分类器的感知边界。研究发现机器分类器更倾向于识别为"兔子"，而人类感知与用于合成的CLIP嵌入更为一致，且引导尺度对人类敏感度的影响强于机器分类器。该框架展示了受控模糊性如何作为诊断工具，连接人类心理物理分析、图像分类与生成模型，为理解人机对齐、模型鲁棒性和可解释性提供新视角。

方法概述 / Method

作者利用CLIP文本-图像嵌入空间的语义连续性，在成对概念（如"鸭子-兔子"）之间进行插值，生成从纯概念A到纯概念B的模糊图像连续谱。该框架结合心理物理学实验范式，通过控制生成模型的引导尺度（guidance scale），系统性地量化和比较人类观察者与机器分类器的语义决策边界位置。

实验结果 / Results

实验表明机器分类器存在系统性偏差——相比人类更倾向于将模糊图像判定为"兔子"；人类感知与CLIP嵌入空间的几何结构更为对齐，而机器分类器的决策边界则发生偏移。此外，增加引导尺度对人类感知敏感度的提升效果显著强于对机器分类器的影响，揭示了人机感知机制的差异性。

应用价值 / Applications

该框架可广泛应用于评估视觉-语言模型的语义对齐质量、诊断生成模型的概念边界缺陷，以及优化人机交互系统的感知一致性。在AI安全领域，该方法有助于识别模型偏见；在认知科学领域，为比较人类与人工神经网络的表征机制提供了可控的实验范式；同时可指导更鲁棒、可解释的图像合成算法设计。
