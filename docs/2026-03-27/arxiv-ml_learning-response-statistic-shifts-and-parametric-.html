Learning Response-Statistic Shifts and Parametric Roll Episodes from Wave--Vessel Time Series via LSTM Functional Models

arXiv Machine Learning 2026-03-27
Read Original Article →

论文摘要 / Abstract

本文开发了一种数据驱动的代理模型，利用堆叠LSTM网络学习从入射波-运动时间序列到船舶运动的非线性因果函数映射。该模型能够复现参数横摇（parametric roll）现象及其伴随的响应统计特性突变，且训练数据来源灵活，可来自物理实验或高保真数值模拟。研究通过URANS数值波浪水池生成训练数据，验证了模型在追踪大幅横摇起始与增长、捕捉概率密度函数变化方面的有效性，并比较了不同损失函数在平均误差与尾部保真度之间的权衡。

方法概述 / Method

研究采用堆叠长短期记忆（LSTM）神经网络构建函数式代理模型，以波浪高程时间序列作为输入，预测船舶运动响应。训练数据基于URANS数值波浪水池生成，包含三种海况下各49组随机相位实现的长峰不规则波，固定航速选取为使船舶遭遇参数横摇激励的条件。模型评估采用时域精度指标和分布保真度指标，并对比了均方误差（MSE）、相对熵目标函数及振幅加权变体等不同损失函数设计。

实验结果 / Results

在最恶劣海况下，LSTM代理模型成功追踪了与参数激励一致的大幅横摇起始与增长过程，并准确捕捉了横摇概率密度函数的相应变化。对比实验表明，相对熵和振幅加权损失函数虽略微增加平均误差，但显著改善了分布尾部的保真度，这对船舶可操作性和风险评估至关重要。模型在保留测试集上展现出良好的泛化能力和分布外样本预测性能。

应用价值 / Applications

该研究为船舶设计与运营安全提供了高效的计算工具，可在设计阶段通过高保真模拟数据训练代理模型，快速预测参数横摇风险，替代耗时的CFD计算。模型还可集成至实时决策支持系统，用于海上航行状态监测与预警。此外，该数据驱动框架具有数据源无关性，可迁移应用于物理实验数据，支持智能船舶和自主航行系统的开发。
