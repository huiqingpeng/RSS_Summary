PoiCGAN: A Targeted Poisoning Based on Feature-Label Joint Perturbation in Federated Learning

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本文针对联邦学习（FL）中的中毒攻击问题，提出了一种名为PoiCGAN的新型目标性中毒攻击方法。现有中毒攻击方法难以绕过基于模型异常检测的性能测试和防御机制，导致攻击实用性受限。
PoiCGAN通过修改条件生成对抗网络（CGAN）中判别器和生成器的输入来影响训练过程，生成理想的中毒生成器，该生成器不仅能产生特定中毒样本，还能自动执行标签翻转。实验结果表明，该方法相比基线方法攻击成功率提高83.97%，主任务准确率下降不到8.87%，且中毒样本和恶意模型具有高隐蔽性。

方法概述 / Method

PoiCGAN基于特征-标签联合扰动的思想，通过对CGAN的判别器和生成器输入进行协同修改，引导生成器学习产生既具有特定目标特征又伴随自动标签翻转的中毒样本。该方法将中毒样本生成与标签操纵统一到一个生成框架中，实现了攻击效果与隐蔽性的平衡。

实验结果 / Results

在多个数据集上的实验表明，PoiCGAN的攻击成功率比基线方法高出83.97%，同时主任务准确率仅下降不到8.87%。此外，生成的中毒样本和恶意模型表现出高度的隐蔽性，能够有效规避基于模型异常检测的防御机制。

应用价值 / Applications

该研究对工业图像分类等联邦学习应用场景的安全防护具有重要意义。一方面，它揭示了现有联邦学习防御机制的潜在漏洞，为设计更鲁棒的安全方案提供了参考；另一方面，其高隐蔽性攻击特性也可用于安全测试和红队评估，帮助工业界在实际部署前识别和修复联邦学习系统的安全风险。
