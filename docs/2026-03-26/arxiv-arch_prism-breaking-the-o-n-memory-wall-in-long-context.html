PRISM: Breaking the O(n) Memory Wall in Long-Context LLM Inference via O(1) Photonic Block Selection

arXiv Computer Architecture 2026-03-26
Read Original Article →

论文摘要 / Abstract

长上下文大语言模型推理的瓶颈并非计算能力，而是每步解码时扫描KV缓存的O(n)内存带宽成本。本文首次发现粗粒度块选择任务（决定获取哪些KV块的内存受限相似性搜索）与光子广播加权范式结构匹配，提出PRISM系统——一种薄膜铌酸锂（TFLN）相似性引擎。该系统的光子评估成本保持O(1)，而电子扫描成本随上下文长度线性增长，在64K上下文实现16倍流量降低，相比GPU基线获得四个数量级的能效优势。

方法概述 / Method

PRISM利用光子技术的广播-加权特性：查询通过无源分路广播至所有候选块，静态签名匹配电光微环谐振器（MRR）编程，且仅需排序结果（精度放宽至4-6位）。系统采用薄膜铌酸锂平台实现内积相似性计算，将块选择从内存受限的电子扫描转为常数时间的光子评估。

实验结果 / Results

在Qwen2.5-7B上的硬件受损针堆实验显示，从4K到64K令牌均保持100%准确率（k=32）；64K上下文下实现16倍流量缩减。能效方面，PRISM在实际上下文长度（n≥4K）相比GPU基线获得四个数量级（10,000倍）的能耗优势，且优势随上下文长度增加而扩大。

应用价值 / Applications

PRISM可部署于数据中心长上下文LLM推理场景，突破当前O(n)内存墙限制，支持更高效的检索增强生成（RAG）、长文档分析和多轮对话系统。光子硬件的低能耗特性使其特别适合大规模部署，为下一代AI基础设施提供可持续的能效解决方案。
