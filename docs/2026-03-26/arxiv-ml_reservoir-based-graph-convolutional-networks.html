论文摘要 / Abstract

本文提出了RGC-Net（基于储层的图卷积网络），通过将储层计算与结构化图卷积相结合，解决了传统GCN在捕获长距离依赖时面临的计算成本高和过度平滑问题。该模型采用固定随机储层权重和泄漏积分器来增强特征保留能力，在图分类和图生成任务上实现了最先进的性能。实验表明，RGC-Net具有更快的收敛速度和更少的过度平滑现象，并成功应用于动态脑连接网络生成任务。

方法概述 / Method

RGC-Net核心创新在于重新设计的卷积框架：使用固定随机的储层权重替代传统可训练参数，并引入泄漏积分器机制来平衡历史状态与新输入信息的融合。该架构通过迭代消息传递动态实现稳定的信息传播，无需大量参数调优即可有效聚合多跳邻域信息。

实验结果 / Results

RGC-Net在图分类和生成任务上均达到 state-of-the-art 性能，相比传统GCN展现出更快的训练收敛速度；在动态脑图演化预测任务中，基于RGC-Net构建的Transformer模型成功实现了脑连接网络生成，同时显著缓解了深层网络中的过度平滑问题。

应用价值 / Applications

该研究为复杂动态图数据的分析提供了高效工具，特别适用于神经科学领域的脑功能连接网络建模与演化预测；其轻量化的储层设计降低了计算资源需求，可推广至社交网络分析、分子结构生成等需要处理大规模动态图结构的实际场景。
