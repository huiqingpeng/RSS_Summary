PowerFlow-DNN: Compiler-Directed Fine-Grained Power Orchestration for End-to-End Edge AI Inference

arXiv Computer Architecture 2026-03-26

论文摘要 / Abstract

本文提出了PowerFlow-DNN，一种编译器指导的细粒度功耗编排框架，用于端到端边缘AI推理。针对现有层级功耗控制方法因层间耦合决策导致意外开销的问题，该框架将截止时间约束的实时周期性推理形式化为统一的层间功耗调度问题，自动发现满足截止时间且考虑端到端层间影响的最小能耗功耗状态调度方案。在TSMC 40nm工艺的DNN加速器VLSI实现上评估表明，该方法在超过10^160种可能的功耗状态分配组合空间中，仍能找到接近最优的解，能耗比无功耗编排的激进基线降低高达37%，且求解器速度提升达2.14倍。

方法概述 / Method

PowerFlow-DNN采用编译器指导的方法，构建了一个严格的数学问题形式化框架，将DVFS（动态电压频率调节）和电源门控等细粒度功耗管理机制联合优化。该方法通过构建结构化的分层状态图来表示层间功耗状态的组合空间，并设计轻量级剪枝策略实现高效优化，同时考虑有限电压轨和状态转换开销等实际约束条件。

实验结果 / Results

在TSMC 40nm技术实现的DNN加速器上，PowerFlow-DNN在离散化问题形式下发现的解与精确ILP（整数线性规划）神谕相比能耗差距仅0.68%，相比无功耗编排的激进基线最高节能37%。尽管搜索空间超过10^160种可能的功耗状态分配，该框架通过结构化分层状态图实现高效优化，求解器速度提升最高达2.14倍。

应用价值 / Applications

该研究适用于电池容量严格受限的超低功耗边缘AI设备，如可穿戴设备、物联网传感器和移动终端等场景，能够在满足实时推理截止时间的前提下显著降低能耗。其编译器自动化的功耗编排方法为边缘AI系统的能效优化提供了可扩展的解决方案，有助于延长设备续航时间并支持更复杂的智能应用部署。
