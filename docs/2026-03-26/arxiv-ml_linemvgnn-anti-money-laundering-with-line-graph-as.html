LineMVGNN: Anti-Money Laundering with Line-Graph-Assisted Multi-View Graph Neural Networks

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本文提出了LineMVGNN，一种用于反洗钱（AML）检测的新型空间图神经网络方法。该模型通过轻量级多视图图神经网络（MVGNN）模块实现交易图中节点的双向消息传递，并引入线图视图增强交易信息传播。实验表明，该方法在以太坊钓鱼交易网络和金融支付交易数据集上均优于现有最优方法，同时具备更好的可扩展性、对抗鲁棒性和可解释性。

方法概述 / Method

LineMVGNN采用空间域方法处理有向交易图，将支付（payment）和收款（receipt）交易作为两个独立视图进行建模。核心创新在于构建原始交易图的线图（line graph），将边转换为节点以显式捕获交易关系，并通过双向消息传递机制融合多视图信息，克服了谱图神经网络对多维边特征支持不足和可扩展性受限的问题。

实验结果 / Results

研究在两个真实账户级交易数据集上进行验证：以太坊钓鱼交易网络数据集和行业合作伙伴的金融支付数据集。实验结果显示，LineMVGNN在洗钱检测任务上显著超越现有最先进方法，验证了线图辅助多视图学习的有效性。此外，该方法展现出良好的可扩展性和对抗攻击鲁棒性。

应用价值 / Applications

该研究可直接部署于金融机构的实时反洗钱监测系统，提升可疑交易识别的准确率和效率。其可解释性设计有助于满足金融监管合规要求，而良好的可扩展性使其能够处理大规模金融交易网络，为全球金融安全和经济保护提供技术支撑。
