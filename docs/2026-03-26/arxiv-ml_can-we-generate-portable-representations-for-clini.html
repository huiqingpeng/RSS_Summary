Can we generate portable representations for clinical time series data using LLMs?

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本研究探索了利用大型语言模型（LLM）为临床时间序列数据生成可迁移的患者嵌入表示，以解决临床机器学习模型在不同医院间部署时面临的分布偏移问题。作者将不规则的ICU时间序列数据转换为简洁的自然语言摘要，再通过冻结的文本嵌入模型生成固定长度的向量表示。实验表明，该方法在多个临床预测任务上与现有基线方法性能相当，同时在跨医院迁移时表现出更小的性能下降，且能提升少样本学习能力而不增加额外的隐私风险。

方法概述 / Method

该方法首先使用冻结的LLM将不规则的ICU时间序列数据映射为结构化的自然语言摘要，然后利用冻结的文本嵌入模型将这些摘要转换为固定长度的向量表示。这些向量可直接作为多种下游预测模型的输入，无需针对特定医院重新训练或微调。

实验结果 / Results

在MIMIC-IV、HIRID和PPICU三个数据集上的多项临床预测任务中，该方法与网格插补、自监督表示学习和时间序列基础模型等基线方法在分布内场景性能相当，但在跨医院迁移时相对性能下降更小。研究发现结构化提示设计对于降低预测模型方差至关重要，且该方法不会增加年龄或性别等人口统计学属性的可恢复性。

应用价值 / Applications

该研究为临床预测模型的大规模部署提供了新途径，通过降低工程开销和减少跨医院重新训练的需求，有望加速生产级预测模型在临床环境中的推广应用。该方法特别适用于需要快速适应新医疗环境、数据标注稀缺的场景，同时兼顾了隐私保护要求。
