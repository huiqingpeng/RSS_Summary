Synthetic Mixed Training: Scaling Parametric Knowledge Acquisition Beyond RAG

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本文提出了**合成混合训练（Synthetic Mixed Training）**，一种突破检索增强生成（RAG）性能瓶颈的新方法。研究发现，单纯增加合成数据量或使用更强的生成器会产生收益递减，而通过结合**合成问答对（synthetic QAs）**和**合成文档（synthetic documents）**这两种互补的训练信号，可实现对数线性的性能提升。该方法使Llama 8B模型在QuaLITY长文档阅读理解基准上相对RAG提升4.4%，在六个跨模型、跨基准测试设置中的五个超越RAG，与RAG结合时更是获得9.1%的性能增益。

方法概述 / Method

核心方法包含两个创新点：（1）**合成混合训练**，将合成问答对与合成文档联合训练，利用前者提供直接的任务监督信号、后者提供丰富的上下文知识表示；（2）**焦点重写（Focal Rewriting）**，一种简单的合成文档生成技术，通过显式地将文档生成条件化于特定问题，提升合成文档的多样性并产生更陡峭的对数线性扩展曲线。

实验结果 / Results

在QuaLITY、LongHealth、FinanceBench三个基准上，合成混合训练在六种设置中的五种超越RAG，平均相对提升2.6%；最终方案训练的Llama 8B在QuaLITY上相对RAG提升4.4%；当与RAG结合使用时，性能进一步提升9.1%。关键发现是：混合训练打破了对数线性扩展的"RAG天花板"，而单一合成数据类型则出现收益递减。

应用价值 / Applications

该方法适用于**数据受限领域的知识密集型任务**，如专业领域问答（医疗、金融）、长文档理解等场景，可替代或增强传统RAG系统。其实际价值在于：降低对外部检索系统的依赖、提升推理效率（纯参数化推理）、为资源受限场景提供可扩展的知识注入方案，同时保持与RAG结合时的协同增效潜力。
