Stochastic Dimension-Free Zeroth-Order Estimator for High-Dimensional and High-Order PINNs

arXiv Machine Learning 2026-03-26

论文摘要 / Abstract

本文针对高维高阶物理信息神经网络（PINNs）面临的计算瓶颈问题，提出了随机无维度零阶估计器（SDZE）。该框架通过公共随机数同步（CRNS）技术消除了零阶优化中的方差爆炸问题，并引入隐式无矩阵子空间投影将参数探索方差从O(P)降至O(r)，实现了空间和内存的维度无关复杂度。实验表明，SDZE可在单张NVIDIA A100 GPU上训练千万维度的PINNs，在速度和内存效率上显著优于现有方法。

方法概述 / Method

SDZE采用双管齐下的技术方案：首先利用CRNS锁定空间随机种子，代数消除零阶扰动带来的O(1/ε²)方差爆炸；其次通过隐式矩阵自由子空间投影，在低维子空间（维度r）中进行参数探索，将方差降至O(r)同时保持O(1)的优化器内存开销。该方法统一了随机空间估计与零阶优化，避免了反向传播的需求。

实验结果 / Results

SDZE成功实现了千万维度PINNs的单GPU训练，突破了现有方法在规模和内存上的限制。与最先进的基线方法相比，该方法在训练速度和内存效率方面均取得显著提升，验证了其在极端高维场景下的可扩展性优势。

应用价值 / Applications

SDZE为科学计算和工程模拟中的高维高阶偏微分方程求解提供了高效工具，特别适用于传统数值方法难以处理的复杂物理系统（如高维量子力学、流体动力学和金融风险模型）。该方法大幅降低了对计算资源的依赖，使大规模PINNs训练在常规硬件上成为可能，有望推动物理信息机器学习在更广泛领域的实际应用。
