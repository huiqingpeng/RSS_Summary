The Alignment Tax: Response Homogenization in Aligned LLMs and Its Implications for Uncertainty Estimation

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本文揭示了RLHF对齐语言模型存在"响应同质化"（response homogenization）现象：在TruthfulQA上，40-79%的问题在10次独立采样中产生单一语义簇，导致基于采样的不确定性估计方法完全失效（AUROC=0.500）。研究发现这种"对齐税"具有任务依赖性，并通过基础模型与指令模型的对比实验以及训练阶段消融实验，确定DPO（直接偏好优化）而非SFT（监督微调）是导致该现象的主要原因。作者最终提出了一种低成本级联策略（UCBD），通过正交不确定性信号的组合实现选择性预测，在GSM8K上将50%覆盖率下的准确率从84.4%提升至93.2%。

方法概述 / Method

本研究采用多维度实验设计，通过语义聚类分析（使用Jaccard相似度、嵌入相似度和NLI三种基线方法）量化响应同质化程度，并以单簇率（Single Cluster Rate, SCR）为核心指标。实验涵盖22项实验、5个基准测试、4个模型家族和3种模型规模（3B-14B），通过跨嵌入器验证和跨数据集验证排除耦合偏差，确保发现的稳健性。

实验结果 / Results

关键发现包括：（1）对齐模型在TruthfulQA上表现出28.5%的单簇率，而基础模型仅1.0%（p < 10^{-6}）；（2）训练阶段消融显示SCR从Base的0.0% → SFT的1.5% → DPO的4.0%，证实DPO是主因；（3）在受影响的题目上，采样熵等基于采样的不确定性方法AUROC跌至0.500（随机水平），而自由token熵仍保持0.603的信号；（4）UCBD级联策略在GSM8K上实现57%的成本节约，将50%覆盖率下的准确率提升至93.2%。

应用价值 / Applications

该研究对部署对齐LLM的实际应用具有重要指导意义：首先，警示从业者避免依赖采样-based不确定性估计进行可靠性判断，尤其在知识密集型任务中；其次，提出的UCBD级联框架为构建高效、低成本的不确定性量化系统提供了可行方案，适用于需要高可靠性决策的场景（如医疗、法律问答）；最后，研究为优化对齐训练流程（如改进DPO算法）指明了方向，有助于开发既保持有益性又保留适当响应多样性的模型。
