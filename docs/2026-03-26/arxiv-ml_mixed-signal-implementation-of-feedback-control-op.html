Mixed-signal implementation of feedback-control optimizer for single-layer Spiking Neural Networks

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本研究提出了一种基于反馈控制优化器的混合信号神经形态处理器概念验证实现，用于单层脉冲神经网络（SNN）的片上训练。该工作在循环训练（In-The-Loop, ITL）设置下评估了该方法，在二元分类任务和非线性阴阳问题（Yin-Yang problem）上均实现了与数值模拟和基于梯度基线方法相当的性能。研究结果表明，在真实混合信号约束条件下，反馈驱动的在线学习是可行的，为将此类学习规则直接嵌入硅片以实现自主自适应神经形态计算提供了协同设计途径。

方法概述 / Method

论文采用反馈控制优化器（feedback-control optimizer）作为核心训练机制，替代传统的基于梯度的反向传播算法，以适配神经形态硬件的物理约束。该方法在混合信号神经形态处理器上进行实现，通过硬件在环（ITL）的训练架构，将模拟神经形态电路与数字控制系统相结合，实现实时的权重更新。

实验结果 / Results

实验在二元分类任务和非线性阴阳问题两个基准任务上进行验证，结果表明该混合信号实现能够达到与纯数值模拟以及传统梯度基线方法相媲美的性能水平。研究成功证明了在真实混合信号硬件约束（如器件失配、噪声、非理想特性）下，反馈控制优化器仍能有效完成单层SNN的在线学习。

应用价值 / Applications

该研究为边缘计算和嵌入式智能系统提供了低功耗、自适应的神经形态计算解决方案，特别适用于需要实时学习和环境适应的场景，如机器人控制、传感器信号处理和物联网设备。通过将学习规则直接嵌入硅片，可避免片外训练的数据传输开销，实现真正自主、能效优化的神经形态智能系统。
