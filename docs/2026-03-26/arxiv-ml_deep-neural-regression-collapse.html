论文摘要 / Abstract

本文将神经坍缩（Neural Collapse）现象从分类任务扩展到回归任务，建立了深度神经回归坍缩（Deep NRC）理论，证明NRC不仅发生在最后一层，也出现在深层网络的中间层。研究发现坍缩层的特征位于与目标维度对应的子空间中，特征协方差与目标协方差对齐，层权重的输入子空间与特征子空间对齐，且特征的线性预测误差接近模型的整体预测误差。此外， exhibiting Deep NRC的模型能够学习低秩目标的内在维度，且权重衰减对诱导Deep NRC具有必要性。

方法概述 / Method

本文通过理论分析和实证验证相结合的方法，系统研究了深度神经网络在回归任务中的神经坍缩现象。作者在不同类型的模型架构（包括多层感知机和残差网络）的多层网络中测量NRC指标，分析特征子空间结构、协方差对齐性质以及权重矩阵的奇异值分布，并考察权重衰减等训练超参数对Deep NRC出现的影响。

实验结果 / Results

实验证实Deep NRC在多种网络架构的多个中间层中普遍存在，表现为特征有效维度坍缩至目标维度、特征协方差与目标协方差的谱结构高度一致、以及层权重的前导奇异向量与特征主成分对齐。研究表明，具有Deep NRC的模型能够自动识别并学习低秩目标数据的内在维度，而移除权重衰减会显著削弱甚至消除Deep NRC现象。

应用价值 / Applications

该研究为理解深度回归网络的内部学习机制提供了理论框架，有助于指导网络架构设计和训练策略优化，特别是在处理具有低秩结构的高维回归问题时。
Deep NRC理论可用于开发模型压缩、知识蒸馏和表示学习等方法，同时为分析神经网络的泛化能力和隐式正则化效应提供了新的视角。
