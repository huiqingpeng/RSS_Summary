A Deep Dive into Scaling RL for Code Generation with Synthetic Data and Curricula

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本文研究了如何通过合成数据和课程学习来扩展强化学习（RL）在代码生成任务中的应用。作者提出了一种可扩展的多轮合成数据生成流程，其中教师模型根据学生的上下文表现迭代优化问题，生成结构化的难度递进，而无需对教师模型进行微调。实验表明，这种多轮方法显著提高了有效合成问题的产出率，并自然产生支持课程学习的"垫脚石"（同一核心任务的易难变体）。研究系统分析了任务难度、课程调度和环境多样性在RL训练中的交互作用，发现合成数据增强能持续提升域内代码性能，并在大多数情况下改善域外数学表现。

方法概述 / Method

论文采用多轮合成数据生成管道，教师模型基于学生模型的上下文性能摘要迭代精炼问题，自动生成从简单到困难的结构化课程。该方法无需对教师模型进行微调，即可产生同一核心任务的多种难度变体（"垫脚石"），支持课程式RL训练。

实验结果 / Results

研究在Llama3.1-8B Instruct和Qwen3-8B Base模型家族上进行系统实验，并在Qwen2.5-32B上进行了额外扩展实验。结果表明，合成数据增强在域内代码任务上持续带来性能提升，在大多数域外数学任务上也有改善，同时揭示了课程设计和数据多样性如何共同塑造RL训练动态。

应用价值 / Applications

该研究为大规模语言模型的代码生成能力改进提供了可扩展的数据合成方案，特别适用于需要逐步学习复杂编程任务的场景。其课程学习框架可应用于教育编程平台、自动化代码辅助工具以及需要多难度层级训练数据的AI编程教育系统，降低了对人工标注高质量训练数据的依赖。
