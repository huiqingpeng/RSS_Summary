Large Language Model Guided Incentive Aware Reward Design for Cooperative Multi-Agent Reinforcement Learning

arXiv Machine Learning 2026-03-26

论文摘要 / Abstract

本研究提出了一种自动化奖励设计框架，利用大语言模型（LLM）从环境检测中合成可执行的奖励程序，以解决合作多智能体系统中辅助奖励设计的难题。该框架通过形式化有效性约束限制候选程序，并在固定计算预算下通过稀疏任务回报评估策略性能进行筛选。在Overcooked-AI的四种不同布局上的实验表明，迭代搜索生成的奖励程序能持续提升任务回报和交付数量，尤其在交互瓶颈主导的环境中增益最为显著，同时诊断分析显示合成奖励增强了动作选择的相互依赖性并改善了协调密集型任务中的信号对齐。

方法概述 / Method

该框架采用大语言模型作为核心生成器，将环境检测信息转化为可执行的奖励程序代码；通过定义形式化的"有效性包络"（validity envelope）对候选程序进行语法和语义约束，确保其可执行性和合理性。候选奖励程序通过在固定计算预算下从头训练策略进行评估，最终仅依据稀疏任务回报（而非密集奖励信号）进行筛选，实现完全自动化的奖励搜索与优化。

实验结果 / Results

在四种具有不同走廊拥堵程度、交接依赖性和结构不对称性的Overcooked-AI布局上，迭代搜索的每一代均持续产生更优的任务回报和食材交付数量；在交互瓶颈最严重的环境中观察到最显著的性能提升。对合成奖励组件的诊断分析表明，优化后的塑形奖励促进了智能体间动作选择的相互依赖性，并在需要高度协调的任务中实现了更好的信号对齐。

应用价值 / Applications

该研究为合作多智能体强化学习提供了自动化奖励工程解决方案，可显著降低手动设计奖励函数的人力成本和设计风险，特别适用于机器人协作、自动驾驶车队协调、分布式物流系统等需要复杂多智能体协调的现实场景。在计算预算受限的实际部署环境中，该方法能够生成与稀疏任务目标对齐的有效塑形信号，提升学习效率和最终协调性能。
