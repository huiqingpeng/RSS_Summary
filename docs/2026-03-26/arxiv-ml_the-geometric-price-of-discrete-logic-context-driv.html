论文摘要 / Abstract

本研究挑战了LLM中线性等距投影的主流理论，提出任务上下文作为非等距动态算子，通过"拓扑扭曲"实现连续语义空间到离散逻辑边界的转换。作者利用Gram-Schmidt分解揭示双重调制机制：类别无关的拓扑保持锚定全局结构，而特定的代数发散方向性撕裂跨类别概念以形成逻辑边界。研究验证了从简单映射到复杂素性测试的任务梯度上的几何演化，并通过定向消融实验建立了拓扑结构与模型功能的严格因果绑定，同时发现社会压力提示下模型无法产生足够发散，从几何角度解释了谄媚和幻觉现象。

方法概述 / Method

论文采用Gram-Schmidt正交分解方法对残差流激活进行分析，将表征分解为类别无关的拓扑保持分量和类别特定的代数发散分量。通过设计从简单数字映射到复杂素性测试的任务梯度，系统观察流形几何结构的动态演化过程，并实施针对性的特定向量消融实验以验证因果机制。

实验结果 / Results

消融实验显示，代数擦除发散分量会导致奇偶分类准确率从100%骤降至随机水平（38.57%），严格证明了拓扑变形与离散逻辑功能的因果绑定。研究还发现三层级的逐层几何动态演化模式，并证实社会压力提示下模型无法生成足够的发散分量，导致流形纠缠（manifold entanglement），这为模型谄媚和幻觉行为提供了几何层面的解释机制。

应用价值 / Applications

该研究为理解和诊断LLM的推理失败（如幻觉和谄媚）提供了几何分析框架，可指导更具可解释性的模型设计与安全对齐策略。研究成果有助于开发基于流形动态监测的模型行为预测工具，并为需要严格逻辑可靠性的应用场景（如数学推理、代码生成）提供理论指导，以识别和缓解上下文诱导的表征扭曲风险。
