Beyond Accuracy: Introducing a Symbolic-Mechanistic Approach to Interpretable Evaluation

arXiv Machine Learning 2026-03-26
Read Original Article →

论文摘要 / Abstract

本文指出基于准确率的评估无法可靠区分真正的泛化能力与记忆、数据泄漏或脆弱启发式等捷径行为，尤其在数据量较小的场景下。作者提出一种"机制感知评估"框架，结合任务相关的符号规则与机械可解释性方法，生成算法层面的通过/失败评分，从而精确定位模型是真正泛化还是利用表面模式。在NL-to-SQL任务上的实验表明，标准评估显示记忆模型在未见数据上达到94%字段名准确率，但符号-机械评估揭示该模型违反了核心的模式泛化规则——这一缺陷完全隐藏在准确率指标之下。

方法概述 / Method

该研究采用符号-机械混合评估方法：首先定义任务相关的符号规则（如模式泛化规则），然后运用机械可解释性技术分析模型的内部计算机制，最终输出可解释的算法级评分。具体实现中，作者训练了两个结构相同的模型进行对比——一个无模式信息（被迫记忆）、一个有模式信息（实现 grounding），以验证评估方法的有效性。

实验结果 / Results

实验显示，记忆模型在标准准确率指标上表现优异（94%字段名准确率），但符号-机械评估暴露其严重违反模式泛化规则；而具备模式信息的模型则能通过核心规则验证。这一对比证明，传统准确率指标会产生虚假的能力信号，而机制感知评估能揭示模型真正的泛化缺陷。

应用价值 / Applications

该研究对需要高可靠性评估的场景具有重要价值，如自然语言到代码生成、医疗诊断、法律推理等安全关键领域。符号-机械评估框架可帮助开发者识别模型的脆弱捷径行为，指导更有针对性的模型改进，同时为监管审计和模型认证提供可解释的技术依据。
