论文摘要 / Abstract

本文提出了一种自适应混合精度量化机制APreQEL，用于解决大型语言模型在边缘设备部署时面临的内存、延迟和精度权衡问题。该机制通过分析各层对模型性能的贡献，并结合目标硬件平台上不同量化类型的表现，为每层分配合适的量化精度。相比统一量化方法，APreQEL能够解锁更多配置设计，显著扩展边缘资源受限设备上高效部署LLM的解决方案空间。

方法概述 / Method

APreQEL采用层敏感度分析方法评估不同层对量化精度的响应差异，同时建立硬件感知性能模型来预测各类量化方案在目标平台上的内存占用和计算吞吐量。基于用户指定的优化优先级（内存、延迟或精度），该方法通过联合优化算法为网络各层自适应分配最优量化位宽。

实验结果 / Results

论文表明APreQEL能够在用户定义的约束条件下实现帕累托最优的部署配置，突破了统一量化方法无法达到的性能边界。该方法在保持模型精度的同时，显著降低了边缘设备上的内存消耗和推理延迟，为资源受限场景下的LLM部署提供了更灵活的精度-效率权衡方案。

应用价值 / Applications

APreQEL适用于智能手机、物联网设备、边缘服务器等资源受限场景下的LLM本地部署，能够在保障数据隐私和实时响应的同时优化硬件利用率。该研究为边缘AI应用的模型压缩和部署决策提供了系统化的工程框架，特别适用于对延迟敏感或内存严格的消费级AI产品。
