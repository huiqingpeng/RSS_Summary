论文摘要 / Abstract

本研究将机械可解释性（mechanistic interpretability）框架从序列预测架构扩展到具身控制系统，以婴儿运动学习为模型系统。研究发现，基础归纳偏置会产生因果控制回路，学习到的门控机制收敛于理论上 motivated 的不确定性阈值。研究揭示了仲裁门控中的清晰相变现象，其承诺行为可用闭式指数移动平均替代函数描述，并发现上下文窗口k是回路形成的关键参数——低于最小阈值（k≤4）时仲裁机制无法形成，高于阈值（k≥8）时门控置信度随log k渐近缩放。

方法概述 / Method

研究采用机械可解释性分析方法，通过分解模型行为为竞争算法来解析具身控制系统的内部机制。具体通过构建传感器-运动认知发展模型，分析学习过程中门控机制的收敛动态，并建立二维相图来刻画任务需求依赖的路径仲裁行为。

实验结果 / Results

实验发现上下文窗口k存在关键阈值效应：k≤4时仲裁机制无法形成，k≥8时门控置信度按log k渐近增长。研究识别出仲裁门控中的相变现象，其承诺行为可用闭式指数移动平均替代函数精确描述。二维相图进一步揭示，只有当预测误差保持在任务容忍窗口内时，前瞻性执行才变得有利。

应用价值 / Applications

本研究为认知发展提供了机械层面的解释框架，并为可解释具身智能体的设计提供原则性指导。研究成果可应用于开发具有透明决策机制的机器人控制系统，以及构建更符合人类认知发展规律的强化学习智能体，特别是在需要平衡反应式与前瞻性控制策略的复杂任务环境中。
