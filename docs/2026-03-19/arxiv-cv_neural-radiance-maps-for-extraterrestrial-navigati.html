论文摘要 / Abstract

本文提出了一种将神经辐射场（NeRFs）应用于地外行星自主导航的新方法，以解决火星车等探测器缺乏全局地图、难以进行在线路径重规划的问题。作者构建了一个基于NeRF地图的规划框架，通过核岭回归将局部地形代价观测插值到全局区域，实现局部与全局信息的融合。该方法在高保真仿真环境中验证，相比基线方法具有更低的路径成本和更高的成功率。

方法概述 / Method

该方法利用NeRF从稀疏二维图像构建紧凑的3D场景表示作为全局地图，并从中提取地形特征；采用核岭回归（kernel ridge regression）将在线运行中发现的不可通行区域的局部代价观测，插值推广到全局区域，从而支持实时路径重规划。

实验结果 / Results

实验在高保真仿真环境中进行，结果表明该方法相比多种基线方法实现了更低的路径规划成本和更高的任务成功率；具体而言，NeRF地图的紧凑存储特性使其适合星载计算资源受限的环境，同时在线重规划能力显著提升了导航鲁棒性。

应用价值 / Applications

该研究可直接应用于火星、月球等地外天面的探测器自主导航任务，加速科学探测目标的实现；NeRF地图的高效存储特性特别适合星载计算资源受限的场景，而在线重规划能力使探测器能够应对未知地形风险，提升任务安全性与效率。
