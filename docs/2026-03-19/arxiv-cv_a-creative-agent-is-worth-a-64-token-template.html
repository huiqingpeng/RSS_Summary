论文摘要 / Abstract

本文提出了CAT（Creative Agent Tokenization）框架，旨在解决文本到图像（T2I）模型在处理模糊创意提示时创造力不足的问题。核心创新在于通过一个"创意分词器"（Creative Tokenizer）将智能体对"创造力"的内在理解封装为可重用的64-token模板，无需重复推理即可注入创意语义。实验表明，该方法在建筑设计、家具设计和自然融合等任务上，实现了3.7倍加速和4.8倍计算成本降低，同时生成图像的人类偏好度和图文对齐度均优于现有最优方法。

方法概述 / Method

CAT框架的核心是创意分词器，它接收模糊提示的嵌入表示，输出生成可重用的token模板。该分词器通过创意语义解耦进行训练，利用部分重叠概念对之间的关系来捕获智能体的潜在创意表征，使模板能够直接与自然语言提示拼接，无需实例特定的迭代推理或提示增强。

实验结果 / Results

在建筑设计、家具设计和自然融合三个创意生成任务上的广泛实验表明，CAT相比现有T2I模型和创意生成方法具有显著优势：推理速度提升3.7倍，计算成本降低4.8倍，同时生成图像在人类偏好评估和文本-图像对齐度方面均达到更优水平。

应用价值 / Applications

CAT框架为创意产业提供了高效可扩展的解决方案，特别适用于需要频繁生成创意概念的设计领域，如建筑概念设计、产品设计和艺术创作。该方法通过将昂贵的智能体推理转化为可重用的token模板，使创意生成从"按需推理"转变为"一次学习、多次复用"，大幅降低了创意AI应用的部署成本和响应延迟。
