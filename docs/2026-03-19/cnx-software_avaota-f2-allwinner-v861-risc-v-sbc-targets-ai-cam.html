Avaota F2 – Allwinner V861 RISC-V SBC targets AI cameras with PTZ and audio support

CNX Software 2026-03-19
Read Original Article →

是什么 / What it is

Avaota F2 是一款基于全志 V861 双核 64 位 RISC-V SoC 的开源硬件单板计算机，专为 AI 摄像头应用设计，支持 4K 视频、PTZ 云台电机控制和音频功能，是 Avaota F1 的升级版本。
The Avaota F2 is an open-source hardware single-board computer based on Allwinner's V861 dual-core 64-bit RISC-V SoC, designed for AI camera applications with support for 4K video, PTZ motor control, and audio features, serving as an upgrade to the Avaota F1.

为什么重要 / Why it matters

这标志着 RISC-V 架构在边缘 AI 视觉领域的进一步渗透，1 TOPS 的 NPU 算力结合低功耗设计使其适合智能安防、工业检测等场景；同时开源硬件模式降低了开发者进入门槛，有助于构建本土化的 AIoT 生态。
This marks further penetration of the RISC-V architecture into edge AI vision, where the 1 TOPS NPU combined with low-power design suits smart security and industrial inspection scenarios; the open-source hardware model also lowers developer barriers and helps build localized AIoT ecosystems.

我能用什么 / How I can use it

开发者可关注其 GitHub 仓库获取硬件设计文件，利用 Tina Linux SDK 进行固件开发，或基于 OpenWrt 系统快速搭建智能摄像头原型；对于 PTZ 云台控制、AI-ISP 图像处理等特性，可参考中文文档进行功能验证，待淘宝上架后评估实际项目可行性。
Developers can monitor its GitHub repository for hardware design files, use the Tina Linux SDK for firmware development, or quickly build smart camera prototypes based on OpenWrt; for PTZ control and AI-ISP features, refer to Chinese documentation for validation, and evaluate project feasibility once it becomes available on Taobao.
