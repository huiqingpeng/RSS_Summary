Online Learning for Supervisory Switching Control

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了部分观测线性动态系统的监督切换控制问题，旨在从N个候选控制器中识别并部署最优控制器，其中部分控制器可能使系统失稳。针对经典基于估计器的监督控制缺乏有限时间性能保证、而现有非渐近方法又要求系统稳定等不兼容假设的问题，作者提出了一种将多臂老虎机算法适配到控制理论框架的新型非渐近分析方法。该数据驱动算法利用系统可观测性设计评分准则以隔离状态历史影响，实现了对失稳控制器的检测和精确系统识别，两种算法变体均具有与维度无关的有限时间保证，能在$\mathcal{O}(N \log N)$步内识别最优控制器，同时实现对系统扰动的有限$L_2$增益。

方法概述 / Method

论文将多臂老虎机（multi-armed bandit）算法适配到控制理论场景，设计了基于数据驱动的评分准则来评估候选控制器。该方法利用系统的可观测性（observability）将状态历史的影响进行隔离，从而能够在测试过程中检测可能导致系统失稳的控制器，并同时完成系统辨识。

实验结果 / Results

作者提出了两种算法变体，均具有维度无关的有限时间理论保证：算法能够在$\mathcal{O}(N \log N)$的时间步内识别出最适合的控制器，同时在整个学习过程中保持对系统扰动的有限$L_2$增益，确保闭环系统的稳定性与性能。

应用价值 / Applications

该研究适用于系统动态未知且需要从多个候选控制器中在线选择最优策略的控制场景，如自适应控制系统、机器人控制、工业过程控制等。其价值在于突破了传统方法要求系统稳定的限制，允许安全地测试潜在失稳的控制器，为实际工程中控制器调参与切换提供了具有理论保证的实用工具。
