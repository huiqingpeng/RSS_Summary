论文摘要 / Abstract

本文提出了一种模型无关的序数分类方法"分类器池化"（Classifier Pooling），可将任意非序数分类方法以序数方式应用。该方法在多个真实数据集上验证有效，尤其在数据量较小或类别较多时显著优于非序数方法。作者还提供了开源Python实现，以促进现代机器学习算法在序数数据处理中的应用。

方法概述 / Method

分类器池化方法通过组合多个二分类器来构建序数分类器，每个二分类器负责区分相邻类别之间的边界。该方法具有模型无关性，可灵活集成各种现代机器学习算法（如梯度提升树、神经网络等），无需修改底层分类器的内部结构。

实验结果 / Results

实验表明，该方法在多个领域的真实数据集上通常优于非序数分类方法，特别是在训练样本量相对较小或输出类别数量较多的场景下优势更为明显。具体性能指标显示序数方法能够更好地利用类别间的有序关系，提升预测准确性。

应用价值 / Applications

该研究具有重要的临床应用价值，因为医学领域普遍存在序数数据（如疾病分期、疼痛评分等）。开源软件包降低了现代序数分类技术的使用门槛，使领域专家能够便捷地应用先进的机器学习算法处理序数预测任务，推动临床决策支持系统的发展。
