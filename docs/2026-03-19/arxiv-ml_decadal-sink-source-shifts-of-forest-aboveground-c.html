Decadal sink-source shifts of forest aboveground carbon since 1988

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究利用多源卫星观测与概率深度学习模型，重建了1988-2021年全球森林地上碳（AGC）的0.25度分辨率、不确定性量化的长期记录。研究发现，尽管全球森林总体固碳6.2 PgC，但湿润热带森林和北方森林自2000年代初以来已逐渐转变为碳源。热带AGC变率与大气CO₂增长率的负相关性显著增强（2011-2021年r=-0.63），表明热带森林对全球碳循环的调节作用日益增强。巴西亚马逊地区原始森林对AGC年际损失的贡献从1990年代的33%上升至2010年代的76%，超过毁林区域。

方法概述 / Method

研究整合多源卫星观测数据，采用概率深度学习模型进行数据融合与重建，生成具有不确定性量化的全球森林地上碳长期序列。该方法实现了0.25度空间分辨率、覆盖1988-2021年的 harmonized AGC 数据集，突破了传统观测在时空连续性和不确定性表征方面的局限。

实验结果 / Results

全球森林在34年间净固碳6.2 PgC，但关键生物群系出现显著的源-汇转变：湿润热带和北方森林自2000年代初逐步成为碳源；热带AGC与大气CO₂增长率的负相关性从早期较弱关系增强至2011-2021年的r=-0.63；巴西亚马逊原始森林对碳损失的年际变率贡献率从33%跃升至76%，而毁林贡献则从60%骤降至13%。

应用价值 / Applications

该研究为追踪人为气候变化下的新兴碳源-汇转变提供了基准数据集和方法论框架，可直接服务于全球碳循环模型验证、国家温室气体清单编制及《巴黎协定》全球盘点。研究成果对识别森林碳汇脆弱热点区域、指导基于自然的气候解决方案（NCS）投资优先级具有重要政策意义，尤其为热带雨林保护策略的制定提供了关键科学依据。
