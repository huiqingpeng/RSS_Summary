MOBODY: Model Based Off-Dynamics Offline Reinforcement Learning

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了**非动态离线强化学习（off-dynamics offline RL）**问题，即利用动态不匹配的离线源数据和有限目标数据学习策略。现有方法通过惩罚奖励或丢弃高动态偏移区域的源数据来优化策略，导致无法探索目标域中位于高偏移区域的高奖励状态。为此，作者提出**MOBODY**算法，通过学习目标动态模型来探索目标域，而非仅依赖低动态偏移数据；并引入目标Q加权行为克隆损失避免分布外动作。在MuJoCo和Adroit基准测试上，MOBODY显著优于现有方法，尤其在动态偏移严重或最优轨迹位于高偏移区域的挑战性场景中表现突出。

方法概述 / Method

MOBODY采用**基于模型的方法**，核心创新在于动态学习模块：观察到相同下一状态在不同域需要不同动作，因此为源域和目标域分别设计**独立的动作编码器**，将不同动作映射到共享隐空间，同时共享状态表示和统一的状态转移函数。策略优化阶段引入**目标Q加权行为克隆损失**，引导策略选择具有高目标域Q值的动作，而非高源域Q值或盲目模仿离线数据中的所有动作。

实验结果 / Results

MOBODY在广泛的MuJoCo和Adroit基准测试中**持续优于最先进的非动态RL基线方法**，以及基于不同动态学习基线的策略学习方法。实验表明，在现有方法难以处理的挑战性场景（动态偏移显著或最优轨迹位于高动态偏移区域）中，MOBODY的优势尤为明显，验证了该方法在复杂非动态环境下的有效性和鲁棒性。

应用价值 / Applications

该研究适用于**源域与目标域动态模型存在差异的离线强化学习场景**，如机器人控制中仿真到真实（sim-to-real）迁移、不同硬件平台间的策略迁移、或动态特性随时间变化的系统控制。
MOBODY通过有效利用源域数据并学习目标域动态，减少了对昂贵目标域数据的依赖，为实际部署中数据收集成本高昂或危险的强化学习应用提供了高效解决方案。
