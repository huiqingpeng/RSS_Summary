论文摘要 / Abstract

本研究提出了一种领域知识驱动的可解释提升机（EBM）框架，用于提高侧向扩展（lateral spreading）预测的物理一致性。该方法通过修改学习到的形状函数来纠正非物理行为，同时保留数据驱动的模式。在2011年基督城地震数据集上的应用表明，该框架能够生成更具物理一致性的全局和局部解释，尽管准确性有4-5%的可接受折损。

研究者采用领域知识约束的可解释提升机方法，通过识别并修正EBM学习到的形状函数中的非物理趋势，将专业领域知识（如地质力学原理）嵌入机器学习模型中。具体实现方式是对原始EBM的特征贡献函数进行后处理修正，确保其符合已知的物理规律。

实验结果 / Results

在2011年基督城地震数据集上的验证显示，修正后的EBM成功消除了原始模型中的非物理趋势，同时保持了较高的预测性能。模型在物理一致性方面显著提升，预测准确率的下降控制在4-5%的合理范围内，实现了可解释性与可靠性的有效平衡。
