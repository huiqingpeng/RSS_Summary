Semi-supervised Shelter Mapping for WASH Accessibility Assessment in Rohingya Refugee Camps

arXiv Computer Vision 2026-03-19

论文摘要 / Abstract

本研究针对罗兴亚难民营中水资源、卫生设施与卫生服务（WASH）可及性不足这一重大公共卫生问题，开发了基于Segment Anything Model（SAM）的半监督分割框架，用于从2017-2025年多时相亚米级遥感影像中绘制避难所分布图。研究发现，2020年后避难所扩张趋于稳定，但人均居住空间因人口持续增长而缩减约14%；同时WASH可及性在2022-2025年间下降，女性可及性比男性低约27%。该研究展示了遥感驱动的人工智能诊断可为长期流离失所环境中的WASH投资优先级决策提供公平性导向的证据支持。

方法概述 / Method

论文采用半监督语义分割方法，以SAM为基础模型，结合少量标注数据与大量未标注的多时相高分辨率遥感影像进行训练，以应对难民营复杂环境中避难检测的挑战。研究还采用改进的网络型两步移动搜索法（2SFCA）评估WASH设施的空间可及性，并引入性别区分的安全惩罚因子以量化不同性别的服务获取差异。

实验结果 / Results

所提出的半监督框架在避难所检测任务上较强基线方法F1分数提升4.9%，有效改善了复杂营地环境中的识别精度。时空分析揭示：2020年后避难所面积增长停滞，但人均空间持续压缩；WASH可及性指标显示设施负荷加重并超出全球基准，性别差异化分析进一步暴露女性面临显著的空间不平等。

应用价值 / Applications

该研究为难民营及类似拥挤人居环境的公共卫生规划提供了可扩展的监测工具，支持基于实证的WASH资源优化配置。研究成果可直接服务于国际人道主义组织的投资决策，帮助识别弱势群体（尤其是女性）的服务缺口，并为长期流离失所情境下的健康风险缓解与公平性提升提供数据驱动的政策依据。
