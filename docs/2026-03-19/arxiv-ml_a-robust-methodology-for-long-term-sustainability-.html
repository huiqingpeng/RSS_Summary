A robust methodology for long-term sustainability evaluation of Machine Learning models

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文针对机器学习模型长期可持续性评估缺乏标准化协议的问题，提出了一种新的评估方法。现有绿色AI审计实践存在三个主要缺陷：过度关注epoch/batch学习设置、未正式建模模型适应和重训练的长期成本、以及仅测量孤立实验而非真实AI生命周期的环境影响。作者基于在线机器学习概念，提出通过增量/持续模型重训练并行于真实数据采集来测量可持续性与性能的新协议，并通过实验发现传统静态训练-测试评估无法可靠捕捉动态数据集下的真实维护成本，且高环境成本有时并不能带来相应的性能提升。

方法概述 / Method

该方法借鉴在线机器学习的核心思想，设计了一种增量式/持续性的模型重训练评估框架，使模型更新与真实世界数据获取同步进行。与传统静态的"训练-测试"范式不同，此协议模拟了ML模型在实际部署环境中面临的连续数据流和概念漂移场景，从而能够动态追踪模型在整个生命周期内的累计环境成本和性能变化。

实验结果 / Results

实验在多种机器学习任务和模型类型上验证了传统静态评估方法的局限性：这些方法会高估、低估或不稳定地估计模型维护的实际成本。关键发现表明，在长期真实ML生命周期中，更高的环境成本（如更多计算资源消耗）有时仅带来极少甚至零性能收益，揭示了成本-效益权衡在动态环境中的复杂性。

应用价值 / Applications

该研究为AI系统的绿色监管和可持续性审计提供了标准化、模型无关的评估工具，特别适用于需要长期部署的机器学习应用（如推荐系统、金融风控、物联网预测等）。研究结果可帮助企业和政策制定者在模型选型时做出更明智的决策，避免"过度计算"造成的资源浪费，推动真正环境友好且经济高效的AI系统开发。
