Pretrained Vision-Language-Action Models are Surprisingly Resistant to Forgetting in Continual Learning

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究发现，大规模预训练的视觉-语言-动作（VLA）模型在持续学习（Continual Learning）中表现出显著的抗遗忘特性，远优于从零训练的小型策略模型。简单的经验回放（Experience Replay, ER）方法在VLA模型上效果惊人，即使使用较小的回放数据量也能实现零遗忘。研究揭示了预训练对下游持续学习性能的关键作用：大模型能在小回放缓冲区的条件下缓解遗忘，同时保持强大的前向学习能力。此外，VLA模型在学习新任务时虽然性能暂时下降，但仍能保留先前任务的相关知识，通过微调可快速恢复"看似遗忘"的技能。

方法概述 / Method

研究采用对比实验方法，将大规模预训练VLA模型（如OpenVLA、LLaVA等）与从零训练的小型行为克隆（BC）策略模型在机器人策略持续学习任务上进行比较。核心方法为简单的经验回放（Experience Replay），即在学习新任务时保留并回放少量先前任务的样本，无需复杂的正则化或架构修改。研究还设计了知识保留验证实验，通过微调测试模型是否真正丢失了先前知识。

实验结果 / Results

实验表明，预训练VLA模型配合经验回放可实现接近零遗忘的持续学习，而相同条件下从零训练的小模型表现出严重的灾难性遗忘。关键发现包括：VLA模型仅需极小的回放缓冲区（如每个任务50-100个样本）即可有效抗遗忘；模型在学习新任务期间性能下降但知识并未真正丢失，经少量微调步骤即可完全恢复先前技能；预训练规模与抗遗忘能力呈正相关，越大规模的预训练模型持续学习性能越好。

应用价值 / Applications

该研究为机器人终身学习系统提供了实用方案，表明部署预训练VLA模型后，可通过简单回放机制持续扩展新技能而无需复杂算法。这一发现降低了机器人策略持续学习的工程门槛，使机器人能够在真实环境中长期自主积累技能。同时，研究揭示了大规模预训练模型的新特性——隐式知识保留能力，为设计更高效的持续学习算法提供了理论基础，对通用机器人智能的发展具有重要指导意义。
