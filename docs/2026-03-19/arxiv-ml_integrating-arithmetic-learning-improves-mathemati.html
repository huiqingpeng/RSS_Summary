Integrating Arithmetic Learning Improves Mathematical Reasoning in Smaller Models

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了如何通过算术学习提升小型语言模型的数学推理能力。研究发现，尽管大型模型在数学推理任务上表现优异，但小型模型即使经过知识蒸馏和数据增强仍难以准确执行算术计算。作者利用程序生成的合成算术数据集，探索了中间微调和指令微调混合两种集成方式，实验表明这两种方法都能显著增强小型模型的算术能力和数学推理性能。

方法概述 / Method

本研究采用程序生成的合成算术数据集，设计了两种训练策略：（1）中间微调，即先在算术数据集上微调，再在推理数据集上训练；（2）将算术数据集融入指令微调混合数据，使模型同时学习算术技能和通用指令遵循能力。实验在GSM8k、MultiArith等多个数学推理基准上进行评估。

实验结果 / Results

实验结果表明，无论是通过针对性微调还是指令微调混合的方式集成算术数据集，都能有效提升小型模型的算术计算能力和数学推理表现。该方法在多个推理基准上均展现出性能改进，证明了算术技能学习对数学推理的促进作用。

应用价值 / Applications

该研究为资源受限场景下开发高效的小型数学推理模型提供了实用方案，可应用于教育辅导、智能解题助手等需要本地部署的场景。通过低成本的数据合成和训练策略优化，使小型模型能够在数学推理任务上接近大型模型的性能，具有重要的实际部署价值。
