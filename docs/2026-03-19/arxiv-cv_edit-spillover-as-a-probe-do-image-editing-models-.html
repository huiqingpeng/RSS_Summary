Edit Spillover as a Probe: Do Image Editing Models Implicitly Understand World Relations?

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文发现图像编辑模型中存在"编辑溢出"（edit spillover）现象——模型会修改与编辑区域语义相关但未指定的内容，并提出将这一现象作为探测模型世界知识的新方法。作者构建了EditSpilloverProbe框架，包含溢出分类体系、自动检测分类流程以及真实中文文本编辑任务基准数据集EditSpilloverBench。通过对5种代表性模型的系统评估，研究发现溢出率在不同架构间差异显著（3.3倍），绝对语义溢出量可反映模型的世界理解能力，且空间衰减分析表明语义溢出比例保持恒定，为"溢出反映真实世界理解而非空间扩散"提供了直接证据。

方法概述 / Method

作者提出EditSpilloverProbe系统框架，将编辑溢出重新定义为探测图像编辑模型世界知识的自然探针。该框架包含四部分：建立空间型、语义型、混合型、随机型的溢出分类体系；开发自动化的溢出检测与分类流程；构建基于真实世界中文文本编辑任务的基准数据集EditSpilloverBench；设计空间衰减分析方法来区分语义理解与注意力泄漏。

实验结果 / Results

对5种代表性编辑模型的评估显示：溢出率从3.49%到11.46%不等，架构间差异达3.3倍；nano_banana产生最多语义溢出（每图27.8处），而qwen_2511编辑控制最精确但语义溢出较低（每图16.3处），揭示了编辑控制与世界理解之间的权衡；空间衰减分析表明溢出面积密度随距离指数衰减，但语义相关溢出比例恒定在40%-58%，证明语义溢出源于真实世界理解而非空间扩散效应。

应用价值 / Applications

该研究为评估图像编辑模型的隐式世界知识提供了新范式，可指导模型架构设计与能力权衡；溢出分类体系和自动检测工具可用于诊断和改进编辑模型的可控性；基准数据集EditSpilloverBench支持中文场景下的模型评估，对需要精确局部编辑的实际应用（如文档处理、内容创作）具有重要参考价值。
