CrowdGaussian: Reconstructing High-Fidelity 3D Gaussians for Human Crowd from a Single Image

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了CrowdGaussian，首个从单张图像直接重建多人3D高斯泼溅（3DGS）表示的统一框架。针对人群场景中的严重遮挡、低清晰度和外观多样性三大挑战，作者设计了自监督自适应流程和自校准学习（SCL）训练策略，使预训练的大型人体模型能够从遮挡输入中重建完整3D人体，并通过扩散模型自适应优化渲染质量。实验表明，该方法能够生成照片级真实感、几何一致的多人场景重建。

方法概述 / Method

CrowdGaussian采用两阶段技术方案：首先通过自监督自适应流程，利用预训练的大型人体模型（如SMPL-X）处理严重遮挡，推断出完整的人体几何和外观；其次引入自校准学习（SCL）策略，训练单步扩散模型通过混合身份保留样本与干净/损坏图像对，将粗糙渲染自适应细化为最优质量，并将优化结果蒸馏回3DGS表示。

实验结果 / Results

论文通过大量实验验证了CrowdGaussian在多人场景重建中的优越性，能够生成高保真、几何连贯的3D高斯表示；相比现有单视角人体重建方法，该方法在遮挡处理、外观一致性和渲染质量方面均有显著提升，实现了从单张人群图像到可实时渲染的3D场景的有效重建。

应用价值 / Applications

该研究在虚拟现实、增强现实、游戏开发和影视特效制作中具有重要应用价值，可实现从单张照片快速生成可交互的3D人群场景；同时适用于智能监控分析、体育事件重建和大型活动场景的数字孪生构建，为从有限视觉输入创建沉浸式3D体验提供了高效解决方案。
