论文摘要 / Abstract

本研究开发了一种机器学习框架，用于快速预测线性嵌段共聚物体系的超额自由能。研究团队利用耗散粒子动力学模拟构建包含异质相互作用能、同质相互作用能和键合弹簧能等描述符的数据集，训练前馈神经网络学习这些描述符与分层BAR方法计算的自由能之间的关系。结果表明，神经网络模型能够准确复现参考自由能，即使在直接BAR估计因相空间重叠不足而失效的情况下，神经网络预测仍与参考值保持一致，证明了物理信息机器学习模型可作为昂贵自由能计算的高效替代方案。

方法概述 / Method

研究采用耗散粒子动力学（DPD）模拟自由连接链聚合物，收集每条链的能量统计量作为输入特征，包括异质相互作用能、同质相互作用能和键合弹簧能。使用分层Bennett Acceptance Ratio（BAR）程序计算自由能作为标签，训练前馈神经网络建立能量描述符与自由能之间的映射关系。

实验结果 / Results

神经网络模型在多种链长、组成和密度条件下均能准确预测自由能，包括对训练时未见的聚合物结构也能良好泛化。在相空间重叠较差的区域（直接BAR估计失效的工况），神经网络预测仍与参考值保持一致，展现出比传统BAR方法更强的鲁棒性。

应用价值 / Applications

该方法可显著加速聚合物体系的自由能计算，避免传统方法中需要大量中间模拟步骤的高昂计算成本。该框架为嵌段共聚物的相行为和热力学稳定性分析提供了高效工具，可推广至更复杂的聚合物体系设计和高通量材料筛选。
