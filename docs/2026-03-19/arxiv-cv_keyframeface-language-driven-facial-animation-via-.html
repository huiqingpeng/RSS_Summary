KeyframeFace: Language-Driven Facial Animation via Semantic Keyframes

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了KeyframeFace框架，通过语义关键帧而非连续运动轨迹来实现语言驱动的面部动画生成。该方法将动画表示为可解释的ARKit面部控制空间中的语义关键帧序列，利用大语言模型（LLM）先验生成与文本描述和情感线索对齐的关键帧。实验表明，结合语义关键帧监督和语言先验显著提升了表情保真度和语义对齐度，并构建了一个包含2,100个表情脚本的多模态数据集支持该方法。

方法概述 / Method

KeyframeFace采用基于关键帧的范式，在ARKit面部控制空间中生成稀疏、语义可解释的面部关键帧，而非直接回归密集运动轨迹。该方法利用大语言模型（LLM）的语言先验能力，根据上下文文本描述和情感提示生成语义对齐的关键帧序列，实现了高层语义意图与低层运动的解耦。

实验结果 / Results

实验表明，引入语义关键帧监督和语言先验的方法在表情保真度和语义对齐方面显著优于不使用面部动作语义的方法。通过新构建的多模态数据集（含2,100个表情脚本、单目视频、逐帧ARKit系数和人工标注语义关键帧）验证了该方法的有效性。

应用价值 / Applications

该方法可显著提升计算机图形学（CG）行业中数字角色面部动画的内容创作效率和可访问性，使动画师能够通过自然语言描述直接控制面部表情。其可解释的关键帧表示支持精确的语义编辑，适用于影视制作、游戏开发、虚拟主播等需要精细面部动画控制的场景。
