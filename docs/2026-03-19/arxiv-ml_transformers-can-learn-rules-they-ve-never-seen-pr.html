Transformers Can Learn Rules They've Never Seen: Proof of Computation Beyond Interpolation

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文通过两个严格控制的实验，证明了Transformer能够学习训练数据中从未出现的规则，而非仅仅依赖基于相似性的插值。第一个实验利用XOR元胞自动机，通过构造性地移除特定局部输入模式，证明Transformer可以恢复线性不可分的规则；第二个实验要求模型在符号运算链中生成中间推导步骤。研究提供了Transformer能够发现并显式表达未见过规则的存在性证明，排除了最强形式的"仅插值"假设。

方法概述 / Method

实验一采用具有纯XOR转移规则的元胞自动机，系统性地从训练中移除特定局部输入模式，利用XOR的线性不可分性使得基于相似性的预测必然失败；实验二研究整数上的符号运算符链，要求模型以证明格式输出中间步骤和最终答案，并将一个运算符对完全留出。此外，研究通过电路提取技术分析模型内部计算，并构造性地证明标准Transformer块可实现精确的局部布尔规则。

实验结果 / Results

在XOR元胞自动机实验中，两层Transformer在留出模式上达到100%准确率（47/60次运行收敛），而无展开训练时准确率仅63.1%（输出偏置水平），软展开训练达96.7%；在符号运算链实验中，Transformer在所有49个留出运算符对上超越所有插值基线（平均41.8%，最高78.6%），而核岭回归平均仅4.3%，KNN和MLP在所有对上均为0%。移除中间步骤监督会显著降低性能。

应用价值 / Applications

该研究为理解大型语言模型的泛化能力提供了理论基础，表明Transformer具备超越训练数据插值的真正规则学习能力。这对需要逻辑推理、数学证明和符号操作的AI应用具有重要意义，如自动定理证明、程序合成和科学发现系统。研究同时提示，多步约束传播和显式中间监督可能是激发此类能力的关键设计要素。
