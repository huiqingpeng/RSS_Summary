论文摘要 / Abstract

本文提出了知识免疫框架（KIF），一种表征感知的机器遗忘方法，通过靶向内部激活签名而非表面输出来区分真正的知识擦除与行为抑制。该方法在Llama、Mistral、Qwen和DeepSeek等模型（3B-14B参数）上实现了近乎理想的擦除效果（FQ≈0.99），同时保持模型效用（MU=0.62），有效打破了稳定性-擦除权衡。研究还揭示了标准模型与推理优先模型在遗忘机制上的本质架构差异，并建立了首个系统诊断机制级遗忘行为的双指标评估协议。

方法概述 / Method

KIF采用动态抑制与参数高效适应相结合的策略，通过识别并擦除特定主题的激活签名来实现持久性遗忘，无需完整模型重训练。该方法核心在于将"知识签名擦除"与"表面行为压制"分离，通过表征层面的干预确保潜在能力被真正移除而非隐藏。框架包含动态抑制模块和参数高效适配组件，实现对内部神经表征的精准靶向。

实验结果 / Results

在标准基础模型上，KIF实现了尺度无关的真实擦除，效用漂移<3%；而在推理优先模型上观察到根本性的架构分歧。定量结果显示遗忘质量FQ接近理想值0.99（对比理论最优1.00），模型效用MU达到0.62的基准水平。综合评估表明，该方法成功打破了先前所有工作受限于的稳定性-擦除权衡，同时双指标评估协议验证了表面泄漏与潜在痕迹持续性的有效区分。

应用价值 / Applications

该研究为GDPR合规和AI模型安全提供了可部署的技术方案，解决了当前机器遗忘方法中"拒绝回答"与"真正遗忘"混淆的关键问题。其表征感知的诊断工具和评估协议可用于审计各类大语言模型的知识移除完整性，特别适用于需要严格隐私保证和法律合规的场景，如医疗、金融等敏感领域的模型部署与更新。
