AgriChat: A Multimodal Large Language Model for Agriculture Image Understanding

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文针对农业领域多模态大语言模型（MLLMs）部署面临的数据稀缺与领域专业知识不足的双重挑战，提出了Vision-to-Verified-Knowledge（V2VK）生成式AI标注框架，通过视觉描述与网络增强科学检索相结合，构建了包含3,000多个农业类别、超过60.7万条视觉问答的AgriMM基准数据集。基于该可验证数据开发的AgriChat模型在细粒度植物物种识别、病害症状识别、作物计数和成熟度评估等多项任务上显著优于现有开源模型，证明了视觉细节保留与网络验证知识相结合是实现可靠农业AI的有效路径。

方法概述 / Method

V2VK管道采用两阶段方法：首先利用视觉描述模型生成图像候选描述，然后通过web-augmented科学检索从经同行评审的植物病理学文献中验证和扩充知识，从而消除生物幻觉并确保数据可溯源。在此数据基础上，AgriChat通过多任务学习进行训练，整合细粒度分类、视觉问答和详细农业评估等能力。

实验结果 / Results

AgriChat在内部和外部基准测试上均优于其他开源MLLMs，在超过3,000个农业类别的细粒度识别任务中展现出广泛的领域知识覆盖能力。实验同时揭示了当前农业MLLMs在不同任务类型和评估条件下的能力边界与局限性，验证了网络验证知识对提升模型可信度的关键作用。

应用价值 / Applications

该研究为精准农业提供了可信赖的AI决策支持工具，可直接应用于田间作物健康监测、病虫害早期预警、产量预估和采收时机优化等场景。
V2VK框架的方法论还可迁移至其他需要高度专业知识的科学领域，推动领域专用大模型的可信数据构建与知识验证机制发展。
