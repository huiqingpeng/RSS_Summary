Coded Information Retrieval for Block-Structured DNA-Based Data Storage

arXiv Information Theory 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了分块结构化数据的编码信息检索问题，该问题源于DNA存储系统，其中数据库被划分为多个必须作为原子单元恢复的文件。作者形式化定义了块结构化检索问题，将k个信息符号划分为两个大小分别为s₁和s₂的文件，目标是刻画所有[n,k]线性码的可实现期望检索时间对(E₁, E₂)的集合。研究推导了基于恢复集互斥性的线性下界族和基于列投影的非线性几何界，提出了双曲约束s₁/E₁ + s₂/E₂ ≤ 1的猜想，并分析了多种显式编码的性能，最终刻画了n→∞时的极限可实现区域。

方法概述 / Method

论文采用信息论与编码理论相结合的方法，通过建立恢复集的互斥性条件推导线性下界，并利用列投影技术构建非线性几何边界。对于具体编码分析，作者运用组合计数论证（特别是子集计数方法）来比较不同编码方案的期望检索时间，为全局MDS码的支配关系提供了简化证明。

实验结果 / Results

研究证明了专用文件MDS码（file-dedicated MDS codes）在其编码族内具有MDS最优性，并满足双曲约束条件；对于全局MDS码，建立了比例局部MDS分配支配性的严格证明，相比近期文献显著简化了证明过程并扩展至非对称情形。当n→∞时，专用文件MDS码渐近达到双曲边界，该边界被猜想为极限可实现区域的精确边界。

应用价值 / Applications

该研究对DNA数据存储系统具有直接应用价值，为设计高效的多文件检索编码方案提供了理论指导，特别是在需要独立快速访问不同文件块的场景下。研究成果可优化冷数据存储架构中的读取延迟权衡，并为合成生物学与分子信息存储领域的工程实现奠定编码理论基础。
