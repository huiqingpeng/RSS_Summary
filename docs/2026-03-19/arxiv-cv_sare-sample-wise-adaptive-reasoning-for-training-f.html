SARE: Sample-wise Adaptive Reasoning for Training-free Fine-grained Visual Recognition

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了SARE（Sample-wise Adaptive REasoning），一种用于无需训练的细粒度视觉识别（FGVR）的自适应推理框架。针对现有方法对所有样本采用统一推理流程、忽视识别难度差异以及缺乏错误经验复用机制的问题，SARE通过级联式设计和自反思经验机制，实现了根据样本难度自适应调用推理模块，并利用历史失败案例提供可迁移的判别指导。在14个数据集上的实验表明，SARE在达到最先进性能的同时显著降低了计算开销。

方法概述 / Method

SARE采用级联架构，首先通过快速候选检索处理简单样本，仅在必要时触发细粒度推理模块以应对困难样本。核心创新在于引入自反思经验机制：系统无需参数更新即可记录并分析历史推理失败案例，将错误特定的经验转化为可迁移的判别指导，用于辅助后续相似困难样本的推理过程。

实验结果 / Results

SARE在14个细粒度视觉识别数据集上取得了最先进的性能表现，同时大幅减少了计算开销。该框架通过自适应地跳过不必要的复杂推理，在保证识别精度的前提下显著提升了推理效率，实现了精度与效率的更好平衡。

应用价值 / Applications

SARE适用于需要高效细粒度视觉识别的实际场景，如野生动物物种鉴定、医学影像分析、工业缺陷检测和商品分类等。其无需训练的特性使其能够快速部署到新的细粒度领域，而自适应推理机制则特别适合资源受限或实时性要求高的边缘计算环境。
