论文摘要 / Abstract

本文针对接地多模态命名实体识别（GMNER）任务，提出了一种无需候选框的查询引导网络（QGN）。现有方法依赖预训练通用目标检测器，存在与文本实体不对齐、易忽略细粒度区域的问题。
QGN通过文本引导和跨模态交互统一多模态推理与解码，在开放域场景中实现了准确的实体接地，并在主流基准上取得了最优性能。

方法概述 / Method

QGN摒弃了传统的两阶段范式（先检测后匹配），采用端到端的查询引导架构。该方法通过文本查询直接引导视觉特征的提取与定位，实现跨模态的深度融合与联合解码，无需依赖预训练检测器生成的候选框。

实验结果 / Results

QGN在广泛使用的GMNER基准数据集上进行了大量实验，与现有模型相比取得了顶尖性能。实验表明，该方法有效解决了预训练检测器与文本实体之间的对齐问题，在开放域场景中展现出更强的鲁棒性和准确性。

应用价值 / Applications

该研究适用于需要文本-图像联合理解的多模态信息抽取场景，如社交媒体内容分析、新闻图文审核、智能问答系统等。
QGN的免候选框设计降低了对预训练检测器的依赖，提升了系统在开放域环境下的泛化能力和实用性。
