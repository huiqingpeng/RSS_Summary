LED: A Benchmark for Evaluating Layout Error Detection in Document Analysis

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了LED（Layout Error Detection），一个用于评估文档布局分析（DLA）中结构错误检测能力的基准测试。针对传统基于重叠的指标（如IoU、mAP）无法捕捉区域合并、分割、遗漏等逻辑不一致问题，LED定义了八种标准化错误类型，并提供定量规则和注入算法进行真实错误模拟。实验表明，该基准能够对多模态模型的结构理解能力进行细粒度、可解释的评估，揭示不同模态和架构的明显弱点。

方法概述 / Method

LED首先定义了八种文档布局错误类型（缺失、幻觉、尺寸错误、分割、合并、重叠、重复和误分类），并为每种类型设计了基于几何关系和语义规则的定量检测算法。基于此，研究团队构建了LED-Dataset数据集，并设计了三个层次化的评估任务：文档级错误检测、文档级错误类型分类和元素级错误类型分类。

实验结果 / Results

实验对当前最先进的多模态大模型进行了系统评估，结果显示现有模型在结构推理方面存在显著不足，尤其是在处理复杂布局错误（如合并和分割）时表现较差。
LED的评估框架能够清晰区分不同模态（纯文本、视觉、多模态）和架构在结构理解能力上的差异，提供了传统指标无法揭示的诊断信息。

应用价值 / Applications

LED为文档理解模型的开发和部署提供了重要的诊断工具，特别适用于需要高可靠性文档处理的场景，如法律文件分析、学术论文解析、财务报表自动化处理等。该基准有助于研究者识别模型弱点、指导模型改进，并为用户选择合适的文档AI系统提供可解释的性能参考。
