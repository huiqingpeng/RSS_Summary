FineViT: Progressively Unlocking Fine-Grained Perception with Dense Recaptions

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了FineViT，一种专为细粒度视觉感知设计的新型视觉编码器，旨在解决多模态大语言模型（MLLMs）中CLIP编码器因低分辨率预训练和粗糙网络爬取数据导致的性能瓶颈问题。通过采用渐进式训练范式——首先在数十亿全局重描述图像-文本对上以高原生分辨率从头训练建立语义基础，随后通过LLM对齐利用4500万高质量局部描述增强局部感知能力——FineViT实现了最先进的零样本识别和检索性能，尤其在长上下文检索任务中表现突出，并在集成到MLLMs时持续超越SigLIP2和Qwen-ViT等现有编码器。

方法概述 / Method

FineViT采用两阶段渐进训练策略：第一阶段在高分辨率下使用全局重描述（global recaptions）进行大规模预训练，构建细节丰富的语义基础；第二阶段通过大语言模型对齐，利用精心构建的FineCap-450M数据集（包含超过4.5亿高质量局部描述）增强局部感知能力。该方法以密集重描述替代传统的粗糙网络爬取数据，系统性地缓解视觉信息损失。

实验结果 / Results

FineViT在零样本识别和检索任务上达到最先进的性能，尤其在长上下文检索方面表现优异；当集成到多模态大语言模型中时，FineViT一致性地优于SigLIP2和Qwen-ViT等主流多模态视觉编码器，验证了渐进式训练策略的有效性。

应用价值 / Applications

FineViT可作为细粒度视觉感知的强大新基线，适用于需要高精度视觉理解的多模态大语言模型系统，如视觉问答、图像描述生成、视觉检索等任务；其高分辨率处理能力和细粒度感知特性使其在医学影像分析、卫星图像解读、工业质检等对细节敏感的实际场景中具有重要应用价值。
