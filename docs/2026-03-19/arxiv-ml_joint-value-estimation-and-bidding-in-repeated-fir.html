论文摘要 / Abstract

本文研究了重复第一价格拍卖（FPA）中的后悔最小化问题，其中竞拍者每轮拍卖后仅能观察到输赢结果。该设定反映了在线展示广告等实际场景，其中广告展示的真实价值取决于赢得与未赢得拍卖两种潜在结果之间的差异（如点击率或转化率）。作者将因果推断框架引入该问题，提出在仅有处理效应（treatment effect）依赖于可观测特征的复杂设定下，联合估计私有价值和优化竞价策略的算法，并在完全信息反馈（知晓最高对手出价）和二值反馈（仅知输赢）两种设定下均实现了近最优的后悔界。特别地，该框架中处理是主动选择的，从而消除了因果推断中常见的重叠条件（overlap condition）要求。

方法概述 / Method

论文将因果推断与在线学习相结合，构建了联合价值估计与竞价优化的框架。针对两种不同的最高对手出价（HOB）反馈机制——完全信息反馈和二值反馈——分别设计了对应的算法。核心方法在于利用竞拍者主动选择出价（即主动选择"处理"）的特性，通过在线学习同时学习价值模型和优化竞价策略，无需传统因果推断所需的重叠假设。

实验结果 / Results

论文理论证明了所提算法在两种反馈设定下均能达到近最优的后悔界（near-optimal regret bounds）。具体而言，算法在联合估计私有价值和优化竞价的过程中，能够有效控制累积后悔，保证学习效率。实验结果验证了该框架在处理效应具有简单结构、但结果变量依赖关系复杂的设定下的有效性。

应用价值 / Applications

该研究直接适用于在线展示广告等程序化广告场景，帮助广告主在仅能获得部分反馈信息（如是否赢得展示机会）的情况下，同时学习广告展示的真实价值并优化出价策略。研究成果可提升广告主的预算使用效率和投放效果，也为其他存在潜在结果推断问题的重复决策场景（如动态定价、资源分配）提供了方法论基础。
