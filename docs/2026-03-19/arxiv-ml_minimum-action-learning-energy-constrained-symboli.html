Minimum-Action Learning: Energy-Constrained Symbolic Model Selection for Physical Law Identification from Noisy Data

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文提出了最小作用量学习（Minimum-Action Learning, MAL），一种从含噪声观测数据中识别物理定律的符号模型选择框架。
MAL通过最小化"三重作用量"泛函（结合轨迹重建、架构稀疏性和能量守恒约束）来选择符号力定律。配合一种宽模板加速度匹配技术，该方法将噪声方差降低10,000倍，使极低信噪比（~0.02）下的不可解问题转化为可学习问题（~1.6）。在开普勒引力与胡克定律基准测试中，MAL以约0.07 kWh能耗实现100%的物理定律识别，较纯预测误差基线节能40%。

方法概述 / Method

MAL的核心是三重作用量泛函优化：数据保真项（轨迹重建）、稀疏正则化项（架构简洁性）和能量守恒约束项（物理一致性）。关键预处理步骤采用宽模板加速度匹配技术，通过空间平均大幅降低噪声方差，为后续符号回归奠定基础。优化过程在预定义的基函数库上进行组合搜索，同时利用能量守恒作为模型选择的判别准则。

实验结果 / Results

宽模板技术将有效信噪比从0.02提升至1.6，是使问题可解的关键。在开普勒引力实验中，MAL恢复的开普勒指数为p = 3.01 ± 0.01，原始正确基选择率为40%；胡克定律实验达90%。引入基于能量守恒的判别准则后，两种场景的管道级识别率均达100%。基库敏感性分析显示：相近干扰项（如r^{-2.5}和r^{-1.5}）会使选择率降至20%，而远距添加项无害；即使正确基缺失，守恒诊断仍具信息量。

应用价值 / Applications

MAL适用于科学发现场景中的物理定律自动提取，尤其是天文观测、实验物理等噪声强、数据受限的领域。相比SINDy变体、哈密顿神经网络和拉格朗日神经网络，MAL在可解释性与能量约束方面形成独特定位，为符号化、物理一致的模型选择提供了计算高效的解决方案，有助于降低科学机器学习的数据需求与能耗成本。
