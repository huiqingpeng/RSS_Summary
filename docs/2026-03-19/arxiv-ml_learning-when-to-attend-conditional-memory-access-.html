Learning When to Attend: Conditional Memory Access for Long-Context LLMs

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出L2A（Learning To Attend）方法，通过令牌级条件判断决定何时调用全局注意力，解决长上下文语言模型中注意力机制二次复杂度带来的高昂训练成本问题。该方法在Qwen 2.5和Qwen 3模型上将有效上下文长度从32K扩展至128K，性能与标准长上下文训练相当（差距<3%），同时跳过约80%令牌的全局注意力计算。此外，L2A支持训练后对高度稀疏的全局注意力层进行剪枝，KV缓存内存可减少50%，并通过定制Triton内核实现高达2倍的训练吞吐量和首token时间提升。

方法概述 / Method

L2A引入一个轻量级决策层，对每个输入令牌进行二分类判断：仅需局部上下文即可处理的令牌使用高效局部注意力，而需要长程依赖的令牌才触发全局注意力。该设计基于关键观察——大多数令牌实际上不需要完整的全局注意力。作者开发了专门的Triton GPU内核，以高效支持这种细粒度的条件注意力路由机制。

实验结果 / Results

在Qwen系列模型上的评估显示，L2A在128K上下文长度下达到与完全全局注意力训练相近的性能（差距控制在3%以内），同时将全局注意力计算量减少约80%。定制内核实现带来约2倍的训练吞吐量提升和首token生成加速。后训练剪枝可进一步将KV缓存压缩50%，而性能损失可忽略不计。

应用价值 / Applications

L2A为长上下文大语言模型的高效部署提供了实用方案，特别适用于需要处理超长文档的检索增强生成（RAG）、多轮对话和代码分析等场景。该方法显著降低训练和推理成本，使在资源受限环境下部署128K+上下文模型成为可能，同时其可剪枝特性进一步优化了内存受限的推理场景。
