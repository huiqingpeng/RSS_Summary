论文摘要 / Abstract

本文提出了一种条件逆繁殖数学习框架（CIRL），用于从传染病发病率数据中估计时变基本再生数（R_t）。该方法通过从历史发病率模式和时间信息到潜在再生数的条件映射来解决逆问题，避免了传统方法对强参数化流行病学模型的依赖。
CIRL将流行病学结构（以更新方程作为前向算子）与灵活的概率统计建模相结合，在保持动力学一致性的同时，对观测噪声、传播机制突变和零膨胀发病率具有鲁棒性。在合成疫情数据和真实SARS、COVID-19数据上的实验验证了该方法的有效性。

方法概述 / Method

CIRL框架采用"软约束"策略，以更新方程（renewal equation）作为前向算子来嵌入流行病学先验知识，而非强加严格的参数化假设。该方法利用神经网络学习从历史发病率序列和显式时间编码到潜在再生数的条件映射，通过似然-based统计建模实现数据驱动的时间表示与流行病学约束的融合。

实验结果 / Results

实验表明，CIRL在具有受控机制转变的合成疫情数据上，能够快速检测传播模式突变且估计精度优于基线方法；在真实SARS和COVID-19数据集上，该方法对零膨胀观测和观测噪声表现出强鲁棒性，同时保持了估计结果的流行病学可解释性。

应用价值 / Applications

该研究可直接应用于传染病实时监测预警系统，为公共卫生决策提供可靠的R_t估计，特别是在干预措施或行为变化导致传播动态非平稳的复杂场景下。其灵活的概率框架还可扩展至其他具有类似逆问题特征的疾病动力学参数推断任务。
