Anisotropic Permeability Tensor Prediction from Porous Media Microstructure via Physics-Informed Progressive Transfer Learning with Hybrid CNN-Transformer

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本研究提出了一种物理信息深度学习框架，用于从孔隙尺度微观结构图像预测各向异性渗透率张量，解决了直接数值模拟耗时过长（每样本数小时）的瓶颈问题。该框架结合了MaxViT混合CNN-Transformer架构、渐进式迁移学习和可微物理约束，在包含20,000个合成多孔介质样本的数据集上进行训练，测试集上达到方差加权R²=0.9960，相比纯监督基线将未解释方差降低了33%。研究提炼出三条可迁移的科学机器学习原则：大规模视觉预训练可有效跨域迁移；物理约束以可微架构组件形式集成最为稳健；基于诊断性失效模式分析的渐进训练可实现性能增益的明确归因。

方法概述 / Method

论文采用MaxViT混合CNN-Transformer架构，其多轴注意力机制通过块局部操作捕捉颗粒尺度孔喉几何、通过网格全局操作捕捉代表性单元体积（REV）尺度连通性统计，满足渗透率张量预测的空间层次需求。训练采用三阶段渐进课程：第一阶段以ImageNet预训练权重为起点，结合D4等变增强和张量变换；第二阶段引入组件加权损失优先优化非对角耦合项；第三阶段冻结主干网络，通过特征线性调制（FiLM）进行孔隙度条件化的迁移学习。同时，通过可微惩罚项强制满足Onsager互易性和正定性物理约束。

实验结果 / Results

在4,000个样本的独立测试集上，模型整体方差加权R²达到0.9960，其中对角分量K_xx的R²=0.9967，非对角耦合分量K_xy的R²=0.9758。相比纯监督基线，该方法将未解释方差降低了33%，显著提升了各向异性渗透率张量的预测精度，尤其对表征交叉渗透耦合的非对角分量预测改善明显。

应用价值 / Applications

该方法可加速地下流动建模中的渗透率计算，将传统需要数小时的直接数值模拟替代为快速推理，支撑大规模不确定性量化和油藏优化工作流。其提出的物理信息机器学习原则（跨域迁移学习、可微物理约束集成、渐进式诊断训练）具有广泛适用性，可推广至其他多尺度科学计算问题，如复合材料性能预测、电池电极传输特性建模等领域。
