Asymptotically ideal Disjunctive Hierarchical Secret Sharing Scheme with an Explicit Construction

论文摘要 / Abstract

本文提出了一种**渐近理想的析取分层秘密共享（DHSS）方案**的显式构造方法。现有理想的DHSS方案往往存在份额尺寸过大的问题，而非理想方案虽能实现小份额尺寸，却无法同时保证安全性和渐近理想性。作者通过结合多项式、多重线性齐次递推关系和单向函数，首次实现了兼具小份额尺寸、多项式时间 dealer 计算复杂度和计算安全性的渐近理想DHSS方案。

方法概述 / Method

该方案的核心技术是将**多项式插值**、**多重线性齐次递推关系**与**单向函数**相结合：利用递推关系生成层次化的份额结构，通过多项式编码实现阈值控制，并借助单向函数提供计算安全性保障。这种组合构造使得份额尺寸得以压缩，同时 dealer 能在多项式时间内完成份额分发。

实验结果 / Results

论文从理论上证明了该方案满足**渐近理想性**（share size 趋近于最优）和**计算安全性**；与现有理想DHSS方案相比，显著降低了份额尺寸；与现有非理想方案相比，同时实现了安全性和渐近理想性的保证，填补了该领域的理论空白。

应用价值 / Applications

该方案适用于需要**分层访问控制**的敏感数据保护场景，如企业分级密钥管理、云存储中不同信任级别用户的访问授权、以及区块链多重签名等。其小份额尺寸特性特别适合带宽受限或存储资源紧张的环境，而显式构造便于实际部署实现。
