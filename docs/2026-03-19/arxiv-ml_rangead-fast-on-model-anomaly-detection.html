RangeAD: Fast On-Model Anomaly Detection

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了"On-Model AD"这一新的异常检测范式，利用主模型本身编码的目标分布信息进行异常检测，而非运行独立的检测模型。在此框架下，作者设计了RangeAD算法，通过利用主模型神经元的输出范围来实现高效检测。该方法在高维任务上实现了更优的检测性能，同时显著降低了推理开销，展示了On-Model AD作为实用高效异常检测框架的潜力。

方法概述 / Method

RangeAD的核心方法是提取主模型各神经元的输出范围（neuron-wise output ranges）作为异常检测的依据，通过比较输入样本在主模型各层神经元的激活值是否超出正常范围来判断异常。这种方法无需训练额外的检测模型，直接复用主模型的内部表示，实现了与主模型紧密集成的轻量级检测机制。

实验结果 / Results

实验表明RangeAD在高维异常检测任务上优于传统方法，同时推理成本大幅降低；该方法在保持检测精度的同时，避免了运行独立AD模型带来的额外计算开销，实现了效率与性能的双重提升。

应用价值 / Applications

RangeAD适用于需要实时过滤异常输入或检测分布漂移的实际ML系统部署场景，如在线服务中的输入验证、模型监控和安全性保障；其低推理开销特性特别适合资源受限的边缘设备或高吞吐量生产环境，为实际部署中的异常检测提供了经济高效的解决方案。
