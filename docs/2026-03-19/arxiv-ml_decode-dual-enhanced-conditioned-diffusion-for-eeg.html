DECODE: Dual-Enhanced Conditioned Diffusion for EEG Forecasting

论文摘要 / Abstract

本文提出了DECODE（Dual-Enhanced COnditioned Diffusion），一种用于EEG信号预测的新型条件扩散模型框架。该框架创新性地将自然语言描述的语义指导与历史信号的时间动态相结合，以生成事件特定的神经响应。在真实驾驶任务数据集上的评估表明，DECODE在75个时间步长的预测范围内达到了亚微伏级精度（MAE = 0.626微伏），并具有良好的不确定性校准能力，为可解释的脑机接口和零样本泛化到新行为开辟了新的可能性。

方法概述 / Method

DECODE采用条件扩散模型架构，利用预训练语言模型将认知事件的丰富文本描述作为条件信息注入扩散过程，同时通过基于历史的Langevin动力学保持时间一致性。该双增强机制实现了高层语义描述与低层神经动态的有效桥接，使模型能够生成生理合理的事件特定EEG轨迹。

实验结果 / Results

在包含五种不同驾驶行为的现实世界数据集上，DECODE实现了0.626微伏的平均绝对误差（MAE），预测 horizon 达75个时间步长。模型同时保持了良好校准的不确定性估计，证明了其在捕获神经动力学随机性和行为任务语义上下文方面的有效性。

应用价值 / Applications

DECODE为脑机接口（BCI）领域建立了新的范式，支持基于自然语言描述的零样本泛化到新认知行为，显著提升了系统的可解释性。该框架可应用于神经科学研究中的情境依赖神经活动预测、辅助驾驶系统中的认知状态监测，以及开发更具适应性和透明度的神经假肢与脑机交互系统。
