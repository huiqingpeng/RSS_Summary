Self-Conditioned Denoising for Atomistic Representation Learning

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了自条件去噪（Self-Conditioned Denoising, SCD），一种适用于原子尺度表示学习的自监督预训练方法，能够跨越小分子、蛋白质、周期性材料及非平衡构型等多种领域进行条件去噪。实验表明，在控制主干网络架构和预训练数据集的前提下，SCD显著优于现有自监督学习方法，并可匹敌或超越基于DFT力-能量标签的有监督预训练。研究还发现，经SCD预训练的小型快速图神经网络（GNN）能在多领域下游任务上达到或超过更大规模模型在更大标注/未标注数据集上预训练的性能。

方法概述 / Method

SCD采用自嵌入作为条件信号，通过去噪扩散目标函数重建原子构型，实现对任意原子数据领域的通用预训练；该方法与具体主干网络架构解耦，可灵活适配不同的图神经网络结构。核心创新在于利用模型自身的嵌入表示来条件化去噪过程，从而摆脱对力-能量标注或平衡态几何结构的依赖。

实验结果 / Results

SCD在多个下游基准测试上显著超越现有自监督学习方法，性能与有监督力-能量预训练相当甚至更优；小型GNN经SCD预训练后，在分子、材料和蛋白质等多领域任务中展现出与更大模型竞争的能力，且预训练数据效率更高。

应用价值 / Applications

SCD为物理科学领域的基础模型开发提供了高效、可扩展的自监督预训练范式，可降低对昂贵DFT计算的依赖；该方法适用于药物发现、材料设计和蛋白质结构预测等场景，使资源受限的研究者能以更小模型和更少数据实现高性能原子尺度建模。
