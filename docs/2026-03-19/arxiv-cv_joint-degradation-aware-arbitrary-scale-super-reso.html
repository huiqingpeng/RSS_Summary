Joint Degradation-Aware Arbitrary-Scale Super-Resolution for Variable-Rate Extreme Image Compression

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出ASSR-EIC框架，通过任意尺度超分辨率（ASSR）实现可变码率的极端图像压缩（EIC），解决了现有扩散压缩方法需为每个码率单独训练模型的问题。该框架在编码端引入任意尺度下采样模块实现可控码率降低，在解码端采用联合退化感知的扩散超分辨率解码器实现单模型自适应重建。实验表明，该方法在极端压缩场景下达到最优性能，同时支持灵活的码率控制和自适应重建。

方法概述 / Method

论文设计了一个双分支架构：编码端通过任意尺度下采样模块生成不同分辨率的潜在表示以控制码率；解码端采用扩散模型作为联合退化感知ASSR解码器，并引入全局压缩-重缩放适配器提供整体码率指导，以及局部压缩-重缩放调制器动态平衡生成与保真行为。此外，采用双语义增强设计进一步提升重建质量。

实验结果 / Results

ASSR-EIC在极端图像压缩任务上达到最先进的性能，能够在单一模型内实现可变码率压缩和自适应重建；相比需要多模型训练的现有扩散压缩方法，显著降低了计算开销，同时在不同压缩和重缩放设置下均能保持高保真度和高真实感的重建效果。

应用价值 / Applications

该研究适用于带宽受限场景下的图像传输与存储，如移动网络图像分享、卫星图像传输、云端图像归档等需要极低码率的应用；其可变码率特性使单一模型能够适应不同网络条件和质量需求，大幅提升了实际部署的灵活性和效率。
