Efficient Exploration at Scale

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了一种在线学习算法，显著提升了基于人类反馈的强化学习（RLHF）的数据效率。该算法在接收选择数据时增量更新奖励模型和语言模型：奖励模型拟合选择数据，语言模型则通过改进的REINFORCE算法进行更新，强化信号由奖励模型提供。在Gemma大语言模型上的实验表明，该算法仅用不到20K标签即可达到离线RLHF使用200K标签的性能，实现10倍以上的数据效率提升；外推至更大规模，预期1M标签可匹配离线RLHF的1B标签效果，实现1000倍增益。

方法概述 / Method

论文采用在线学习框架，核心组件包括：（1）对每条强化信号添加小的正向偏置（affirmative nudge）；（2）使用认知神经网络（epistemic neural network）建模奖励不确定性；（3）引入信息导向探索（information-directed exploration）策略。这三个关键设计共同驱动了数据效率的显著提升。

实验结果 / Results

在Gemma大语言模型上的实证结果显示，该在线算法使用少于20K人类标签即可匹配离线RLHF在200K标签上的性能表现，实现超过10倍的数据效率提升。基于实验结果的外推预测表明，该方法在1M标签规模下有望达到离线RLHF使用1B标签的水平，即潜在1000倍的效率增益。

应用价值 / Applications

该研究对大规模语言模型的对齐训练具有重要价值，可大幅降低RLHF过程中昂贵的人类标注成本，使资源受限的组织也能进行高质量的模型微调。此外，高效探索机制可加速模型迭代周期，为需要快速适应新领域或用户偏好的应用场景（如个性化助手、专业领域模型）提供实用解决方案。
