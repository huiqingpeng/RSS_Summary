论文摘要 / Abstract

本文提出了一种新型单层量子神经网络，通过标准递归块基（SRBB）近似任意酉演化，并将CNOT门数量以指数级渐近减少。该研究利用李代数及其拓扑特征实现酉算子的可扩展参数化，将原本仅具理论意义的SRBB可扩展方案重新设计为高效的算法实现。提出的CNOT门约简算法仅需单层近似即可完成酉算子合成，并在PennyLane库中验证了高达6量子比特的稀疏和稠密酉矩阵的近似性能。

方法概述 / Method

论文采用基于李代数的代数方法，利用SRBB的结构特性实现酉算子的可扩展参数化；通过拓扑分析设计CNOT门约简算法，将原始多层变分量子电路压缩为单层结构。该方法将2量子比特算子作为原始缩放方案的特例处理，并结合梯度优化与Nelder-Mead无梯度优化两种方法进行参数训练。

实验结果 / Results

实验在PennyLane平台上测试了高达6量子比特的各类酉矩阵（包括稀疏和稠密矩阵），使用多种度量指标评估近似效果；单层CNOT约简网络在真实量子硬件上进行了验证，并与文献中其他近似和分解方法进行了对比。结果表明该单层结构在保持近似精度的同时显著降低了电路深度。

应用价值 / Applications

该研究为量子计算中的酉算子合成提供了高效的硬件实现方案，可应用于量子机器学习、量子模拟和量子算法编译等领域；单层浅层电路结构特别适合当前含噪声中等规模量子（NISQ）设备的硬件约束，有助于提高量子程序在真实量子处理器上的执行保真度和效率。
