论文摘要 / Abstract

本文提出了一种病理感知多视角对比学习框架，用于解决从减少导联集重建12导联心电图（ECG）的病态逆问题。该框架通过在病理流形上正则化潜在空间，整合高保真时域波形与通过监督对比对齐学习的病理感知嵌入，从而滤除解剖学"干扰"变量。在PTB-XL数据集上，该方法在患者独立设置下实现了相比最先进模型约76%的RMSE降低，并在PTB诊断数据库的跨数据集评估中展现出优越的泛化能力。

方法概述 / Method

论文采用病理感知多视角对比学习方法，将时域波形重建与病理嵌入学习相结合，通过监督对比对齐最大化潜在表示与临床标签之间的互信息。该架构利用病理流形作为正则化约束，使模型学习区分与心脏疾病相关的形态学特征和个体解剖变异带来的无关噪声。

实验结果 / Results

在PTB-XL数据集的患者独立重建任务中，该方法相比现有最优模型实现了约76%的RMSE降低。跨数据集评估在PTB诊断数据库上验证了方法的优越泛化性能，证明其在不同数据分布下的稳健性。

应用价值 / Applications

该研究为便携式心电图监测设备提供了诊断级重建能力，使减少导联的硬件设计（如可穿戴设备）能够输出完整12导联信号，兼顾便携性与临床诊断精度。这对于远程医疗、家庭监护和急救场景中的心律失常及心肌缺血检测具有重要实用价值。
