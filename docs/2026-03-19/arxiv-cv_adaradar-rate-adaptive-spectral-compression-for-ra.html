AdaRadar: Rate Adaptive Spectral Compression for Radar-based Perception

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了AdaRadar，一种面向雷达感知的自适应速率频谱压缩方法，用于解决高维原始雷达数据与低带宽计算引擎通信链路之间的瓶颈问题。该方法通过检测置信度对压缩率的代理梯度进行梯度下降，动态调整压缩比，并采用零阶梯度近似处理不可微的剪枝和量化操作。实验表明，该方案在RADIal、CARRADA和Radatron数据集上实现了超过100倍的特征尺寸压缩，性能下降仅约1个百分点。

方法概述 / Method

AdaRadar采用离散余弦变换（DCT）处理雷达数据立方体，利用雷达特征图高度集中于少数频率分量的特性进行选择性系数剪枝，并通过缩放量化保留每个雷达 patches 的动态范围。核心创新在于基于检测置信度的在线自适应反馈机制，使用零阶梯度估计避免在带宽受限链路上传输大型梯度张量。

实验结果 / Results

在多个公开雷达数据集（RADIal、CARRADA、Radatron）上的验证表明，该方法可实现超过100倍的特征压缩率，同时目标检测性能下降控制在约1%以内。自适应机制能够在不同或对抗性条件下动态优化压缩策略，显著优于固定压缩比的图像域方法。

应用价值 / Applications

该研究直接适用于自动驾驶系统中的雷达感知模块，可缓解边缘雷达传感器与中央NPU之间的数据传输瓶颈，支持高分辨率雷达数据的实时处理。其自适应特性使其在复杂多变的实际驾驶环境中具有鲁棒性，为全天候、全场景自动驾驶感知提供了高效的边缘-云端协同计算方案。
