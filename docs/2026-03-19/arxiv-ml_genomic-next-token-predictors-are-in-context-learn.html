Genomic Next-Token Predictors are In-Context Learners

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究首次证明基因组序列模型能够自发涌现上下文学习（ICL）能力。通过对Evo2基因组模型（基于核苷酸预测训练）的系统评估，作者发现该模型与语言模型类似，能够随着上下文示例数量的增加而呈现对数线性的模式归纳能力提升。这一发现挑战了ICL仅源于人类语言独特统计特性的观点，表明大规模预测训练本身即可催生跨模态的元学习能力。

方法概述 / Method

研究团队构建了符号推理任务的对比实验框架，将同一类抽象推理任务分别实例化为语言形式和基因组形式（A/T/C/G序列）。通过控制任务结构和示例数量，实现了基因组模型（Evo2）与语言模型在ICL能力上的直接定量比较，并系统测量模型在不同数量的上下文演示下的性能变化。

实验结果 / Results

实验表明，Evo2基因组模型展现出与语言模型相似的ICL特性：随着上下文示例数量增加，模型在模式归纳任务上的准确率呈现对数线性增长。这一规律在语言和基因组两种任务形式中均成立，且无需任何针对特定任务的微调，证实了ICL能力是通过大规模下一token预测训练有机涌现的。

应用价值 / Applications

该研究为生物信息学领域开辟了新的方法论路径：基因组模型可通过上下文示例快速适应新任务（如调控元件识别、突变效应预测），无需昂贵的重新训练。同时，这一发现支持了"统一模态无关的上下文学习"理论框架，为设计更通用的序列学习架构（适用于蛋白质、化学分子等其他符号域）提供了理论基础。
