Hi-GMAE: Hierarchical Graph Masked Autoencoders

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了分层图掩码自编码器（Hi-GMAE），一种新颖的多尺度自监督学习框架，用于解决现有单尺度图掩码自编码器（GMAEs）无法捕获图数据层次结构的问题。
Hi-GMAE通过图池化构建多尺度图层次结构，并设计了从粗到细的掩码策略和渐进恢复策略，以实现对不同粒度层次图信息的有效学习。在17个图数据集上的实验表明，Hi-GMAE在两项图学习任务中 consistently 优于29种最先进的自监督学习方法。

方法概述 / Method

Hi-GMAE首先利用图池化操作构建多尺度图层次结构，从而在原子-官能团-分子等不同粒度级别探索图结构。为保证跨尺度子图掩码的一致性，论文提出了一种从粗到细的掩码策略，即在最粗尺度上初始化掩码并逐步反投影到更细尺度；同时结合渐进恢复策略来缓解完全掩码子图带来的学习困难。

实验结果 / Results

Hi-GMAE在涵盖两项图学习任务的17个图数据集上进行了广泛评估，与29种最先进的自监督基线方法相比，Hi-GMAE在捕获全面图信息方面表现出 consistently 更优的性能，验证了层次化多尺度建模对图自监督学习的有效性。

应用价值 / Applications

Hi-GMAE特别适用于具有明显层次结构的图数据，如分子图（原子-官能团-分子）、社交网络和知识图谱等领域。该方法通过自监督预训练学习高质量的图表示，可提升分子性质预测、药物发现、图分类和节点分类等下游任务的性能，为缺乏大量标注数据的图学习应用提供了有效的表示学习方案。
