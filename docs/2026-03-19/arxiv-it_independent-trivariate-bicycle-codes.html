Independent Trivariate Bicycle Codes

arXiv Information Theory 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了六种独立三变量自行车（ITB）码，将Bravyi等人的双变量自行车码框架扩展至三个循环维度。通过在三维环面上使用非对称多项式对，作者构造了四种量子纠错码，其中包括一个$[[140,6,14]]$码，其$kd^2/n = 8.40$。在码容量设置下，该码实现了8.0%的伪阈值，超越了Voss等人最佳多变量自行车码的7.9%和$kd^2/n = 2.67$。研究还展示了两种自对偶码$[[54,14,5]]$和$[[128,20,8]]$。

方法概述 / Method

该方法将双变量自行车码扩展至三个循环维度，利用三维环面上的非对称多项式对构造量子低密度奇偶校验（LDPC）码。通过引入第三个循环维度，显著扩展了代数量子纠错码的设计空间，同时保持了良好的编码参数和可实现性。

实验结果 / Results

在电路级退极化噪声下，$[[140,6,14]]$码达到0.59%的伪阈值，$[[84,6,10]]$码达到0.53%；在SI1000超导噪声模型中，$[[140,6,14]]$码在$p = 0.20\%$时每轮每可观测量的错误率为$5.6 \times 10^{-5}$。两种自对偶码$[[54,14,5]]$和$[[128,20,8]]$分别达到$kd^2/n = 6.48$和$10.0$，且稳定子权重仅为8。

应用价值 / Applications

这些结果为实用容错量子计算提供了具有竞争力的候选编码方案，特别是在超导量子硬件平台上。高伪阈值和低错误率表明ITB码在实际噪声环境下具有良好的容错性能，第三个循环维度的引入为设计更优的代数量子LDPC码开辟了新途径，有望推动大规模量子计算的实现。
