Verification and Validation of Physics-Informed Surrogate Component Models for Dynamic Power-System Simulation

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文针对物理信息机器学习代理模型在电力系统动态仿真中的应用，提出了一个验证与确认（V&V）框架。核心贡献在于将代理模型嵌入微分-代数方程（DAE）求解器的"在仿真器内使用"场景形式化为V&V问题，并推导出有限时间范围的误差上界，该上界关联了组件输出误差、代数耦合敏感性、动态误差放大和仿真时长。研究发现，单独的高精度代理模型并不能保证其在完整系统仿真中的准确性，最大偏差往往出现在高应力运行区域，且小的方程残差不一定意味着小的状态轨迹误差。

方法概述 / Method

论文采用理论推导与案例研究相结合的方法：首先建立数学框架，推导有限时间范围内的误差边界，量化组件输出误差如何通过代数耦合传播并放大；随后研究两种互补设置——基于模型的验证（对照参考组件求解器）和基于数据的验证（通过保形预测校准与仿真器交换的组件输出变量）。案例研究针对二阶、四阶和六阶同步电机模型，使用物理信息神经网络（PINN）作为代理模型。

实验结果 / Results

实验结果表明，独立组件上表现良好的代理模型在嵌入完整系统仿真器后可能出现显著误差；误差分布呈现明显的运行条件依赖性，最大偏差集中在系统受应力较高的运行区域（如重载或故障后状态）；此外，代理模型在微分方程残差上的良好表现（小残差）并不能可靠预测实际状态轨迹的误差大小，揭示了传统评估指标的局限性。

应用价值 / Applications

该研究为电力系统数字孪生和实时仿真提供了关键的模型可信度评估工具，可指导运营商在采用机器学习加速仿真时识别高风险运行场景；框架的通用性使其可扩展至其他基于DAE的物理系统（如流体网络、机械系统）；保形预测校准方法为安全关键应用中的不确定性量化和风险感知决策提供了理论基础，有助于在高渗透率可再生能源电网的稳定性分析中建立对AI代理模型的信任。
