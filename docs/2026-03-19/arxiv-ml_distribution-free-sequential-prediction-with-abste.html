Distribution-Free Sequential Prediction with Abstentions

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文研究了带有弃权机制的序列预测问题，其中对抗者可以在独立同分布（i.i.d.）样本流中注入任意数量的对抗样本，但学习者在每轮可以选择弃权而不对预测负责——若该样本确实被污染则不会受到惩罚。针对分布未知的场景，作者提出了基于提升（boosting）弱学习器的AbstainBoost算法，证明了一般VC类函数在分布无关的弃权学习中对遗忘型对抗者可实现次线性误差，并对自适应对抗者在结构化函数类（如线性分类器）上也有类似保证，同时建立了揭示误分类误差与错误弃权次数之间多项式权衡关系的下界。

方法概述 / Method

作者采用boosting框架构建AbstainBoost算法，通过迭代组合多个弱学习器来构建强预测器；该算法核心在于设计了一种机制，使得学习者能够在面对不确定样本时主动选择弃权，从而将对抗污染的影响转化为可控的弃权决策，而非直接承担误分类风险。

实验结果 / Results

理论分析表明，对于一般VC类，AbstainBoost在分布无关设置下对遗忘型对抗者保证次线性累积误差；对于自适应对抗者，在包括线性分类器在内的结构化函数类上同样可获得类似保证。下界结果进一步揭示了误分类误差与错误弃权次数之间存在固有的多项式权衡关系，即无法同时使两者都任意小。

应用价值 / Applications

该研究适用于存在数据污染风险的在线决策场景，如金融欺诈检测、网络安全入侵识别、医疗诊断辅助系统等，其中系统需要在不确定时主动拒绝预测以保证可靠性；同时，分布无关的特性使其更具实用价值，无需预先收集大量干净数据来估计分布，降低了部署门槛。
