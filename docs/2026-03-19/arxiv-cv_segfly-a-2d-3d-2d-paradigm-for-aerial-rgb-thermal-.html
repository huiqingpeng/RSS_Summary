SegFly: A 2D-3D-2D Paradigm for Aerial RGB-Thermal Semantic Segmentation at Scale

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了SegFly，一种面向无人机(UAV)的2D-3D-2D几何驱动范式，用于大规模RGB-T热成像语义分割。该方法利用高重叠航拍图像中的多视图冗余，将少量人工标注的RGB图像(少于3%)提升到语义3D点云，再重投影到所有视图，自动生成97%的RGB标签和100%的热成像标签，标注准确率达91%和88%。基于此框架，作者构建了包含20,000余张高分辨率RGB图像和15,000余对几何对齐RGB-T图像的大规模基准数据集，并建立了Firefly基线模型，证明传统架构和视觉基础模型均能从SegFly监督中显著受益。

方法概述 / Method

核心方法是2D-3D-2D范式：首先将少量2D RGB图像通过多视图几何提升到3D语义点云，然后将该点云重投影回所有2D视图以生成密集伪真值标签。该框架还扩展到跨模态图像配准，利用3D几何作为中间对齐空间，实现无需硬件同步的全自动像素级RGB-T配准(87%配准准确率)。

实验结果 / Results

自动标注方法在RGB和热成像模态上分别达到91%和88%的标注准确率，仅需不到3%的人工RGB标注。跨模态配准实现87%的配准准确率。在SegFly基准上，Firefly基线模型以及多种传统架构和视觉基础模型(VFMs)均展现出显著性能提升，验证了该几何驱动范式对多模态场景理解的有效性。

应用价值 / Applications

该研究为无人机航拍场景理解提供了可扩展的多模态数据标注解决方案，降低了人工标注成本，适用于城市、工业和农村等多样化环境的智能监测。
RGB-T语义分割能力可支持搜救、消防、基础设施巡检、农业监测等需要全天候(昼夜)感知的应用场景，推动自主无人机系统在复杂环境中的部署。
