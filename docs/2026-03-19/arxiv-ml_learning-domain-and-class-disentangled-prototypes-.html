Learning Domain- and Class-Disentangled Prototypes for Domain-Generalized EEG Emotion Recognition

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了一种面向完全未见目标域的EEG情绪识别多域聚合迁移学习框架MAT，通过特征解耦模块分离类不变域特征与域不变类特征，并引入基于最大均值差异（MMD）的层次化域聚合机制和自适应原型更新策略，有效解决了跨被试差异、目标域数据依赖和标签噪声等问题。在SEED、SEED-IV和SEED-V三个公开数据集上的实验表明，MAT在未见目标域上的识别准确率较最先进方法分别提升2.87%、3.84%和2.05%，为真实场景下的跨被试情绪识别提供了新方向。

方法概述 / Method

MAT框架核心包含三个模块：特征解耦模块通过分离类相关与域相关特征实现可解释性更强的EEG表征；层次化域聚合（HDA）机制利用MMD构建超级域以建模跨被试共享分布结构；自适应原型更新持续优化域原型和类原型以捕获稳定内在表征。此外，成对学习策略将分类任务重构为样本对相似度估计，有效缓解标签噪声影响。

实验结果 / Results

在三个标准EEG情绪数据集（SEED三分类、SEED-IV四分类、SEED-V五分类）的跨被试域泛化实验中，MAT均取得最优性能，相比当前最先进方法准确率提升2.05%-3.84%不等。实验验证了特征解耦、层次化域聚合和成对学习策略各组件的有效性，证明该方法在完全未见目标域上具有强泛化能力和抗噪声鲁棒性。

应用价值 / Applications

该研究可直接应用于情感脑机接口（aBCIs）的实际部署，解决临床和日常场景中因新用户无标注数据导致的系统失效问题，支持即插即用的跨被试情绪识别系统。其域泛化框架还可迁移至其他生理信号（如fNIRS、ECG）的跨个体识别任务，为普适化医疗健康监测和人机交互系统提供技术基础。
