Discovering Decoupled Functional Modules in Large Language Models

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了首个针对大型语言模型（LLM）功能模块发现问题的系统性研究框架ULCMOD，通过无监督方式同时实现神经元聚类与输入样本主题发现。该框架引入新颖的目标函数和迭代解耦算法（IterD），实验表明发现的功能模块具有高质量解耦性、语义一致性和层次化空间组织特性，显著提升了多项下游任务性能，为LLM可解释性研究填补了关键空白。

方法概述 / Method

论文将功能模块发现问题形式化为联合优化任务，设计ULCMOD框架同时学习神经元分配矩阵和输入样本的主题分布。核心采用Iterative Decoupling（IterD）算法交替优化：固定神经元分配时推断样本主题，固定主题时更新神经元聚类，通过跨层信息传递实现全局一致的模块发现。

实验结果 / Results

实验表明ULCMOD发现的模块在语义相关性指标上显著优于基线方法，模块激活模式呈现清晰的语义 specialization（如数学推理、代码生成、事实知识等）。下游任务评估显示，基于模块的干预和编辑在知识编辑、毒性内容抑制等任务上取得更优性能，同时保持模型原有能力。

应用价值 / Applications

该研究为LLM机制可解释性提供了新工具，可应用于模型诊断（定位故障功能区域）、安全对齐（精准编辑有害行为模块）、模型压缩（按模块剪枝）及持续学习（模块化知识更新）。发现的层次化模块组织规律还可指导更高效的网络架构设计。
