ChopGrad: Pixel-Wise Losses for Latent Video Diffusion via Truncated Backpropagation

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文提出ChopGrad，一种针对视频扩散模型的截断反向传播方案，通过将梯度计算限制在局部帧窗口内，解决了视频生成中像素级损失训练时内存成本随帧数线性增长的问题。该方法将训练内存从与视频帧数相关的线性复杂度降至常数复杂度，同时保持全局一致性。
ChopGrad在视频超分辨率、视频修复、神经渲染场景增强和可控驾驶视频生成等多个像素级损失的条件视频生成任务上取得了与现有最先进方法相媲美的效果。

方法概述 / Method

ChopGrad采用截断反向传播策略，在视频解码过程中仅对局部时间窗口内的帧计算梯度，而非整个视频序列，从而避免递归帧处理导致的激活值累积问题。作者对该近似方法进行了理论分析，证明其在保持全局时间一致性的同时实现了高效的像素级损失优化。

实验结果 / Results

ChopGrad成功将训练内存消耗从随帧数线性增长降低为恒定值，使得长视频和高分辨率视频的微调计算可行。实验表明，该方法在多个条件视频生成任务中表现优异，包括视频超分辨率、视频修复、神经渲染场景增强和可控驾驶视频生成，性能与现有最先进的视频扩散模型相当。

应用价值 / Applications

ChopGrad为需要像素级监督信号的视频生成任务提供了高效的训练框架，特别适用于长视频和高分辨率场景下的模型微调。其实际应用涵盖视频质量增强（超分辨率、修复）、神经渲染内容优化以及自动驾驶仿真中的可控视频生成等领域，显著降低了计算资源门槛。
