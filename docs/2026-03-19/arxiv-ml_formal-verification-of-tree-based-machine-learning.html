Formal verification of tree-based machine learning models for lateral spreading

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文针对地质灾害预测中的机器学习模型物理一致性问题，提出了一种基于可满足性模理论（SMT）的形式化验证方法。研究将训练好的树集成模型编码为逻辑公式，对四个岩土工程物理规范（地下水位深度、PGA单调性、距离安全性和平坦地面安全性）进行全输入域的严格验证。研究发现无约束的EBM模型（准确率80.1%）违反全部四项规范，而完全约束的EBM（准确率67.2%）满足其中三项，揭示了准确率与物理一致性之间的持久权衡关系。该研究建立了"验证-修复-验证"的工程循环，为安全关键岩土工程应用中的机器学习模型部署提供了形式化认证框架。

方法概述 / Method

本研究采用SMT求解器将训练后的树集成模型（XGBoost和EBM）编码为可判定的逻辑公式，将物理规范形式化为逻辑约束条件。验证过程通过SMT求解器在全输入域上进行穷尽式检查，要么生成违反规范的具体反例，要么证明规范在所有情况下均成立。

实验结果 / Results

在2011年基督城地震侧向扩展数据集（7,291个场地，4个特征）上的实验表明，33种模型变体的帕累托分析显示没有任何变体能同时实现超过80%的准确率和完全规范合规。
SHAP分析进一步揭示，规范违反的反例中，违规特征可能排名最后，证明事后解释方法无法替代形式化验证。

应用价值 / Applications

该方法为地震液化侧向扩展等安全关键地质灾害预测提供了可认证的机器学习部署方案，特别适用于需要严格物理一致性的岩土工程风险评估。形式化验证框架可推广至其他依赖树集成模型的科学和工程领域，支持高风险决策中的模型可信度认证。
