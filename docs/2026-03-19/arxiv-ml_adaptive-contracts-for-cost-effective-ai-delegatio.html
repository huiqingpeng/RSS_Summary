Adaptive Contracts for Cost-Effective AI Delegation

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了在AI委托任务中，当评估存在噪声时，按绩效付费的合同会导致预期支付上升的问题。作者提出了一种自适应合同机制，允许在观察到初步粗略信号后，选择性地进行详细评估以节约资源。论文提供了计算最优自适应合同的高效算法，分析了随机化自适应合同模型，并通过问答和代码生成数据集实证展示了自适应机制相对于非自适应基线的优势。

方法概述 / Method

论文采用机制设计理论，构建了基于两层信号结构的自适应合同框架：第一层使用低成本粗略评估筛选输出，第二层仅在必要时启用高成本精细评估。作者针对自然假设或小规模核心问题维度开发了高效优化算法，并探讨了随机化自适应合同作为替代方案。

实验结果 / Results

在问答和代码生成数据集上的实证研究表明，自适应合同显著优于非自适应基线，能够在保证委托质量的同时降低总体评估成本。理论分析证明，一般无结构情况下的最优自适应合同计算具有近似难度，但在结构化场景下可实现高效求解。

应用价值 / Applications

该研究适用于企业向AI服务商外包文本生成任务（如内容创作、代码生成、客服应答）的场景，帮助委托方在评估准确性与成本之间取得平衡。自适应合同机制可减少企业在AI质量监控上的支出，促进更具成本效益的人机协作模式。
