WPT: World-to-Policy Transfer via Online World Model Distillation

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了WPT（World-to-Policy Transfer），一种通过在线世界模型蒸馏实现世界知识向策略迁移的训练范式。该方法通过可训练的奖励模型将世界模型预测的未来动态与候选轨迹对齐，从而将世界知识注入教师策略；随后通过策略蒸馏和世界奖励蒸馏将教师的推理能力迁移至轻量级学生策略，在提升规划性能的同时保证实时部署能力。实验表明，WPT在开放环路测试中达到0.11的碰撞率，在闭环测试中实现79.23的驾驶得分，超越现有基于世界模型和模仿学习的方法，且学生策略推理速度提升达4.9倍。

方法概述 / Method

WPT采用两阶段蒸馏框架：首先构建可训练的世界奖励模型，利用世界模型预测的未来状态对候选轨迹进行评分，指导教师策略学习世界知识；其次设计策略蒸馏与世界奖励蒸馏机制，将教师策略的推理能力迁移至轻量级学生网络，实现知识压缩与推理加速。

实验结果 / Results

WPT在开放环路和闭环自动驾驶基准测试中均达到最优性能：开放环路碰撞率为0.11，闭环驾驶得分达79.23，显著优于现有世界模型方法和模仿学习方法；蒸馏后的学生策略推理速度提升4.9倍，同时保留了大部分性能增益，验证了该方法在准确性与实时性之间的有效权衡。

应用价值 / Applications

WPT可广泛应用于自动驾驶等实时决策系统，解决复杂世界模型推理开销大、难以部署的问题；通过知识蒸馏实现高性能轻量级策略，使先进的世界模型能力能够在边缘计算设备上实时运行，推动自动驾驶系统的实际落地与规模化部署。
