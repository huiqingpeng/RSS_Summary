From Street Form to Spatial Justice: Explaining Urban Exercise Inequality via a Triadic SHAP-Informed Framework

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本研究构建了一个理论导向且可解释的分析框架，用于诊断街道层面的运动剥夺问题。该框架整合列斐伏尔的空间三元组理论、多源城市数据与SHAP可解释机器学习方法，以深圳为案例揭示了"构想空间"属性对运动强度影响最大，但局部剥夺机制因情境而异。研究识别出七种剥夺模式类型，并定位出高需求低支持的街道段作为优先干预区域，为促进日常体育活动的空间正义提供了理论分析框架与实践诊断工具。

方法概述 / Method

研究采用列斐伏尔的空间三元组理论（空间实践、空间表征、表征空间）作为理论框架，整合多源城市数据（包括街景图像、POI数据、手机信令数据等）构建街道形态与运动行为的关联模型。运用SHAP（SHapley Additive exPlanations）方法对机器学习模型进行事后解释，量化不同空间维度特征对运动强度的贡献度，并识别局部化的剥夺机制。

实验结果 / Results

研究发现"构想空间"（conceived space）属性对运动强度的整体影响最强，但不同区域的剥夺机制存在显著异质性。基于SHAP分析，研究归纳出七种剥夺模式类型（typology），并成功识别出大量"高需求-低支持"错配的街道段。这些优先干预区域的空间分布揭示了城市运动基础设施配置的不平等格局，为精准化空间治理提供了数据支撑。

应用价值 / Applications

该框架可应用于城市健康基础设施的公平性评估与规划决策，帮助规划者识别运动服务盲区并优化资源配置。研究成果支持"15分钟运动圈"等健康城市政策的精准落地，为应对城市慢性病防控、健康不平等及空间正义议题提供可操作的诊断工具与循证依据，具有向其他快速城市化地区推广的潜力。
