Unsupervised Symbolic Anomaly Detection

论文摘要 / Abstract

本文提出了SYRAN，一种基于符号回归的无监督异常检测方法。该方法不采用不透明的高维模型来编码正常模式，而是学习一组人类可读的方程来描述"符号不变量"——即在正常数据上近似保持恒定的函数。偏离这些不变量的程度产生异常分数，使检测逻辑天生具有可解释性，无需事后解释。实验表明，SYRAN既能提供与已知科学或医学关系对应的可解释方程，又能保持与最先进方法相当的异常检测性能。

方法概述 / Method

SYRAN利用符号回归技术自动发现描述正常数据内在约束关系的数学方程（符号不变量），通过构建方程组集成来刻画数据的正常行为模式。异常检测时，计算新样本与这些不变量的偏离程度作为异常分数。

实验结果 / Results

SYRAN在多个数据集上展现出与深度学习等最先进方法相当的异常检测准确率，同时生成的符号方程能够对应到领域专家可理解的物理或医学规律（如心率与血压的已知关系），实现了性能与可解释性的统一。

应用价值 / Applications

该方法适用于需要模型透明度的关键领域，如医疗诊断（解释为何标记某患者为异常）、工业设备监控（提供可验证的故障预警规则）和科学发现（自动揭示数据中的守恒定律），帮助领域专家建立对AI决策的信任。
