Knowledge Localization in Mixture-of-Experts LLMs Using Cross-Lingual Inconsistency

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了一种利用跨语言不一致性进行混合专家（MoE）大语言模型可解释性研究的新方法。研究者发现现代LLM在不同语言中表现存在显著差异，并将这一现象转化为知识定位工具，通过对比模型在"成功"和"失败"语言中的路由行为来识别对特定知识起关键作用的专家组件。实验表明，仅禁用约20个专家（占总6000个的0.3%）就能使模型在超过40%的情况下无法正确回答问题，证明了该方法的有效性和可扩展性。

方法概述 / Method

该方法分为两个阶段：首先用困难的事实性问题在多种语言中查询模型，生成"成功"和"失败"的激活桶；然后对MoE路由器的logits应用统计对比分析，识别对知识回答至关重要的专家。通过对比不同语言条件下的路由模式差异，精确定位功能相关的模型组件。

实验结果 / Results

研究发现仅需禁用约20个专家（占6000个专家的极小比例），模型在超过40%的情况下便无法正确回答知识性问题，表明这些被定位的专家对知识回忆具有关键必要性。该方法实现了高效的知识定位，以极小的干预成本显著影响模型性能。

应用价值 / Applications

该研究为复杂MoE架构LLM的可解释性提供了现实可行的解决方案，有助于理解大模型内部知识存储机制。实际应用包括：模型编辑与知识更新、特定知识的精准干预、多语言模型性能优化，以及为AI安全研究提供定位潜在有害知识的能力。
