A Novel end-to-end Digital Health System Using Deep Learning-based ECG Analysis

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究提出了AI-HEART，一个基于云的数字化健康信息系统，用于管理、分析长程动态心电图（ECG）记录并辅助临床决策。该平台实现了端到端处理流程，包括多导联ECG数据摄取、信号预处理、基于深度神经网络的波形分割、噪声/质量检测以及心跳和节律级别的多分类心律失常识别。研究通过结合大规模临床标注数据集、专家参与筛选和生成式数据增强来解决类别不平衡问题，并在三导联动态ECG数据上验证了系统的技术可行性和临床实用价值。

方法概述 / Method

AI-HEART采用端到端的深度学习流水线，整合多种专用神经网络分别处理波形分割（wave delineation）、噪声检测和心律失常多分类任务。为应对真实世界信号变异性和类别不平衡，模型开发结合了大规模临床标注数据、专家在环（expert-in-the-loop）数据筛选，以及对稀有节律的生成式数据增强技术。

实验结果 / Results

在三导联动态ECG数据上的实证评估表明：波形分割精度足以支持自动化间期测量；噪声检测能有效标记低质量信号段；心律失常分类在常见和稀有节律上均实现了高特异性和具有临床实用价值的宏平均性能。系统同时支持可追溯的输出、审计友好的数据存储以及临床医生反馈驱动的模型改进。

应用价值 / Applications

AI-HEART为将AI整合到常规ECG服务提供了可扩展的部署方案，适用于医院、诊所等医疗机构的心电监测与诊断 workflow。该系统通过支持临床医生审阅编辑、捕获反馈以实现可控的模型迭代优化，具有提升诊断效率、减轻医生工作负担、促进医疗资源公平可及的潜力，可作为数字健康信息系统在心血管慢病管理和远程医疗中推广应用。
