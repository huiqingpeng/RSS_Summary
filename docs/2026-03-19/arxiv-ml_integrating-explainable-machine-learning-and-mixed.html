Integrating Explainable Machine Learning and Mixed-Integer Optimization for Personalized Sleep Quality Intervention

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究提出了一种个性化预测-处方框架，将可解释机器学习与混合整数优化相结合，用于睡眠质量的个性化干预设计。该框架通过监督分类器预测睡眠质量，利用SHAP特征归因量化可改变因素的影响，并将其纳入混合整数优化模型以识别最小且可行的行为调整。实验表明该框架具有强预测性能（测试F1分数0.9544，准确率0.9366），能够生成简洁的个性化建议，通常仅需一到两项高影响力行为调整。

方法概述 / Method

论文采用"预测-解释-优化"三阶段方法：首先训练监督分类器预测主观睡眠结果；其次应用SHAP（SHapley Additive exPlanations）进行特征归因，识别可修改因素的重要性；最后构建混合整数优化模型，将SHAP重要性作为目标函数系数，并通过惩罚机制建模行为改变的阻力，在约束条件下寻找最优干预方案。

实验结果 / Results

该框架在测试集上达到F1分数0.9544和准确率0.9366的强预测性能。敏感性和帕累托分析显示预期改善与干预强度之间存在明显权衡，随额外改变增加呈现收益递减效应。个体层面分析表明，模型通常生成简洁建议，推荐一到两项高影响力行为调整，或在预期收益较小时建议不作出改变。

应用价值 / Applications

该研究为数字健康领域提供了从预测洞察到可执行干预的完整决策支持系统，适用于移动健康应用、远程医疗平台和可穿戴设备集成的睡眠管理场景。通过生成个性化、最小侵入性的行为建议，该框架有助于提升用户依从性，为临床医生和健康管理师提供数据驱动的精准干预工具，推动睡眠医学向精准化和预防性方向发展。
