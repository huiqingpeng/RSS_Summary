GigaWorld-Policy: An Efficient Action-Centered World--Action Model

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了GigaWorld-Policy，一种以动作为中心的World-Action模型（WAM），解决了现有方法中联合推理未来视觉动态与动作导致的高推理开销、以及视觉与运动表征纠缠的问题。该模型将策略学习分解为两个耦合组件：基于当前观测预测未来动作序列，并基于预测动作生成未来视频。通过因果设计使未来视频token不影响动作token，实现可选的视频生成和更快的动作推理。实验表明，该模型在真实机器人平台上比领先基线Motus快9倍，任务成功率提升7%；在RoboTwin 2.0上相比pi-0.5性能提升95%。

方法概述 / Method

GigaWorld-Policy采用双组件耦合架构：动作预测模块基于当前观测生成未来动作序列，视频生成模块则基于预测动作和相同观测生成未来视频。核心创新在于因果掩码设计，阻断未来视频token向动作token的信息流动，使视频生成在推理阶段变为可选。模型首先在大规模机器人数据集上预训练动作中心的视频生成模型，再适配为机器人策略学习的主干网络。

实验结果 / Results

在真实机器人平台实验中，GigaWorld-Policy推理速度达到Motus的9倍，同时任务成功率提高7%；在RoboTwin 2.0基准上，相比pi-0.5模型性能提升95%。这些结果表明该模型在保持高精度的同时显著降低了计算开销，实现了效率与性能的双重优化。

应用价值 / Applications

该研究适用于需要实时决策的机器人操控任务，如工业装配、家庭服务机器人等场景，其高效的推理能力支持边缘设备部署。可选的视频生成功能还可用于策略可解释性分析和安全验证，为机器人学习系统提供物理合理的动作约束和可视化调试工具。
