论文摘要 / Abstract

本文深入研究了对角线性网络（具有线性激活函数和对角权重矩阵的神经网络）的隐式正则化特性。作者发现，从小初始化开始训练的对角线性网络，其完整训练轨迹与Lasso正则化路径密切相关，其中训练时间扮演着逆正则化参数的角色。在Lasso正则化路径满足单调性假设时，这种联系是精确的；一般情况下则呈现近似联系。该研究通过理论分析和仿真实验验证了这一结论，为理解梯度下降隐式正则化提供了新的视角。

方法概述 / Method

论文采用理论分析与数值模拟相结合的方法，研究对角线性网络的梯度下降动力学。核心方法是建立对角线性网络训练轨迹与Lasso正则化路径之间的数学对应关系，通过分析权重矩阵的对角结构特性，推导出训练时间与正则化参数的等价性。

实验结果 / Results

实验结果表明，对角线性网络的训练轨迹与Lasso正则化路径高度吻合；在单调性假设下，两者等价性得到严格证明；非单调情况下，近似联系依然成立。数值模拟验证了理论预测，展示了从小初始化开始的训练过程确实收敛到最小ℓ1范数解。

应用价值 / Applications

该研究为理解神经网络的隐式正则化机制提供了可严格分析的简化模型，有助于设计具有可控稀疏性的深度学习架构。理论框架可指导实践中的初始化策略选择和训练过程监控，特别是在需要稀疏解的高维统计学习场景中具有潜在应用价值。
