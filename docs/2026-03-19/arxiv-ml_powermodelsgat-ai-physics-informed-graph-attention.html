arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文提出了PowerModelsGAT-AI，一种物理信息图注意力网络，用于实时求解交流潮流方程。该模型通过总线类型感知掩码处理不同总线类型，并采用可学习权重平衡多损失项（包括功率失配惩罚）。研究在14个基准系统（4至6,470节点）上验证了模型，在N-2条件下训练统一模型实现了电压幅值平均归一化绝对误差0.89%、电压角度R²>0.99的精度；同时通过经验回放与弹性权重整合策略解决了持续学习中的灾难性遗忘问题，使新系统适配时基系统误差增幅控制在2%以内。

方法概述 / Method

该研究采用图注意力网络架构，引入物理信息约束设计损失函数，包含功率失配惩罚项以确保预测满足基尔霍夫定律。模型通过总线类型感知掩码机制灵活处理PQ、PV、平衡等不同总线类型，并使用可学习权重自动平衡各损失项的贡献。针对持续学习场景，结合经验回放与弹性权重整合（EWC）策略保护重要参数，防止模型适配新系统时遗忘旧知识。

实验结果 / Results

统一模型在13个系统的N-2条件下训练后，跨系统泛化性能优异：电压幅值预测误差仅0.89%，电压角度决定系数R²超过0.99。持续学习实验中，标准微调导致基系统误差激增超1000%，而所提策略将误差增幅压制在2%以下，部分基系统性能甚至得到提升。可解释性分析揭示注意力权重与物理参数显著相关（电纳r=0.38，热极限r=0.22），验证模型捕获了物理规律。

应用价值 / Applications

该研究为电力系统实时运行与控制提供了高效可靠的AI求解器，可替代传统牛顿-拉夫逊法在紧急状态下的计算瓶颈。持续学习能力使模型能动态适应电网拓扑变化与新系统接入，适用于跨区域互联电网的协同优化。物理可解释性设计增强了调度员对AI决策的信任度，支撑高比例可再生能源渗透下的智能电网安全分析应用。
