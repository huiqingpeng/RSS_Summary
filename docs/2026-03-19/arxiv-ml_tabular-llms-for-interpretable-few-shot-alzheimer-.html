Tabular LLMs for Interpretable Few-Shot Alzheimer's Disease Prediction with Multimodal Biomedical Data

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了TAP-GPT，首个面向阿尔茨海默病（AD）预测的表格型大语言模型框架，通过领域适配TableGPT2实现少样本表格分类。该模型在四种ADNI衍生数据集上超越了传统机器学习基线和通用LLM，且无需数据填补即可处理缺失数据，同时生成符合AD生物学知识的可解释推理，为临床决策支持系统奠定了基础。

方法概述 / Method

TAP-GPT基于TableGPT2进行领域微调，采用表格提示（tabular prompts）而非纯文本输入来处理结构化生物标志物数据。模型整合了QT-PAD生物标志物及结构MRI、淀粉样蛋白PET、tau PET等多模态数据，通过特征选择缓解高维输入退化，并支持自反思迭代优化。

实验结果 / Results

在少样本设置下，TAP-GPT在双分类AD任务中 consistently 优于传统机器学习方法和其基础模型，与最先进的通用LLM相比具有竞争力。研究表明特征选择有效应对高维数据挑战，且模型在模拟和真实缺失数据场景下均保持稳定性能，自反思机制进一步提升了推理稳定性。

应用价值 / Applications

该研究为临床AD诊断提供了可解释、数据高效的AI工具，特别适用于样本稀缺、数据不完整的真实医疗环境。
TAP-GPT的结构化推理能力支持多智能体临床决策系统开发，其模态感知解释有助于医生理解预测依据，推动精准神经退行性疾病诊疗的落地应用。
