SLAM Adversarial Lab: An Extensible Framework for Visual SLAM Robustness Evaluation under Adverse Conditions

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了SLAM对抗性实验室（SAL），这是一个用于评估视觉SLAM系统在恶劣天气（如雾和雨）等对抗性条件下鲁棒性的模块化框架。
SAL将每种对抗性条件建模为扰动，能够将现有数据集转换为对抗性数据集，并使用易于理解的物理单位（如雾能见度米数）来定义严重程度等级。该框架采用可扩展架构，通过通用接口解耦数据集、扰动和SLAM算法，使用户无需重写集成代码即可添加新组件。此外，SAL还包含自动搜索程序，可确定导致SLAM系统失效的扰动严重程度阈值。

方法概述 / Method

SAL采用模块化设计，将数据集、扰动函数和SLAM算法分离为独立组件，通过统一接口实现灵活组合。框架支持多种扰动类型（包括天气、相机和视频传输扰动），并提供基于物理单位的严重程度量化机制。核心功能还包括二分搜索算法，用于自动定位SLAM系统从成功运行到失效的临界扰动强度。

实验结果 / Results

作者在三个数据集上对七种主流SLAM算法进行了全面评估，涵盖天气（雾、雨）、相机（模糊、噪声）和视频传输（压缩、丢包）三类扰动。实验揭示了不同SLAM系统在各种对抗条件下的失效阈值差异，为算法鲁棒性比较提供了量化基准。具体性能指标和对比分析详见论文原文。

应用价值 / Applications

SAL为自动驾驶、无人机导航等依赖视觉SLAM的机器人系统提供了标准化的鲁棒性测试平台，有助于在部署前识别算法的脆弱性边界。该框架的可扩展性使其能够快速适应新兴扰动类型和SLAM算法，支持学术界和工业界开展系统化的可靠性评估与安全验证工作。
