The Program Hypergraph: Multi-Way Relational Structure for Geometric Algebra, Spatial Compute, and Physics-Aware Compilation

arXiv Computer Architecture 2026-03-19

论文摘要 / Abstract

本文提出了程序超图（Program Hypergraph, PHG），作为程序语义图（PSG）的扩展，将二元边关系推广为任意元数的超边，以解决两类关键问题：空间数据流架构中的多路瓦片共置与路由约束，以及几何代数中分级多路乘积无法忠实表示为二元操作序列的问题。研究表明，克利福德代数的级（grade）可作为维度类型系统（DTS）阿贝尔群框架的自然维度轴，级推断能够自动推导几何乘积的稀疏性，从而消除克利福德代数神经网络的主要性能障碍。该工作在Fidelity编译框架中实现了PHG，使得几何正确性、内存放置、数值精度选择和硬件分区可从单一图结构联合推导。

方法概述 / Method

论文将现有的程序语义图从二元边关系扩展为超图形式，允许超边连接任意数量的节点，以自然表达多路关系。该方法将几何代数的级（grade）整合进维度类型系统的阿贝尔群框架，通过级推断实现几何乘积稀疏性的自动推导，并将网格拓扑的k-单形结构直接实例化为超边形式，在Fidelity编译框架中统一处理几何计算、内存管理和硬件映射。

实验结果 / Results

本文评估了现有的几何代数库生态系统，识别出当前系统普遍存在的类型理论缺口，并证明PHG能够在Fidelity编译框架中填补这一缺口。通过级推断机制，PHG消除了克利福德代数神经网络因缺乏手动特化而导致的性能障碍，实现了几何乘积稀疏性的自动优化，同时支持空间计算架构中的多路约束表达。

应用价值 / Applications

PHG可应用于异构计算编译器设计，特别是支持几何代数神经网络的高效编译与优化，以及空间数据流架构（如FPGA和专用AI加速器）的自动映射。该框架为物理感知编译提供了统一的中间表示，使得开发者能够在设计时获得关于几何正确性、内存布局和硬件分区的交互式反馈，适用于机器人学、计算机图形学和科学计算等领域的高性能计算系统开发。
