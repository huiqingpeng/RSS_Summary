PaAgent: Portrait-Aware Image Restoration Agent via Subjective-Objective Reinforcement Learning

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了PaAgent，一种基于主观-客观强化学习的人像感知图像修复智能体。该智能体通过构建自演化的人像知识库（portrait bank）并结合检索增强生成（RAG）技术，实现对图像修复工具的智能选择与调用，避免了传统方法中对最优工具的穷举搜索。此外，论文提出了一种融合图像质量分数和语义洞察的主观-客观强化学习策略，以增强复杂场景下的退化感知能力。在8个图像修复基准数据集上的实验表明，PaAgent在6种单一退化和8种混合退化场景中均表现出色。

方法概述 / Method

PaAgent的核心方法包括三个关键组件：（1）自演化的人像知识库，通过持续总结各类图像修复工具的特征、修复后的图像以及退化图像来不断丰富和更新；（2）基于RAG的工具选择机制，通过从知识库中检索相关洞察来为输入图像选择最优修复工具；（3）主观-客观强化学习策略，将图像质量客观评分与语义层面的主观洞察相结合生成奖励信号，以准确感知部分和非均匀退化。

实验结果 / Results

论文在涵盖6种单一退化和8种混合退化场景的8个图像修复基准数据集上进行了广泛实验。结果表明，PaAgent在复杂的图像修复任务中具有显著优势，能够有效处理多样化的退化类型。具体性能指标显示该方法在自动化工具选择和修复质量方面均优于现有基线方法。

应用价值 / Applications

PaAgent可广泛应用于需要自动化图像修复的实际场景，如老照片修复、监控视频增强、移动端图像处理以及专业摄影后期制作等。该方法通过智能体自主决策替代人工选择修复工具，大幅提升了图像修复的工作效率和智能化水平，为构建通用的图像修复系统提供了新思路。
