Towards Inclusive Communication: A Unified Framework for Generating Spoken Language from Sign, Lip, and Audio

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了首个能够统一处理手语、唇语和音频多种模态组合的口语文本生成框架，解决了现有音频中心系统排斥听障人士的问题。该研究实现了三个核心目标：设计模态无关的统一架构、探索唇语作为手语非手动线索的协同作用、达到或超越单任务专用模型的性能。实验表明，显式建模唇语作为独立模态能显著提升手语翻译性能，揭示了重要的语言学洞见。

方法概述 / Method

论文采用统一的模态无关架构，能够灵活处理异构输入（手语视频、唇语视频、音频及其组合）。该框架通过共享编码器处理不同模态，并特别将唇语建模为独立模态以捕捉手语中的非手动线索，实现跨模态的信息融合与互补。

实验结果 / Results

该统一框架在手语翻译（SLT）、视觉语音识别（VSR）、自动语音识别（ASR）和视听语音识别等任务上均达到或超越任务专用最先进的模型性能。分析表明，显式引入唇语模态能显著提升手语翻译效果，验证了唇语作为非手动线索在手语理解中的关键作用。

应用价值 / Applications

该研究为包容性通信提供了技术基础，可应用于听障人士的辅助交流系统、嘈杂环境下的鲁棒语音识别、多模态人机交互等场景。统一框架的灵活性使其能够适应不同用户的感知能力组合（如仅手语、手语+唇语、音频+视觉等），推动无障碍技术的普适化发展。
