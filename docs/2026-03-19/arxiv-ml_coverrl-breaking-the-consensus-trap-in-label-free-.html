CoVerRL: Breaking the Consensus Trap in Label-Free Reasoning via Generator-Verifier Co-Evolution

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文揭示了无标签强化学习中的"共识陷阱"（consensus trap）问题：当模型通过多数投票自举训练时，输出多样性崩溃会导致系统性错误被自信地强化。为此，作者提出CoVerRL框架，让单一模型交替扮演生成器和验证器角色，通过协同进化打破这一陷阱。实验表明，该方法在数学推理基准上比无标签基线提升4.7-5.9%，且自验证准确率从55%提升至85%以上。

方法概述 / Method

CoVerRL采用生成器-验证器协同进化机制：同一模型通过角色切换，先用多数投票训练验证器能力，再利用改进的验证器过滤伪标签中的自洽错误。这种交替优化形成良性循环，使奖励准确率在整个训练过程中保持高位，避免了传统自举方法中错误累积的恶性循环。

实验结果 / Results

在Qwen和Llama模型家族上的实验显示，CoVerRL在数学推理基准上显著优于无标签基线方法，性能提升达4.7-5.9%。关键的是，模型的自验证准确率从初始约55%大幅提升至超过85%，证明生成与验证两种能力实现了真正的协同进化而非此消彼长。

应用价值 / Applications

该研究为大规模语言模型在缺乏标注数据场景下的推理能力提升提供了可靠路径，特别适用于数学问题求解、代码生成等需要复杂推理的领域。其无需真实标签的特性降低了对昂贵人工标注的依赖，同时通过内置的自我验证机制增强了模型的可解释性和可靠性，对部署安全关键型AI应用具有重要价值。
