SkyReels-V4: Multi-modal Video-Audio Generation, Inpainting and Editing model

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

SkyReels-V4 是一个统一的多模态视频基础模型，能够联合生成视频与音频，并支持图像修复和编辑功能。该模型采用双流多模态扩散Transformer（MMDiT）架构，视频分支与音频分支共享基于多模态大语言模型（MLLM）的文本编码器，实现时间同步的音视频生成。该模型支持文本、图像、视频片段、掩码和音频参考等多种输入模态，可生成分辨率高达1080p、帧率32 FPS、时长15秒的电影级高质量视频。

方法概述 / Method

SkyReels-V4 的核心架构为双流MMDiT，其中视频分支通过通道拼接公式统一了图像生成视频、视频扩展、视频编辑等多种修复风格任务，并结合MLLM的多模态指令跟随能力与上下文学习实现精细视觉引导；音频分支则利用音频参考引导声音生成。为提升计算效率，模型采用联合生成低分辨率全序列与高分辨率关键帧的策略，再通过专用的超分辨率和帧插值模型完成最终输出。

实验结果 / Results

论文未在提供的摘要中详细列出具体的定量实验结果或性能指标对比，但强调了SkyReels-V4在1080p高分辨率、15秒长时长生成条件下的高效性与高质量，能够实现多镜头、电影级视频生成与同步音频输出，是首个同时支持多模态输入、联合音视频生成、并统一处理生成/修复/编辑任务的视频基础模型。

应用价值 / Applications

SkyReels-V4 可广泛应用于影视内容创作、广告制作、短视频生成等专业视频生产领域，大幅降低高质量音视频内容的制作门槛与成本。其统一的多模态交互界面支持灵活的创意控制，适用于需要精细视觉引导和音频同步的复杂场景，如电影预可视化、虚拟制片、个性化内容定制等。
