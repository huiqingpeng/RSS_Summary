Personalized Fall Detection by Balancing Data with Selective Feedback Using Contrastive Learning

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文针对个性化跌倒检测模型中真实跌倒数据稀缺、非跌倒反馈样本占主导导致的数据不平衡问题，提出了结合半监督聚类与对比学习的个性化框架，通过识别和平衡最具信息量的用户反馈样本来提升模型性能。实验表明，从头训练（TFS）策略效果最佳，较基线提升25%，少样本学习（FSL）次之，提升7%，验证了选择性个性化在实际部署中的有效性。

方法概述 / Method

该研究采用半监督聚类与对比学习相结合的方法，从用户反馈中智能筛选高信息量的样本以平衡跌倒与非跌倒数据；并在三种再训练策略（从头训练TFS、迁移学习TL、少样本学习FSL）下评估框架的适应性。

实验结果 / Results

十名参与者的实时实验显示，TFS策略性能最优，较基线提升高达25%；FSL策略位列第二，提升7%；结果表明该框架能有效改善数据不平衡问题，且在不同学习范式下均展现出良好的适应能力。

应用价值 / Applications

该研究适用于老年人居家监护、医疗机构实时监测等场景，通过个性化模型减少对大量跌倒样本的依赖，降低部署成本；其选择性反馈机制可推广至其他数据稀缺的异常检测任务，如医疗诊断预警和工业安全监测。
