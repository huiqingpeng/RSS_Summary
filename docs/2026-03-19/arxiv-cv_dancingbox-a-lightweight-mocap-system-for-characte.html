DancingBox: A Lightweight MoCap System for Character Animation from Physical Proxies

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

DancingBox 提出了一种轻量级的基于视觉的动作捕捉系统，通过将日常物品作为"数字木偶"的物理代理，使新手用户能够使用单摄像头创建3D角色动画。该系统将粗略的代理物体运动通过边界框表示输入生成式运动模型，并结合大规模数据集学习的人体运动先验， refine 为逼真的角色动画。用户研究表明，该系统支持从毛绒玩具到香蕉等多样化代理物体的直观创意动画制作，显著降低了动画创作的入门门槛。

方法概述 / Method

DancingBox 采用单摄像头跟踪用户操控的日常物品（物理代理），提取其边界框运动轨迹作为粗糙输入信号。为克服缺乏配对代理-动画训练数据的难题，研究者通过将现有动作捕捉序列转换为代理表示来合成训练数据对，进而训练一个以边界框为条件、融合人体运动先验的生成式运动模型。

实验结果 / Results

用户研究验证了 DancingBox 的可用性和创意表达能力，证明非专业用户能够利用多样化的日常物品（如毛绒玩具、香蕉等）直观地进行角色动画创作。系统成功将粗糙的物理代理运动转换为逼真的角色动画，在保持轻量级单摄像头设置的同时实现了专业级的动画输出质量。

应用价值 / Applications

DancingBox 可广泛应用于独立游戏开发、社交媒体内容创作、教育娱乐等场景，使缺乏专业动画技能的普通用户能够快速制作3D角色动画。该系统消除了对昂贵动作捕捉设备和专业演员的需求，为个人创作者、小型工作室和教育领域提供了低成本的动画制作解决方案。
