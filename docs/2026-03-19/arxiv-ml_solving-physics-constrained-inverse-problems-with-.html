论文摘要 / Abstract

本研究提出了一种基于条件流匹配（conditional flow matching）的框架，用于求解物理约束的贝叶斯反问题。该方法仅需从推断变量与测量值的联合分布中采样，无需显式计算先验和似然密度，即可训练神经网络学习概率流常微分方程的速度场，直接将样本从源分布传输到以观测为条件的后验分布。研究还揭示了有限训练数据下过度训练会导致的退化行为（包括方差崩溃和选择性记忆现象），并通过早停策略有效缓解，最终在多个物理反问题中验证了方法对复杂多模态后验分布的准确捕捉能力。

方法概述 / Method

论文采用条件流匹配算法，通过训练神经网络学习速度场来构建概率流ODE，实现从源分布（如高斯分布或数据驱动先验）到条件后验分布的确定性映射。该黑箱框架无需可微前向模型或特定噪声假设，适用于非线性、高维及不可微的物理前向模型。

实验结果 / Results

数值实验验证了过度训练确实会导致方差崩溃和选择性记忆现象（生成样本聚集在相似观测的训练数据点附近），而基于测试损失监控的标准早停准则可有效抑制此类退化。在多个物理反问题中，条件流匹配方法在保持计算效率的同时，准确捕捉了复杂的多模态后验分布，且不同源分布选择（高斯先验与数据驱动先验）对性能有显著影响。

应用价值 / Applications

该方法可广泛应用于科学计算和工程中的物理约束反问题，如医学成像、地球物理反演、材料参数识别等场景，其黑箱特性使其能灵活适配复杂的物理模拟器。相比传统方法，该框架在避免繁琐的似然计算的同时，提供了对后验不确定性的完整刻画，具有重要的实际应用价值。
