论文摘要 / Abstract

本文提出了一种物理信息生成对抗网络（GAN）框架，将量子资源态生成重新表述为逆向设计任务。通过将特定任务的效用函数嵌入训练过程，模型学习生成针对量子隐形传态和纠缠广播优化的有效双量子比特态。研究表明，与仅依赖损失函数的方法相比，显式强制施加厄米性、迹为一和正定性等结构约束可获得更高的保真度和训练稳定性，对Werner类和Bell对角态的理论资源边界复现保真度超过98%，为约束驱动的量子态自动发现建立了轻量且有效的方法。

方法概述 / Method

该研究采用物理信息GAN架构，通过生成器直接输出密度矩阵元素，并在网络结构中嵌入量子力学约束（厄米性、迹为一、正定性）以确保物理有效性。训练过程中结合对抗损失与任务特定的效用函数（如隐形传态保真度或纠缠广播容量），实现资源态的定向优化生成。

实验结果 / Results

实验对比了基于分解的生成架构与直接生成架构，证实结构约束强制方法在保真度和训练稳定性上均优于纯损失约束方法。模型成功复现了Werner类和Bell对角态的理论资源边界，生成态保真度达到98%以上，验证了对抗学习在量子资源态发现中的有效性。

应用价值 / Applications

该框架为量子信息处理应用中的定制化量子资源自动设计提供了可扩展的基础，可直接应用于量子隐形传态和纠缠广播等任务，并有望支持高效量子网络的自动化设计与优化。
