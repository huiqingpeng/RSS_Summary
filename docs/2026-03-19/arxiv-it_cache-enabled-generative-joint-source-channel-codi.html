论文摘要 / Abstract

本文提出了一种面向演进式语义通信的缓存增强生成式联合信源信道编码框架（CAGI-JSCC），旨在解决现有基于学习的语义通信方法端到端训练成本高、灵活性差的问题。该框架利用预训练的SemanticStyleGAN模型实现免训练语义通信，并通过将无线信道特性显式融入GAN反演过程来适应动态信道环境。此外，作者设计了缓存增强动态码本（CDC）机制，通过在收发两端缓存解耦的语义组件来复用历史传输内容，随着经验积累持续降低冗余传输。实验表明，该系统在图像传输任务中实现了1/224的平均带宽压缩比，单张图像最低可达1/1024，显著优于基线方法的1/128。

方法概述 / Method

CAGI-JSCC框架采用通道感知的GAN反演技术，将无线信道特性（如衰落、噪声）嵌入到预训练SemanticStyleGAN的反演优化过程中，使语义编码天然适应信道变化而无需重新训练。核心创新在于CDC机制：利用StyleGAN的解耦潜在空间特性，将图像分解为可复用的语义组件（如风格、结构），并在收发端建立同步缓存，新传输仅需发送与缓存差异的增量信息。

实验结果 / Results

图像传输实验显示，CAGI-JSCC在保持与基线相当感知质量的同时，平均带宽压缩比（BCR）达到1/224，最优情况下单张图像BCR低至1/1024，相比传统方法（BCR=1/128）实现约2-8倍的压缩效率提升。随着缓存语义组件的积累，系统冗余传输持续减少，验证了"演进式"通信的有效性。

应用价值 / Applications

该研究适用于带宽受限的动态无线环境（如卫星通信、边缘物联网），可显著降低重复内容传输开销。其免训练特性使模型能快速部署到新场景，而语义级缓存机制特别适合视频监控、智能巡检等具有时序相关性和内容重复性的应用，为6G语义通信网络的绿色高效传输提供新范式。
