论文摘要 / Abstract

本文提出了RGD-Blast，一种用于高保真、长期爆炸波预测的稳健且可泛化的深度代理模型，解决了传统数值模拟计算成本高昂以及现有机器学习模型在复杂城市布局和分布外场景中精度下降、自回归预测误差累积的问题。该模型通过多尺度模块捕捉全局流动模式与局部边界相互作用，并引入动态-静态特征耦合机制融合时变压力场与静态源点和布局特征，显著提升了分布外泛化能力。实验表明，RGD-Blast在实现两个数量级加速的同时保持了可比精度，在未见过建筑布局上的280个连续时间步预测中平均RMSE低于0.01、R²超过0.89。

方法概述 / Method

RGD-Blast采用多尺度模块（multi-scale module）同时捕捉爆炸波传播的全局流动模式和局部边界相互作用，以缓解自回归预测过程中的误差累积问题；同时设计了动态-静态特征耦合机制（dynamic-static feature coupling mechanism），将时变压力场与静态的爆炸源位置及城市布局特征进行融合，从而增强模型对分布外场景的泛化能力。

实验结果 / Results

RGD-Blast相比传统数值方法实现了两个数量级（two-order-of-magnitude）的推理加速，同时保持了相当的预测精度；在未见过的建筑布局泛化测试中，模型在280个连续时间步上的平均RMSE低于0.01，R²超过0.89；此外，在不同爆炸源位置和炸药当量条件下的额外评估进一步验证了模型的强泛化性能。
