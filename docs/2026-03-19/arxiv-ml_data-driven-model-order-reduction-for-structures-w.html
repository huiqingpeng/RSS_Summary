论文摘要 / Abstract

本文提出了一种基于动态模态分解（DMD）的数据驱动模型降阶方法，用于处理分段线性非线性系统的动力学预测问题。该方法利用系统脉冲响应获取状态变量快照，提取动态模态构建投影基向量，并通过Galerkin投影将原系统运动方程降维至低维子空间。研究通过悬臂梁弹性碰撞和脱粘层合板两个典型算例验证了方法的有效性，表明所构建的降阶模型能准确预测非线性受迫响应。

方法概述 / Method

核心方法采用动态模态分解（DMD）结合Galerkin投影：首先采集系统脉冲响应数据生成状态快照矩阵，通过DMD算法提取主导动态模态作为降阶基；随后将原高维运动方程投影至DMD模态张成的子空间，必要时补充经典约束模态以精确捕捉接触非线性特性，最终获得自由度显著减少的降阶模型。

实验结果 / Results

研究针对两类分段线性系统——端部带弹性限位的悬臂梁以及含局部脱粘的粘接板组件——构建了降阶模型（ROMs），并开展谐波激励下的非线性受迫响应分析。结果表明：纯DMD模态或DMD模态与约束模态组合所构建的ROMs均能复现原系统的非线性动力学行为，在大幅降低计算成本的同时保持了较高的预测精度。

应用价值 / Applications

该方法适用于含接触、碰撞、间隙等分段线性非线性特征的工程结构动力学分析与实时仿真，如航空发动机叶片-机匣碰摩、机械关节间隙振动、复合材料脱粘损伤监测等场景。数据驱动的特性使其可整合实验测量数据，为复杂非线性系统的数字孪生、健康管理和在线控制提供高效可靠的动力学代理模型。
