论文摘要 / Abstract

本文提出了一种特征保留的人脸匿名化框架，在保护隐私的同时保留面部表情、光照方向和肤色等关键属性。该方法通过扩展DeepPrivacy，引入密集面部关键点以更好地保留表情，并采用轻量级后处理模块确保光照和肤色的一致性。在CelebA-HQ数据集上的实验表明，该方法在表情保真度、光照一致性和肤色保留方面显著优于现有基线方法，同时保持良好的图像真实感和抗重识别能力。

方法概述 / Method

该方法基于DeepPrivacy进行扩展，首先利用密集面部关键点（dense facial landmarks）来指导生成过程，以更好地保留面部表情信息；其次引入轻量级后处理模块，对生成结果进行光照方向和皮肤颜色的调整，确保光度一致性。此外，论文还设计了专门的评估指标来量化表情保真度、光照一致性和颜色保留程度。

实验结果 / Results

在CelebA-HQ数据集上的实验表明，该方法生成的匿名化人脸在图像真实感、姿态准确性方面表现良好，同时在表情保真度、光照一致性和肤色保留方面显著优于现有最先进基线方法。实验结果验证了特征感知匿名化在提升隐私保护数据可用性方面的有效性。

应用价值 / Applications

该研究适用于需要保护个人隐私但同时依赖面部特征分析的场景，如社交媒体图像分享、大规模人脸数据集构建、医学图像分析（如皮肤病诊断）以及情感计算等应用。通过保留光照和肤色信息，该方法还可支持重光照（relighting）和颜色恒常性（color constancy）等计算机视觉任务，为更公平、可信的隐私保护面部数据提供了技术基础。
