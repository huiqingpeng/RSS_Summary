PCA-Seg: Revisiting Cost Aggregation for Open-Vocabulary Semantic and Part Segmentation

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文针对开放词汇语义和部分分割（OSPS）任务中现有方法存在的类别级语义与空间上下文知识干扰问题，提出了一种简单有效的并行成本聚合范式PCA-Seg。该范式通过专家驱动的感知学习（EPL）模块和特征正交化解耦（FOD）策略，使模型能够从成本体中捕获更丰富的视觉-语言对齐信息。在八个基准数据集上的大量实验表明，PCA-Seg的每个并行块仅增加0.35M参数，却达到了最先进的OSPS性能。

方法概述 / Method

PCA-Seg采用并行结构替代传统的串行空间-类别聚合方式，核心组件包括：（1）专家驱动的感知学习（EPL）模块，通过多专家解析器从多个视角提取互补特征，并利用系数映射器自适应学习像素级权重以融合特征；（2）特征正交化解耦（FOD）策略，通过正交化语义流和上下文流来减少两者之间的冗余，促进多样化知识学习。

实验结果 / Results

PCA-Seg在八个基准数据集上取得了最先进的性能，同时保持了极高的参数效率——每个并行块仅引入0.35M额外参数。实验验证了并行聚合结构相比串行结构能有效缓解知识干扰，且EPL模块和FOD策略的协同作用显著提升了视觉-语言对齐质量。

应用价值 / Applications

该研究可广泛应用于需要灵活理解新类别和物体部件的计算机视觉场景，如自动驾驶中的可行驶区域与障碍物识别、机器人视觉中的物体操作与抓取、以及交互式图像编辑和内容创作工具。其开放词汇特性使其能够快速适应新领域而无需重新训练，具有重要的实用价值。
