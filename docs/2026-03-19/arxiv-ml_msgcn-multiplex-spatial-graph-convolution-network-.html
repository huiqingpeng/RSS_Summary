论文摘要 / Abstract

本文提出了多层空间图卷积网络（MSGCN），用于解决多层网络中的层间链接权重预测问题。该方法将空间图卷积推广到多层网络，能够捕捉跨多层节点的几何结构信息。实验结果表明，MSGCN在多种网络结构中展现出稳健、准确且可泛化的链接权重预测性能，并在伦敦交通网络的实际应用中成功预测了乘客流量。

方法概述 / Method

MSGCN通过空间嵌入方式整合多层网络信息，将传统空间图卷积扩展到多层网络场景。该方法能够显式建模跨层节点的几何关系，从而有效捕获层间链接的复杂模式。

实验结果 / Results

在具有已知层间链接信息的多组数据集上，MSGCN表现出鲁棒且准确的链接权重预测能力，且具有良好的跨网络结构泛化性。在伦敦交通网络案例中，该方法成功预测了非直接连接站点间的乘客流量（即层间链接权重）。

应用价值 / Applications

MSGCN可应用于交通网络流量预测、多层社交网络关系强度估计、以及基础设施网络负载预测等场景。该方法为理解复杂多层系统中跨层交互强度提供了有效工具，特别适用于需要量化层间连接强度的实际决策支持系统。
