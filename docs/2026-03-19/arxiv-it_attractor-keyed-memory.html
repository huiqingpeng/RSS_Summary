Attractor-Keyed Memory

arXiv Information Theory 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出"吸引子键控存储"（Attractor-Keyed Memory）概念，利用物理选择器（如激光模式选择、伊辛机基态收敛等）在决策瞬间产生的高维特征信号（全场振幅、多模干涉图样等）直接编码信息载荷，将选择与存储访问合并为单一事件。该方法通过单次SVD分解校准即可认证设备能力并限定最坏情况误差，消除了传统稀疏路由架构中主导的延迟和能耗开销。研究完整推导了误差分解公式，给出了伊辛机选择器的具体构造，并在三种测量模态下验证了预测标度律，同时提出了可证伪的四步实验协议以指导首次硬件验证。

方法概述 / Method

核心方法基于两个关键假设：特征信号在多次试验中可重复（stereotyped）且不同路由间线性无关。通过测量设备响应构建字典矩阵Φ，执行单次奇异值分解（SVD）以认证最小奇异值σ_min(Φ)所决定的解码保真度，并建立线性解码器将高维物理特征直接映射至任意目标载荷，绕过传统"选择-索引-取数"的多级流程。

实验结果 / Results

研究在合成散斑特征模拟中验证了理论预测的三类标度关系：解码误差随字典条件数恶化、路由可靠性由信噪比余量Δ/T_eff控制、以及两种误差通道的独立可诊断性。
SVD预处理被证明可在任务选定前即完成设备能力认证与最坏误差边界确定，为伊辛机选择器提供了具体构造参数。

应用价值 / Applications

该技术可应用于光学计算、神经形态计算及稀疏路由硬件（如光子网络、量子退火系统），通过消除存储访问延迟突破"内存墙"瓶颈。潜在场景包括超高速模式识别、低能耗决策系统，以及需要纳秒级响应的物理层信息处理；但其实际部署取决于真实器件能否满足特征可重复性这一核心开放问题。
