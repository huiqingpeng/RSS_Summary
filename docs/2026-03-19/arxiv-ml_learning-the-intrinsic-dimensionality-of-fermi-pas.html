Learning the Intrinsic Dimensionality of Fermi-Pasta-Ulam-Tsingou Trajectories: A Nonlinear Approach using a Deep Autoencoder Model

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究利用深度自编码器（DAE）分析Fermi-Pasta-Ulam-Tsingou（FPUT）β模型在弱非线性区域（β ≲ 1）的高维轨迹数据，发现400万数据点的轨迹嵌入在64维相空间中，实际位于一个二维的非线性黎曼流形上。研究还发现当β = 1.1时，内在维度增至3维，这与β模型特有的对称性破缺现象（偶数波数k = 2, 4的额外能量模式被激发）相吻合，而传统的PCA方法无法检测到这一关键转变。

方法概述 / Method

作者采用深度自编码器（DAE）作为非线性降维工具，通过学习数据在低维潜在空间的紧凑表示来推断FPUT系统轨迹的内在维度（ID）。该方法与主成分分析（PCA）结合参与率（PR）方法的线性方法进行对比，以展示非线性方法在捕捉复杂动力学结构方面的优势。

实验结果 / Results

DAE揭示FPUT轨迹在弱非线性区域（β ≲ 1）的内在维度为m* = 2，而PCA仅能提供ID的上界估计。当非线性参数β = 1.1时，DAE检测到内在维度跃升至m* = 3，精确对应系统发生对称性破缺的临界点；相比之下，PCA完全无法识别这一重要的动力学相变。

应用价值 / Applications

该研究为复杂多体系统的降维和本征动力学分析提供了新的非线性工具，可推广至其他具有能量递归或模式激发的非线性晶格系统。
DAE方法在检测传统线性方法无法识别的对称性破缺和相变现象方面具有独特优势，对统计力学、非线性动力学及机器学习在物理系统中的应用具有重要参考价值。
