Robust estimation of heterogeneous treatment effects in randomized trials leveraging external data

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文提出了一种名为QR-learner的模型无关学习方法，用于在随机对照试验中利用外部数据（来自其他试验或观察性研究）估计条件平均处理效应（CATE），以解决传统试验因统计功效不足而难以发现个体层面治疗效应异质性的问题。该方法具有稳健性：既能相对于仅使用试验数据的CATE学习器降低均方误差，又能保证即使外部数据与试验人群不一致时仍能恢复真实的CATE。此外，作者还提出了一种将QR-learner与试验专用CATE学习器相结合的集成程序，该程序在渐近意义上能够匹配或超越两个组成学习器的均方误差表现。

方法概述 / Method

QR-learner采用分位数回归框架，通过构建辅助模型来利用外部数据的信息，同时保持对试验目标人群的一致性。该方法无需对外部数据与试验数据之间的分布关系做出强假设，通过设计特定的损失函数和加权机制，使得估计量在存在外部数据偏差时仍具有稳健性。作者进一步提出了一个自适应组合策略，通过数据驱动的方式动态调整QR-learner与试验专用学习器的权重，以优化最终的估计性能。

实验结果 / Results

模拟研究表明，QR-learner在多种设定下均能有效降低CATE估计的均方误差，尤其在样本量有限或外部数据存在一定程度偏倚时表现突出。所提出的组合程序在渐近理论中被证明具有 oracle 性质，即能够自动选择更优的学习器或取得两者的加权优势。在真实数据集的应用中，该方法在CATE估计精度和检测异质性效应的统计功效方面均显示出显著改进。

应用价值 / Applications

该研究对精准医疗和个性化决策具有重要价值，可帮助研究人员在保持随机试验内部有效性的同时，充分利用历史数据或其他来源的外部信息，提高个体化治疗效果预测的准确性。该方法可广泛应用于药物临床试验、医疗政策评估以及任何需要识别治疗效应异质性以指导个性化干预的场景，有助于克服传统试验样本量受限的瓶颈，加速精准医学的实现。
