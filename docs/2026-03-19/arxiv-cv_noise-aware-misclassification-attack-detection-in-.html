论文摘要 / Abstract

本文针对边缘AI场景中的协作式DNN推理安全问题，提出了一种噪声感知的误分类攻击检测框架。该框架利用变分自编码器（VAE）捕获对抗性操纵引起的偏差，并引入鲁棒的噪声感知特征来刻画环境噪声的典型行为，从而在提升检测准确率的同时降低误报率。实验表明，该方法在真实噪声条件下对主流目标分类DNN具有鲁棒性（AUROC最高达90%），但也揭示了特征相似性和高噪声水平带来的局限性。

方法概述 / Method

论文采用半灰盒（semi-gray-box）检测范式，结合变分自编码器构建异常检测框架，通过建模正常数据的潜在分布来识别异常输入。核心创新在于设计了噪声感知特征提取机制，将环境噪声的统计特性融入检测过程，使模型能够区分良性噪声扰动与恶意对抗扰动。

实验结果 / Results

评估结果显示，所提检测方法在多种DNN配置下均表现出良好的鲁棒性，AUROC指标最高可达90%；该方法在真实噪声环境中有效平衡了检测准确率与误报率。然而，当对抗特征与正常特征高度相似或环境噪声水平过高时，检测性能会出现明显下降。

应用价值 / Applications

该研究可直接应用于资源受限边缘设备的安全推理场景，如智能安防监控、自动驾驶感知系统和工业视觉检测等，为边缘AI服务提供实时攻击检测能力。通过在不增加终端计算负担的前提下保障推理安全性，该框架有助于推动可信边缘计算的规模化部署。
