TechImage-Bench: Rubric-Based Evaluation for Technical Image Generation

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了技术图像生成任务，即模型需根据详细描述合成信息密集、科学精确的插图，而非仅生成视觉可信的图片。作者提出了TechImage-Bench，首个基于评分标准的基准测试，涵盖生物示意图、工程/专利图纸和通用技术插图三类任务，包含654张来自真实教材和技术报告的图片。研究发现，尽管现有文本到图像模型在开放域表现强劲，但在科学精细度上存在显著差距，最佳基线模型仅达到0.801的评分标准准确率和0.576的标准得分；同时，作者证明该评分标准可提供可操作的监督信号，通过迭代编辑将强生成器的准确率从0.660提升至0.865。

方法概述 / Method

本文采用大型多模态模型（LMM）从参考图像及其周围文本中自动提取层次化评分标准，将正确性分解为6,076条标准和44,131个二元检查项。评估阶段使用基于LMM的自动评判器，配合原则性惩罚机制将子问题结果聚合为可解释的标准得分。此外，作者设计了反馈驱动的迭代编辑流程，将失败的检查项反馈给编辑模型进行图像精修。

实验结果 / Results

在TechImage-Bench上的基准测试显示，现有文本到图像模型在技术图像生成方面表现不佳：最佳基线模型总体评分标准准确率仅为0.801，标准得分仅0.576，生物示意图类别表现最差（0.660准确率/0.382标准得分）。通过利用评分标准进行迭代编辑，强生成器的性能显著提升——评分标准准确率从0.660提高到0.865，标准得分从0.382提高到0.697，证明了该评估框架不仅能诊断问题，还能有效指导模型改进。

应用价值 / Applications

TechImage-Bench为科学教育、工程设计和专利文档等领域提供了严格的技术图像生成评估工具，可确保AI生成的教材插图、工程图纸等符合专业规范。该基准的评分标准可作为可扩展的监督信号，直接用于提升生成模型的规范忠实度，推动自动化学术出版、技术文档生成等实际应用的发展，同时减少人工审核成本。
