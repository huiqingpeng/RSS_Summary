论文摘要 / Abstract

本文研究了多臂场景下的序贯假设检验问题，其中统计学家在每个时间步需要从多个数据源（臂）中选择一个获取数据。针对复合全局零假设（所有臂均为零假设）和复合备择假设（至少一个臂非零）的检验问题，作者提出了一种最优性准则：即使多个臂非零，所设计的e-过程和序贯检验也应达到具有"神谕"知识（知晓哪个臂产生最强证据）时的性能水平。研究将log-最优性和期望拒绝时间最优性推广到多臂情形，并获得了匹配的下界和上界。

方法概述 / Method

论文采用基于赌博（betting）的序贯检验框架，核心方法是设计一种改进的上置信界（UCB-like）算法来处理不可观测但可充分估计的奖励。在技术层面，作者推导了关于Kelly意义下最优财富增长率的非渐近集中不等式，作为算法设计和最优性分析的关键工具。

实验结果 / Results

论文主要贡献在于理论层面的最优性结果：建立了多臂情形下log-最优性和期望拒绝时间最优性的形式化定义，并证明了所提算法能够达到与具有臂选择先验知识的神谕策略相匹配的性能界限，上下界相吻合。

应用价值 / Applications

该研究在药物临床试验中具有重要应用价值，例如需要同时测试多种治疗剂量（多臂）并快速识别有效剂量的场景。此外，任何涉及多源数据动态选择的高效假设检验问题，如A/B测试的扩展、在线实验设计和自适应传感等，都可受益于这种具有最优保证的序贯检验方法。
