论文摘要 / Abstract

本文研究了从不完整图信号中联合推断时变网络拓扑并插补缺失数据的难题。作者提出了一种统一的非凸优化框架，能够同时恢复一系列图拉普拉斯矩阵并重建未观测的信号条目。该集成方法实现了图域与信号域之间的双向信息流，在高缺失数据场景下展现出卓越的鲁棒性，并通过fused-lasso型正则化捕捉真实的网络动态特性。

方法概述 / Method

论文采用融合图学习与时变图信号补全的联合优化框架，引入fused-lasso型正则化项惩罚拉普拉斯序列的连续大变化以促进时间平滑性。针对该非凸优化问题，设计了基于交替方向乘子法（ADMM）的高效求解算法，利用问题结构特性为图和信号子问题提供闭式解。

实验结果 / Results

数值实验验证了该方法在收敛速度和图学习与信号恢复的联合精度方面均显著优于最先进的基线方法。理论分析证明了ADMM方案收敛到稳定点，并导出了图估计量的非渐近统计保证，给出了关于样本量、信号平滑度和图内在时变性的高概率误差界。

应用价值 / Applications
