Fine-Grained Uncertainty Quantification via Collisions

arXiv Machine Learning 2026-03-19

论文摘要 / Abstract

本文提出了一种基于类别碰撞（class collisions）的偶然不确定性量化新指标，通过定义碰撞矩阵（collision matrix）来精细刻画分类问题中每对类别之间的固有区分难度。作者建立了一套从独热标签数据估计碰撞矩阵的完整方法，包括训练成对对比模型、估计行Gramian矩阵并唯一恢复碰撞矩阵，并证明了该方法可用于估计任意点的后验类别概率分布。

方法概述 / Method

论文采用三阶段方法估计碰撞矩阵：首先训练一个成对对比模型来判断两个输入是否属于同一类别；然后利用该模型估计碰撞矩阵的行Gramian矩阵 G=SS^T；最后在合理假设下，从G唯一恢复出非负矩阵S。这一恢复结果本身是关于非负矩阵理论的新贡献。

实验结果 / Results

实验验证了碰撞矩阵估计方法和类别后验分布估计的有效性，在多个数据集上展示了所提方法的实际性能。结果表明，基于碰撞矩阵的估计能够准确捕捉类别间的混淆程度，并与贝叶斯错误率（BER）等现有不确定性量化方法建立理论联系。

应用价值 / Applications

碰撞矩阵可应用于需要精细不确定性量化的场景，如医疗诊断中区分易混淆疾病、自动驾驶中识别相似交通标志、以及模型部署时的风险决策。该方法仅需标准分类标签即可估计，无需额外标注成本，具有广泛的实用价值。
