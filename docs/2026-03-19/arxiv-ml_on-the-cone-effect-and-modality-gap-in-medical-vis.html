On the Cone Effect and Modality Gap in Medical Vision-Language Embeddings

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究探讨了视觉-语言模型（VLMs）中的"锥形效应"和"模态差距"现象在医学领域的实际影响。作者提出了一种轻量级的后处理机制，通过单一超参数λ在保持预训练编码器冻结的同时控制跨模态分离程度。实验结果表明，适度减小模态差距能提升下游多模态任务性能，医学数据集对此更为敏感，但完全消除差距并非最优，任务相关的中间分离度效果最佳。

方法概述 / Method

论文设计了一种可微分的后处理方法，在推理阶段通过可学习的缩放参数λ调节图像和文本嵌入的相对位置，从而连续控制模态差距大小。该方法无需重新训练预训练VLM编码器，即可系统分析不同模态分离程度对下游监督多模态学习的影响。

实验结果 / Results

在通用模型（CLIP、SigLIP）和医学专用模型（BioMedCLIP、MedSigLIP）上的评估显示，减小过度的模态差距能一致提升性能，且医学数据集对差距调节的敏感度显著高于自然图像数据集。关键发现是存在任务依赖的最优λ值，完全坍缩模态差距（λ=0）并非 universally 最优，中等程度的分离往往取得最佳效果。

应用价值 / Applications

该研究为医学多模态AI系统提供了实用的优化策略：通过简单调节单一超参数即可提升诊断报告生成、医学影像-文本检索等任务的性能。研究将模态差距重新定位为可调谐的表示属性而非需要最小化的缺陷，为医学VLM的部署和微调提供了理论指导和实用工具。
