论文摘要 / Abstract

本文提出了**可证明安全模型更新**的框架，旨在解决安全关键环境中机器学习模型持续更新时的安全性验证问题。作者将问题形式化为计算参数空间中**最大局部不变域（LID）**——一个所有参数点均满足给定规范的连通区域。通过将精确LID计算松弛为参数化抽象域（正交体、zonotope），作者得到了可处理的原始-对偶形式，从而实现与数据和算法无关的高效安全认证。实验表明，该方法在持续学习和基础模型微调基准上，既能匹敌或超越启发式基线防止遗忘的效果，又能提供形式化安全保证。

核心方法是将安全模型更新问题转化为**最大局部不变域（LID）计算**，并采用**参数化抽象域松弛**（正交体或zonotope）使问题可处理。该方法通过**原始-对偶优化框架**高效认证参数更新，将任意更新投影到安全域内；同时支持计算多个近似最优LID、引入正则化偏置以及利用前瞻数据缓冲区。
