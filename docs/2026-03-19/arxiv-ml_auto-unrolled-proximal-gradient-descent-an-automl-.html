Auto-Unrolled Proximal Gradient Descent: An AutoML Approach to Interpretable Waveform Optimization

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本研究将自动化机器学习（AutoML）与基于模型的深度展开（DU）相结合，用于无线波束成形和波形优化。作者将迭代近端梯度下降（PGD）算法转换为深度神经网络，使每层参数可学习而非预先设定，并引入混合层实现可学习的线性梯度变换。所提出的自动展开PGD（Auto-PGD）仅需5个展开层和100个训练样本即可达到传统200次迭代PGD求解器98.8%的频谱效率，显著降低了训练数据和推理成本，同时保持高可解释性。

方法概述 / Method

该方法将PGD算法展开为深度神经网络，每层参数通过数据驱动方式学习；通过引入混合层在投影前执行可学习的线性梯度变换增强架构；采用AutoGluon结合树结构Parzen估计器（TPE）进行超参数优化，搜索空间涵盖网络深度、步长初始化、优化器、学习率调度器、层类型和梯度后激活函数。

实验结果 / Results

Auto-PGD仅用5个展开层和100个训练样本即达到传统200次迭代PGD求解器98.8%的频谱效率；解决了梯度归一化问题以确保训练和评估阶段性能一致；通过逐层和速率日志实现了模型透明性。

应用价值 / Applications

该研究适用于无线通信系统中的实时波束成形和波形优化，特别适合训练数据稀缺或计算资源受限的场景；相比传统黑盒神经网络，该方法兼具模型驱动方法的可解释性和数据驱动方法的性能优势，为6G等下一代无线系统的低延迟、高效信号处理提供了实用解决方案。
