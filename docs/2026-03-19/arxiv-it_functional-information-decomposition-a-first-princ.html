Functional Information Decomposition: A First-Principles Approach to Analyzing Functional Relationships

arXiv Information Theory 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出**功能信息分解（Functional Information Decomposition, FID）**框架，旨在解决多变量系统中输入如何共同决定输出的核心难题。现有方法依赖观测概率分布，易将系统内在功能逻辑与有限数据的统计伪影混淆，导致不同系统产生相同观测而难以区分真正的高阶交互。
FID通过定义相对于完整输入-输出映射的信息组件，在完全指定映射时提供唯一的独立与协同贡献分解；在仅有部分观测时，通过采样兼容函数刻画所有一致分解的空间，使推断极限显式化。

方法概述 / Method

FID采用**第一性原理方法**，将信息分解建立在系统的完整功能映射（输入-输出关系）之上，而非仅依赖观测数据的统计分布。该方法引入**几何视角**来阐明信息组件的结构起源，并通过**函数采样技术**在部分观测条件下探索所有与数据一致的可能分解，从而量化推断的不确定性。

实验结果 / Results

作者在三个跨学科场景验证FID：经典逻辑函数（如XOR）、康威生命游戏（Conway's Game of Life）以及基于基因表达的癌症药物反应预测。实验表明FID能够有效分离功能架构与观测分布，揭示传统方法无法捕捉的真实高阶交互；并提供开源实现以促进应用。

应用价值 / Applications

FID为复杂系统分析提供原则性基础，适用于**完全观测系统**（如已知规则的细胞自动机、布尔网络）和**部分观测系统**（如生物医学中基因-药物响应关系）。其分离功能逻辑与统计噪声的能力，对神经科学（神经编码）、系统生物学（基因调控网络）及机器学习（可解释性）等领域具有重要价值。
