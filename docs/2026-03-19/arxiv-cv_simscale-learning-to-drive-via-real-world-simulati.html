SimScale: Learning to Drive via Real-World Simulation at Scale

arXiv Computer Vision 2026-03-19

论文摘要 / Abstract

本文提出了SimScale，一种大规模真实世界仿真框架，用于解决自动驾驶中安全关键和分布外场景数据稀缺的问题。该框架利用神经渲染技术结合反应式环境，基于现有驾驶日志生成大规模未见状态的高保真多视角观测数据，并设计了伪专家轨迹生成机制为新模拟状态提供动作监督。研究发现，简单的真实数据与模拟数据联合训练策略即可显著提升多种规划方法在挑战性真实世界基准上的鲁棒性和泛化能力（navhard提升+8.6 EPDMS，navtest提升+2.9），且仅增加模拟数据即可实现策略性能的持续扩展。

方法概述 / Method

SimScale采用先进的神经渲染技术构建反应式仿真环境，通过对自车轨迹进行扰动来生成受控的高保真多视角视觉观测；同时设计了伪专家轨迹生成机制，为模拟出的新状态提供高质量的动作监督信号，形成完整的"模拟-标注"数据生产管线。

实验结果 / Results

在真实世界自动驾驶基准测试中，基于SimScale模拟数据的联合训练使多种规划方法获得显著性能提升：navhard基准上EPDMS指标提升达+8.6，navtest基准上提升+2.9；关键的是，策略改进仅通过增加模拟数据即可平滑扩展，无需额外真实世界数据流入。

应用价值 / Applications

SimScale为自动驾驶系统开发提供了可扩展的数据增强方案，特别适用于安全关键场景训练和边缘案例覆盖，能够降低对昂贵且危险的真实世界数据采集的依赖；该框架的模拟数据可平滑扩展特性使其适用于快速迭代训练不同架构的驾驶策略，加速自动驾驶技术的商业化落地。
