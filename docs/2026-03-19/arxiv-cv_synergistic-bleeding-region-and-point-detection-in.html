Synergistic Bleeding Region and Point Detection in Laparoscopic Surgical Videos

arXiv Computer Vision 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文针对腹腔镜手术中出血检测的基准数据缺失问题，首次构建了真实世界的手术出血检测数据集SurgBlood，包含95个视频片段的5,330帧图像及其出血区域和出血点标注。基于此数据集，作者开发了双任务协同在线检测器BlooDet，能够同时检测出血区域和定位出血点，通过区域检测量化失血辅助决策，通过点定位帮助外科医生快速识别出血源并及时止血，从而提高手术成功率。

方法概述 / Method

BlooDet基于Segment Anything Model 2（SAM 2）构建双分支双向引导架构：掩码分支通过自适应边缘和点提示嵌入检测出血区域；点分支利用掩码记忆诱导出血点记忆建模，并通过帧间光流捕获点运动方向。两个分支通过耦合双向引导机制，探索时空相关性并利用记忆建模推断当前出血状态。

实验结果 / Results

实验结果表明，所提方法在出血检测任务上优于13种对比方法。虽然具体量化指标未在摘要中详述，但该方法在区域分割和点检测两个任务上均展现出领先的性能，验证了双任务协同设计的有效性。

应用价值 / Applications

该研究具有直接的临床应用价值：出血区域检测可实时量化术中失血量，为外科医生提供客观的决策支持；出血点定位功能可帮助快速识别出血源，缩短止血响应时间，降低术后并发症风险。该技术可集成到腹腔镜手术智能辅助系统中，提升手术安全性和成功率。
