Evaluating Feature Dependent Noise in Preference-based Reinforcement Learning

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文研究了基于偏好的强化学习（PbRL）中与特征相关的噪声问题。作者首次形式化了"目标特征依赖噪声"的概念，提出了轨迹特征噪声、轨迹相似性噪声、边际依赖噪声和语言模型噪声等多种变体。实验发现，在某些特征依赖噪声设置下，最先进的抗噪声PbRL方法性能显著下降，而无显式去噪的PbRL方法在多数情况下反而表现更好。此外，语言模型产生的噪声表现出与特征依赖噪声相似的特性，为模拟真实人类偏好提供了新视角。

方法概述 / Method

作者系统性地定义并实现了四类特征依赖噪声：基于轨迹特征的噪声（噪声概率与特定状态特征相关）、基于轨迹相似性的噪声（相似轨迹对更易产生噪声）、基于偏好边际的噪声（偏好差异小的轨迹对更易出错），以及基于语言模型的噪声（利用LLM模拟人类标注者的噪声模式）。通过在DMControl和Meta-world的连续控制任务中注入这些噪声，评估了现有PbRL方法的鲁棒性。

实验结果 / Results

实验表明，特征依赖噪声会严重损害现有抗噪声PbRL方法的性能，尤其在轨迹特征噪声和语言模型噪声设置下；令人意外的是，标准PbRL方法（如PPO+Preference）在多数噪声设置中优于专门的抗噪声方法（如IPR、RIME）；语言模型生成的噪声呈现出与特征依赖噪声高度相似的模式，能够有效模拟真实人类标注中的系统性偏差。

应用价值 / Applications

该研究为设计更鲁棒的PbRL系统提供了重要指导，特别是在需要大规模人类反馈的场景（如大语言模型对齐、机器人学习）中，提醒研究者关注噪声的结构特性而非仅假设均匀分布；提出的语言模型噪声框架可用于低成本生成 realistic 的噪声标注数据，用于预训练和评估抗噪声算法；同时也为理解人类偏好的复杂性和改进人机协作标注流程提供了理论基础。
