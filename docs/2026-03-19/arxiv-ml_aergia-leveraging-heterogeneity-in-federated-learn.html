Aergia: Leveraging Heterogeneity in Federated Learning Systems

arXiv Machine Learning 2026-03-19
Read Original Article →

论文摘要 / Abstract

本文提出了Aergia，一种利用联邦学习系统中异构性的新方法，通过让慢速客户端冻结计算密集型的模型部分并将其训练任务卸载给更快的客户端，显著加速训练过程。与FedAvg和TiFL相比，Aergia在异构环境下分别减少了27%和53%的训练时间，同时保持了高准确率。该方法通过可信执行环境私有评估客户端数据集相似性，并由联邦服务器协调卸载决策。

方法概述 / Method

Aergia的核心机制是让慢速客户端冻结其模型中计算开销最大的层，仅训练剩余部分，同时将冻结部分的训练任务卸载给计算能力更强的客户端。联邦服务器基于客户端报告的训练速度和数据集相似性（通过可信执行环境保护隐私）来动态决策任务分配，实现计算资源的优化调度。

实验结果 / Results

实验表明Aergia在异构设置下能有效维持模型准确率，同时大幅缩短训练时间——相比经典FedAvg算法减少27%训练时间，相比现有异构优化方法TiFL减少53%训练时间。该方法在不同异构程度和客户端分布场景下均展现出稳定的性能优势。

应用价值 / Applications

Aergia适用于实际部署中设备能力差异显著的联邦学习场景，如移动边缘计算、物联网网络和跨组织协作机器学习。该研究为缓解"掉队者"问题提供了实用解决方案，有助于在保护数据隐私的前提下提升大规模分布式学习的效率，对资源受限环境下的AI应用部署具有重要价值。
