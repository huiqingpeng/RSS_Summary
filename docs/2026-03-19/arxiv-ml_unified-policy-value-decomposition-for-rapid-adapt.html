论文摘要 / Abstract

本文提出了一种统一策略价值分解框架，通过共享低维目标嵌入向量实现策略函数和价值函数的联合表示，使智能体能够无需重新训练表征即可快速适应新任务。该框架基于双线性演员-评论家分解，在预训练阶段联合学习结构化价值基和兼容的策略基，测试时通过单次前向传播零样本估计目标嵌入系数。研究在MuJoCo Ant多方向 locomotion 任务上验证了该方法的有效性，表明共享低维目标嵌入为高维控制中的快速结构化适应提供了一种通用机制。

方法概述 / Method

论文采用双线性演员-评论家架构，将价值函数分解为Q = Σ_k G_k(g) y_k(s,a)，其中G_k(g)是目标条件系数向量，y_k(s,a)为学习的价值基函数；策略网络通过相同的系数G_k(g)加权组合原始策略。该方法扩展了Successor Features框架，实现价值基与策略基的联合学习，并通过乘法门控机制（类比大脑L5锥体神经元的增益调制）实现上下文信号对状态依赖基的调制。

实验结果 / Results

研究在MuJoCo Ant环境中训练Soft Actor-Critic智能体完成八个连续目标向量指定的多方向移动任务。实验表明，双线性结构使各策略头能够专精于特定方向子集，而共享的系数层可在它们之间泛化，通过在目标嵌入空间插值实现对新方向的适应；测试时基函数冻结，仅通过单次前向传播估计G_k(g)即可完成零样本适应，无需任何梯度更新。

应用价值 / Applications

该方法适用于需要快速任务适应的高维连续控制场景，如机器人运动控制、多目标导航和技能迁移等。其生物学合理的增益调制机制为设计高效迁移学习系统提供了新思路，共享低维嵌入的模块化设计可显著降低新任务适应的计算成本，对实时自适应系统和终身学习智能体具有重要价值。
