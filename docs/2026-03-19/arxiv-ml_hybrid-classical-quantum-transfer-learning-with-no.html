论文摘要 / Abstract

本研究提出了一类紧凑的量子迁移学习架构，将变分量子分类器连接到冻结的卷积骨干网络用于图像分类。研究在PennyLane和Qiskit平台上实现了多种经典-量子混合模型，并在异构图像数据集上与经典迁移学习基线进行系统比较。实验表明，所提出的量子迁移学习架构在保持竞争性或更优准确率的同时，显著减少了训练时间和能耗，其中PennyLane实现方案在准确率与计算效率之间提供了最佳权衡。

方法概述 / Method

该研究采用混合经典-量子迁移学习方法，利用预训练的经典深度卷积网络进行特征提取，将提取的特征输入轻量化的变分量子电路进行分类。研究在理想模拟、基于IBM量子硬件校准的噪声模拟以及真实IBM量子硬件三种环境下，使用PennyLane和Qiskit框架对多种架构变体进行了全面评估。

实验结果 / Results

实验结果显示，量子迁移学习模型在多个异构图像数据集上达到了与经典基线相当甚至更高的分类准确率，同时训练时间和能源消耗显著降低。
PennyLane-based实现方案在准确率和计算效率之间展现出最优的权衡性能，即使在含噪声的NISQ（噪声中等规模量子）设备设置下仍保持实用价值。

应用价值 / Applications

该研究为NISQ时代的实用化量子机器学习提供了可行路径，特别适用于需要快速部署且计算资源受限的图像分类场景。混合架构通过将特征提取保留在经典域、仅将轻量分类任务交由量子电路处理，有效降低了量子硬件的比特数和电路深度需求，为在现有 noisy 量子设备上实现机器学习优势开辟了实际应用前景。
