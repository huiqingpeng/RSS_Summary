论文摘要 / Abstract

本文研究了基于配对闪光-非闪光成像的非接触式指纹活体检测方法，作为轻量级主动感知机制应对无接触生物识别中的欺骗攻击挑战。通过实证分析发现，闪光照明能够增强材料与结构相关的特性（如脊线可见度、次表面散射、微几何结构和表面油脂），而非闪光图像提供基线外观上下文。研究采用可解释性指标分析光照诱导差异，有效区分真实指纹与打印、数字和模具等呈现攻击，同时探讨了该方法对成像设置、数据规模和新型高保真欺骗的敏感性局限。

方法概述 / Method

论文提出配对闪光-非闪光采集作为主动感知机制，利用两种照明条件下的互补信息进行分析。具体采用四种可解释性度量指标：通道间相关性、镜面反射特性、纹理真实度和差分成像，以捕捉材料依赖的光照响应差异，实现无需深度学习的轻量化特征设计。

实验结果 / Results

实验表明闪光照明能够显著凸显真实指纹与欺骗材料在物理特性上的差异，所提出的光照感知特征在多种欺骗类型（打印、数字屏幕、模具）上展现出判别能力。研究同时揭示了该方法的局限性：性能受成像参数设置影响、需要一定规模的数据集支持，且面临高保真新型欺骗的挑战。

应用价值 / Applications

该方法可集成于智能手机、门禁系统等非接触式指纹识别设备，提供卫生便捷的活体检测方案。其轻量级特性适合资源受限的边缘计算场景，而可解释的特征设计有助于满足金融、司法等高安全领域对算法透明度的监管要求，为物理信息驱动的防伪技术开辟新方向。
