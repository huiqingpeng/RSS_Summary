LEGO Machine Plays Tic-Tac-Toe Without Electronics

Hackaday 2026-03-16
Read Original Article →

是什么 / What it is

这是一台完全由乐高积木搭建的纯机械井字棋（Tic-Tac-Toe）游戏机，由Joost van Velzen设计。它不使用任何电子元件，而是通过52个机械逻辑门和204位机械内存来处理游戏状态并做出应对。
This is a purely mechanical Tic-Tac-Toe playing machine built entirely from LEGO bricks, designed by Joost van Velzen. It uses no electronics whatsoever, instead processing game states and making moves through 52 mechanical logic gates and 204 bits of mechanical memory.

为什么重要 / Why it matters

这个项目展示了机械计算的可能性，为理解计算机基本原理提供了直观的物理模型。它同时融合了复古美学（18世纪自动机风格）与现代创客文化，在LEGO Ideas平台获得广泛关注，体现了低技术解决方案的独特魅力。
This project demonstrates the possibility of mechanical computing, offering an intuitive physical model for understanding fundamental computer principles. It blends retro aesthetics (18th-century automaton style) with modern maker culture, gaining significant attention on LEGO Ideas and showcasing the unique appeal of low-tech solutions.

我能用什么 / How I can use it
