A Theoretical Framework for Energy-Aware Gradient Pruning in Federated Learning

arXiv Machine Learning 2026-03-25
Read Original Article →

论文摘要 / Abstract

本文针对联邦学习（FL）中边缘设备的通信与能耗限制问题，提出了一种能量感知的梯度剪枝理论框架。现有Top-K幅度剪枝方法忽视了硬件层面的能耗差异，假设所有参数更新的传输和内存开销相同。作者将剪枝过程形式化为能量约束投影问题，提出成本加权幅度剪枝（CWMP）策略，并证明其是该约束问题的最优贪心解。在非独立同分布CIFAR-10基准上的实验表明，CWMP在性能-能耗帕累托前沿上显著优于传统Top-K基线。

方法概述 / Method

论文将梯度剪枝重新建模为能量约束下的投影优化问题，显式考虑后向传播阶段内存密集型操作与计算高效操作之间的硬件级成本差异。提出的CWMP选择规则根据参数更新的幅度与其物理成本的比值进行优先级排序，而非仅依据幅度大小，从而实现能量效率与模型性能的平衡。

实验结果 / Results

在非IID分布的CIFAR-10数据集上的数值实验表明，CWMP方法能够持续建立优于Top-K基线的性能-能耗帕累托前沿，即在相同能耗预算下实现更高的模型精度，或在达到相同精度时显著降低能耗消耗。

应用价值 / Applications

该研究为资源受限的边缘计算场景（如智能手机、物联网设备参与的联邦学习）提供了理论指导，使梯度压缩策略能够根据实际硬件特性进行能耗优化。
CWMP可集成到现有联邦学习系统中，在延长边缘设备电池寿命的同时维持模型训练质量，对大规模分布式机器学习部署具有重要实用价值。
