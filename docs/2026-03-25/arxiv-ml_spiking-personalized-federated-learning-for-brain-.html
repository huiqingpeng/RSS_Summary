论文摘要 / Abstract

本文提出了一种新型沉浸式通信框架，利用脑机接口（BCI）获取脑信号以推断用户中心状态（如意图和感知相关的不适），从而在强个体差异性下实现更个性化且稳健的沉浸式适应。该研究开发了个性化联邦学习（PFL）模型来处理脑信号数据，既能适应神经多样性的脑信号，又能防止敏感脑信息泄露。通过将脉冲神经网络（SNN）嵌入PFL，解决了能量受限沉浸式终端（如头戴显示器）上持续设备端学习与推理的能耗瓶颈。

方法概述 / Method

该研究将脉冲神经网络（SNN）与个性化联邦学习（PFL）相结合，利用SNN稀疏、事件驱动的脉冲计算特性降低计算和能耗成本。具体而言，SNN-enabled PFL框架在保护用户隐私的联邦学习架构下，实现了对神经多样性脑信号数据的高效个性化处理，同时满足沉浸式终端的低功耗需求。

实验结果 / Results

在真实脑信号数据集上的实验表明，该方法在保持最佳整体识别准确率的同时，相比传统基于人工神经网络的个性化基线方法，推理能耗降低了6.46倍。这一结果验证了SNN-enabled PFL在准确性与能效之间的有效权衡。

应用价值 / Applications

该研究适用于脑机接口驱动的沉浸式通信场景，如虚拟现实/增强现实头戴设备，可实现基于用户意图和舒适度的实时个性化内容适配。其低功耗特性特别适合电池供电的移动沉浸式终端，同时联邦学习架构保障了用户脑信号数据的隐私安全，为神经数据敏感的普适计算应用提供了可行方案。
