MIOFlow 2.0: A unified framework for inferring cellular stochastic dynamics from single cell and spatial transcriptomics data

arXiv Machine Learning 2026-03-25
Read Original Article →

论文摘要 / Abstract

MIOFlow 2.0 是一个统一的计算框架，用于从单细胞转录组学和空间转录组学数据中推断细胞随机动态。该框架通过整合流形学习、最优传输和神经微分方程，同时建模细胞命运的随机分支、非保守性群体增殖变化以及空间环境信号的影响。相比现有确定性方法，MIOFlow 2.0 能够捕捉真实的概率性分支过程、群体动态变化和生态位依赖的信号传导，在合成数据集、胚状体分化和蝾螈脑再生等验证中展现出更优的轨迹推断精度，并能揭示细胞转换的隐藏驱动因素。

方法概述 / Method

MIOFlow 2.0 采用三模块架构：使用神经随机微分方程（Neural SDEs）建模随机性与细胞命运分支；通过基于非平衡最优传输初始化的学习增长率模型处理非保守性群体变化；构建联合潜在空间整合基因表达与空间特征（如局部细胞类型组成和信号传导）。整个框架在 PHATE 距离匹配自编码器的潜在空间中运行，确保轨迹尊重数据的内在几何结构。

实验结果 / Results

实验表明，基于神经微分方程的轨迹学习方法在表达能力上优于现有生成模型，包括无模拟的流匹配方法。在合成数据集、胚状体分化数据集以及空间分辨的蝾螈脑再生数据上的验证显示，MIOFlow 2.0 显著提升了轨迹推断准确性，并成功揭示了细胞状态转换的隐藏驱动因素，如特定的信号传导生态位。

应用价值 / Applications

MIOFlow 2.0 在发育生物学、再生医学和疾病研究中具有重要应用价值，能够整合单细胞与空间转录组学数据以揭示组织尺度的细胞动态。该框架特别适用于分析包含复杂随机决策、增殖动态和空间异质性的生物过程，为理解胚胎发育、组织再生和肿瘤微环境中的细胞命运调控机制提供了强有力的工具。
