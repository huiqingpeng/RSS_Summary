论文摘要 / Abstract

本文针对基于群体的结构健康监测（PBSHM）中异构结构间知识迁移的难题，提出通过中间结构桥接高度差异系统间信息差距的迁移学习方法。核心思想是通过连续改变材料属性、几何形状等参数，使一种结构能够平滑变形为另一种结构。研究通过两个案例验证了该方法：案例1实现了模拟异构桥梁设计间的参数化与迁移；案例2则通过有限元模型链，在简化的"桥梁"与"飞机"物理表征之间实现了迁移，证明了即使高度不同的系统也能实现正向迁移。

方法概述 / Method

本研究采用结构插值方法进行迁移学习，通过构建参数化的中间结构序列来连接源域和目标域。具体而言，该方法通过系统性地调整结构的关键参数（如材料属性、几何构型），生成一系列连续变形的中间模型，形成从源结构到目标结构的渐进式迁移路径，而非直接进行跨域迁移。

实验结果 / Results

案例1成功展示了模拟异构桥梁设计之间的有效迁移；案例2更具挑战性，在"桥梁"与"飞机"这两个高度差异的简化物理系统之间实现了正向知识迁移。实验结果表明，通过有限元模型构建的插值结构链，可以克服传统基于结构相似性预测所认为的不可迁移障碍，验证了"在某些情况下，高度不同的系统之间确实可以实现正向迁移"这一反直觉结论。

应用价值 / Applications

该方法为结构健康监测领域提供了处理异构结构群体的新范式，特别适用于工程实践中缺乏同类历史数据、需要跨结构类型借鉴知识的场景。潜在应用包括：复杂基础设施（如桥梁、建筑、航空器）的跨类型损伤识别与预测、数字孪生模型间的知识共享，以及降低新结构类型监测系统的数据依赖和开发成本。
