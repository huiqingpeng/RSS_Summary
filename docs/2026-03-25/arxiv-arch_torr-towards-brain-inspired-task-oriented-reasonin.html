TorR: Towards Brain-Inspired Task-Oriented Reasoning via Cache-Oriented Algorithm-Architecture Co-design

arXiv Computer Architecture 2026-03-25
Read Original Article →

论文摘要 / Abstract

本文提出TorR，一种面向任务导向目标检测（TOOD）的脑启发式算法-架构协同设计框架。
TorR用超维计算（HDC）关联推理器替代CLIP风格的密集对齐计算，通过时间相干性和缓存复用机制显著降低内存流量和计算开销。在TSMC 28nm工艺下，TorR实现了毫秒级能效（60FPS约50mJ，30FPS约113mJ），同时在五个任务提示上保持有竞争力的AP@0.5（平均44.27%），较视觉语言模型基线能耗降低数个数量级。

方法概述 / Method

算法层面，TorR将特征对齐重新表述为HDC相似度计算和图组合，引入三种部分相似性复用机制：（i）带每类分数累积的查询缓存，（ii）仅少量超向量位变化时的精确δ更新，以及（iii）高系统负载下的相似度/负载门控旁路。架构层面，采用可扩展通道、位切片的项目存储器，配合bank/精度门控和轻量级控制器，根据目标数量动态调度旁路/δ/完整路径以满足RT-30/RT-60实时目标。

实验结果 / Results

TorR在28nm工艺综合和周期精确仿真验证下，以毫焦耳级能耗实现实时吞吐（60FPS约50mJ/窗，30FPS约113mJ/窗），延迟抖动低。在五个任务提示上，TorR取得平均44.27%的AP@0.5，与强VLM基线相比处于有界差距内，但能耗降低数个数量级。设计支持部署时可配置参数（有效维度D'、阈值、精度），可在边缘预算约束下权衡精度、延迟和能耗。

应用价值 / Applications

TorR适用于实时、功耗受限的边缘设备场景，如自主机器人、无人机和智能摄像头等需要开放词汇、提示驱动语义理解的任务导向目标检测应用。其可配置性使系统能够根据具体边缘预算动态调整性能-能效权衡，为资源受限环境下的视觉语言推理提供了实用的部署方案。
