EmbBERT: Attention Under 2 MB Memory

arXiv Computer Architecture 2026-03-25
Read Original Article →

论文摘要 / Abstract

本文提出了EmbBERT，一种专为极端内存约束环境设计的微型语言模型（TLM），仅需2 MB总内存即可运行。该模型通过紧凑的嵌入层、精简的前馈模块和高效注意力机制实现了架构级优化，在TinyNLP基准和GLUE套件上取得了与内存需求10倍于己的最先进模型相当的准确率，并显著优于同等规模的BERT和MAMBA精简版本。此外，EmbBERT支持8位量化，可进一步将内存压缩至781 kB，展现出良好的量化鲁棒性和架构可扩展性。

方法概述 / Method

EmbBERT采用系统性的架构精简策略，针对嵌入层、前馈网络和注意力机制三大核心组件进行重新设计，以实现极致的内存效率。模型通过紧凑的嵌入表示、流线型前馈块以及优化后的轻量级注意力机制，在严格内存预算下平衡表达能力与计算开销。完整的预训练流程和组件级消融验证确保了模型设计的有效性。

实验结果 / Results

在TinyNLP基准和GLUE套件上的广泛实验表明，EmbBERT以2 MB内存实现了与10倍内存需求的SotA模型可比的准确率，并一致性地超越同等参数规模的BERT和MAMBA变体。8位量化后模型仅需781 kB内存，且架构可灵活扩展至亚MB到数十MB范围。消融研究证实了各组件及预训练策略的积极贡献。

应用价值 / Applications

EmbBERT适用于可穿戴设备、物联网终端等超资源受限边缘计算场景，使复杂的NLP能力首次在仅数MB内存的微控制器上部署成为可能。该研究为微型机器学习（TinyML）领域提供了实用的架构设计范式，推动了智能传感、实时语音交互、低功耗文本处理等应用的普及，同时其可扩展性也支持向更大内存预算场景的平滑迁移。
