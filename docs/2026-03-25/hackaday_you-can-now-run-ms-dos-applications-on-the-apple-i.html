You Can Now Run MS-DOS Applications on the Apple IIe

Hackaday 2026-03-25
Read Original Article →

是什么 / What it is

一位开发者成功将 MS-DOS 2.0 移植到了 Apple II 电脑上，通过 AD8088 等 x86 协处理器扩展卡，让这款 1983 年的 8 位苹果电脑能够运行 DOS 应用程序。该项目经历了大量调试工作，解决了协处理器与主机 6502 CPU 之间的通信、内存重定位等问题，现已公开发布。
A developer has successfully ported MS-DOS 2.0 to the Apple II computer, enabling this 1983 8-bit machine to run DOS applications via an x86 co-processor expansion card like the AD8088. After extensive debugging to solve communication between the co-processor and host 6502 CPU, memory relocation, and other issues, the project has been released to the public. ---

为什么重要 / Why it matters

这一项目复原了历史上曾存在但已失传的软件功能，展示了复古计算社区对技术遗产的 preservation 热情。它打破了 Apple II 与 IBM PC 生态之间的壁垒，让单一硬件平台能同时运行两大阵营的软件，具有独特的历史和技术价值。
This project restores software functionality that existed historically but was lost to time, demonstrating the retrocomputing community's passion for preserving technical heritage. It breaks down barriers between the Apple II and IBM PC ecosystems, enabling a single hardware platform to run software from both camps—offering unique historical and technical value. ---

我能用什么 / How I can use it

如果你拥有 Apple IIe 和 AD8088 协处理器卡，可直接下载该项目体验在 8 位机上运行 DOS 软件的独特乐趣；复古计算爱好者可研究其桥接程序设计，学习异构 CPU 通信与内存管理技巧；开发者则可借鉴这种跨平台移植思路，探索其他历史软硬件的兼容性扩展方案。
If you own an Apple IIe with an AD8088 co-processor card, you can download the project to experience running DOS software on 8-bit hardware; retrocomputing enthusiasts can study its bridge program design to learn techniques for heterogeneous CPU communication and memory management; developers can draw inspiration from this cross-platform porting approach to explore compatibility extensions for other historical hardware and software.
