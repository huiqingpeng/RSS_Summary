Geometric Mixture-of-Experts with Curvature-Guided Adaptive Routing for Graph Representation Learning

论文摘要 / Abstract

本文提出了几何混合专家框架（GeoMoE），用于解决图结构数据中复杂拓扑异质性的建模难题。该方法利用Ollivier-Ricci曲率（ORC）作为内在几何先验，自适应地融合不同黎曼空间中的节点表示。通过设计图感知门控网络和曲率引导的对齐损失，实现了可解释且几何一致的自适应路由。在六个基准数据集上的实验表明，GeoMoE在多种图类型上均优于最先进的基线方法。

方法概述 / Method

GeoMoE的核心是一个基于曲率的混合专家架构：首先使用Ollivier-Ricci曲率计算节点级几何特性，作为路由决策的几何先验；然后设计图感知门控网络为每个节点分配专家融合权重，并通过曲率引导的对齐损失正则化路由过程；最后引入曲率感知对比学习目标，根据曲率一致性构建正负样本对以增强几何判别性。

实验结果 / Results

论文在六个涵盖同构/异构图、社交网络和分子图的基准数据集上进行了评估，GeoMoE在所有数据集上均取得最优性能，显著超越了现有的单曲率和混合曲率基线方法。消融实验验证了ORC引导路由、曲率对齐损失和对比学习三个组件各自的有效性。

应用价值 / Applications

该方法适用于具有复杂多尺度拓扑结构的图表示学习任务，如社交网络分析（用户关系的多层次结构）、分子性质预测（化学键的空间异质性）和知识图谱推理。
GeoMoE通过显式的几何先验提供了可解释的路由机制，有助于理解模型决策背后的拓扑原理，为需要几何可解释性的科学计算和药物发现等应用提供了新工具。
