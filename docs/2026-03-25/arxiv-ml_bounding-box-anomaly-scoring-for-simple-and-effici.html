论文摘要 / Abstract

本文提出了边界框异常评分（BBAS），一种用于分布外（OOD）检测的后处理方法。该方法通过边界框抽象来表示分布内数据的特征空间支持，结合了基于区间越界的分级异常评分、适配卷积层的监控变量，以及解耦的聚类和边界框构建策略。实验表明，BBAS在图像分类基准上实现了分布内与分布外样本的鲁棒分离，同时保持了边界框方法的简洁性、紧凑性和可更新性。

方法概述 / Method

BBAS利用轴对齐的边界框对预训练网络隐藏层激活进行紧凑摘要，以近似分布内数据的支持区域。该方法采用分级评分机制量化激活值超出边界框区间的程度，并针对卷积层设计专门的监控变量，通过解耦聚类与边界框构建实现多层级丰富表示。

实验结果 / Results

在图像分类基准数据集上的实验表明，BBAS能够有效区分分布内和分布外样本，提供稳健的OOD检测性能。该方法在保持模型紧凑性的同时，达到了与更复杂的基于参考的方法（如k近邻）相竞争的检测效果，显著优于传统的参数化方法（如马氏距离）。

BBAS可部署于需要高可靠性深度学习预测的实际场景，如自动驾驶中的异常物体识别、医疗影像中的未知病变检测，以及工业质检中的缺陷识别。其紧凑性和可更新性使其特别适合资源受限的嵌入式设备和需要持续学习的在线系统。
