Understanding Behavior Cloning with Action Quantization

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文研究了行为克隆（Behavior Cloning）中动作量化（Action Quantization）的理论基础，这是将自回归模型（如Transformer）应用于连续控制任务的关键技术。作者分析了量化误差在时间维度上的传播机制及其与统计样本复杂度的相互作用，证明了在动力学稳定且策略满足概率平滑条件下，基于量化动作和对数损失的行为克隆能够实现最优样本复杂度，且量化误差仅产生多项式水平依赖。此外，论文还提出了无需策略平滑性假设的基于模型增强方法来改进误差界，并建立了量化误差与统计复杂度联合作用的基本极限。

方法概述 / Method

本文采用理论分析的方法，通过建立数学框架来刻画动作量化在行为克隆中的影响。具体而言，作者引入概率平滑性条件（probabilistic smoothness condition）来约束策略的连续性，并分析不同量化方案（如均匀量化、自适应量化）对该条件的满足情况；同时提出了一种模型增强（model-based augmentation）技术，通过引入环境动力学模型来显式修正量化误差。

实验结果 / Results

理论结果表明：在稳定动力学和概率平滑策略假设下，量化行为克隆的样本复杂度达到$\tilde{O}(1/\epsilon^2)$，匹配现有下界；量化误差导致的累积次优性（suboptimality）随水平长度$H$呈多项式增长$O(H \cdot \Delta^2)$，其中$\Delta$为量化精度；模型增强方法可将误差界改进至$O(H \cdot \Delta^4)$或完全消除对策略平滑性的依赖。作者还刻画了不同量化方案满足理论要求的充分必要条件。

应用价值 / Applications

本研究为机器人控制、自动驾驶和视觉-语言-动作（VLA）系统等领域的实际部署提供了理论指导，帮助从业者理解如何设计量化策略以平衡模型表达能力与学习稳定性。提出的模型增强方法可直接应用于现有自回归策略学习框架（如RT-1、RT-2等），在不增加样本复杂度的前提下提升策略性能，特别适用于高维连续动作空间中的模仿学习任务。
