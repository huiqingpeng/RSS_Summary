论文摘要 / Abstract

本文是一篇关于属性图聚类（Attributed Graph Clustering, AGC）的综述论文，核心贡献在于提出了"编码-聚类-优化"（Encode-Cluster-Optimize）的统一分类框架，将现有方法分解为三个正交可组合的模块。作者批判性地指出当前AGC研究存在"学术单一文化"问题：过度依赖小规模同质引用网络、使用不适用于无监督任务的监督指标、以及忽视计算可扩展性。论文基于工业部署的实际约束（大规模数据、异质性、特征噪声等），提出了包含监督语义对齐、无监督结构完整性和效率分析的综合评估标准，并指明了异质性鲁棒编码器、可扩展联合优化等未来研究方向。

方法概述 / Method

论文采用综述与框架构建的方法论，首先通过"编码-聚类-优化"三维分类法对现有AGC算法进行系统化解构——表示编码模块涵盖GNN和自监督学习等方法，聚类投影模块涉及软/硬分配策略，优化策略模块包括交替优化与端到端联合优化。其次，作者构建了配套基准测试平台，在真实工业场景数据集上开展大规模实证研究，以验证不同方法组合的实际效能。

实验结果 / Results

通过 companion benchmark 的广泛实验，论文揭示了学术基准与工业需求之间的显著性能差距：现有方法在异质图和大规模图上的聚类质量急剧下降，且计算效率远不能满足生产环境要求。实验表明，简单的模块组合（如特定编码器与优化策略的配对）往往优于复杂端到端架构，同时强调无监督模型选择准则的缺失是当前关键瓶颈。

应用价值 / Applications

本研究对工业界的图数据挖掘应用具有直接指导价值，包括社交网络用户分群、电商商品聚类、金融风控中的社区发现等场景。提出的评估标准和工程策略可帮助企业规避"论文算法落地即失效"的困境，而开源的基准测试平台为算法选型提供了可靠依据。对学术界而言，该综述打破方法论的碎片化现状，为设计满足生产级要求的下一代AGC方法提供了清晰的技术路线图。
