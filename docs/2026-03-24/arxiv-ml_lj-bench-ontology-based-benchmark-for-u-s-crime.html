LJ-Bench: Ontology-Based Benchmark for U.S. Crime

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文提出了LJ-Bench，首个基于法律本体论的全面基准测试，用于评估大型语言模型（LLMs）对各类非法活动的鲁棒性。该研究构建了以《模范刑法典》为法律基础、以加州法律为实例的犯罪相关概念本体论，涵盖76种分类组织的犯罪类型。实验发现，LLMs对社会危害性攻击的敏感性高于对直接针对个人的攻击，揭示了不同犯罪类别中模型脆弱性的差异模式。

方法概述 / Method

研究采用本体工程方法，将《模范刑法典》的法律框架形式化为结构化知识表示（LJ-Ontology），并以加州法律进行实例化。基于该本体论，研究者系统性地设计了覆盖76种犯罪类型的测试用例，构建了多维度攻击评估框架。

实验结果 / Results

基准测试揭示了LLM安全机制的关键漏洞：模型在面对针对社会整体危害的查询时表现出更高的易感性，相比直接针对个人伤害的攻击更容易被突破。这一发现表明现有安全对齐策略在保护社会层面危害方面存在系统性不足。

应用价值 / Applications

LJ-Bench为LLM安全研究提供了法律 grounded 的标准化评估工具，可支持红队测试、安全对齐优化和监管合规验证。该基准有助于推动更鲁棒、可信的LLM开发，特别适用于需要严格法律合规性的司法咨询、法律教育等高风险应用场景。
