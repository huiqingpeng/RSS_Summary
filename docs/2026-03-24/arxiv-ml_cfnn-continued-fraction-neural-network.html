CFNN: Continued Fraction Neural Network

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文提出了连分数神经网络（CFNN），通过将连分数与梯度优化相结合，为神经网络引入"有理数归纳偏置"，以解决多层感知机（MLP）因频谱偏置而难以刻画高曲率特征和奇点的问题。
CFNN能够以极少的参数捕捉复杂渐近行为和间断特性，并提供了指数收敛和稳定性的形式化逼近边界。实验表明，CFNN在精度上持续超越MLP，参数减少一至两个数量级，噪声鲁棒性和物理一致性提升高达47倍。

方法概述 / Method

CFNN的核心是将连分数这一经典数学工具嵌入神经网络架构，利用其天然的有理函数逼近能力来建模非线性流形奇点。为克服递归计算中的数值不稳定问题，论文开发了三种实现变体：CFNN-Boost（提升式）、CFNN-MoE（混合专家）和CFNN-Hybrid（混合架构），分别针对不同场景优化计算稳定性与表达能力。

实验结果 / Results

基准测试显示，CFNN在多个任务上以MLP 1%-10%的参数规模实现了更高精度，参数效率提升一至两个数量级；在噪声环境下表现出高达47倍的鲁棒性改进，同时保持更好的物理一致性（如守恒律、边界条件等）。理论分析证明了CFNN具有指数收敛速率和严格的稳定性保证。

应用价值 / Applications

CFNN建立了"灰盒"建模范式，兼具黑箱方法的灵活性和白盒模型的可解释性，特别适用于科学计算中需要精确刻画奇点、渐近行为和物理约束的场景，如流体力学间断问题、量子力学势函数拟合、金融衍生品定价模型等AI驱动的科学研究领域。
