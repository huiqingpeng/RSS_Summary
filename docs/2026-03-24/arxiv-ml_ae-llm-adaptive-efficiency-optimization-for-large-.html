AE-LLM: Adaptive Efficiency Optimization for Large Language Models

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文提出了AE-LLM，一个统一的自适应效率优化框架，用于自动选择和组合最优的大语言模型效率技术。该框架通过多目标优化同时考虑准确性、延迟、内存占用和能耗，并开发高效搜索算法在架构、微调和推理阶段的组合空间中识别帕累托最优配置。实验表明，AE-LLM在15个模型（0.5B-70B参数）和10个多样化任务上平均实现2.8倍的效率提升，同时保持与基线相差1.2%以内的竞争性准确率，且能泛化到视觉-语言模型。

方法概述 / Method

AE-LLM采用多目标优化框架，联合优化准确性、延迟、内存和能耗四个关键指标，同时考虑硬件约束和任务需求。研究团队开发了高效搜索算法，探索架构设计（如高效注意力机制、MoE）、参数高效微调（PEFT）和推理优化（如量化）等技术的组合空间，以识别特定部署场景下的帕累托最优配置。

实验结果 / Results

在涵盖15个模型（0.5B至70B参数）和10个多样化任务的广泛实验中，AE-LLM相比静态效率配置平均实现2.8倍的效率指标提升，同时保持准确率损失在1.2%以内。该框架还成功泛化到视觉-语言模型，实现了类似的效率增益，证明了其跨模态的通用性。

应用价值 / Applications

AE-LLM为实际部署LLM的从业者提供了自动化工具，可根据具体硬件环境、资源预算和任务特性动态选择最优效率技术组合，显著降低计算成本、内存需求和能耗。该框架特别适用于需要在边缘设备、云端服务器或多样化应用场景中灵活部署大模型的场景，助力实现LLM的高效可持续应用。
