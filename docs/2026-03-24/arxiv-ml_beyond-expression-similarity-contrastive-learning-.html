Beyond Expression Similarity: Contrastive Learning Recovers Functional Gene Associations from Protein Interaction Structure

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本研究将预测性联想记忆（PAM）框架中的对比联想学习（CAL）方法从自然语言处理迁移至分子生物学领域，验证蛋白质相互作用结构能否提供超越基因表达相似性的功能基因关联。研究在四个跨生物学域实验中发现，CAL在基因扰动数据上取得显著优于表达相似性的性能（AUC 0.908 vs 0.518），并揭示了三项关键跨域发现：生物学中的归纳迁移成功而文本领域失败、CAL得分与相互作用度呈负相关、以及高质量关联优于大规模噪声训练集。

方法概述 / Method

研究采用对比多层感知机（contrastive MLP）架构，以STRING数据库中的蛋白质-蛋白质相互作用作为共现标注进行训练，通过对比正负样本对来学习基因间的功能关联表示。方法核心在于利用蛋白质相互作用网络的结构信息而非基因表达谱相似性，通过节点不相交划分评估归纳迁移能力，并设计药物敏感性实验生成信息性负样本以优化边界条件。

实验结果 / Results

在Replogle K562 CRISPRi数据集（2,285个基因）上，CAL的跨边界AUC达0.908，远超表达相似性的0.518；DepMap数据集（17,725个基因）经负采样校正后AUC提升至0.947。节点不相交分割的归纳迁移实验取得0.826的AUC（提升+0.127），而相同设置在文本领域失败（±0.10）。
CAL得分与蛋白质相互作用度呈显著负相关（Spearman r = -0.590），且结果在训练种子和阈值选择下高度稳定（标准差<0.001）。

应用价值 / Applications

该方法为功能基因组学研究提供了不依赖表达相似性的基因关联预测工具，特别适用于识别研究不足、相互作用谱集中的基因功能关系，有望加速靶点发现和疾病机制解析。研究还建立了跨域迁移学习的生物学验证范式，为物理约束型关联网络（如分子相互作用）与偶然共现型关联（如文本共现）的可迁移性差异提供了理论洞察，指导领域特定表示学习策略的设计。
