Spatio-Temporal Grid Intelligence: A Hybrid Graph Neural Network and LSTM Framework for Robust Electricity Theft Detection

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本研究提出了一种融合时空智能的混合AI框架，用于解决电力窃电（非技术性损耗NTL）检测中的复杂动态问题。该框架整合时间序列异常检测、监督机器学习和图神经网络（GNN），通过LSTM自编码器、随机森林分类器和GNN的协同工作，在高度不平衡的数据集上实现了高精度窃电识别。实验表明，相比单一异常检测模型仅0.20的F1分数，该混合融合方法达到93.7%的整体准确率，并通过精确率-召回率阈值校准实现了0.55的精确率和0.50的召回率，有效降低了误报率。

方法概述 / Method

该研究采用三层混合架构：首先使用LSTM自编码器进行时间序列异常评分以捕捉时序行为模式，其次利用随机森林分类器处理表格特征（包括滚动平均、电压降估计和电网不平衡指数）进行判别分析，最后通过GNN建模配电网络的拓扑空间依赖关系。三种模型的输出通过决策层融合，形成综合的窃电风险评估。

实验结果 / Results

关键实验结果显示：单一LSTM异常检测模型在窃电识别上表现较差（F1=0.20），而提出的混合融合框架将整体准确率提升至93.7%；经过精确率-召回率曲线阈值优化后，系统在保持0.50召回率的同时将窃电检测精确率提升至0.55，显著优于单一模型方法，有效平衡了漏检与误报之间的权衡。

应用价值 / Applications

该研究为智能电网运营商提供了可扩展的风险导向型主动窃电检测解决方案，特别适用于配电网络拓扑复杂、数据高度不平衡的实际场景。通过整合电网拓扑感知与时序监督分析，该技术可提升电网可靠性、减少非技术性损耗造成的财务损失，并为智能电表数据分析平台的建设提供核心技术支撑。
