Bayesian Scattering: A Principled Baseline for Uncertainty on Image Data

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文提出了**贝叶斯散射（Bayesian Scattering）**方法，为图像数据的不确定性量化提供了一个可解释、数学基础扎实的基线方法，类似于贝叶斯线性回归在表格数据中的作用。该方法将小波散射变换（一种深度非学习特征提取器）与简单的概率头相结合，利用基于几何原理而非学习得到的特征来避免对训练分布的过拟合，从而在显著分布偏移下仍能提供合理的不确定性估计。研究在医学影像机构迁移、跨国财富映射和分子性质贝叶斯优化等多样化任务上验证了该方法的有效性。

方法概述 / Method

核心方法是将**小波散射变换（wavelet scattering transform）**作为特征提取器，其通过级联的小波变换和模运算构建深度特征，完全基于几何原理而非数据驱动学习；随后在这些固定特征上施加简单的概率建模头（如高斯过程或贝叶斯线性模型），形成端到端的不确定性量化框架。这种"非学习特征 + 概率推断"的组合避免了神经网络对训练分布的过拟合问题。

实验结果 / Results

实验表明，贝叶斯散射在多种分布偏移场景下展现出稳健的不确定性估计能力，包括：医学影像中的跨机构域迁移、卫星图像的跨国财富预测迁移，以及分子性质的贝叶斯优化任务。结果显示该方法可作为复杂深度不确定性量化方法的有力基线，在保持可解释性的同时提供可靠的预测不确定性。

应用价值 / Applications

该方法适用于**高风险决策场景**中需要可靠不确定性估计的图像分析任务，如医疗诊断中的跨医院模型部署、全球发展政策中的卫星图像分析，以及科学发现中的分子设计优化。其可解释性和数学严谨性使其特别适合作为复杂方法的比较基准，以及在数据有限或分布偏移严重的实际环境中快速部署。
