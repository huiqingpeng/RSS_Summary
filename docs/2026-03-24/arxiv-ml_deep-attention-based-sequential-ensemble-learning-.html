Deep Attention-based Sequential Ensemble Learning for BLE-Based Indoor Localization in Care Facilities

论文摘要 / Abstract

本文提出了深度注意力序列集成学习框架（DASEL），将室内定位重新概念化为序列学习问题，而非传统方法中的独立观测处理。该框架通过双向GRU网络、注意力机制和多向滑动窗口等技术捕捉人员移动轨迹，在护理机构真实数据集上取得了0.4438的宏F1分数，较最优传统基线提升53.1%。研究有效解决了传统机器学习方法在BLE室内定位中忽视时间依赖性的根本局限。

方法概述 / Method

DASEL框架整合了四项核心技术：基于频率的特征工程提取BLE信号时频特性；双向GRU网络结合注意力机制建模序列双向依赖关系；多向滑动窗口策略捕获不同时间尺度的移动模式；置信度加权时序平滑对预测结果进行后处理优化。该方法通过端到端的序列学习范式，充分利用人员移动的连续性和方向性特征。

实验结果 / Results

在护理机构真实数据集上采用4折时间交叉验证，DASEL取得0.4438的宏F1分数，显著优于最佳传统基线（0.2898），相对提升达53.1%。实验结果表明，引入序列建模和注意力机制能有效捕捉人员移动的时空规律，显著改善定位精度。

应用价值 / Applications

该研究可直接应用于护理机构的智能管理系统，实现工作人员实时定位、动态任务分配和工作负荷优化，提升护理服务质量和运营效率。其序列学习框架亦可推广至其他BLE室内定位场景，如医院患者监护、仓储物流追踪和商场客流分析等需要连续位置感知的应用领域。
