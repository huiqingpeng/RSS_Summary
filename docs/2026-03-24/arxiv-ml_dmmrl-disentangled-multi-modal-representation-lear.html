DMMRL: Disentangled Multi-Modal Representation Learning via Variational Autoencoders for Molecular Property Prediction

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文提出了DMMRL（解耦多模态表示学习）框架，利用变分自编码器将分子表示解耦为共享的（结构相关）和私有的（模态特定）潜在空间，以提升分子性质预测的可解释性和准确性。该方法通过变分解耦机制分离最具信息量的特征，并结合正交性和对齐正则化促进统计独立性和跨模态一致性。在七个基准数据集上的实验表明，DMMRL优于现有最先进方法。

方法概述 / Method

DMMRL采用变分自编码器架构，通过编码器将分子图、序列和几何等多模态数据分别映射到共享潜在空间和各模态特定的私有潜在空间。模型引入正交性约束确保共享与私有表示的统计独立性，同时通过对齐正则化保证跨模态一致性，最后使用门控注意力融合模块自适应地整合共享表示以捕获复杂的模态间关系。

实验结果 / Results

DMMRL在七个分子性质预测基准数据集上进行了全面验证，实验结果显示其性能显著优于当前最先进的多模态分子表示学习方法。消融实验证明了变分解耦机制、正交性正则化和门控注意力融合模块各自对整体性能的贡献，验证了各组件设计的有效性。

应用价值 / Applications

该研究可直接应用于药物发现和材料科学中的分子性质预测任务，如药物活性筛选、毒性预测和材料性能优化。解耦的表示结构增强了模型的可解释性，有助于研究人员理解分子结构-性质关系；同时，分离的共享表示具有良好的迁移能力，可支持跨数据集和跨任务的迁移学习应用。
