Thinking in Different Spaces: Domain-Specific Latent Geometry Survives Cross-Architecture Translation

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本研究探讨了独立训练的语言模型是否会收敛到几何兼容的潜在表示，以及这种兼容性是否可用于在推理时纠正模型行为而无需更新权重。作者发现，通过线性投影将大教师模型的激活向量映射到小教师模型的坐标系中，可以在20种异构的师生模型配对中实现跨架构翻译，其中Ridge投影在言语推理上达到R²=0.50，在数学推理上达到R²=0.40。关键发现是：领域特定的子空间几何是语言模型的普遍属性——投影矩阵在跨领域转移时会灾难性崩溃（平均R²=-3.83），且几何对齐质量与行为纠正率几乎无关（r=-0.07）。

方法概述 / Method

研究者采用线性投影方法，学习一个投影矩阵将教师模型的激活向量映射到学生模型的坐标系统中，然后在生成过程中通过将学生内部状态替换为翻译后的教师表示来干预学生的残差流。实验设计了一个完全交叉的20组异构师生配对矩阵，涵盖混合专家模型、密集模型、代码专用模型和合成训练架构，并使用Ridge回归、排列控制和L1正则化作为对比条件。

实验结果 / Results

Ridge投影在言语推理任务上 consistently 达到R²=0.50，在数学推理上达到R²=0.40，而排列控制和L1正则化分别降至R²=-0.22和R²=0.01。行为纠正率在TruthfulQA上为14.0%-50.0%（平均25.2%），在GSM8K上为8.5%-43.3%（平均25.5%）。双分离实验证实，投影矩阵在跨推理领域转移时平均R²=-3.83，且学生模型表现出领域反转的特征敏感性轮廓——最易引导的言语学生成为最难引导的数学学生。

应用价值 / Applications

该研究为模型对齐和知识迁移提供了新思路：无需微调即可通过表示空间干预纠正模型行为，这对部署后模型安全更新和实时纠错具有潜力。同时，领域特定子空间几何的发现提示，跨领域知识转移需要专门的适配机制，这对多任务学习和模型融合架构设计具有指导意义。方法还可用于理解不同架构模型间的表示对应关系，促进可解释性研究。
