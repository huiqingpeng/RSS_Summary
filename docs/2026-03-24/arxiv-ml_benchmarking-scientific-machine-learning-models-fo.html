Benchmarking Scientific Machine Learning Models for Air Quality Data

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本研究针对北德克萨斯地区（达拉斯县）构建了可解释且全面的空气质量指数（AQI）预测基准，通过对比经典时间序列模型、机器学习与深度学习方法的性能，发现深度学习模型优于简单基线模型，而物理引导方法能提升预测稳定性并确保污染物与AQI关系的物理一致性。研究最终提出了物理引导的最优模型选择指南，为城市区域多时间尺度（1-30天）的AQI预测提供了实用参考。

方法概述 / Method

研究采用美国环保署（EPA）2022-2024年PM2.5和O₃日观测数据，构建1/7/14/30天滞后预测数据集；基准模型包括线性回归、SARIMAX、MLP和LSTM，并创新性地提出物理引导变体（MLP+Physics和LSTM+Physics），通过加权损失函数将EPA断点AQI计算公式作为一致性约束嵌入模型训练。

实验结果 / Results

实验采用时间顺序划分训练测试集，以MAE和RMSE评估显示：深度学习模型整体优于传统方法，物理引导在短预测周期（短周期）及PM2.5、O₃预测中收益最大，显著提升了预测的稳定性和物理一致性，但未明确报告具体数值提升幅度。

应用价值 / Applications

该基准为快速发展的城市区域公共卫生防护提供了模型选择决策支持，特别适用于资源受限地区需要平衡预测精度与计算效率的场景；物理引导框架可推广至其他受物理规律约束的环境预测任务，为科学机器学习在空气质量管理中的落地应用建立了方法论范例。
