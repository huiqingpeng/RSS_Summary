CAMA: Exploring Collusive Adversarial Attacks in c-MARL

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文首次系统性地研究了协作多智能体强化学习（c-MARL）中的共谋对抗攻击问题，提出了三种共谋攻击模式：集体恶意智能体、伪装恶意智能体和间谍恶意智能体，并设计了统一框架CAMA。研究从破坏性、隐蔽性和攻击成本三个维度对攻击效果进行了理论分析，实验表明三种攻击具有叠加的对抗协同效应，能在保持高隐蔽性和长期稳定性的同时强化攻击效果，填补了c-MARL领域共谋对抗学习的研究空白。

方法概述 / Method

论文设计了CAMA（Collusive Adversarial Multi-agent Attack）统一框架，通过智能体观测信息融合和攻击触发控制技术实现策略层面的共谋攻击；三种攻击模式分别对应不同的恶意智能体组织策略，包括集体协作干扰、身份伪装渗透和内部信息窃取。

实验结果 / Results

研究在四个SMAC II地图上进行多维度实验验证，结果表明三种共谋攻击模式之间存在加性对抗协同效应，能够在长时程运行中维持高隐蔽性和稳定性；相较于单智能体扰动攻击和白盒攻击，CAMA框架下的共谋攻击显著提升了攻击成功率并降低了检测风险。

应用价值 / Applications

该研究对无人机集群、社交机器人、具身智能等实际部署的c-MARL系统安全具有重要警示意义，为防御方提供了识别和抵御多智能体共谋攻击的理论基础；同时也可用于安全测试和鲁棒性评估，推动可信多智能体系统的发展。
