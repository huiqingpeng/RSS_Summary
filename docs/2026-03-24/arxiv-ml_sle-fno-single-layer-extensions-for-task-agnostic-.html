SLE-FNO: Single-Layer Extensions for Task-Agnostic Continual Learning in Fourier Neural Operators

arXiv Machine Learning 2026-03-24
Read Original Article →

论文摘要 / Abstract

本文提出了一种名为SLE-FNO的新型架构方法，将单层扩展（Single-Layer Extension, SLE）与傅里叶神经算子（FNO）相结合，以解决科学机器学习中的持续学习（Continual Learning）问题。该方法旨在使模型能够适应分布偏移的新任务，同时避免灾难性遗忘，无需重新访问先前数据。在脉动性动脉瘤血流的图像到图像回归任务中，SLE-FNO在四个顺序且分布外（out-of-distribution）的CFD模拟配置上进行了评估，展现出在可塑性与稳定性之间的最佳平衡，实现了零遗忘且仅需极少额外参数。

方法概述 / Method

SLE-FNO通过在FNO架构中添加轻量级的单层扩展模块来实现持续学习能力，该模块允许模型在学习新任务时动态扩展网络容量，同时保持原有参数不变以保护先前知识。与基于正则化（EWC、LwF）、梯度投影（OGD、GEM）、重放（replay）以及低秩适应（LoRA）等方法不同，SLE采用架构扩展策略，仅在需要时添加少量任务特定参数，实现任务无关的持续学习。

实验结果 / Results

实验基于230个计算流体力学（CFD）模拟，涵盖四种不同的几何构型、边界条件或流动状态配置。结果表明，基于重放的方法和架构方法（PiggyBack、LoRA和SLE-FNO）在知识保留方面表现最佳；其中SLE-FNO在塑料性与稳定性之间取得了最强平衡，实现了零遗忘（zero forgetting），且仅引入极少量的额外参数，优于包括弹性权重巩固（EWC）、无遗忘学习（LwF）、正交梯度下降（OGD）和梯度情景记忆（GEM）在内的多种经典持续学习方法。

应用价值 / Applications

该研究为科学机器学习中的代理模型（surrogate model）部署提供了重要价值，特别是在流体力学、生物医学工程（如动脉瘤血流预测）等领域，当实验条件或模拟工况随时间变化、出现分布偏移时，模型无需重新训练即可持续适应新场景。
SLE-FNO的零遗忘特性和参数高效性使其适用于资源受限环境下的实时模型更新，为数字孪生、实时仿真和个性化医疗等应用提供了可行的技术路径。
