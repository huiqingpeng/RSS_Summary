Learning from Label Proportions with Dual-proportion Constraints

arXiv Machine Learning 2026-03-24

论文摘要 / Abstract

本文针对标签比例学习（LLP）这一弱监督学习问题，提出了一种利用双重比例约束（LLP-DC）的训练方法。该方法同时在包级别和实例级别强制执行比例约束：包级别训练使平均预测与给定比例对齐，实例级别则通过最小费用最大流算法生成满足比例约束的硬伪标签。在多个基准数据集上的大量实验表明，LLP-DC在不同数据集和包大小设置下均优于先前的LLP方法。

方法概述 / Method

论文提出的LLP-DC方法通过双重约束机制进行训练：在包级别，通过损失函数使模型预测的平均值与给定的标签比例保持一致；在实例级别，利用最小费用最大流算法为每个包内的实例分配硬伪标签，确保这些伪标签的比例与给定约束匹配。这种双重约束设计使模型能够从粗粒度的比例标注中学习细粒度的实例级预测。

实验结果 / Results

实验在多个基准数据集上进行，涵盖了不同的包大小配置。结果表明，LLP-DC方法在所有测试场景下均一致性地超越了现有的LLP方法，显示出良好的泛化能力和鲁棒性。具体的性能指标和对比细节可参考原文中的实验表格。

应用价值 / Applications

该研究在隐私保护和标注成本受限的场景中具有重要应用价值，例如医疗诊断（仅提供人群级别的疾病比例而非个体诊断）、社交媒体分析（聚合数据的情感比例）、以及大规模数据标注（降低细粒度标注的人力成本）。
