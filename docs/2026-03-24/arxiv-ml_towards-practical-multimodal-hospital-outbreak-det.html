Towards Practical Multimodal Hospital Outbreak Detection

论文摘要 / Abstract

本研究针对医院感染暴发的快速识别问题，探索了三种全基因组测序（WGS）的替代方案：MALDI-TOF质谱、抗菌药物耐药（AR）模式和电子健康记录（EHR）。研究提出了一种机器学习方法，能够从多模态数据中学习判别性特征以支持暴发检测，并通过多物种评估证明模态整合可显著提升检测性能。此外，研究提出了分层监测范式以减少对WGS的依赖，并通过EHR分析识别出与侵入性操作和高频工作流程相关的高风险污染途径。

方法概述 / Method

研究采用机器学习框架整合三种替代WGS的模态数据（MALDI-TOF质谱、AR模式、EHR），自动学习用于暴发检测的判别性特征。该方法通过多模态融合策略实现跨物种的暴发检测，并构建了分层监测体系以优化资源配置。

实验结果 / Results

多物种评估表明，整合多模态数据能够显著提升医院暴发检测的性能。基于EHR的进一步分析成功识别出特定临床操作（尤其是涉及侵入性设备和高频工作流程的操作）相关的潜在高风险污染途径。

应用价值 / Applications

该研究为资源有限的医疗机构提供了成本效益更高的常规监测替代方案，通过分层监测范式减少了对昂贵WGS的依赖。同时，EHR驱动的风险识别为感染预防团队提供了可操作的干预目标，支持主动性的风险缓解策略制定。
