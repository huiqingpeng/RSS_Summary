论文摘要 / Abstract

本文重新审视了医疗领域基础模型的预训练范式，提出在高质量监督信号可用的临床预测场景中，直接对齐预测目标可能比生成式预训练提供更强的归纳偏置。作者提出了一种监督深度学习框架，通过最大化类间分离与类内方差的比值来显式塑造表征几何结构，在死亡率预测和再入院预测等多个纵向电子健康记录任务上，该方法在匹配模型容量的情况下持续优于掩码、自回归和对比预训练基线，同时提升了判别能力、校准性能和样本效率。

该研究提出了一种以结果为导向的监督表征学习方法，核心思想是通过优化类间分离与类内方差的比值（类似于Fisher判别准则）来显式塑造表征空间的几何结构，使模型容量集中于临床有意义的维度。该方法将传统两阶段（预训练+微调）流程简化为单阶段优化，直接利用结局标签进行端到端的判别式表征学习。

实验结果 / Results

在多个纵向电子健康记录预测任务（包括死亡率和再入院预测）中，该方法在匹配模型容量的条件下，一致性地优于掩码语言建模、自回归建模和对比学习等自监督预训练基线。实验表明，该方法不仅提升了判别性能（discrimination）和校准性能（calibration），还显著提高了样本效率，同时简化了训练流程。

应用价值 / Applications

该研究对低熵、结果驱动的医疗预测领域具有重要实践意义，表明在高质量临床标签可用的场景下，监督学习可作为表征学习的统计最优驱动，挑战了"大规模自监督预训练是强临床性能先决条件"的普遍假设。这一发现可指导临床预测模型的开发策略优化，尤其适用于标注质量高但数据规模有限的专科医疗场景，有助于降低计算成本并加速模型部署。
