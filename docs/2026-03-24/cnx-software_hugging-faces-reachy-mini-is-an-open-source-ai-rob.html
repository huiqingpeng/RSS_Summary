Hugging Face’s Reachy Mini is an open-source AI robot for your computer or Raspberry Pi CM4

CNX Software 2026-03-24

是什么 / What it is

Hugging Face推出的Reachy Mini是一款面向桌面环境的开源AI机器人，提供两个版本：Lite版需连接电脑使用，Wireless版则内置Raspberry Pi CM4实现独立运行。该机器人配备摄像头、四麦克风阵列、扬声器以及9个伺服电机驱动的6自由度头部、可旋转机身和可动天线，专为将AI应用与物理世界交互而设计。
Hugging Face, best known for AI software, has launched the Reachy Mini, an open-source desktop robot available in two versions: the Lite model requiring a computer connection and the Wireless model featuring a built-in Raspberry Pi CM4 for standalone operation. The robot is equipped with a camera, four microphones, a speaker, and nine servo motors controlling a 6-DoF head, rotatable body, and animated antennas, designed specifically for deploying AI applications that interface with the physical world.

为什么重要 / Why it matters

这标志着Hugging Face从纯软件向软硬件一体化AI生态扩展，降低了开发者进入实体机器人领域的门槛；开源特性结合Python/JS SDK与LLM集成能力，有望加速"AI+机器人"应用的民主化创新，推动人机交互从屏幕走向物理空间。
This marks Hugging Face's strategic expansion from pure software into integrated hardware-software AI ecosystems, significantly lowering barriers for developers entering physical robotics. The open-source nature, combined with Python/JS SDK and LLM integration capabilities, is expected to democratize "AI + robotics" innovation and push human-computer interaction from screens into physical space.
