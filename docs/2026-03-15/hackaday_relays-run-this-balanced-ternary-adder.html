是什么 / What it is

这篇文章介绍了一个由 Jeroen Brinkman 构建的**平衡三进制加法器**，它使用继电器逻辑而非传统的二进制电路，采用 -、0、+ 三个状态（对应 -5V、0V、+5V）进行计算，每个数字位称为"trit"（三进制比特）。
This article introduces a **balanced ternary adder** built by Jeroen Brinkman that uses relay logic instead of traditional binary circuits, employing three states of -, 0, and + (corresponding to -5V, 0V, +5V) for computation, with each digit called a "trit" (ternary bit).

这个项目展示了**非二进制计算系统的可行性**，打破了"计算机必须用 0 和 1"的思维定式，为低功耗、高效率计算提供了替代思路；同时用继电器这种可见、可听的机械元件实现，使计算过程直观易懂，极具教育价值。
This project demonstrates the **feasibility of non-binary computing systems**, breaking the assumption that "computers must use 0 and 1," offering alternative approaches for low-power, high-efficiency computing; meanwhile, using relays—visible and audible mechanical components—makes the computation process intuitive and highly educational.
