Incorporating contextual information into KGWAS for interpretable GWAS discovery

arXiv Machine Learning 2026-03-30
Read Original Article →

论文摘要 / Abstract

本研究针对知识图谱全基因组关联分析（KGWAS）框架进行改进，提出通过整合细胞类型特异性上下文信息来提升疾病机制发现的可解释性。研究发现，原始KGWAS中使用的大型通用知识图谱可以被大幅剪枝而不损失统计效能，且整合perturb-seq数据衍生的基因-基因关系能进一步提升性能。最重要的是，使用基于直接perturb-seq证据的稀疏、上下文特异性知识图谱能够产生更一致且生物学上更稳健的疾病关键网络。

方法概述 / Method

本研究采用知识图谱剪枝策略，将通用知识图谱替换为细胞类型特异性的稀疏知识图谱，并整合perturb-seq（大规模基因扰动测序）数据中的基因-基因相互作用关系作为上下文信息。通过对比不同知识图谱配置下的统计效能和生物学可解释性，评估上下文信息整合对KGWAS框架的优化效果。

实验结果 / Results

研究表明，通用知识图谱经大幅剪枝后，下游任务的统计效能并无损失；引入perturb-seq衍生的基因-基因关系后，KGWAS性能进一步提升。使用稀疏的、基于直接perturb-seq证据的上下文特异性知识图谱，相比原始通用知识图谱，能够识别出更一致且生物学稳健性更强的疾病关键网络。

应用价值 / Applications

该研究为GWAS向因果机制转化提供了更精准的方法学工具，特别适用于治疗靶点优先级排序和疾病机制解析。通过整合细胞类型特异性上下文信息，该方法可提高复杂疾病遗传研究的生物学可解释性，对精准医学和药物研发具有重要应用价值，尤其是在需要区分不同细胞类型贡献的疾病研究中。
