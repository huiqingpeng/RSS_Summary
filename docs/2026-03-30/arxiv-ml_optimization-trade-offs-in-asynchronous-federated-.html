Optimization Trade-offs in Asynchronous Federated Learning: A Stochastic Networks Approach

arXiv Machine Learning 2026-03-30

论文摘要 / Abstract

本文针对异步联邦学习中的优化权衡问题，提出了一个随机队列网络框架来建模广义异步SGD（Generalized AsyncSGD）。该框架首次给出了更新吞吐量的闭式表达式，以及达到ε-平稳点所需的通信轮复杂度和期望墙钟时间的上界，形式化刻画了梯度陈旧性与收敛速度之间的权衡。此外，研究还量化了随机时序下的能耗，揭示了收敛速度与能效之间的额外权衡，并提出了基于梯度的路由与并发联合优化策略。

方法概述 / Method

作者利用乘积型网络理论（product-form network theory）构建随机队列网络框架，联合建模客户端和服务器的随机计算时间以及上下行通信延迟。基于该框架，推导出更新吞吐量、通信复杂度和墙钟收敛时间的闭式上界，并扩展至能耗分析，最终设计梯度优化策略实现路由与并发度的联合优化。

实验结果 / Results

在EMNIST数据集上的实验表明，相比标准AsyncSGD，所提策略实现了29%–46%的收敛时间降低和36%–49%的能耗降低。这些结果验证了理论分析所揭示的权衡关系，并证明了联合优化策略在提升异步联邦学习效率方面的有效性。

应用价值 / Applications

该研究为资源受限的边缘计算场景中的联邦学习系统提供了理论指导和优化工具，特别适用于移动设备参与的大规模分布式训练。通过量化收敛速度与能耗的权衡，该方法可帮助设计更高效的异步聚合策略，在物联网、智慧城市等延迟敏感且能效关键的应用中实现可持续的协作机器学习。
