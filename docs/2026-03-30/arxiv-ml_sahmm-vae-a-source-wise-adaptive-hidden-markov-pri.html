SAHMM-VAE: A Source-Wise Adaptive Hidden Markov Prior Variational Autoencoder for Unsupervised Blind Source Separation

arXiv Machine Learning 2026-03-30
Read Original Article →

论文摘要 / Abstract

本文提出SAHMM-VAE，一种用于无监督盲源分离的源级自适应隐马尔可夫先验变分自编码器。该方法突破传统做法，为每个潜在维度分配独立的自适应状态切换先验，使不同维度在训练过程中被拉向不同源特有的时间组织结构。源分离被直接嵌入变分学习过程本身，而非作为外部后处理步骤，通过编码器、解码器、后验参数和源级先验参数的联合优化，实现潜在源轨迹与异构HMM先验的渐进对齐，从而完成源分离。实验表明该方法能在无监督条件下实现源恢复，同时学习到有意义的源级切换结构。

方法概述 / Method

SAHMM-VAE采用耦合优化框架，编码器学习近似混合变换的逆映射进行推断，解码器充当生成式混合模型，两者与后验参数及源级先验参数联合训练。框架包含三个分支实例：高斯发射HMM先验、马尔可夫切换自回归HMM先验，以及具有状态级自回归流变换的HMM状态流先验。

实验结果 / Results

实验验证了所提框架在无监督源恢复任务中的有效性，同时成功学习到可解释的源级状态切换结构。该方法将结构化先验VAE从平滑、混合及流式潜在先验扩展至自适应切换先验，为可解释且潜在可识别的潜在源建模奠定了基础。

应用价值 / Applications

该方法适用于音频分离、神经信号解混、金融时间序列分解等需要无监督盲源分离的场景，其源级自适应机制提供了比传统VAE更丰富的结构建模能力。研究成果为开发可解释、可识别的潜在变量模型提供了理论基础和实用工具，对推进结构化深度生成模型在科学数据分析中的应用具有重要价值。
