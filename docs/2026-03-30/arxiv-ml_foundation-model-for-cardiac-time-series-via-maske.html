论文摘要 / Abstract

本文提出了潜在注意力掩码自编码器（LAMAE）基础模型，专门用于学习心电图（ECG）的可迁移表示。该方法通过自监督预训练显式利用ECG导联间的强结构冗余性，学习跨导联连接机制，实现导联表示的置换不变聚合和自适应加权。在Mimic-IV-ECG数据库上的实验表明，该方法在ICD-10代码预测任务上优于独立导联掩码建模和基于对齐的基线方法。

方法概述 / Method

LAMAE采用潜在注意力机制建模导联间的高阶交互，通过掩码自编码器架构进行自监督预训练。核心创新在于将结构冗余作为监督信号，使模型能够学习跨导联的显式连接机制，而非将各导联视为独立通道。

实验结果 / Results

在Mimic-IV-ECG数据库上的实证研究表明，利用跨导联连接能有效提升表示质量和可迁移性。该方法在ICD-10代码预测任务上表现优异，显著超越了独立导联掩码建模和基于对齐的对比学习基线。

应用价值 / Applications

该研究为心血管疾病的自动化诊断提供了更强大的ECG表示学习方法，可提升临床决策支持系统的准确性。其导联感知的设计思想可推广至其他多通道生理信号分析，具有广泛的医学AI应用前景。
