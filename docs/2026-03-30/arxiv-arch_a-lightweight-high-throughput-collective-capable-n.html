A Lightweight High-Throughput Collective-Capable NoC for Large-Scale ML Accelerators

arXiv Computer Architecture 2026-03-30
Read Original Article →

论文摘要 / Abstract

本文针对大规模机器学习加速器的需求，提出了一种轻量级、支持集合通信的高吞吐网络芯片（NoC）。该设计通过引入直接计算访问（DCA）新范式，使互连结构能够直接访问核心计算资源，以仅16.5%的路由器面积开销实现高效的网内归约操作。实验表明，该架构在GEMM工作负载中可实现最高3.8倍的多播性能提升和2.4倍的归约性能提升，同时节省高达1.17倍的能耗。

方法概述 / Method

论文提出了一种专为下一代ML加速器协同设计的轻量级集合通信能力NoC架构，核心创新是Direct Compute Access（DCA）机制，该机制赋予互连网络直接访问核心计算资源的权限，从而实现高吞吐量的网内归约操作。该设计同时支持高效的屏障同步、可扩展的高带宽多播和归约操作，且仅需较小的硬件开销（路由器面积增加16.5%）。

实验结果 / Results

通过网内硬件加速，该架构在涉及1至32 KiB数据的多播和归约操作上分别实现了2.9倍和2.5倍的几何平均加速。在GEMM工作负载中，通过将通信移出关键路径，该架构可高效扩展到大型网格结构，相比基线单播NoC架构，多播和归约支持分别带来最高3.8倍和2.4倍的预估性能提升，以及最高1.17倍的预估能耗节省。

应用价值 / Applications

该研究对大规模机器学习加速器的设计具有重要价值，特别是在需要高效集合通信的场景中，如分布式训练中的参数同步和梯度聚合。随着技术缩放使单芯片集成数千计算单元成为可能，该轻量级NoC方案能够有效弥合分布式系统与片上系统之间的界限，为下一代AI芯片提供高吞吐、低能耗的通信基础设施，支撑更大规模模型的训练和推理需求。
