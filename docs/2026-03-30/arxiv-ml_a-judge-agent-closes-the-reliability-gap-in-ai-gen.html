A Judge Agent Closes the Reliability Gap in AI-Generated Scientific Simulation

arXiv Machine Learning 2026-03-30
Read Original Article →

论文摘要 / Abstract

本文提出了一种"裁判智能体"(Judge Agent)，通过自动化经典数学验证（适定性、收敛性和误差认证）来大幅提升AI生成科学模拟代码的可靠性。在涵盖12个科学领域的134个测试案例中，该方法将静默失败率从42%降至1.5%；在一项前瞻性基准测试中，12位独立科学家提交的72项盲测任务实现了89%的成功率（95%置信区间：[80%, 95%]），并提供自动化误差边界，而无裁判智能体时成功率仅为53%。

方法概述 / Method

该研究构建了一个裁判智能体，能够自动执行数学验证流程，包括检查问题的适定性、数值方法的收敛性以及计算结果的误差认证。研究团队还形式化了"可模拟性类别S"（simulability class S）来界定可认证与不可认证问题的边界，并开发了一种结构化规范格式，使任何科学计算问题都能被机器读取且与求解器无关。

实验结果 / Results

在134个跨领域测试案例中，裁判智能体将静默失败率从42%显著降低至1.5%；前瞻性盲测基准显示成功率从53%提升至89%。在临床CT重建这一唯一具备统计效力的实验（n=200）中，该流程达到了专家质量的99%。残余的1.5%失败案例集中于分岔点（bifurcation points），即数学可认证性失效的边界区域。

应用价值 / Applications

该研究为科学计算领域提供了可信赖的AI代码生成解决方案，特别适用于需要严格误差控制的场景，如医学成像、物理模拟和工程计算。通过自动化数学验证，该方法使非专业用户也能获得专家级的可靠模拟代码，同时开放的基准测试和规范格式有助于推动整个领域的标准化和可重复性研究。
