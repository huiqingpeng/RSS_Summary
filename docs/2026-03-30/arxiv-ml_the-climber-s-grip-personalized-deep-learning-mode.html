The Climber's Grip -- Personalized Deep Learning Models for Fear and Muscle Activity in Climbing

arXiv Machine Learning 2026-03-30
Read Original Article →

论文摘要 / Abstract

本研究探讨了攀岩运动中感知恐惧与肌肉活动之间的心理生理关系，通过结合统计建模和深度学习技术分析19名攀岩者在先锋攀和顶绳攀过程中的肌电（EMG）、心电（ECG）及手臂运动数据。研究发现，将随机效应整合到深度学习模型中可显著提升预测性能（降低MSE、MAE和RMSE），并首次揭示了肌肉疲劳与先锋攀登中恐惧感增加存在显著相关性。该研究为理解运动中心理与生理状态的复杂交互提供了新的方法论框架。

方法概述 / Method

研究者采用线性混合效应模型分析感知恐惧与生理指标的关系，并进一步扩展至融合随机效应的个性化深度学习方法以捕捉非线性动态特征。实验采集了多模态生理信号（EMG、ECG）和运动数据，并结合攀岩不同阶段的主观恐惧评分进行建模。

实验结果 / Results

引入随机效应的个性化深度学习模型在MSE、MAE和RMSE等误差指标上均优于基准模型；线性混合效应模型分析表明，肌肉疲劳与先锋攀登中的恐惧感知呈显著正相关，而顶绳攀登中未观察到类似效应。这一发现提示攀登方式（风险水平）调节了生理-心理关联的强度。

应用价值 / Applications

该研究为攀岩运动的安全训练和心理技能开发提供了数据驱动的支持，可帮助教练识别高风险情境下的生理疲劳信号以预防事故。方法学上，融合统计随机效应与深度学习的框架可推广至其他需要个性化建模的运动心理学及人机交互场景，如极限运动风险评估或康复训练监测。
