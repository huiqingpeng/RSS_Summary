Are LLM-Enhanced Graph Neural Networks Robust against Poisoning Attacks?

arXiv Machine Learning 2026-03-30
Read Original Article →

论文摘要 / Abstract

本文首次系统研究了LLM增强型图神经网络（LLM-enhanced GNNs）在面对投毒攻击时的鲁棒性。研究发现，与基于浅层嵌入的基线模型相比，LLM增强的GNN在各种攻击设置下表现出显著更高的准确率和更低的相对准确率下降（RDA）。作者进一步分析了这种鲁棒性的关键因素，并从攻防两方面提出了未来研究方向，包括一种新的组合攻击和图净化防御方法。

方法概述 / Method

作者构建了一个全面的鲁棒性评估框架，系统评估LLM增强型GNN在投毒攻击下的表现。该框架组合了8种基于LLM/LM的特征增强器与3种代表性GNN骨干网络，涵盖6种结构投毒攻击（包括目标性和非目标性攻击）和3种文本投毒攻击（字符、词、句子级别），并在4个真实数据集上进行评估，其中包含一个LLM出现后发布的数据集以避免预训练数据泄露。

实验结果 / Results

实验结果表明，LLM增强的GNN在多种攻击设置下均优于浅层嵌入基线，具有更高的准确率和更低的RDA。深入分析揭示，这种鲁棒性主要源于LLM生成的节点表示能够有效编码结构信息和标签信息。基于这些发现，作者提出了新的组合攻击策略和图净化防御机制。

应用价值 / Applications

该研究为安全关键领域的图神经网络应用提供了重要指导，如社交网络分析、推荐系统和知识图谱等场景。评估框架和开源代码为后续研究LLM增强GNN的安全性提供了标准化工具，有助于推动更鲁棒的图学习系统的发展，特别是在需要抵御恶意数据操纵的实际部署环境中。
