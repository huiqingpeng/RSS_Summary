论文摘要 / Abstract

本文提出了 Somax，一个与 Optax 原生兼容的可组合软件栈，用于曲率感知训练（curvature-aware training）。
Somax 将二阶优化方法中的隐藏设计选择（如曲率算子、估计器、线性求解器、预处理器和阻尼策略）显式化为可替换的一等模块，并通过单一 JIT 编译步骤执行。该设计通过静态规划分离计划与执行，复用中间结果以减少冗余计算，实验表明组合选择和规划机制显著影响扩展行为和达到目标精度的时间。

方法概述 / Method

Somax 采用模块化架构，将曲率感知训练的各个组件（曲率算子、估计器、求解器等）封装为可组合的一等模块，用户可通过统一接口灵活替换。系统在执行前推导静态执行计划（包括计算节奏和依赖关系），然后通过专用执行路径运行，实现跨模块的中间结果复用，并与 Optax 的标准梯度变换（动量、权重衰减、学习率调度等）无缝集成。

实验结果 / Results

系统级消融实验表明，不同的模块组合选择对扩展行为和达到目标精度的时间有实质性影响；此外，预规划机制相比无规划的组合方式显著降低了每步计算开销，避免了冗余重计算。

应用价值 / Applications

Somax 为深度学习研究者提供了一套生产级的二阶优化工具，降低了曲率感知方法的使用门槛，适用于需要更快收敛和更好优化稳定性的场景（如大规模语言模型训练、科学计算和强化学习）。其可组合设计也便于研究人员快速实验不同的二阶优化变体，加速算法创新。
