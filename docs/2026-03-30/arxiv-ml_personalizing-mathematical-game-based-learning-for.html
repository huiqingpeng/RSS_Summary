Personalizing Mathematical Game-based Learning for Children: A Preliminary Study

arXiv Machine Learning 2026-03-30
Read Original Article →

论文摘要 / Abstract

本研究针对游戏化数学学习中个性化关卡设计的挑战，提出了一种基于自适应学习理论的AI框架。该框架通过机器学习分类器自动识别和筛选玩家生成的有效游戏关卡，解决了传统游戏化学习系统需要大量高质量关卡且难以匹配学习者能力水平的问题。研究发现随机森林模型在四种分类算法中表现最优，为将AI集成到游戏关卡设计流程中提供了初步证据。

方法概述 / Method

研究者开发了一个名为"Creative Mode"的游戏工具，收集206个由专家和高级玩家创建的数学游戏关卡作为数据集。研究提取游戏特征并比较了四种机器学习分类模型（k近邻、决策树、支持向量机和随机森林），以预测关卡的有效性。

实验结果 / Results

随机森林模型在四种对比算法中表现最佳，能够有效识别有效的玩家生成关卡。该模型通过提取游戏特征实现了对关卡质量的自动分类，为后续个性化推荐奠定了基础。

应用价值 / Applications

该研究为教育游戏开发者提供了AI辅助的关卡筛选工具，可降低人工审核成本并提升内容质量。研究成果可应用于自适应学习系统，根据儿童的学习能力自动匹配个性化数学游戏关卡，从而增强学习者的内在动机和批判性思维能力。
