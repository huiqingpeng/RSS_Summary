Interpretable long-term traffic modelling on national road networks using theory-informed deep learning

arXiv Machine Learning 2026-03-30

论文摘要 / Abstract

本文提出了DeepDemand，一种理论驱动的深度学习框架，用于长期交通流量预测。该框架将出行需求理论的关键组件嵌入神经网络，结合外部社会经济特征和道路网络结构，解决了传统交通模型在可解释性、可迁移性和预测精度之间的权衡问题。在英国战略道路网络（2017-2024年，5088个路段）上的实验表明，该方法在随机交叉验证（R²=0.718）和空间交叉验证（R²=0.665）中均优于多种基线模型，并展现出良好的地理可迁移性和可解释性。

方法概述 / Method

DeepDemand采用双源Dijkstra算法提取局部起讫点（OD）区域并筛选OD对，结合可微分架构建模OD交互和出行时间阻抗函数。该方法将交通理论（如重力模型、出行时间衰减）嵌入深度学习框架，使模型既能捕捉复杂非线性模式，又保持理论可解释性。

实验结果 / Results

在八年观测数据上，DeepDemand随机交叉验证R²达0.718、MAE为7406辆，显著优于线性回归、Ridge、随机森林和传统重力模型基线。空间交叉验证R²为0.665，表明模型具有较强地理泛化能力。可解释性分析揭示了稳定的非线性出行时间衰减模式、关键社会经济驱动因素，以及与主要就业中心和交通枢纽一致的多中心OD交互结构。

应用价值 / Applications

该研究为交通规划部门提供了兼具预测精度和理论可解释性的长期交通建模工具，支持基础设施投资决策、区域发展规划和政策评估。其良好的空间可迁移性使得模型可推广至未观测区域，降低数据收集成本，适用于国家级道路网络的战略规划。
