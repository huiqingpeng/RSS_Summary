From Astronomy to Astrology: Testing the Illusion of Zodiac-Based Personality Prediction with Machine Learning

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本研究使用机器学习框架系统检验了基于星座的人格预测的有效性。研究者构建了一个合成数据集，模拟实际占星系统中星座与性格特征之间的模糊重叠关系，并训练了多种分类器进行预测。实验结果表明，所有模型的预测性能均接近随机水平，证明星座系统缺乏可靠的预测结构，其表面上的"准确性"实际上源于特征普遍性、类别重叠、认知偏差以及解释灵活性等因素。作者最终得出结论：星座系统是文化上持久的叙事框架，而非科学的人格预测工具。

方法概述 / Method

研究者设计了一个受控实验，构建合成数据集：为虚拟个体分配星座标签，并从100个通用人类特质池中为每个星座分配10个描述符，且不同星座的描述符存在故意设计的重叠。随后使用逻辑回归、随机森林和神经网络三种分类器，基于星座特征和干扰协变量预测人格标签，并以标签打乱的对照实验验证结果。

实验结果 / Results

所有分类器在星座预测人格任务上的表现均处于或接近随机期望水平；即使将标签随机打乱，模型仍能获得相当的准确率，表明数据中不存在可被学习的真实预测结构。这一结果在多种模型架构下保持一致，无法通过增加模型复杂度得到改善。

应用价值 / Applications

本研究为批判性审视占星学等流行信念系统提供了严谨的实证方法论框架，可应用于科学素养教育、认知偏差研究以及伪科学辨析等领域。其"幽默学术练习"的定位也展示了如何用正式科研方法解构文化现象，对社会科学研究方法和公众科学传播具有启发意义。
