是什么 / What it is

NVIDIA Mission Control 3.0 是一款专为 AI 工厂设计的集成软件栈，基于 NVIDIA 参考架构构建，提供统一的控制平面来管理大规模 GPU 集群。该版本引入了分层式 API 驱动架构、多租户隔离、智能电力编排和预测性 AIOps 功能，旨在解决 AI 工厂在性能、电力限制和运营可见性方面的核心挑战。
NVIDIA Mission Control 3.0 is an integrated software stack designed for AI factories, built on NVIDIA reference architectures with a unified control plane for managing large-scale GPU clusters. This version introduces a layered API-driven architecture, multi-tenant isolation, intelligent power orchestration, and predictive AIOps capabilities to address core challenges in performance, power constraints, and operational visibility for AI factories. ---

为什么重要 / Why it matters

AI 工厂的经济效益直接取决于可用 GPU 时间和 token 产出效率，1% 的 GPU 时间损失可能导致每小时数百万 token 的损失。随着 AI 工厂规模扩展至数千 GPU 并运行多样化关键任务工作负载，网络拥塞、电力限制、长尾延迟和可见性不足等问题的成本呈指数级增长，传统以利用率为核心的 KPI 已无法满足需求。
The economic performance of AI factories directly depends on usable GPU time and token production efficiency, where a 1% drop in GPU time can mean millions of tokens lost per hour. As AI factories scale to thousands of GPUs running diverse mission-critical workloads, the costs of network congestion, power constraints, long-tail latency, and limited visibility grow exponentially—making traditional utilization-focused KPIs inadequate. ---

我能用什么 / How I can use it

企业可采用 Mission Control 3.0 的模块化架构快速集成最新 NVIDIA 硬件，并通过 API 将能力嵌入自有生态系统；利用多租户隔离功能在共享基础设施上安全地服务多个组织，降低总体拥有成本；通过电力感知调度优化 token 产出与能效平衡，结合预测性 AIOps 实现从被动监控到主动预防的运维转型。
Organizations can adopt Mission Control 3.0's modular architecture to rapidly integrate latest NVIDIA hardware and embed capabilities into their own ecosystems via APIs; leverage multi-tenant isolation to securely serve multiple organizations on shared infrastructure, reducing TCO; and optimize token production versus energy efficiency through power-aware scheduling while transforming operations from reactive monitoring to proactive prevention with predictive AIOps.
