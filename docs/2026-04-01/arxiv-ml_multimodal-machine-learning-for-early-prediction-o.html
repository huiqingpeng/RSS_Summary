Multimodal Machine Learning for Early Prediction of Metastasis in a Swedish Multi-Cancer Cohort

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本研究提出了一种多模态机器学习框架，用于在诊断前一个月预测癌症患者的转移风险。研究分析了来自瑞典卡罗林斯卡大学医院四个癌症队列（乳腺癌、结肠癌、肺癌、前列腺癌）的电子健康记录数据，涵盖结构化数据（人口统计、合并症、实验室结果、用药）和非结构化临床文本。通过比较传统与深度学习分类器在单模态和多模态组合中的表现，发现中间融合策略在多数癌症类型中取得了最佳预测性能，并采用多模态SHAP方法解释模型推理过程。

方法概述 / Method

研究采用多模态融合策略整合电子健康记录中的六类数据，包括人口统计、合并症、实验室结果、药物、临床文本等。实验比较了早期融合、中间融合和晚期融合三种策略，以及传统机器学习模型（如逻辑回归、随机森林）与深度学习模型（如神经网络、Transformer）的性能。研究遵循TRIPOD 2a设计指南，采用80-20的开发-验证数据分割确保评估的严谨性，并开发了多模态适配的SHAP方法进行模型可解释性分析。

实验结果 / Results

中间融合策略在乳腺癌（F1=0.845）、结肠癌（F1=0.786）和前列腺癌（F1=0.845）中取得最高F1分数，肺癌中则以文本单模态模型表现最佳（F1=0.829），中间融合次之（F1=0.819）。深度学习分类器持续优于传统模型。样本量最小的结肠癌队列性能最低，凸显充足训练数据的重要性。
SHAP分析揭示不同癌症类型的模态重要性存在差异，如临床文本在肺癌预测中权重较高。

应用价值 / Applications

该框架可整合至医院电子健康记录系统，为临床医生提供转移风险的早期预警，支持个性化治疗决策和患者分层管理。多模态设计与可解释性分析使其适用于不同医疗场景，有助于优化筛查资源配置、改善患者预后，并可为其他疾病的预测建模提供方法学参考。研究结果对数据规模与模型选择的关联性分析，也为类似研究的实验设计提供实践指导。
