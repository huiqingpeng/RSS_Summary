论文摘要 / Abstract

本文针对多模态主动学习中的独特挑战（包括缺失模态、模态难度差异和交互结构变化）提出了系统性评估框架。通过合成数据集隔离这些潜在陷阱，研究发现模型会发展出不平衡表示，主要依赖单一模态而忽略其他模态，且现有查询方法无法缓解该问题。多模态策略并未一致性地优于单模态策略，凸显了开发模态感知查询策略的迫切需求。

方法概述 / Method

作者构建了一个基于合成数据集的基准测试框架，能够独立控制多模态主动学习的各种陷阱因素，从而实现无混杂噪声的系统评估。该框架对比了单模态与多模态查询策略，并在两个真实数据集上验证了研究发现。

实验结果 / Results

实验表明，模型在多模态主动学习过程中持续产生表示不平衡现象，倾向于过度依赖易学习的模态；现有的主动学习方法未能有效应对这一问题，且专门的多模态查询策略相比单模态策略并未展现出稳定优势。

应用价值 / Applications
