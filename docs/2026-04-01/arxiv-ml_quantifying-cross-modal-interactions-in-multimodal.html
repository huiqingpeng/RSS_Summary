Quantifying Cross-Modal Interactions in Multimodal Glioma Survival Prediction via InterSHAP: Evidence for Additive Signal Integration

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本研究挑战了多模态深度学习在癌症预后中必然受益于跨模态协同作用的普遍假设。作者将InterSHAP（基于Shapley交互指数的度量方法）从分类任务适配到Cox比例风险模型，用于量化胶质瘤生存预测中的跨模态交互作用。通过对TCGA-GBM和TCGA-LGG数据集（n=575）的四种融合架构进行分析，研究发现预测性能与跨模态交互强度呈反比关系：C-index从0.64提升至0.82的架构，其跨模态交互反而从4.8%降至3.0%。方差分解显示各架构中WSI贡献约40%、RNA贡献约55%、交互贡献仅约4%，表明性能提升源于互补信号的累加聚合而非学习到的协同作用。

方法概述 / Method

研究将InterSHAP从分类任务扩展到生存分析的Cox比例风险模型，以量化全切片图像（WSI）和RNA-seq特征融合时的跨模态交互作用。评估了四种融合架构：早期融合、中期融合、晚期融合以及基于注意力机制的融合，通过Shapley值分解将预测方差归因于各单模态及其交互成分。

实验结果 / Results

实验显示高性能架构（C-index 0.82）的跨模态交互强度（3.0%）低于低性能架构（C-index 0.64，交互4.8%），揭示预测性能与交互强度呈负相关。方差分解结果表明，无论架构复杂度如何，WSI和RNA-seq的贡献比例保持稳定（约40%和55%），而交互作用始终维持在低水平（约4%），说明性能提升主要来自有效的特征互补而非模态间的协同学习。

应用价值 / Applications

InterSHAP为多模态融合模型提供了实用的审计工具，可用于比较不同融合策略并指导架构选择。研究发现对隐私保护型联邦学习部署具有启示意义——由于模态间交互较弱，可能支持在保护各模态隐私的前提下进行分布式建模。此外，该研究重新界定了架构复杂度在多模态融合中的作用，提示简单高效的融合策略可能优于复杂的交互学习机制。
