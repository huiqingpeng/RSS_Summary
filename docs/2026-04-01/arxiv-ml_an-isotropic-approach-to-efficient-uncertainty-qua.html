An Isotropic Approach to Efficient Uncertainty Quantification with Gradient Norms

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本文提出了一种轻量化的神经网络预测不确定性量化方法，通过一阶泰勒展开将不确定性表示为预测梯度与参数协方差的函数，并结合参数协方差的各向同性假设，将认知不确定性简化为梯度范数的平方、偶然不确定性简化为点预测的伯努利方差。该方法仅需单次前向-反向传播即可计算，无需训练数据访问，且理论分析表明各向同性假设能避免非训练数据构建的协方差估计所引入的结构化失真。实验表明该估计与MCMC参考值高度一致，且在大语言模型问答任务中揭示了不同基准测试上不确定性信号的本质差异。

方法概述 / Method

核心方法基于两个关键近似：首先利用一阶泰勒展开将预测不确定性分解为梯度与参数协方差的交互项，随后假设参数协方差为各向同性（即协方差矩阵为标量乘以单位矩阵），从而将复杂的参数级不确定性计算简化为梯度范数运算。该方法可直接应用于未经修改的预训练模型，仅需一次前向传播计算预测和反向传播获取梯度。

实验结果 / Results

在合成问题上，该方法与MCMC参考估计的相关性随模型规模增大而增强，验证了各向同性假设在大规模网络中的合理性。在大语言模型问答实验中，组合不确定性估计在TruthfulQA（涉及答案间真实冲突）上达到最高平均AUROC，但在TriviaQA（事实性回忆任务）上表现接近随机水平，表明参数级不确定性与自我评估方法捕获的是不同类型的信号。

应用价值 / Applications

该方法为大规模语言模型（如LLM）提供了一种计算可行的不确定性量化工具，无需访问原始训练数据或昂贵采样，适用于模型部署后的可靠性评估与风险预警。特别是在需要检测模型"幻觉"或答案冲突的场景（如TruthfulQA所示），该不确定性估计可有效预测答案正确性，为高风险应用场景（医疗、法律问答）中的模型可信度判断提供依据。
