Deep Learning-Based Anomaly Detection in Spacecraft Telemetry on Edge Devices

arXiv Computer Architecture 2026-04-01
Read Original Article →

论文摘要 / Abstract

本研究针对航天器异常检测任务，探索了三种深度学习方案（预测阈值法、直接分类法和图像分类法），并利用多目标神经架构优化技术将其部署于边缘设备。实验表明预测阈值法在基线测试中表现最优（CEF0.5达92.7%），经Pareto最优架构优化后，该模型在保持88.8%检测性能的同时，将内存占用降低97.1%至59KB、运算量减少99.4%。研究证实复杂异常检测模型可在CubeSat等严苛硬件约束下实现近实时星上检测。

方法概述 / Method

论文系统比较了三种异常检测范式：基于时间序列预测的阈值判定、端到端直接分类以及将序列转换为图像后的视觉分类方法；采用多目标神经架构搜索（NAS）技术，以检测性能（CEF0.5）、内存占用和计算量（MACs）为优化目标，在Pareto前沿上寻找最优模型配置。

实验结果 / Results

基线实验中预测阈值法以92.7% CEF0.5显著优于其他方案；优化后的轻量化模型实现性能-资源权衡突破——预测阈值模型保留88.8%检测精度的同时，内存骤降至59KB（压缩比34:1），运算量削减至原模型的0.6%；所有优化模型仅需CubeSat星载计算机0.36-6.25%的RAM资源。

应用价值 / Applications

该研究为航天器自主健康管理系统提供了可行的星上智能解决方案，使原本依赖地面站的事后异常分析转变为实时在轨检测，大幅缩短故障响应时间；技术路径可推广至卫星星座、深空探测器等通信延迟高或地面站覆盖受限的场景，在保障任务安全的同时降低对星地链路带宽的依赖。
