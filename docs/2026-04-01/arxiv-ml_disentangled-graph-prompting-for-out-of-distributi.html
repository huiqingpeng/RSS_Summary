Disentangled Graph Prompting for Out-Of-Distribution Detection

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本文针对图神经网络（GNN）在分布外（OOD）检测中的安全问题，提出了一种解耦图提示（Disentangled Graph Prompting, DGP）方法。该方法遵循预训练+提示范式，利用预训练GNN编码器，通过设计类别特定和类别无关的两类提示生成器来捕捉细粒度的分布内（ID）模式。在十个数据集上的实验表明，DGP相比最优基线取得了3.63%的相对AUC提升，有效解决了端到端训练缺乏OOD监督信号导致的次优问题。

方法概述 / Method

DGP包含两个提示生成器：类别特定提示生成器为每个ID类别生成专属提示图，类别无关提示生成器生成跨类别共享的通用提示图，两者通过修改输入图的边权重实现。为训练提示生成器并避免平凡解，作者设计了多种有效的损失函数，包括保持ID样本区分度的分类损失和防止模式坍塌的多样性正则化。

实验结果 / Results

在十个图数据集上的广泛实验验证了DGP的优越性，相比最佳图OOD检测基线实现了3.63%的相对AUC提升。消融实验表明类别特定和类别无关提示的协同作用至关重要，超参数实验进一步证明了方法的稳定性和对关键超参数的鲁棒性。

应用价值 / Applications

DGP可应用于药物分子筛选、社交网络异常检测、金融风控等图数据安全关键领域，在这些场景中系统需要识别训练时未见过的异常样本以规避安全风险。该方法利用预训练模型和轻量提示学习，降低了部署成本，适用于资源受限的工业环境。
