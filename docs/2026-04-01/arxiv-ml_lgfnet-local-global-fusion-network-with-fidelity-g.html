LGFNet: Local-Global Fusion Network with Fidelity Gap Delta Learning for Multi-Source Aerodynamics

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本文提出了局部-全局融合网络（LGFNet），用于融合计算流体力学（CFD）数据、风洞试验数据和飞行试验数据，以同时捕获局部流动结构的高分辨率细节和全局气动趋势的长程依赖关系。该网络通过空间感知层（滑动窗口机制）和关系推理层（自注意力机制）实现多尺度特征分解，并引入保真度差距增量学习（FGDL）策略，将CFD数据作为"低频载体"显式逼近非线性差异。实验结果表明，LGFNet在多种气动场景中实现了最先进的精度和不确定性降低性能。

方法概述 / Method

LGFNet采用双分支架构：空间感知层利用滑动窗口机制提取细粒度局部特征（如激波），关系推理层基于自注意力机制捕获长程流动信息，实现局部-全局特征的协同融合。
FGDL策略将CFD数据视为基准低频信号，通过增量学习显式建模高保真试验数据与CFD仿真之间的非线性差距，避免非物理平滑的同时继承仿真基线的物理趋势。

实验结果 / Results

LGFNet在多样化气动场景中取得了 state-of-the-art 的性能表现，在预测精度方面显著优于现有方法，同时有效降低了预测不确定性。实验验证了该方法在平衡高分辨率局部保真度与宽范围全局依赖性方面的有效性，能够准确捕捉激波等间断特征以及长程拓扑相关性。

应用价值 / Applications

该研究可直接应用于飞行器气动数据库构建、数字孪生系统开发以及飞行包线内的气动性能预测，为航空航天工程中的多源异构数据融合提供高效解决方案。
FGDL策略的增量学习框架具有通用性，可拓展至其他物理仿真与试验数据融合场景，如汽车空气动力学、风力机设计等领域。
