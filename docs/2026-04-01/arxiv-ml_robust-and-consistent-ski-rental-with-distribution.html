Robust and Consistent Ski Rental with Distributional Advice

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本文针对经典的滑雪租赁问题（Ski Rental Problem），提出了一种系统性的框架来整合完整分布预测（distributional advice），以解决传统最坏情况算法和近期"学习增强"方法仅利用点估计预测的局限性。作者分别针对确定性算法和随机算法设计了新策略：确定性场景下提出了具有可调参数的Clamp Policy，随机场景下设计了满足鲁棒性约束的Water-Filling Algorithm。理论分析和实验结果表明，该框架在保证严格鲁棒性的同时，显著提升了预测准确时的性能一致性。

方法概述 / Method

在确定性设置中，作者首先形式化了完美分布预测下的最优阈值购买策略，并设计了Clamp Policy通过将购买阈值限制在安全范围内来处理不完美的预测；在随机设置中，通过Water-Filling Algorithm刻画最优停止分布，该算法在严格满足鲁棒性约束的前提下优化期望成本。两种方法均通过可调参数实现鲁棒性与一致性的权衡。

实验结果 / Results

实验在高斯分布、几何分布和双峰分布等多种分布类型上进行，结果表明本文提出的分布预测框架相比现有的点预测基线方法，在一致性（预测准确时的性能）方面有显著提升，同时保持了可比的鲁棒性（最坏情况保证）。特别地，当预测分布准确时，确定性算法的期望竞争比能够达到经典最优随机算法的理论下界。

应用价值 / Applications

该研究适用于各类需要在不确定环境下做出"租赁vs购买"决策的在线优化场景，如云计算中的按需实例与预留实例选择、网络带宽租赁、设备维护决策等。相比仅提供点估计的预测方法，完整分布预测的整合使决策者能够更精细地量化不确定性，在利用机器学习预测的同时获得可证明的理论保证，对实际系统中预测质量波动较大的情况尤为有价值。
