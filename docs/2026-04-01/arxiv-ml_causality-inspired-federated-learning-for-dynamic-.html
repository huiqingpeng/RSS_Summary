论文摘要 / Abstract

本文针对动态时空图联邦学习（FSTG）中的核心挑战，揭示了现有方法因假设所有特征同等可迁移而导致的虚假表征纠缠、客户端特异性干扰和负迁移的恶性循环。为此，作者提出了基于因果启发的框架SC-FSGL，通过表征级干预显式解耦可迁移的因果知识与客户端特异性噪声。在五个异构时空图数据集上的实验表明，该方法显著优于现有最优方法。

方法概述 / Method

SC-FSGL框架包含两个核心模块：条件分离模块通过客户端条件掩码模拟软干预，实现不变时空因果因子与虚假信号的解耦；因果码本模块则通过聚类因果原型并采用对比学习对齐局部表征，促进跨客户端一致性并支持多样化时空模式的知识共享。

实验结果 / Results

该研究在五个具有不同异构特性的时空图数据集上进行了验证，实验结果表明SC-FSGL consistently优于当前最先进的方法，有效缓解了客户端异构性带来的表征纠缠问题，提升了模型在动态时空图联邦学习场景下的泛化性能。

应用价值 / Applications

该研究适用于智慧城市交通预测、分布式传感器网络分析、多站点气候建模等需要保护数据隐私的动态图学习场景，为跨组织协作的时空数据分析提供了鲁棒的联邦学习解决方案，同时其因果解耦思想可推广至其他存在显著分布异构性的联邦学习任务。
