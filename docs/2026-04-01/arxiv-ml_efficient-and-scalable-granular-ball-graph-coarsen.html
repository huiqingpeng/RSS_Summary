Efficient and Scalable Granular-ball Graph Coarsening Method for Large-scale Graph Node Classification

论文摘要 / Abstract

本文提出了一种面向大规模图节点分类的高效可扩展粒球图粗化方法（Efficient and Scalable Granular-ball Graph Coarsening Method），旨在解决图卷积网络（GCN）在大规模图数据上训练时计算开销过高的问题。该方法通过多粒度粒球图粗化算法将原图压缩为多个子图，时间复杂度为线性且显著低于现有粗化方法，随后通过随机采样粒球子图构建小批量数据进行GCN训练。实验结果表明，该方法在多个数据集上的节点分类任务中展现出优越的性能，同时显著提升了GCN的训练效率和可扩展性。

方法概述 / Method

本文提出的方法采用两阶段策略：首先利用多粒度粒球图粗化算法对原始图进行层次化压缩，生成由粒球构成的子图，该阶段具有线性时间复杂度；随后从这些粒球子图中随机采样构建小批量（minibatch）数据，用于GCN的批量训练。该方法能够自适应地大幅缩减原始图规模，同时保留图结构的多粒度信息。

实验结果 / Results

实验在多个公开数据集上进行节点分类任务评估，结果表明所提方法在保持分类精度的同时显著降低了计算开销。相较于现有的采样和图粗化方法，该方法在线性时间复杂度下实现了更高效的图粗化，且通过自适应的图规模缩减有效增强了GCN的训练效率和可扩展性。

应用价值 / Applications

该方法适用于大规模图数据的节点分类场景，如社交网络分析、知识图谱推理、推荐系统和生物信息学网络分析等领域。其线性复杂度的粗化算法和高效的批量训练机制使其能够处理亿级节点规模的图数据，为实际工业应用中部署GCN模型提供了可扩展的解决方案，显著降低了计算资源需求并加速了模型训练过程。
