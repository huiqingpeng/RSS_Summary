Task Scarcity and Label Leakage in Relational Transfer Learning

arXiv Machine Learning 2026-04-01
Read Original Article →

论文摘要 / Abstract

本研究揭示了关系型迁移学习中的"标签泄漏"问题：由于每个数据库的预测目标数量有限（任务稀缺），学习到的表示会编码任务特定的捷径，损害跨任务迁移能力。作者提出K-Space架构和梯度投影方法来解决这一问题，在RelBench基准上实现了平均+0.145 AUROC的性能提升。研究表明，有限的任务多样性（而非仅数据量）是制约关系型基础模型的关键因素。

方法概述 / Method

论文采用K-Space模块化架构，将冻结的预训练表格编码器与轻量级消息传递核心相结合。核心创新是梯度投影方法：在表示更新过程中识别并移除能够预测标签的方向，从而抑制标签泄漏。该方法通过约束表示学习，防止模型过度拟合特定任务的预测目标。

实验结果 / Results

在RelBench基准数据集上，梯度投影方法使模型在数据集内迁移任务上平均提升+0.145 AUROC，且经常能恢复到接近单任务性能的水平。实验表明该方法有效缓解了因任务稀缺导致的表示退化问题，验证了标签泄漏假设的正确性。

应用价值 / Applications

该研究对构建关系型基础模型具有重要指导意义，特别是在企业数据库、知识图谱等关系数据丰富的场景中。研究成果可帮助改善数据标注资源有限情况下的模型迁移能力，为金融风控、推荐系统、医疗诊断等需要跨表推理的实际应用提供更鲁棒的表示学习方法。
