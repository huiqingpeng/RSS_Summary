# RSS AI Digest

自动化 RSS 文章抓取 + AI 智能总结，生成双语笔记并发布到 GitHub Pages。

## 功能特性

- 🤖 **AI 总结**: 调用 Kimi/Claude/OpenAI API 生成结构化双语摘要
- 📥 **智能抓取**: 自动提取文章正文，去除广告和噪音
- 🔄 **去重机制**: 基于 URL 的去重，避免重复处理
- 📅 **日期过滤**: 只抓取最近 N 天的文章
- 🌐 **静态网站**: 自动生成 GitHub Pages 静态网站
- ⏰ **定时任务**: 支持 crontab 定时执行

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/huiqingpeng/RSS_Summary.git
cd RSS_Summary
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置

创建 `config/api_keys.yaml` 文件：

```yaml
ai:
  api_key: "your-api-key"
  api_base_url: "https://coding.dashscope.aliyuncs.com/v1"  # 阿里云 DashScope
  model_openai: "kimi-k2.5"
```

编辑 `config/settings.yaml` 调整设置：
- `fetch.max_days_back`: 抓取最近几天的文章 (默认 2)
- `ai.provider`: AI 提供商 (openai/claude)
- `git.auto_push`: 是否自动推送到 GitHub

### 4. 运行

```bash
# 测试运行 (不推送)
python scripts/fetch_and_summarize.py --dry-run

# 正式运行
python scripts/fetch_and_summarize.py
```

### 5. 定时任务 (可选)

```bash
# 编辑 crontab
crontab -e

# 添加定时任务 (每 6 小时执行)
0 0,6,12,18 * * * /usr/bin/python3 /path/to/RSS_Summary/scripts/fetch_and_summarize.py >> /path/to/rss-digest.log 2>&1
```

## AI 输出格式

每篇 AI 总结包含三部分:

```
【是什么 / What it is】
(中文描述) ...
(English) ...

【为什么重要 / Why it matters】
(中文描述) ...
(English) ...

【我能用什么 / How I can use it】
(中文描述) ...
(English) ...
```

## 配置说明

### config/settings.yaml

```yaml
ai:
  provider: "openai"           # 或 "claude"
  model_claude: "claude-opus-4-6"
  model_openai: "kimi-k2.5"
  max_tokens: 1000

fetch:
  max_days_back: 2             # 只抓取最近 2 天

storage:
  seen_file: "data/seen.json"
  articles_dir: "articles"
  summaries_dir: "summaries"

git:
  auto_push: true
  branch: "main"
```

### 环境变量

也可以通过环境变量设置 API 密钥:

```bash
export OPENAI_API_KEY="your-key"
export ANTHROPIC_API_KEY="your-key"
```
