【论文摘要 / Abstract】
本文提出了ReLMXEL，一种基于可解释多智能体在线强化学习的内存控制器框架，通过奖励分解动态优化内存控制器参数以降低延迟和能耗。该框架直接集成于内存控制器内部，利用详细的内存行为指标指导决策，并在多样化工作负载上展现出优于基线配置的一致性能提升。通过将可解释性融入学习过程，ReLMXEL不仅提升了性能，还增强了控制决策的透明度，为更具问责性和自适应性的内存系统设计奠定了基础。

【方法概述 / Method】
ReLMXEL采用多智能体在线强化学习架构，直接部署在内存控制器中运行；通过奖励分解技术将优化目标拆解为可解释的子目标，结合细粒度的内存行为指标进行实时决策；该方法能够根据工作负载特定的内存访问模式自适应调整控制器参数。

【实验结果 / Results】
实验评估在多样化工作负载上进行，结果表明ReLMXEL相比基线配置实现了持续的性能提升；优化效果由工作负载特定的内存访问行为驱动，体现出良好的自适应能力；可解释性机制的引入使得学习过程的决策逻辑透明化，便于分析和验证。

【应用价值 / Applications】
ReLMXEL适用于现代计算系统中对能效和延迟敏感的高性能内存子系统，如数据中心服务器、边缘计算设备和AI加速器；其可解释特性满足关键任务系统对决策透明度和可问责性的要求；该框架为下一代自适应、智能化内存控制器的设计提供了可行的技术路径。
