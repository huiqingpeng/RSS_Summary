【论文摘要 / Abstract】
本文提出了基于树搜索的群体相对策略优化方法（Tree-GRPO），用于解决大语言模型智能体强化学习中的稀疏监督问题。该方法将每个树节点表示为完整的智能体交互步骤，通过共享公共前缀提高采样效率，并利用树结构轨迹构建逐步的过程监督信号。理论分析表明，树内层级的群体相对策略优化目标等价于步骤级直接偏好学习，在11个数据集和3类QA任务上验证了树式强化学习相对于链式强化学习的优越性。

【方法概述 / Method】
Tree-GRPO采用树搜索架构进行分组智能体强化学习，其中每个节点代表完整的交互步骤，共享前缀的多个分支实现高效采样。该方法在树内（intra-tree）和树间（inter-tree）两个层级估计分组相对优势，仅依赖结果奖励即可构建步骤级过程监督信号。

【实验结果 / Results】
实验在11个数据集和3种问答任务类型上进行，结果表明Tree-GRPO显著优于传统的链式强化学习方法。树搜索采样机制在固定token预算或工具调用次数下实现了更多的rollout采样，提升了训练效率和最终性能。

【应用价值 / Applications】
该方法适用于需要长期规划和多轮交互的LLM智能体场景，如复杂问答系统、工具调用智能体和自主决策任务。通过缓解稀疏奖励问题并提高采样效率，Tree-GRPO为构建更可靠、更高效的LLM智能体提供了实用的训练框架。
