【论文摘要 / Abstract】

本文提出了一种基于逆动态博弈的算法，用于从多智能体局部纳什均衡交互的演示数据中学习参数化约束。该方法通过混合整数线性规划（MILP）编码交互智能体的Karush-Kuhn-Tucker（KKT）条件，恢复与局部纳什平稳性一致的约束。理论分析证明了该方法能够学习真实安全集和不安全集的内近似，并成功应用于设计满足底层约束的鲁棒运动规划。

【方法概述 / Method】

核心方法是构建混合整数线性规划（MILP）来编码多智能体动态博弈的KKT最优性条件，将约束学习问题转化为可从演示数据中求解的优化问题。该方法能够处理凸约束和非凸约束，通过局部纳什均衡的假设建立智能体交互的数学模型，从而实现约束参数的辨识。

【实验结果 / Results】

在仿真和硬件实验中，该方法准确推断了多种类别（包括凸和非凸）的交互约束，并成功设计了安全的交互式运动规划。实验验证了算法在非线性动态智能体的交互演示中，能够有效恢复底层约束并生成鲁棒满足这些约束的运动轨迹。

【应用价值 / Applications】

该研究适用于多机器人系统的安全协作、自动驾驶车辆的交互决策、以及人机协作等场景。通过从专家演示中学习隐式约束规则，可避免繁琐的手工约束设计，使系统能够快速适应新的交互环境和安全规范，提升自主系统在动态多智能体环境中的安全性和可部署性。
