【论文摘要 / Abstract】
本文提出了 NEO 框架，将预训练的解码器-only 大语言模型适配为一种无需外部工具、直接基于目录生成的统一模型，能够同时支持搜索、推荐和推理任务。NEO 通过 SID（结构化标识符）表示商品，并训练单一模型在共享序列中交错生成自然语言和类型化商品标识符，实现了"语言可操控性"（language-steerability）。在包含超过 1000 万商品的真实大规模多模态目录上的离线实验表明，NEO 持续优于强任务特定基线，并展现出跨任务迁移能力，为将大规模发现能力整合到单一生成模型中提供了可行路径。

【方法概述 / Method】
NEO 将商品表示为结构化标识符（SIDs）并视为独立模态，通过分阶段对齐和指令微调将离散实体表示整合到 LLM 中。文本提示控制任务类型、目标实体类型和输出格式（ID、文本或混合），同时采用约束解码确保生成目录有效的商品标识符，而不限制自由格式文本的生成。

【实验结果 / Results】
NEO 在包含超过 1000 万商品、涵盖多种媒体类型和发现任务（推荐、搜索、用户理解）的真实大规模目录上进行了评估。实验结果显示，NEO 在所有任务上均持续优于强任务特定基线，并表现出跨任务迁移能力，证明了单一语言可操控生成模型在整合大规模发现能力方面的有效性。

【应用价值 / Applications】
NEO 为构建统一的搜索、推荐和推理系统提供了实用方案，可部署于电商平台、内容流媒体、数字图书馆等需要处理大规模异构商品目录的场景。该框架消除了复杂的外部工具编排，降低了系统复杂度和延迟，同时通过端到端优化提升了可靠性和性能，适用于对实时性和准确性有严格要求的生产环境。
