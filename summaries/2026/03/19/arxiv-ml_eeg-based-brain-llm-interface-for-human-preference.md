【论文摘要 / Abstract】
本文提出了一种基于脑电图（EEG）信号的大脑-大语言模型（Brain-LLM）接口，旨在为言语或运动障碍患者（如渐冻症ALS患者）提供替代性的LLM交互方式。研究团队训练了一个能够从EEG信号预测用户满意度的分类器，并将其集成到测试时缩放（TTS）框架中，利用实时神经反馈动态调整图像生成模型的推理过程。实验结果表明，EEG信号能够有效预测用户偏好，为神经反馈与自适应语言模型推理的融合开辟了新的研究方向。

【方法概述 / Method】
该研究采用两阶段技术方案：首先训练一个EEG分类器来估计用户对生成图像的满意度；随后将该分类器的预测结果嵌入测试时缩放（TTS）框架，使模型能够根据用户实时神经反馈动态调整采样策略和生成过程。

【实验结果 / Results】
实验验证了EEG信号可以可靠地预测用户满意度，表明神经活动携带了实时偏好推断的信息；基于EEG反馈的TTS框架成功实现了对人类偏好的对齐生成，证明了神经信号作为LLM控制输入的可行性。

【应用价值 / Applications】
该技术为言语或运动障碍患者提供了无需语言或肢体动作即可控制AI系统的交互方式，具有显著的辅助医疗和无障碍技术价值；同时，该研究为开发更直观、自适应的人机交互界面奠定了基础，可扩展至创意生成、个性化推荐等需要实时理解用户隐性偏好的应用场景。
