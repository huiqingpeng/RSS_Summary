【论文摘要 / Abstract】

本文研究了**变尺度核（variably scaled kernels）**作为构建非平稳高斯过程的有效工具，通过显式修改数据的相关性结构来突破传统平稳核的局限。该方法利用尺度函数动态调整数据间的相关性，能够有效建模具有突变或不连续性的目标函数。实验表明，基于变尺度核的高斯过程在重建精度上有所提升，且其不确定性估计能更好地反映数据的底层结构。

【方法概述 / Method】

论文通过引入**尺度函数（scaling function）**对核函数进行空间自适应调制，使核的"有效距离"随位置变化，从而构造非平稳高斯过程。该方法通过变尺度核功率函数（variably scaled kernel power function）分析预测不确定性，并建立了与经典非平稳核构造之间的理论联系。

【实验结果 / Results】

数值实验验证了变尺度核高斯过程相比传统方法具有**更高的重建精度**，且其不确定性量化能够**自适应地反映数据的空间异质性结构**（如在数据稀疏或函数突变区域自动增大不确定性）。具体性能指标未在摘要中详细列出，但定性结果表明该方法在处理非平稳数据时优势明显。

【应用价值 / Applications】

该方法适用于**具有空间异质性或突变特征的数据建模场景**，如地质统计学中的矿藏分布预测、工程中的断裂力学仿真、以及任何存在物理不连续性或参数剧变的黑箱函数优化问题。其改进的不确定性估计对风险评估和自适应采样策略具有重要价值。
