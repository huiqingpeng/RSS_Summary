【论文摘要 / Abstract】
本文展示了编码智能体（coding agent）的自我引导（bootstrap）能力：从一个926词的规格说明和由现有智能体（Claude Code）生成的初始实现出发，新生成的智能体能够从零开始正确地重新实现同一规格说明。这一工作将编译器构造中的经典自举序列和Lisp中已知的元循环特性复现到了AI编码智能体领域，并揭示了一个重要实践启示——规格说明而非实现才是稳定的记录性产物。

【方法概述 / Method】
研究者采用元循环（meta-circular）方法，利用现有编码智能体（Claude Code）根据简短的自然语言规格说明生成初始实现，随后以此规格说明为输入，驱动新生成的智能体独立完成相同功能的重新实现，从而验证智能体能否基于规格说明自我重建。

【实验结果 / Results】
实验成功实现了编码智能体的自我引导：新生成的智能体能够仅依据926词的规格说明，从零开始正确地重新实现原智能体的功能，证明了规格说明足以作为智能体再生的充分依据，且实现本身具有原则上可随时重新生成的特性。

【应用价值 / Applications】
该研究对AI系统开发和维护具有深远意义：开发者可将精力集中于优化规格说明这一稳定产物，而非维护易变的代码实现；这为智能体的版本迭代、可审计性和长期可维护性提供了新范式，特别适用于需要高度可靠性和可解释性的关键软件系统开发场景。
