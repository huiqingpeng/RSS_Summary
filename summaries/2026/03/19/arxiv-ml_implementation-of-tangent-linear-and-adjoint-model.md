【论文摘要 / Abstract】
本文提出了 TorchNWP，一个用于高效耦合人工智能组件与传统数值模型的编译库工具。该工具解决了 Fortran 开发的操作数值模型与基于 Python 的深度学习框架之间跨语言兼容性差、耦合灵活性不足和数据传输效率低的问题。基于 LibTorch 优化设计了统一的应用层调用接口，将 PyTorch 框架下的深度学习模型转换为静态二进制格式并提供 C/C++ 接口，通过 Fortran/C/C++ 混合编程实现深度学习模型在数值模型中的部署。在此基础上，实现了基于神经网络的切线性模型和伴随模型，可屏蔽神经网络模型的内部结构，简化四维变分资料同化系统的构建过程。

【方法概述 / Method】
TorchNWP 基于 LibTorch 设计统一的应用层调用接口，将 PyTorch 深度学习模型转换为静态二进制格式并提供 C/C++ 接口；采用 Fortran/C/C++ 混合编程技术，使深度学习模型能够嵌入传统数值模型中运行。该工具支持在异构平台上部署，兼容主流神经网络模型，并能够实现不同并行粒度的映射和高效并行执行。

【实验结果 / Results】
TorchNWP 已成功高效集成到 CMA-GFS 和 MCV 等数值天气预报模型中，应用于基于深度学习的物理参数化方案（如辐射、非地形重力波拖曳）的耦合及其切线性和伴随模型的开发。集成过程仅需对原数值模型进行最小程度的代码修改，显著降低了耦合成本，并有效提高了数值天气预报的准确性和效率。

【应用价值 / Applications】
该工具在数值天气预报领域具有重要应用价值，可加速深度学习物理参数化方案与传统数值模型的融合，支持四维变分资料同化系统的构建。其异构平台部署能力和高效并行执行特性使其适用于高性能计算环境，为人工智能与地球系统模式的深度耦合提供了通用技术框架，可推广至气候模拟、海洋预报等其他数值模拟领域。
