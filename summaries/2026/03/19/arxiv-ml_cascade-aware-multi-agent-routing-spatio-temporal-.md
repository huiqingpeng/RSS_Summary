【论文摘要 / Abstract】
本文识别了现有AI推理系统调度器中的"几何盲区"问题：调度器未考虑失败在不同图结构（树状vs循环）中的传播差异——树状委托结构中失败呈指数级级联，而密集循环图中失败趋于自我限制。作者提出了一种轻量级在线几何控制方法，通过时空传播模型和可学习的几何选择器动态切换欧几里得与双曲风险模型。在Genesis 3基准上，该方法在最困难的非树状区域将胜率从64-72%提升至92%，整体胜率达87.2%。

【方法概述 / Method】
论文提出三层路由风险估计框架：（i）欧几里得时空传播基线，（ii）带时间衰减（可选突发激励）的双曲路线风险模型，（iii）基于结构特征的轻量级几何选择器（9-12-1 MLP，仅133参数）。选择器输入包括六个拓扑统计量和三个几何感知信号：BFS壳层增长斜率、循环秩范数和拟合庞加莱曲率，实现欧几里得与双曲几何的自适应切换。

【实验结果 / Results】
与仅使用原生bandit/LinUCB信号（团队适应度和平均节点负载）的基线相比，完整sidecar系统将整体胜率从50.4%提升至87.2%（+36.8个百分点），在树状区域提升尤为显著（+48至+68个百分点）。固定双曲变体在非树状困难区域仅达64-72%胜率，而自适应切换达到92%，验证了级联敏感性分析的有效性。

【应用价值 / Applications】
该研究适用于高能力AI执行图系统（如Genesis 3等多智能体推理架构），可显著提升复杂任务委托链的可靠性和鲁棒性。时空sidecar设计轻量且即插即用，为需要动态路由的分布式AI系统（如模块化大语言模型、多智能体协作平台）提供了低成本的失败传播缓解方案，特别是在金融风控、自动驾驶决策等级联失效代价高昂的领域。
