【论文摘要 / Abstract】
本文提出了一种基于分数阶连续动力学的无增强多视图图对比学习框架。通过将分数阶导数阶数α∈(0,1]设为可学习参数，模型能够自适应地生成从局部特征（小α）到全局聚合（大α）的连续频谱视图，无需手工设计数据增强即可发现信息丰富的多尺度表示。实验表明，该方法生成的嵌入更具鲁棒性和表达力，优于现有最先进的图对比学习基线。

【方法概述 / Method】
核心方法是构建分数阶神经扩散网络作为图编码器，利用分数阶微分方程控制信息在图结构上的传播过程。通过将分数阶阶数α参数化并纳入端到端训练，模型可根据数据特性自动优化扩散尺度，实现自适应的多视图生成。

【实验结果 / Results】
在标准图学习基准测试上的广泛实验表明，该方法能够产生更稳健、更具表达力的节点和图级嵌入，性能超越当前最先进的图对比学习方法。分数阶动力学的引入有效捕获了多尺度结构模式，提升了表示学习的质量。

【应用价值 / Applications】
该方法适用于各类图数据分析任务，如社交网络分析、分子性质预测、知识图谱推理等。其无增强、自适应的特性降低了对领域知识的依赖，为缺乏先验增强策略的复杂图数据提供了通用的表示学习解决方案。
