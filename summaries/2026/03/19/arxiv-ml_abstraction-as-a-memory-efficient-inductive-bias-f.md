【论文摘要 / Abstract】
本文提出了一种名为抽象增强训练（AAT）的新方法，通过在损失函数层面引入抽象表示学习来缓解持续学习中的灾难性遗忘问题。该方法鼓励模型捕捉跨示例的潜在关系结构，无需使用经验回放（ER）缓冲区即可实现稳定的在线学习。实验表明，AAT在零额外内存开销的情况下，性能达到或超越了强ER基线，证明了结构抽象作为无记忆替代方案的有效性。

【方法概述 / Method】
AAT是一种损失级别的训练修改，通过联合优化具体实例及其抽象表示来引入归纳偏置。研究者在两个基准测试上实现该方法：一是通过实体掩码实现抽象的关系数据集，二是通过共享谚语表达抽象的叙事数据集，以捕捉抽象的多面性本质。

【实验结果 / Results】
AAT在严格在线数据流中实现了稳定学习，完全消除了对回放缓冲区的需求。该方法在性能上可比或超越强经验回放基线，且仅需对训练目标进行最小程度的修改，无需任何额外内存开销。

【应用价值 / Applications】
该研究为资源受限环境下的持续学习提供了实用解决方案，特别适用于无法存储历史数据的边缘设备和实时学习场景。AAT的内存高效特性使其在无限复杂、非平稳的真实世界环境中具有广阔应用前景，为开发更具适应性的智能体开辟了新途径。
