【论文摘要 / Abstract】
本研究是一项概念验证性试点研究，证明了在冻结的编码器-解码器大语言模型（LLM）的连续潜在空间中实现持久记忆的可行性。研究针对单一冻结的Flan-T5-XL骨干网络，仅训练小型适配器，实现了六种跨越三个注入点和四种写入机制的架构方法。实验表明，在LoCoMo遗忘曲线评估中，无记忆基线得分为零，而10倍容量下所有六种方法均产生正的记忆召回曲线，揭示了容量是关键设计参数。

【方法概述 / Method】
研究采用冻结的Flan-T5-XL编码器-解码器模型，仅训练小型适配器来读写密集向量形式的记忆库，而非文本级记忆系统。六种架构方法涵盖编码器输入、编码器输出和解码器输入三个注入点，以及四种不同的写入机制，所有读写操作均为可微的向量运算。

【实验结果 / Results】
在LoCoMo数据集的两档容量规模（1×和10×）评估中，无状态基线完全无法回忆信息（得分为零）；10×容量下六种训练后的适配器均产生正的记忆召回曲线；1×容量下三种方法失效崩溃，表明记忆容量是决定性设计因素。记忆库作为紧凑数值数组，可在不改变骨干网络的情况下任意扩展容量。

【应用价值 / Applications】
该方法支持"会话式学习"——推理时记忆库可持续累积而无需梯度计算，适用于需要长期上下文保持的对话系统和个性化AI助手。由于仅需训练小型适配器且记忆可扩展，该方案为资源受限场景下赋予冻结LLM持久记忆能力提供了可行路径，未来结合更大模型、更多数据和更大记忆规模有望获得更强性能。
