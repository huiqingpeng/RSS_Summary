【论文摘要 / Abstract】
本文提出了一种基于Coxeter群Bruhat序生成CSS量子纠错码的新方法。利用Bruhat序作为正则CW复形的面偏序集这一拓扑性质，作者构造了多种具有可控参数的CSS码族，包括稳定子权重受限的码（如[6006, 924, {≤14,≤7}]和[22880,3432,{≤8,≤16}]）以及具有高度不规则稳定子权重分布的码（如[571,199,{5,5}]）。此外，作者还发展了权重约化方法处理少数重稳定子，并提取了可解释为带元校验CSS码的四项链复形。

【方法概述 / Method】
该方法以有限或无限（可约或不可约）Coxeter群为研究对象，利用其Bruhat序的面偏序集区间结构构造链复形，进而转化为CSS量子纠错码。通过不同的构造方法，可以控制稳定子的权重分布特性；对于出现少数高权重稳定子的情况，作者专门开发了权重约化技术。

【实验结果 / Results】
论文获得了多个具有优良参数的CSS码实例：稳定子权重受限的码包括[6006, 924, {≤14,≤7}]（实际稳定子权重为14和9）和[22880,3432,{≤8,≤16}]（权重16和10）；同时构造了具有高度不规则稳定子权重分布的码如[571,199,{5,5}]。此外，还成功提取了长度为3的四项链复形，可解释为具有元校验结构的CSS码。

【应用价值 / Applications】
该研究为量子纠错码设计提供了基于代数拓扑和群论的系统化构造框架，所生成的CSS码可应用于需要特定稳定子权重约束的量子计算场景。具有可控稳定子权重的码有利于降低量子硬件的实现难度，而元校验结构则为发展更高效的量子纠错协议提供了新的数学基础。该方法适用于广泛的Coxeter群类型，具有良好的可扩展性。
