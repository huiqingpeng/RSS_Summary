【论文摘要 / Abstract】

本文提出χ₀框架，针对长程机器人操作中的分布不一致问题（人类示教分布、策略归纳偏置与测试执行分布之间的偏移），通过三大技术支柱实现资源高效的高可靠性操作。该方法仅用20小时数据和8张A100 GPU，在双臂协作服装操作任务（铺平、折叠、悬挂）上达到24小时连续自主运行，成功率较SOTA方法π₀.₅提升近250%。

【方法概述 / Method】

χ₀构建于三个核心技术：（i）模型算术，通过权重空间融合策略整合多样化示教分布；（ii）阶段优势估计器，提供稳定密集的任务进度信号以解决非阶段方法的数值不稳定性；（iii）训练-部署对齐，结合时空增强、启发式DAgger修正与时序分块平滑来弥合分布差距。

【实验结果 / Results】

实验表明，χ₀在双机器人长程服装操作任务中展现出极高的可靠性，能够从任意初始状态连续运行24小时无需人工干预。与当前最先进的π₀.₅相比，χ₀在数据量（20小时）和计算资源（8×A100）显著减少的情况下，成功率提升近250%，证明了其资源效率与鲁棒性。

【应用价值 / Applications】

χ₀适用于需要高可靠性长程自主操作的工业与服务场景，如自动化服装处理、仓储物流中的柔性物体操作等。其资源高效的特性降低了大规模机器人部署的门槛，为生产级机器人系统的实际落地提供了可行路径，尤其在数据获取困难或计算资源受限的场合具有显著优势。
