【论文摘要 / Abstract】

本文提出了CodeGym，一个可扩展的框架，用于合成多样化、可验证且可控的多轮工具使用强化学习环境。该框架将静态编程问题转换为交互式环境，通过提取原子函数或逻辑为可调用的工具，生成涵盖多种工具执行工作流的可验证任务。研究发现，在CodeGym中训练的模型表现出一致的分布外泛化能力，例如Qwen2.5-32B-Instruct在OOD基准测试τ-Bench上获得了8.7个百分点的绝对准确率提升。

【方法概述 / Method】

CodeGym利用代码执行反映现实世界工作流结构模式的特点，将编程问题作为结构化基础来构建工具使用智能体的训练环境。该框架通过从代码中提取原子函数或逻辑并将其封装为可调用的外部工具，将静态编码问题转化为动态交互式多轮环境，支持智能体主动探索和掌握各种工作流。

【实验结果 / Results】

实验表明，不同规模模型和思维链配置在CodeGym中训练后均表现出一致的分布外泛化性。具体而言，Qwen2.5-32B-Instruct在τ-Bench这一OOD基准上实现了8.7个百分点的绝对准确率提升，显著优于依赖静态轨迹监督微调或窄任务强化学习的传统方法。

【应用价值 / Applications】

CodeGym为训练通用工具使用行为提供了可扩展的强化学习环境，适用于需要智能体与外部工具交互的广泛场景，如自动化编程、复杂任务规划和现实世界工作流执行。该研究有助于推动开发能够适应新工具和未见工作流的鲁棒智能体系统，降低对特定任务标注数据的依赖。
