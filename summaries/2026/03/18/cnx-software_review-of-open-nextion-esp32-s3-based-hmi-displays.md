【是什么 / What it is】

Open Nextion 是 ITEAD 推出的新一代 HMI（人机界面）显示屏产品线，与原版 Nextion 不同，它内置 ESP32-S3 无线微控制器，支持 ESP-IDF 框架和 Arduino IDE 开发，并集成 LVGL 图形库，提供 2.8 英寸和 3.5 英寸两种电容触摸屏型号。

Open Nextion is ITEAD's new HMI (Human-Machine Interface) display product line that, unlike the original Nextion, features a built-in ESP32-S3 wireless microcontroller, supports ESP-IDF framework and Arduino IDE development, and integrates the LVGL graphics library, available in 2.8-inch and 3.5-inch capacitive touchscreen variants.

---

【为什么重要 / Why it matters】

这标志着 HMI 开发从封闭生态向开源生态的转变：开发者不再依赖专有的 Nextion Editor，而是可以使用标准的 ESP32 开发工具和 LVGL 进行完全自定义的 UI 开发，同时内置 Wi-Fi 和蓝牙 5 LE 无线连接能力，大幅降低了物联网终端和智能面板的开发门槛。

This marks a shift from closed to open ecosystem in HMI development: developers no longer rely on the proprietary Nextion Editor, but can use standard ESP32 development tools and LVGL for fully customizable UI development, with built-in Wi-Fi and Bluetooth 5 LE connectivity, significantly lowering the barrier for IoT terminal and smart panel development.

【我能用什么 / How I can use it】

嵌入式开发者可直接用 Arduino IDE 快速原型验证，或用 ESP-IDF 进行高性能专业开发；搭配双麦克风阵列、扬声器和 GPIO 扩展板，可快速搭建带语音交互的物联网控制终端；学生和教育工作者可利用其学习 LVGL 图形开发和 ESP32-S3 无线应用，工业场景则适合作为无线传感器显示面板。

Embedded developers can use Arduino IDE for rapid prototyping or ESP-IDF for high-performance professional development; paired with dual-microphone arrays, speakers, and GPIO expansion boards, it enables quick building of IoT control terminals with voice interaction; students and educators can use it to learn LVGL graphics development and ESP32-S3 wireless applications, while industrial scenarios suit it well as wireless sensor display panels.
