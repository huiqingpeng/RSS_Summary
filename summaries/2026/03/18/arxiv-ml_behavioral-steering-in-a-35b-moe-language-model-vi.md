【论文摘要 / Abstract】

本研究在Qwen 3.5-35B-A3B（350亿参数MoE模型）上训练了九个稀疏自编码器（SAE），通过SAE解码的探针向量实现对五种智能体行为特征的识别与操控。研究发现，虽然可以分离出五个特质向量，但它们主要调节的是单一的"能动性轴"（独立行动vs.服从用户），特质特异性效应仅表现为工具类型组成和剂量-反应形状的次要调制。自主性操控在乘数为2时达到Cohen's d = 1.01的效应量，使模型从78%概率向用户求助转变为主动执行代码和搜索网络。

【方法概述 / Method】

研究者采用两阶段方法：首先在模型残差流上训练SAE，然后在SAE潜在激活上训练线性探针，再将探针权重投影回SAE解码器以获得模型原生激活空间中的连续操控向量。该方法绕过了SAE的top-k离散化，实现了无需重新训练的细粒度推理时行为干预。实验设计涵盖50个场景×36个条件，共1,800次智能体 rollout。

【实验结果 / Results】

关键发现包括：（1）自主性操控效应最强（d = 1.01, p < 0.0001），工具使用向量能操控行为（d = 0.39），而风险校准向量仅产生抑制效应；（2）仅在自回归解码阶段进行操控时效应为零（p > 0.35），为GatedDeltaNet架构中行为承诺在prefill阶段计算提供了因果证据；（3）五种操控向量高度共线，主要加载于单一能动性维度。

【应用价值 / Applications】

该研究为大规模MoE语言模型的可解释对齐提供了实用工具，使开发者能够在不重新训练的情况下精细调节模型的自主行为倾向，对构建安全可控的AI智能体系统具有重要价值。同时，关于prefill阶段计算行为承诺的发现为高效推理时的干预策略设计提供了理论指导。
