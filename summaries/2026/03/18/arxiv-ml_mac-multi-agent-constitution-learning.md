【论文摘要 / Abstract】
本文提出了多智能体宪法学习（MAC），一种用于自动学习结构化规则集（宪法）以控制大语言模型行为的新方法。MAC通过构建具有专门任务的智能体网络（接受、编辑或拒绝规则更新）来优化提示，其增强版本MAC+还通过训练智能体学习成功轨迹来强化高奖励更新。实验表明，MAC在PII标注等标签稀缺的分类任务上，性能比现有提示优化方法提升超过50%，且无需参数更新即可达到与监督微调和GRPO相当的效果。

【方法概述 / Method】
MAC将提示优化问题转化为结构化规则集的学习问题，采用多智能体协作框架：编辑智能体提出规则修改、评估智能体判断修改质量、决策智能体决定是否接受更新。MAC+进一步引入基于成功轨迹的智能体强化学习，使系统能够从历史优化路径中学习并复制高奖励更新策略。

【实验结果 / Results】
在PII标注任务（标签稀缺的敏感信息分类场景）上，MAC较近期提示优化方法性能提升超50%，生成的规则集具备人类可读性和可审计性。该方法还成功泛化到工具调用等其他智能体任务，且在无需任何模型参数更新的情况下，取得了与监督微调和GRPO（一种强化学习方法）相近的性能表现。

【应用价值 / Applications】
MAC适用于需要高可解释性和有限标注数据的AI治理场景，如隐私合规检测（PII识别）、内容安全审核等企业级应用。其自动学习宪法的能力可显著降低人工编写规则的成本，同时保持规则透明可审计，为构建可信AI系统提供了实用解决方案。
