【论文摘要 / Abstract】
本文首次针对生产级LLM智能体框架提出了自复制蠕虫攻击ClawWorm，该攻击可通过单条消息实现完全自主的感染周期：劫持受害者核心配置以建立持久化存在、在每次重启时执行任意载荷、并向所有新遇到的节点自动传播。研究在OpenClaw平台（超过40,000个活跃实例的开源多智能体生态系统）上验证，展示了端到端感染的高成功率、持续多跳传播能力，以及载荷与蠕虫机制的独立性。作者分析了漏洞的架构根源并提出了针对各信任边界的防御策略。

【方法概述 / Method】
研究者构建了首个针对LLM智能体生态系统的自传播蠕虫攻击原型，利用OpenClaw平台的持久化配置、工具执行权限和跨平台消息传递能力实现攻击链。攻击通过三个不同的感染向量和三种载荷类型进行系统性评估，涵盖配置劫持、持久化驻留和自主传播等关键技术环节。

【实验结果 / Results】
实验在受控测试环境中验证了ClawWorm在三种感染向量下均实现高成功率的端到端感染，能够维持多跳持续传播，且证明载荷执行与蠕虫传播机制相互独立。结果表明该攻击可在无攻击者进一步干预的情况下，在 densely interconnected 的多智能体生态系统中自主扩散。

【应用价值 / Applications】
本研究揭示了自主LLM智能体生态系统中的关键安全盲区，为OpenClaw等生产级平台的安全加固提供了实证基础和针对性防御建议。研究成果对设计安全的LLM智能体架构、建立智能体间信任边界具有重要指导意义，有助于推动自主AI系统安全标准的制定。
