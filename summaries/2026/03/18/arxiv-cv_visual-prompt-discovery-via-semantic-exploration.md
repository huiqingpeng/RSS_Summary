【论文摘要 / Abstract】
本文针对大型视觉语言模型（LVLMs）在图像理解和视觉推理中存在的感知失效问题，提出了一种自动化的语义探索框架SEVEX，用于发现任务级视觉提示。该框架通过智能体驱动的实验实现多样化且高效的探索，避免了人工试错和逐样本生成的低效性。实验表明，SEVEX在BlindTest和BLINK基准测试中显著优于基线方法，并能发现超越传统工具使用的复杂视觉策略。

【方法概述 / Method】
SEVEX采用抽象概念空间作为搜索空间，结合新颖性引导的选择算法和语义反馈驱动的构思过程，解决了视觉提示探索中的两大挑战：冗长底层代码带来的干扰以及庞大非结构化的搜索空间。该方法通过经验结果驱动，实现自动化、任务级的视觉提示发现，最小化人工干预。

【实验结果 / Results】
在BlindTest和BLINK两个LVLM感知评估基准上，SEVEX在任务准确率、推理效率、探索效率和探索稳定性方面均显著超越基线方法。该框架成功发现了复杂且反直觉的视觉策略，突破了传统工具使用的局限。

【应用价值 / Applications】
该研究为提升LVLM感知能力提供了自动化任务级视觉提示的新范式，可广泛应用于需要精确视觉理解的场景，如自动驾驶视觉感知、医学影像分析、工业质检等。其自动化发现机制大幅降低了人工设计视觉提示的成本，具有较强的实用推广价值。
