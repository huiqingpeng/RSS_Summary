【论文摘要 / Abstract】

本文提出了一个统一框架来量化机器学习模型个性化对预测准确性和可解释性的影响。研究发现个性化对预测和解释的影响可能 diverge（发散）：即使预测性能不变，模型的可解释性也可能提高或降低。作者还推导了检测个性化效应的假设检验的有限样本误差下界，揭示了数据集统计特征（如群体规模、个人属性数量）如何决定效应的可检测性，并在真实表格数据集上验证了某些场景下效应根本无法被检测。

【方法概述 / Method】

作者构建了一个理论框架，将个性化建模形式化为条件分布的变化，并分别量化其对预测性能（通过风险差异）和解释质量（通过特征归因稳定性）的影响。针对实际应用，他们研究了检测个性化效应的标准假设检验，并基于信息论推导了该检验的有限样本误差概率下界，该下界是群体大小、个人属性维度和期望收益等参数的函数。

【实验结果 / Results】

在真实表格数据集上使用特征归因方法（如SHAP）进行验证，研究揭示了多种场景：有些数据集中个性化显著改善了预测和解释，而另一些则显示预测提升但解释质量下降，或两者均无变化。关键的是，作者识别出" fundamentally untestable"（根本不可检测）的场景——由于群体规模过小或属性维度相对于样本量过高，即使存在真实的个性化效应，也无法在统计上可靠地检测到。

【应用价值 / Applications】

该研究为高风险的个性化应用领域（如精准医疗、个性化教育、金融风控）提供了重要的设计指导：帮助从业者预判给定数据集是否足以支持个性化效应的评估，避免在信息不足的数据上部署可能无法验证的个性化系统。同时强调了在部署个性化模型前必须联合评估预测性能和解释质量，防止"可解释性退化"未被察觉的情况，为构建更透明、可信赖的个性化AI系统奠定基础。
