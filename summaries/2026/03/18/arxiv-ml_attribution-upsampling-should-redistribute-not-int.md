【论文摘要 / Abstract】

本文指出可解释AI中的归因方法依赖于为自然图像设计的上采样技术，而非专门为显著性图设计。标准双线性/双三次插值会通过混叠、振铃和边界渗漏等问题系统性地破坏归因信号，产生虚假的高重要性区域。作者提出归因上采样应被视为质量重分布问题而非插值问题，并提出了通用语义感知上采样方法（USU），通过比率形式的质量重分布算子重新构建上采样过程，可证明地保持归因质量和相对重要性排序。

【方法概述 / Method】

作者将归因上采样重新定义为质量重分布问题，其中模型推导的语义边界必须控制重要性的流动方式。USU方法基于比率形式的质量重分布算子，将传统插值视为孤立操作的问题转化为与模型推理耦合的语义感知过程。该方法通过扩展特征归因的公理传统到上采样领域，形式化了忠实上采样的四个期望性质，并证明插值在结构上违反了其中三个。

【实验结果 / Results】

在具有已知归因先验的模型上进行对照实验，验证了USU的形式化保证；在ImageNet、CIFAR-10和CUB-200数据集上的评估表明，USU在归因忠实性方面实现了一致的改进，并生成了质量更优、语义连贯的解释。理论分析证明，三个被违反的约束条件强制任何重分布算子必须采取比率形式，而第四个约束条件在该族中选择唯一的势函数，从而确定USU。

【应用价值 / Applications】

该研究对需要高可信度模型解释的深度学习应用具有重要价值，如医疗影像诊断、自动驾驶决策解释、金融风控模型审计等关键领域。USU可替代现有的归因可视化流程中的标准上采样步骤，提升解释结果的准确性和可靠性，帮助用户更好地理解模型的真实推理过程，避免被插值伪影误导。
