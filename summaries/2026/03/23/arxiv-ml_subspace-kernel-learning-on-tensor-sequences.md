【论文摘要 / Abstract】

本文提出了不确定性驱动的核张量学习框架（UKTL），用于处理高阶张量序列数据。该框架通过比较张量展开得到的模态子空间来实现表达性强且鲁棒的相似性度量，并引入基于软k-means的动态枢轴张量选择和不确定性感知的子空间加权机制，以提升计算效率和可靠性。在NTU-60、NTU-120和Kinetics-Skeleton等动作识别基准上的实验表明，UKTL达到了最先进的性能，同时具有良好的泛化能力和模态级可解释性。

【方法概述 / Method】

UKTL采用基于Nyström方法的核线性化技术，通过软k-means聚类动态学习枢轴张量，实现大规模张量数据的可扩展处理。核心创新在于不确定性感知的子空间加权机制，该机制根据估计的置信度自适应地降低不可靠模态分量的权重，并通过结构化核组合端到端地整合多路和多模态交互。

【实验结果 / Results】

UKTL在多个动作识别基准数据集（NTU-60、NTU-120、Kinetics-Skeleton）上取得了最先进的性能表现，展现出优越的泛化能力。该方法还能提供有意义的模态级洞察，验证了其在复杂张量序列建模中的有效性和鲁棒性。

【应用价值 / Applications】

UKTL可广泛应用于需要处理结构化多路数据的场景，如基于骨骼关键点的动作识别、多模态视频分析和人体行为理解等任务。该框架为张量序列数据提供了一种可扩展、可解释且计算高效的核学习范式，有助于推动计算机视觉和机器学习领域对高阶结构化数据的建模能力。
