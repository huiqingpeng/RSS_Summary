【是什么 / What it is】

Ohm Lab Neuro N6 是一款基于 STMicro STM32N6 微控制器的紧凑型模块化边缘 AI/AI 视觉开发板，采用 Adafruit Feather 尺寸规格，配备 600 GOPS 神经网络加速器，支持通过扩展板连接卷帘快门、全局快门或热成像相机。它兼容 Arduino 开发环境，并可通过高密度连接器扩展 WiFi、以太网、显示屏等功能。

The Ohm Lab Neuro N6 is a compact, modular Edge AI/AI Vision development board based on the STMicro STM32N6 microcontroller in Adafruit Feather form factor, featuring a 600 GOPS neural network accelerator and supporting rolling shutter, global shutter, or thermal cameras via expansion boards. It is compatible with Arduino IDE and can be expanded with WiFi, Ethernet, displays, and more through high-density connectors.

【为什么重要 / Why it matters】

这款开发板代表了边缘 AI 视觉硬件向模块化、低成本方向发展的趋势，600 GOPS 的算力在微控制器级别实现了以往需要更高功耗处理器才能完成的实时推理任务，同时 73 美元的起售价大幅降低了 AI 视觉原型开发的门槛。其灵活的相机扩展选项使其适用于从消费级物联网到工业检测的多种场景。

This board exemplifies the trend toward modular, low-cost edge AI vision hardware, delivering 600 GOPS compute power for real-time inference tasks that previously required higher-power processors, while the starting price of $73 significantly lowers the barrier for AI vision prototyping. Its flexible camera expansion options make it suitable for diverse applications ranging from consumer IoT to industrial inspection.

【我能用什么 / How I can use it】

开发者可以利用 Arduino IDE 或 STM32Cube IDE 快速部署 YOLOv8 等目标检测模型，结合 NeuroStudio 工具实现即插即用的视觉调试；对于需要完整交互界面的项目，可选择 Vision TFT 扩展套件集成触摸屏、ToF 传感器和音频输出。该套件特别适合构建智能监控、环境感知机器人或工业缺陷检测等需要低延迟本地推理的原型系统。

Developers can rapidly deploy object detection models like YOLOv8 using Arduino IDE or STM32Cube IDE, with plug-and-play visual debugging via the NeuroStudio tool; for projects requiring full interactive interfaces, the Vision TFT expansion kit integrates touchscreen, ToF sensor, and audio output. The kit is particularly well-suited for building prototype systems requiring low-latency local inference, such as smart surveillance, environmental sensing robots, or industrial defect detection.
