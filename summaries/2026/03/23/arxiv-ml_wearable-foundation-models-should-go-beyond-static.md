【论文摘要 / Abstract】

本文指出可穿戴基础模型（WFMs）目前主要依赖静态编码器进行短期、回顾性预测，难以应对慢性病等需要长期追踪的健康问题。作者提出WFMs必须从静态编码器转向支持纵向、预测性健康推理的架构，并明确了实现这一转变所需的三大基础变革：结构化丰富的数据、纵向感知的多模态建模和智能体推理系统。

【方法概述 / Method】

本文是一篇观点性/综述性论文，未提出具体技术实现，而是从系统层面论证WFMs架构转型的必要性。作者通过分析现有WFMs的局限性，构建了一个三层次的变革框架，为下一代可穿戴健康AI的设计提供理论指导。

【实验结果 / Results】

本文不涉及具体实验或性能评估，属于概念性研究。其核心"结果"是确立了三个关键转型方向：整合多模态长期个人轨迹数据、发展长上下文推理与时序抽象能力、以及构建支持规划与决策的智能体系统。

【应用价值 / Applications】

该研究为可穿戴设备从健身追踪向慢性病管理、早期风险预警和个性化健康干预演进提供了路线图。其提出的框架可指导开发能够持续监测、预测并主动支持用户健康的智能系统，对老龄化社会和精准医疗具有重要战略意义。
