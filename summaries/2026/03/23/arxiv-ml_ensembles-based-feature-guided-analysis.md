【论文摘要 / Abstract】
本文提出了基于集成的特征引导分析（EFGA）方法，旨在解决现有特征引导分析（FGA）技术中规则召回率较低的问题。EFGA通过将FGA提取的规则组合成集成模型，利用聚合策略提高规则的适用性。实验表明，EFGA在MNIST和LSC数据集上显著提升了训练和测试召回率（分别提升28.51%/33.15%和25.76%/30.81%），同时仅造成测试精度微小下降（-0.89%/-0.69%），为深度神经网络的可解释性提供了更实用的解决方案。

【方法概述 / Method】
EFGA的核心方法是将FGA提取的多个规则按照用户定义的聚合准则组合成规则集成，通过规则间的互补性扩展其适用场景。本文实现了三种不同的聚合准则，允许在精度与召回率之间进行灵活权衡，且该框架具有可扩展性，支持用户开发自定义聚合策略。

【实验结果 / Results】
在MNIST和LSC两个基准数据集上的评估显示：采用兼顾精度与召回率的聚合准则时，EFGA相比FGA在训练召回率上分别提升28.51%和33.15%，测试召回率提升25.76%和30.81%；而测试精度仅轻微下降0.89%和0.69%。不同聚合准则呈现出精度与召回率之间的替代权衡关系。

【应用价值 / Applications】
EFGA可应用于需要高覆盖率的深度神经网络解释场景，如医疗诊断、自动驾驶等安全关键领域，帮助开发者更全面地理解模型决策边界。该方法为模型调试、故障分析和监管合规提供了更可靠的解释工具，同时其可扩展的聚合框架支持根据具体应用需求定制解释策略。
