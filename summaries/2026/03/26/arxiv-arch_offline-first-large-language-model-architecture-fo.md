【论文摘要 / Abstract】
本文提出了一种面向低连接环境、以离线优先为核心的大型语言模型架构，用于AI辅助学习。该系统通过量化语言模型实现本地推理，并采用硬件感知的模型选择策略，使其能够在仅配备CPU的低配置设备上运行。系统还设计了自适应响应级别功能，可根据学生能力生成不同复杂度（简易英语、初中、高中、技术级）的解释。在有限连接条件下的中学和高校部署评估表明，该系统在老旧硬件上运行稳定、响应时间可接受，且用户对自主学习的支持效果评价积极。

【方法概述 / Method】
该架构采用模型量化技术压缩LLM以实现本地推理，并结合硬件感知选择机制动态匹配设备性能与模型规模。系统内置自适应响应生成模块，通过提示工程或微调方式控制输出复杂度，提供四个递进式解释层级。

【实验结果 / Results】
系统在选定中学和高校的有限连接环境中完成部署测试，结果显示：能够在遗留硬件（legacy hardware）上稳定运行；响应时间达到可接受水平；用户在可用性、感知响应质量和教育影响等维度给出积极反馈，证实了离线LLM用于AI辅助教育的可行性。

【应用价值 / Applications】
该研究为网络基础设施薄弱的发展中地区提供了可落地的AI教育解决方案，特别适用于农村学校、偏远地区高校及网络不稳定场景。离线优先设计降低了技术门槛和运营成本，有助于推动教育公平，使资源受限地区的学生也能获得个性化AI辅导支持。
