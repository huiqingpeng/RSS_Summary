【论文摘要 / Abstract】

本文从最小描述长度（Minimum Description Length, MDL）原则的新视角重新解释了深度神经网络中的"简单性偏置"（simplicity bias）现象。作者将监督学习形式化为最优两部分无损压缩问题，揭示了简单性偏置如何通过模型复杂度（描述假设的成本）与预测能力（描述数据的成本）之间的基本权衡来支配神经网络的特征选择。该理论预测，随着训练数据量的增加，学习器会经历从简单虚假捷径到复杂特征的质量相变，且仅在数据编码成本的降低足以抵消增加的模型复杂度时才会发生。研究进一步识别出不同的数据制度：增加数据可通过排除平凡捷径来促进鲁棒性，而限制数据则可作为一种基于复杂度的正则化手段，防止学习不可靠的复杂环境线索。

【方法概述 / Method】

论文采用理论分析与实验验证相结合的方法。理论层面，运用MDL框架将监督学习重新表述为两部分最优压缩问题：第一部分编码模型（假设）的描述长度，第二部分编码数据在模型下的残差描述长度。实验层面，构建半合成基准测试（semi-synthetic benchmark），对比神经网络的特征选择轨迹与最优两部分压缩器的解轨迹，以验证理论预测。

【实验结果 / Results】

实验验证了理论预测的核心结论：神经网络的特征选择行为与最优两部分压缩器的解轨迹高度一致。具体而言，随着训练数据量的变化，神经网络确实表现出从简单虚假特征向复杂真实特征过渡的质量相变行为，且该过渡发生的临界点与MDL理论预测的"数据编码成本降低足以补偿模型复杂度增加"的条件相符。

【应用价值 / Applications】

该研究为理解神经网络的泛化行为和鲁棒性提供了新的理论工具，有助于指导数据收集策略和正则化设计。实际应用价值包括：（1）帮助机器学习从业者预判特定数据规模下模型可能学习的特征类型，从而主动干预避免虚假关联；（2）为数据高效学习场景（如少样本学习、域适应）提供理论依据，说明在特定情况下限制数据量可能作为一种隐式的复杂度正则化机制。
