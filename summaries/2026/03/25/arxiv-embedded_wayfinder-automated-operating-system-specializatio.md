【论文摘要 / Abstract】
Wayfinder 是一个完全自动化的操作系统配置专业化框架，无需专家知识即可针对特定应用优化 OS 配置。该框架通过神经网络驱动的搜索算法，动态学习配置参数对性能的影响并避免无效配置，在编译时、启动时和运行时三个层面实现专业化。实验表明，Wayfinder 相比默认配置可实现最高 24% 的应用性能提升和 8.5% 的内存使用降低，且知识可在相关应用间迁移。

【方法概述 / Method】
Wayfinder 由自动化 OS 基准测试平台和基于神经网络的搜索算法组成，通过在线学习识别对性能影响最大的配置参数及其有效取值范围。系统支持可选的预训练模型复用机制，将在一个应用上训练得到的模型用于加速相关应用的专业化过程，从而解决配置空间巨大、评估耗时且存在大量无效配置的挑战。

【实验结果 / Results】
在两个操作系统、四个应用和两个目标指标上的评估显示，Wayfinder 能够全自动发现专业化配置，实现最高 24% 的性能提升和 8.5% 的内存缩减。与随机搜索和贝叶斯优化相比，神经网络方法能更快收敛到优质解，并成功实现了相关知识在相关应用间的迁移。

【应用价值 / Applications】
Wayfinder 适用于需要为特定工作负载和硬件环境优化 OS 性能的数据中心、云计算和边缘计算场景，可显著降低 OS 调优的技术门槛和人力成本。该框架的自动化能力使其特别适合需要频繁部署多样化应用的 DevOps 环境，以及资源受限且对性能敏感的嵌入式系统。
