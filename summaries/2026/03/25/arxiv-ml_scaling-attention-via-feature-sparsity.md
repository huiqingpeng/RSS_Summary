【论文摘要 / Abstract】
本文提出了一种新的注意力机制优化方向——特征稀疏性（feature sparsity），以解决Transformer在长上下文扩展中的计算瓶颈问题。作者提出了Sparse Feature Attention (SFA)方法，将查询和键表示为k-稀疏编码，将注意力计算复杂度从Θ(n²d)降低到Θ(n²k²/d)。同时开发了IO感知的FlashSFA内核，可直接在稀疏重叠上操作而无需物化密集分数矩阵。实验表明，SFA在GPT-2和Qwen3预训练中匹配密集基线的同时，速度提升达2.5倍，FLOPs和KV缓存减少近50%，并在长上下文检索任务中保持准确性。

【方法概述 / Method】
SFA的核心思想是在特征维度而非序列维度上引入稀疏性：将高维查询和键向量投影到低维稀疏表示（k-稀疏编码），通过仅计算非零特征的重叠来近似注意力分数。为实现高效计算，作者设计了FlashSFA内核，扩展FlashAttention以直接处理稀疏特征的重叠计算，避免生成完整的n×n注意力矩阵，从而降低内存访问开销。

【实验结果 / Results】
在GPT-2和Qwen3的预训练实验中，SFA与密集注意力基线达到相当的困惑度（perplexity），同时实现最高2.5倍的训练速度提升，FLOPs和KV缓存内存占用减少约50%。在合成长上下文检索任务和下游基准测试上，SFA保持了长距离依赖建模能力，显著优于短嵌入基线（后者因特征多样性坍缩而性能下降），证明了特征稀疏性在保持表达能力的同时实现计算效率的优势。

【应用价值 / Applications】
该研究为超大规模语言模型的长上下文扩展提供了新的技术路径，特别适用于需要处理超长文档（如法律、科学文献）或多模态长序列的应用场景。特征稀疏性与现有的序列级稀疏方法（如局部窗口、核近似）正交互补，可进一步叠加使用以实现更大程度的效率提升，对降低大模型推理成本和扩展上下文窗口具有重要工程价值。
