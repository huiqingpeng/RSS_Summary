【论文摘要 / Abstract】

本文提出了物理信息条件薛定谔桥（PICSB），用于解决偏微分方程（PDE）系统的数据同化问题。该方法能够在仅有稀疏高保真（HF）观测数据、无需完整HF监督的情况下，将低保真（LF）先验信息传输至观测条件化的HF后验分布，且无需测试时优化。通过迭代代理端点刷新机制和直接将PDE残差纳入训练目标，PICSB实现了极快速度的时空场重建，同时在流体PDE基准测试中保持了具有竞争力的精度。

【方法概述 / Method】

PICSB基于薛定谔桥理论构建生成模型，以LF模拟结果为初始分布、HF观测条件化为目标，学习两者之间的最优传输映射。核心创新包括：迭代代理端点刷新方案以替代缺失的HF端点监督，以及在训练过程中直接嵌入PDE物理约束，并通过硬条件化在采样全程强制执行稀疏观测约束。

【实验结果 / Results】

在流体PDE基准测试上的实验表明，PICSB在稀疏HF监督条件下实现了极快的时空场重建速度，同时保持了与需要完整HF监督的现有方法相当甚至更好的精度。该方法消除了传统方法中耗时的测试时优化瓶颈，显著提升了计算效率。

【应用价值 / Applications】

PICSB适用于气象预报、海洋模拟、湍流分析等需要实时或近实时数据同化的时关键应用场景。其实际价值在于：仅需稀疏观测即可训练，大幅降低数据获取成本；推理阶段无需迭代优化，满足实时决策需求；同时严格保持物理一致性和观测约束，确保结果的科学可靠性。
