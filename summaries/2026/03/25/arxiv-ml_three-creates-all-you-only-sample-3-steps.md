【论文摘要 / Abstract】
本文发现标准的时间步条件机制是扩散模型少步采样的关键瓶颈，提出了一种名为多层时间嵌入优化（MTEO）的新方法。该方法冻结预训练扩散主干网络，仅从参考轨迹中蒸馏少量逐层、逐步的时间嵌入参数，实现了与现有ODE求解器的即插即用兼容。实验表明，MTEO在少步采样任务上达到最先进的性能，显著缩小了基于蒸馏的方法与轻量级方法之间的差距，且训练参数量极小、不增加推理开销。

【方法概述 / Method】
MTEO基于层相关的去噪动力学观察，将时间嵌入优化分解到不同网络层，为每个采样步骤学习专属的多层时间嵌入。该方法保持预训练扩散模型的权重完全冻结，仅优化极少量（约0.1%）的时间嵌入参数，通过蒸馏高质量参考轨迹实现高效训练。

【实验结果 / Results】
在多个数据集和骨干网络上，MTEO仅需3步采样即可生成高质量图像，在少步采样设置下达到当前最优性能。该方法成功弥合了需要大量训练资源的蒸馏方法与轻量级加速方法之间的性能差距，同时保持了与标准ODE求解器的兼容性。

【应用价值 / Applications】
MTEO可广泛应用于需要实时图像生成的场景，如交互式内容创作、游戏渲染和移动设备部署。其即插即用特性使其易于集成到现有扩散模型基础设施中，而极低的训练成本和零推理开销使其特别适合资源受限的商业应用和大规模部署。
