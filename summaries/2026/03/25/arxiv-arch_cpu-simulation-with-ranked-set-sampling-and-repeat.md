【论文摘要 / Abstract】
本文重新审视了计算机体系结构仿真中的随机抽样方法，提出将统计学中的排序集抽样（Ranked Set Sampling, RSS）技术应用于CPU仿真。研究表明RSS能够比简单随机抽样产生更紧致的置信区间（宽度减少高达50%），并通过重复子抽样方案进一步将最大观测误差从35%降至10%。该方法在SPEC CPU 2017整数应用上实现了平均误差低于2%、最大误差3.5%的优异性能。

【方法概述 / Method】
论文采用排序集抽样（RSS）技术替代传统简单随机抽样，通过利用样本标准差与样本均值的关系来评估不同微架构配置下的置信水平变化。同时设计了重复子抽样方案，结合两种选择标准来识别具有代表性的仿真区域。

【实验结果 / Results】
RSS技术将置信区间宽度减少高达50%；重复子抽样方案在固定样本量下将最大观测误差从35%降至10%。在SPEC CPU 2017整数应用测试中，模拟30个各100万指令的区域时，平均误差低于2%，最大误差仅3.5%，且更明智的子样本选择策略可进一步提升准确性。

【应用价值 / Applications】
该方法可广泛应用于计算机体系结构研究与设计中的性能评估与建模，显著减少全程序端到端仿真所需的时间成本。对于处理器设计迭代、微架构优化以及新型CPU配置的性能预测具有重要的工程实践价值，能够在保证统计代表性的前提下大幅提升仿真效率。
