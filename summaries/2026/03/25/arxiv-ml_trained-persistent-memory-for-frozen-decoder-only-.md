【论文摘要 / Abstract】

本文探讨了为冻结的仅解码器语言模型（decoder-only LLMs）添加持久记忆的方法。研究者将六种记忆适配方法（前缀、并行交叉注意力、KV扩展、Hebbian记忆、上下文门控分支和基于槽的稀疏写入）应用于冻结的GPT-2模型，仅训练小型记忆适配器θ_mem。实验发现，在1×容量下，具有强架构先验的三种方法（交叉注意力、Hebbian记忆、槽写入）显著优于其他方法（记忆保持率7-18% vs <0.4%），但在10×容量下所有方法收敛，表明差距源于架构设计而非本质限制。该研究确立了持久潜空间记忆作为跨越主要transformer架构家族的通用范式。

【方法概述 / Method】

研究者在冻结的GPT-2骨干网络上训练小型记忆适配器θ_mem，通过共享的写入规则但不同的读取注入方式实现记忆功能。关键创新在于将记忆注入从编码器-解码器架构中的交叉注意力改为仅解码器架构中的自注意力KV前缀或并行分支，探索了六种记忆读取机制以适应无交叉注意力通路的decoder-only设置。

【实验结果 / Results】

在LoCoMo长上下文记忆基准测试上，研究发现显著的归纳偏置二分现象：1×容量时，交叉注意力（M.2）、Hebbian记忆（M.4）和槽写入（M.6）三种方法实现7-18%的记忆保持率和7-10的知识增益ΔK，而前缀、KV扩展和上下文门控分支三种方法几乎完全失败（<0.4%）。但当容量扩大至10×时，所有六种方法性能收敛，证明架构先验的差异可通过规模弥补。

【应用价值 / Applications】

该研究为现有大语言模型（如GPT系列、LLaMA等decoder-only模型）提供了无需重新训练主干网络即可赋予持久记忆能力的实用方案，可显著降低长对话、个性化助手和持续学习等应用中的上下文窗口压力和计算成本。同时，该框架与先前编码器-解码器及脑启发模块工作共同构建了跨架构的通用持久记忆范式，为开发更具状态感知能力的AI系统奠定了基础。
