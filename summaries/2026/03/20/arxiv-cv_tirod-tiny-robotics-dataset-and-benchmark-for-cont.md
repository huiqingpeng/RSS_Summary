【论文摘要 / Abstract】

本文提出了 TiROD（Tiny Robotics Object Detection），这是一个专门针对微型移动机器人平台设计的视频数据集与基准测试框架。该研究旨在评估轻量级持续学习策略在低分辨率、高噪声图像上的目标检测性能，以解决微型机器人在动态环境中面临域迁移时的适应性问题。作者使用 NanoDet 轻量级检测器对多种持续学习策略进行了全面评估，揭示了在资源受限设备上开发鲁棒高效持续学习目标检测器的关键挑战。

【方法概述 / Method】

研究采用真实微型移动机器人（Freenove Smart Car）的板载摄像头采集视频数据，构建了包含多个域和类别的 TiROD 数据集。实验选用 NanoDet 作为基线检测器，系统评估了包括经验回放、知识蒸馏、正则化方法在内的多种持续学习策略在不同场景下的表现。

【实验结果 / Results】

基准测试结果表明，现有的持续学习方法在微型机器人平台上存在显著局限性，特别是在处理低质量视觉输入时难以平衡模型适应性与稳定性。研究发现轻量级检测器在持续学习过程中面临灾难性遗忘和计算效率的双重挑战，为后续算法优化指明了方向。

【应用价值 / Applications】

该研究为微型机器人（如无人机、微型无人车）在真实动态环境中的视觉感知提供了标准化评估工具，可直接应用于自主导航、环境巡检、搜索救援等场景。TiROD 基准有助于推动适用于边缘计算设备的持续学习目标检测技术发展，促进资源受限智能系统的实际部署。
