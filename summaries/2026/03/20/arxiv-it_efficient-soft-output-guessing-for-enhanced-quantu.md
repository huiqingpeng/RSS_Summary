【论文摘要 / Abstract】
本文提出了一种针对量子Tanner码的广义低密度奇偶校验（LDPC）解码框架，该框架采用软输出猜测随机加性噪声解码（SOGRAND）技术。通过对整个分量码进行软输出解码，有效缓解了陷阱集和环路问题，提升了收敛性能。结合有序统计解码（OSD）后处理，SOGRAND在逻辑错误率方面比标准置信传播加OSD基线方法提高了多达三个数量级，为基于Tanner码的新型量子码的可扩展解码提供了可行路径。

【方法概述 / Method】
论文核心方法是将软输出猜测随机加性噪声解码（SOGRAND）应用于量子Tanner码的广义LDPC解码框架中，通过软输出方式解码完整的分量码而非硬判决。该方法进一步与有序统计解码（OSD）后处理相结合，形成完整的解码流程。

【实验结果 / Results】
实验表明，SOGRAND结合OSD后处理在逻辑错误率性能上显著优于标准置信传播加OSD的基线方法，提升幅度达到三个数量级。该方法有效解决了传统解码中的陷阱集和环路问题，改善了收敛特性。

【应用价值 / Applications】
该研究为量子Tanner码的高效解码提供了新的技术方案，适用于未来大规模量子纠错系统。其显著提升的解码性能有助于推动基于Tanner码的量子LDPC码在实际量子计算中的可扩展应用，对实现容错量子计算具有重要意义。
