【论文摘要 / Abstract】
本文提出了一种基于随机游走的新方法来分析码基密码系统的安全性，特别是针对与码的总变差距离密切相关的平滑参数。作者利用公平划分（equitable partitions）推导出了随机游走总变差距离的不等式，并证明该边界推广了现有关于有限阿贝尔群的结果，为码平滑边界分析提供了替代性的理论工具。

【方法概述 / Method】
论文采用随机游走理论替代传统的傅里叶变换方法，通过构建公平划分来刻画随机游走的收敛性质。该方法在局部紧阿贝尔群上建立新的分析框架，避免了依赖傅里叶变换的复杂性。

【实验结果 / Results】
论文主要贡献在于理论推导而非实验验证，证明了所提出的基于随机游走的边界能够涵盖并推广现有针对有限阿贝尔群的平滑边界结果，建立了更一般化的理论不等式。

【应用价值 / Applications】
该研究为后量子密码学中码基密码系统（如基于纠错码的加密方案）的安全性分析提供了新的数学工具，有助于更精确地评估密钥尺寸和安全参数，推动抗量子计算攻击的密码标准化进程。
