【论文摘要 / Abstract】

本文首次将$L_2$-稳定性理论引入异构环境，为模型融合建立了统一的泛化理论框架。研究通过严格解耦合并模型的超额风险为优化误差和泛化误差，揭示了"优化-泛化权衡"的基本规律，解释了过训练专家导致灾难性融合崩溃的理论悖论。该框架不仅统一解释了线性（TA、AdaMerging）和非线性（TIES、DARE）融合算法，还推导出了可定量预测超参数影响的缩放定律，并通过20个视觉分类任务上的数千个微调模型实验得到了验证。

【方法概述 / Method】

论文采用$L_2$-稳定性理论作为核心分析工具，将其扩展到异构微调超参数（如不同学习率、批次大小）的场景。通过建立模型参数平均$\boldsymbol{x}_{avg}$的稳定性分析框架，将超额风险分解为优化误差和泛化误差两个可解释分量，并引入稀疏化算子分析来证明非线性融合算法如何通过抑制任务异质性来收紧泛化界。

【实验结果 / Results】

在ResNet和ViT架构上跨越20个视觉分类任务的广泛实验中，涉及数千个微调模型，理论预测的缩放定律与实证泛化行为高度吻合。实验结果稳健地证实了：过训练专家确实会因优化-泛化权衡失衡而导致融合性能崩溃，而非线性稀疏化方法（TIES、DARE）通过降低任务异质性有效提升了合并模型的泛化性能。

【应用价值 / Applications】

该研究为模型融合实践提供了理论指导原则，使从业者能够基于定量缩放定律战略性地构建"融合友好型"专家模型，避免启发式调参。这一框架可广泛应用于多任务学习、联邦学习、大模型高效适配等场景，特别是在需要聚合多个异构微调模型能力的分布式机器学习系统中，实现无需原始数据和昂贵重计算的模型能力整合。
