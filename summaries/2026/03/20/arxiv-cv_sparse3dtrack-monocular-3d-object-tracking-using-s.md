【论文摘要 / Abstract】
本文提出了首个针对单目3D目标跟踪的稀疏监督框架Sparse3DTrack，解决了现有方法依赖密集3D标注的高成本问题。该方法将跟踪任务分解为2D查询匹配和3D几何估计两个子问题，利用时空一致性增强稀疏标注样本并学习丰富的2D/3D场景表征。模型能够自动生成高质量3D伪标签，将稀疏监督转化为密集3D轨迹标注，在KITTI和nuScenes数据集上实现最高15.50个百分点的性能提升，且每条轨迹最多仅需4个真实标注。

【方法概述 / Method】
该方法采用两阶段架构：首先通过2D查询匹配建立跨帧目标关联，然后进行3D几何估计恢复物体姿态。核心创新在于利用视频序列的时空一致性来扩充有限的标注数据，并基于学习到的表征自动生成3D伪标签，从而将稀疏监督扩展为密集监督信号。

【实验结果 / Results】
在KITTI和nuScenes数据集上的大量实验表明，该方法能显著提升跟踪性能，最高改进达15.50个百分点。关键优势在于极端稀疏标注场景下的有效性——每条轨迹仅需不超过4个真实3D标注即可达到接近全监督方法的性能。

【应用价值 / Applications】
该研究大幅降低了3D目标跟踪的数据标注成本，使现有全监督跟踪器能在标注资源受限的场景中有效部署。特别适用于自动驾驶、机器人导航等需要实时理解场景动态但难以获取密集3D标注的实际应用，推动了单目3D跟踪技术的可扩展性和实用性。
