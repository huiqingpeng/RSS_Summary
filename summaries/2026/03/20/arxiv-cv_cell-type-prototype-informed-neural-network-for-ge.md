【论文摘要 / Abstract】
本文提出了一种细胞类型原型引导神经网络（CPNN），用于从病理图像估计基因表达谱。现有方法将基因表达视为整体信号，忽略了其源于细胞水平表达聚合的生物学事实。CPNN利用公开的单细胞RNA测序数据估计细胞类型原型（反映稳定基因共变的平均表达谱），直接从图像学习细胞类型组成权重，并建模原型与观测表达之间的关系，实现了生物学可解释且结构正则化的预测框架。在三个全切片级别和三个空间转录组数据集上，CPNN均取得最优的Spearman相关性表现。

【方法概述 / Method】
CPNN首先利用单细胞RNA测序数据估计细胞类型原型，以捕获稳定的基因-基因共变关系；随后从病理图像直接学习细胞类型组成权重，并建立原型与观测到的整体或空间表达之间的映射关系。该方法通过引入细胞分辨率的先验指导，将生物学知识显式嵌入神经网络架构中。

【实验结果 / Results】
CPNN在三个全切片级别数据集和三个空间转录组数据集上进行了评估，在所有设置中均取得了最高的Spearman相关系数。此外，通过可视化推断的细胞类型组成权重，该框架能够提供可解释的生物学洞察，揭示哪些细胞类型驱动了预测的基因表达。

【应用价值 / Applications】
该研究为快速、低成本的分子分析提供了新工具，具有广泛的临床转化潜力，如肿瘤分子分型、预后评估和治疗反应预测。其可解释性特征有助于病理学家理解模型决策背后的细胞生物学机制，促进人工智能辅助诊断系统在精准医学中的可信应用。
