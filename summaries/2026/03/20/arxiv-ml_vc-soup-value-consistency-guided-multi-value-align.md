【论文摘要 / Abstract】
本文提出了VC-Soup，一种基于价值一致性学习的多价值对齐框架，用于解决大型语言模型在同时对齐多个潜在冲突的人类价值时面临的训练成本高昂和性能下降问题。该框架通过设计基于余弦相似度的价值一致性度量来筛选高质量偏好数据，训练出具有更好线性模式连接性的策略模型，并通过线性组合与帕累托过滤获得多价值平衡的最优解。实验和理论分析表明，VC-Soup能有效缓解价值冲突，持续优于现有的多价值对齐方法。

【方法概述 / Method】
VC-Soup采用三阶段方法：首先基于奖励差距向量与全一向量的余弦相似度定义价值一致性指标，量化跨价值的数据连贯性；然后过滤低一致性样本并训练价值一致的策略模型；最后通过线性参数合并与帕累托最优筛选获得多价值平衡解。该方法的核心洞察是价值一致的数据能诱导出保持线性模式连接性的平滑策略，使模型合并更加有效。

【实验结果 / Results】
实验表明VC-Soup在多个价值组合场景下均优于现有方法，包括奖励重加权、提示微调和传统模型合并等基线；理论分析证明了价值一致性与线性模式连接性之间的正向关系，为参数合并的有效性提供了理论支撑；该方法在保持各价值维度性能的同时，显著降低了为每种价值组合单独训练模型的计算开销。

【应用价值 / Applications】
VC-Soup适用于需要同时满足多元人类价值的LLM部署场景，如内容审核（兼顾安全性与有用性）、对话系统（平衡诚实与无害）和决策支持（协调公平与效率）；该框架通过数据筛选和参数复用大幅降低了多目标对齐的边际成本，为构建可扩展、可定制的可信AI系统提供了实用解决方案。
