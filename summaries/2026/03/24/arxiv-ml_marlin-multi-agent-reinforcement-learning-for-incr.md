【论文摘要 / Abstract】
本文提出了 MARLIN，一种基于多智能体强化学习的高效增量有向无环图（DAG）学习方法，用于从观测数据中挖掘因果结构。该方法通过DAG生成策略将连续实值空间映射到DAG空间，并整合状态特定和状态不变两个强化学习智能体来发现因果关系。实验表明，MARLIN在合成数据集和真实数据集上的效率和有效性均优于现有最先进方法，适用于在线应用场景。

【方法概述 / Method】
MARLIN采用双智能体强化学习架构：一个状态特定智能体和一个状态不变智能体协同工作以揭示变量间的因果关系。该方法利用DAG生成策略实现连续空间到离散DAG空间的映射作为批次内策略，并引入因子化动作空间设计以提升并行计算效率，最终构建成增量学习框架。

【实验结果 / Results】
在合成数据集和真实数据集上的大量实验表明，MARLIN在计算效率和因果发现准确性方面均显著优于现有最先进方法。该方法特别适合在线应用需求，能够在保持高精度的同时实现高效的增量式DAG学习。

【应用价值 / Applications】
MARLIN可应用于需要实时因果推理的在线决策系统，如金融风控、医疗诊断和工业流程优化等领域。其高效的增量学习能力使其能够适应动态变化的数据环境，为复杂系统的实时理解与干预提供可靠的因果结构发现工具。
