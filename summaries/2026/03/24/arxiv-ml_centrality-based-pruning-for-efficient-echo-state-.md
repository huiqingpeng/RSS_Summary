【论文摘要 / Abstract】
本文针对回声状态网络（ESNs）中随机初始化的储层存在冗余节点、导致计算开销过大的问题，提出了一种基于图中心性的剪枝方法。该方法将储层视为加权有向图，利用中心性度量识别并移除结构重要性较低的节点。实验表明，该方法能在显著减小储层规模的同时保持甚至提升预测精度，且有效保留了储层的核心动态特性。

【方法概述 / Method】
论文将ESN储层建模为加权有向图，采用图中心性度量（如度中心性、特征向量中心性等）量化各节点的结构重要性，并据此剪除低中心性节点。该方法无需重新训练网络，仅通过拓扑分析实现储层压缩，保持了ESN快速训练和硬件友好的特性。

【实验结果 / Results】
在Mackey-Glass混沌时间序列预测和电力负荷预测任务上的实验显示，所提方法可将储层规模减少30%-70%，同时预测精度与完整网络相当或更优。此外，剪枝后的网络在动态特性评估指标（如谱半径、记忆容量）上表现稳定，验证了其对储层核心功能的保留能力。

【应用价值 / Applications】
该方法适用于资源受限环境下的实时时间序列预测场景，如边缘设备上的电力负荷预测、物联网传感器数据分析和嵌入式系统中的信号处理。通过降低计算和存储需求，该研究有助于推动ESN在移动设备和工业物联网中的实际部署，提升 reservoir computing 方法的工程实用性。
