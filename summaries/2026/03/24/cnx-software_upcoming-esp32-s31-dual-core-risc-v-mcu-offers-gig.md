【是什么 / What it is】
乐鑫科技（Espressif Systems）正在开发一款名为 ESP32-S31 的新型无线微控制器，采用双核 RISC-V 架构（一个高性能核心+一个低功耗核心），集成千兆以太网、WiFi、蓝牙和 802.15.4 等多种连接功能，并配备 62 个 GPIO 引脚。
Espressif Systems is developing a new wireless microcontroller called ESP32-S31 featuring a dual-core RISC-V architecture (one high-performance core plus one low-power core), integrating Gigabit Ethernet, WiFi, Bluetooth, and 802.15.4 connectivity, along with 62 GPIO pins.

【为什么重要 / Why it matters】
ESP32-S31 是目前乐鑫最强大的无线 SoC，GPIO 数量创纪录，且首次在 ESP32 系列中引入千兆以太网，填补了高性能无线 MCU 与边缘计算之间的空白；其多协议无线支持（WiFi + 蓝牙 + Thread/Zigbee/Matter）使其成为物联网和智能家居生态的理想核心。
The ESP32-S31 represents Espressif's most powerful wireless SoC to date with a record number of GPIOs and the first Gigabit Ethernet in the ESP32 family, bridging the gap between high-performance wireless MCUs and edge computing; its multi-protocol wireless support makes it ideal for IoT and smart home ecosystems.

【我能用什么 / How I can use it】
可用于构建需要高带宽有线+无线混合连接的边缘网关、工业自动化控制器或 Matter/Thread 智能家居中枢；其 2D GPU 和 MJPEG 编解码能力也适合带屏幕的交互式终端设备开发。
Ideal for building edge gateways, industrial automation controllers, or Matter/Thread smart home hubs requiring hybrid high-bandwidth wired/wireless connectivity; the 2D GPU and MJPEG codec also suit interactive terminal devices with displays.
