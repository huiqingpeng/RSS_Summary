【论文摘要 / Abstract】
本文研究了文本属性图（Text-Attributed Graphs）中的后门攻击风险，提出了一种仅通过修改节点文本而不改变图结构的攻击方法TAGBD。该攻击通过识别易受影响的训练节点、利用影子图模型生成自然触发文本，并以替换或追加方式注入后门。实验表明该攻击在多个基准数据集上高度有效，可跨不同图模型迁移，且能抵御常见防御机制，揭示了文本单独作为图学习系统攻击渠道的可行性。

【方法概述 / Method】
TAGBD采用三阶段攻击流程：首先基于节点影响力分析筛选关键训练节点，然后利用影子图模型生成语义自然、与原始文本风格一致的触发文本，最后通过文本替换或短句追加的方式隐蔽注入后门触发器。该方法完全依赖文本操作，无需修改图拓扑结构。

【实验结果 / Results】
在三个基准数据集上的实验显示，TAGBD攻击成功率极高，且能有效迁移至GCN、GAT等不同图神经网络架构；即使面对基于异常检测和鲁棒训练的常见防御手段，攻击仍保持较强效力，证明了其隐蔽性和鲁棒性。

【应用价值 / Applications】
该研究对社交媒体分析、学术文献分类、推荐系统等依赖开放文本平台的图学习应用具有重要安全警示意义，提示开发者在设计防御策略时需同时关注图结构异常和节点文本内容的安全性审查。
