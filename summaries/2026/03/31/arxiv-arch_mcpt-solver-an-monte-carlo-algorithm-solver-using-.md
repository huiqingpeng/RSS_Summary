【论文摘要 / Abstract】
本文提出了MCPT-Solver，一种基于自旋器件（MTJ）的蒙特卡洛粒子传输问题专用硬件求解器。针对传统冯·诺依曼架构在执行随机算法时因分支预测和 irregular 内存访问导致的效率低下问题，该工作利用磁性隧道结（MTJ）的随机翻转特性构建高质量真随机数生成器（TRNG），并通过贝叶斯推断网络实现概率可调输出。实验表明，该系统在求解输运问题时均方误差达7.6e-6，相比通用处理器实现显著加速，同时达到185 Mb/s的吞吐量和8.6 pJ/bit的能效，成为首个兼具工艺-电压-温度容限与概率可调特性的自旋TRNG。

【方法概述 / Method】
MCPT-Solver 采用三层架构设计：首先利用MTJ器件的随机开关特性作为熵源构建TRNG，通过优化控制逻辑和写入机制提升生成吞吐量和PVT稳定性；其次设计硬件贝叶斯推断网络实现输出概率的动态调节；最后构建系统级仿真框架完成端到端评估。该方法将物理随机性与概率计算网络深度融合，实现了从随机数生成到概率调控的完整硬件 pipeline。

【实验结果 / Results】
在蒙特卡洛粒子传输问题求解中，MCPT-Solver 达到7.6e-6的均方误差，展现出优异的数值精度；性能指标方面，系统吞吐量为185 Mb/s，面积效率为27.8 μm²/bit，能耗低至8.6 pJ/bit。与通用处理器相比，该专用硬件实现了显著的加速效果，同时在工艺、电压、温度变化条件下保持稳定输出，突破了现有自旋TRNG在可调性和鲁棒性方面的局限。

【应用价值 / Applications】
MCPT-Solver 主要面向科学计算中的蒙特卡洛粒子输运模拟（如核反应堆物理、医学放疗剂量计算、天体辐射传输等），可大幅提升这类随机算法的执行效率。其概率可调TRNG架构还可扩展至概率计算、贝叶斯机器学习、金融蒙特卡洛模拟等需要高质量可控随机数的领域，为后摩尔时代随机计算硬件提供了高能效、高鲁棒性的解决方案。
