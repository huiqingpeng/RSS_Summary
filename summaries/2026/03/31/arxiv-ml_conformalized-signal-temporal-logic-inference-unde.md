【论文摘要 / Abstract】
本文提出了一种在协变量偏移（covariate shift）条件下进行信号时序逻辑（STL）推理的共形化框架。该框架通过估计训练分布与部署分布之间的似然比，并将其整合到基于STL鲁棒性的加权共形预测方案中，从而在分布偏移情况下提供有效性保证。实验结果表明，该方法在保持STL公式可解释性的同时，显著提升了部署时的符号学习可靠性。

【方法概述 / Method】
该方法首先采用无模板、可微分的STL推理方法学习初始模型，然后利用有限的部署端数据集进行细化以促进分布对齐。技术上，通过估计训练分布与部署分布的似然比，构建加权共形预测机制，实现对STL公式鲁棒性的不确定性量化。

【实验结果 / Results】
论文在轨迹数据集上进行了实验验证，结果表明所提框架能够有效处理训练与部署数据之间的协变量偏移问题。相较于传统方法，该框架在分布偏移场景下显著提高了STL推理的可靠性和有效性保证，同时保留了STL公式的可解释性优势。

【应用价值 / Applications】
该研究适用于自动驾驶、机器人控制、工业过程监控等动态系统的时序行为建模与验证场景。在实际部署环境中，当系统面临与训练阶段不同的运行条件时，该方法能够提供可靠的逻辑规则学习和不确定性量化，增强安全关键系统的可信度和鲁棒性。
