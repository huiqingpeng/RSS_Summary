【论文摘要 / Abstract】
本文提出了一种将图像溯源信息（provenance）直接嵌入图像文件的新方法，以解决计算机视觉领域数据溯源信息通常单独存储导致信息丢失的问题。作者设计了一个基于JSON-LD（JavaScript Object Notation for Linked Data）的新型结构化模式，用于管理图像捕获设置、数据预处理步骤及模型架构等关键元数据。该方法确保溯源信息与图像本身内在绑定，提升了系统的可维护性和适应性，并与现有标准保持一致。

【方法概述 / Method】
论文采用JSON-LD格式构建图像溯源信息的结构化模式，将元数据直接嵌入图像文件内部而非外部存储。该模式借鉴并链接到已建立的标准，实现了图像描述与健壮模式的对齐，保证了溯源信息与视觉资源的直接关联。

【实验结果 / Results】
论文主要聚焦于方法论框架的提出，未详细报告具体的量化实验结果或性能基准测试。核心贡献在于验证了JSON-LD嵌入方案在技术可行性上的优势，以及该方案在防止信息丢失、支持合规审计和提升数据可重用性方面的设计有效性。

【应用价值 / Applications】
该研究适用于需要严格数据溯源管理的工业级计算机视觉应用场景，如医疗影像分析、自动驾驶数据集管理和AI模型合规审计等。通过确保数据来源透明可追溯，该方案有助于提升下游模型的可信度、简化数据维护流程，并支持机器学习生命周期中的监管要求。
