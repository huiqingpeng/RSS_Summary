【论文摘要 / Abstract】
本文研究了在成本、GPU资源和并发约束下将查询路由到大型语言模型（LLM）的问题。针对现有逐查询路由方法难以控制批次级成本、尤其在非均匀或对抗性批次下失效的缺陷，作者提出了批次级资源感知路由框架，联合优化每个批次的模型分配，并引入鲁棒变体以应对LLM性能预测的不确定性。实验表明，该方法在严格约束成本与GPU资源的同时，显著提升了路由准确性和吞吐量。

【方法概述 / Method】
该框架采用批次级优化策略，将查询按批次进行统一路由决策，而非逐查询独立处理；通过引入鲁棒优化变体，显式建模性能预测的不确定性，并设计离线实例分配程序，在多个模型间平衡质量与吞吐量。

【实验结果 / Results】
鲁棒版本相比非鲁棒方法准确率提升1-14%（取决于性能估计器）；在对抗性批次条件下，批次级路由比逐查询方法性能提升高达24%；优化后的实例分配相比非优化分配额外带来最高3%的收益，同时严格满足成本和GPU资源约束。

【应用价值 / Applications】
该研究适用于多LLM服务部署场景，如云计算平台的智能模型调度、企业级LLM API网关优化，以及资源受限环境下的高效推理服务，能够在保证服务质量的前提下显著降低运营成本并提升系统吞吐量。
