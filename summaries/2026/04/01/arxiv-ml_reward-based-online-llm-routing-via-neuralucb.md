【论文摘要 / Abstract】
本研究探索了使用 NeuralUCB 算法进行成本感知的大型语言模型（LLM）路由问题。现有路由方法主要分为监督式路由和部分反馈方法，两者在效率和适应性上存在不同权衡。作者实现了基于 NeuralUCB 的路由策略，并在 RouterBench 的模拟在线环境中进行评估，结果表明该方法在效用奖励方面持续优于随机和最小成本基线，同时相比最大质量参考方法显著降低了推理成本。

【方法概述 / Method】
论文采用 NeuralUCB（神经网络上置信界）算法构建在线 LLM 路由策略，将路由问题建模为上下文多臂老虎机问题，利用神经网络近似奖励函数并结合不确定性估计进行探索-利用权衡。该方法在每次交互中根据查询特征动态选择最优 LLM，并通过在线反馈持续更新策略。

【实验结果 / Results】
实验在 RouterBench 基准的模拟在线设置下进行，NeuralUCB 方法在效用奖励指标上稳定超越随机路由和最小成本基线；与始终选择最高质量模型的参考策略相比，该方法在保持竞争力奖励的同时大幅降低了推理成本。研究同时揭示了动作区分度和探索机制方面仍存在待解决的挑战。

【应用价值 / Applications】
该研究为 LLM 路由系统提供了一种高效且自适应的在线学习方案，适用于需要根据查询难度动态分配不同能力（及成本）模型的实际场景，如智能客服、内容生成等需要平衡质量与成本的商业应用，有助于降低大规模 LLM 部署的运营开销。
