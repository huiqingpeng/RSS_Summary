【论文摘要 / Abstract】
本文是一篇立场论文，将多智能体系统的记忆需求重新框定为计算机架构问题。作者区分了共享内存与分布式内存两种范式，提出了包含I/O、缓存和内存的三层记忆层级结构，并识别出两个关键协议缺口：跨智能体的缓存共享和结构化内存访问控制。论文指出多智能体记忆一致性是最紧迫的开放挑战，为构建可靠、可扩展的多智能体系统提供了架构基础。

【方法概述 / Method】
本文采用计算机架构的经典视角分析多智能体记忆问题，借鉴了传统计算机系统中的内存层次结构设计理念。作者将多智能体记忆系统映射为三层架构（I/O层、缓存层、内存层），并引入分布式系统与共享内存系统的对比分析框架，以识别现有LLM多智能体系统在记忆管理方面的协议缺陷。

【实验结果 / Results】
作为一篇立场/愿景论文，本文未报告具体实验结果或定量性能指标。其核心贡献在于提出概念框架和识别研究挑战，特别是明确指出多智能体记忆一致性（multi-agent memory consistency）是当前领域最紧迫的开放性问题，为后续研究指明了方向。

该研究为大规模LLM多智能体协作系统的设计提供了理论基础，可应用于需要多智能体协同工作的场景，如自动化软件开发、科学研究协作、复杂决策支持系统等。通过建立标准化的记忆架构，有望解决当前多智能体系统中因记忆管理混乱导致的可靠性问题，推动从原型系统向生产级应用的转化。
