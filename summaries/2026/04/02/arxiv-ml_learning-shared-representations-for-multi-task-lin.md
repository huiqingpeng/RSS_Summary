【论文摘要 / Abstract】

本文研究了多任务线性赌博机中的共享表示学习问题，其中T个并发的d维线性赌博机任务共享一个r维的低维潜在表示（r ≪ min{d,T}）。作者提出了一种新的基于不确定性的乐观线性（OFUL）算法，通过谱初始化估计共享模型，并在新构建的置信集上进行学习。理论分析表明，该算法的累积遗憾界为Õ(√drNT)，显著优于独立求解T个任务时的Õ(dT√N)遗憾界。

【方法概述 / Method】

论文提出的算法包含三个阶段：首先通过探索阶段收集数据，然后利用谱方法初始化估计共享低秩表示，最后基于新构建的置信集执行OFUL学习。核心思想是将多任务之间的相关性建模为共享的低维潜在空间，并通过联合学习实现知识迁移。

【实验结果 / Results】

理论分析证明了所构建置信集的高概率保证，并推导了改进的累积遗憾界Õ(√drNT)，与独立学习相比实现了√(d/r)倍的加速。数值模拟实验验证了算法在不同问题规模下的有效性，证实了理论预测的性能提升。

【应用价值 / Applications】

该研究适用于需要同时处理多个相关决策任务的场景，如推荐系统中多个用户的个性化推荐、在线广告中多广告位的投放优化，以及医疗试验中多治疗方案的序列决策等，能够显著提高样本效率和决策质量。
