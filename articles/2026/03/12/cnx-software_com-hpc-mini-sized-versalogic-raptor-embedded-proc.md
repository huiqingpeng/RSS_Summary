VersaLogic Raptor (VL-EPU-5530) Embedded Processing Unit is powered by a 13th Gen Intel Core i5-1345URE or Processor U300E SoC and designed for high-performance computing applications in harsh environments with a -40 to +85°C temperature range and MIL-STD-202H shock and vibration qualification.
The COM HPC Mini-sized (95 x 80 x 36mm) board supports up to 64 GB of soldered-down LPDDR5 memory, features vertical USB4, USB 3.0 Type-A, and mini DisplayPort connectors, two 2.5 Gbps Ethernet ports with TSN support, and three M.2 expansion sockets for NVMe SSDs, AI accelerators, or other expansion modules.
Specifications:
- Raptor Lake SoC (one or the other)
- Intel Processor U300E 5-core/6-thread processor with 1x P-core @ 1.1 GHz / 4.3 GHz (Turbo) 4x E-cores up to 3.2 GHz (Turbo), 8MB Cache, 48EU Intel UHD Graphics up to 1.1 GHz; PBP: 15W
- Intel Core i5-1345URE 10-core/12-thread processor with 2x P-cores @ 1.4 GHz / 4.6 GHz (Turbo), 8x E-cores up to 3.4 GHz (Turbo), 12MB cache, 80 EU Intel Iris Xe graphics @ 1.25 GHz; PBP: 15W
- System Memory – Up to 64GB LPDDR5x
- Storage – Up to three M.2 NVMe SSDs (See Expansion section)
- Display Interfaces
- 1x Mini DP++ with support for HDMI signaling (video and audio)
- 1x USB4 DP Type-C ports
- Networking – 2x 2.5GbE ports with latching connector, TSN support
- USB
- 2x USB4 Type-C ports
- 2x USB 3.2 Type-A ports
- Up to 8x USB 2.0 interfaces
- Serial
- 2x RS-232/422/485 COM ports
- Optional CAN Bus through M.2 M key expansion card
- Expansion
- 3x M-Key sockets (2x 2242, 1x 2280)
- 8x GPIO, 1x I2C
- Analog I/O – ADC through M.2 M key expansion card
- Power Supply – 10V to 15V DC
- Dimensions – 95 x 80 x 36 mm (~COM-HPC Mini form factor)
- Weight – 265 grams
- Temperature Range – -40°C to +85°C
- Shock and Vibration – MIL-STD-202H
- Rated for operation at 15,240 meters (50,000 feet) altitude
Versalogic provides support for Ubuntu 64-bit (latest version) and Windows 10/11 64-bit for the board. The company also released the VersaAPI Library of API calls for reading and controlling on-board devices using Visual Studio and C/C++ software development interfaces on Windows and Linux.
The Raptor’s small size makes it suitable for drones and other uncrewed systems (UxVs). The board can be expanded with a range of accessories such as single and dual GbE M.2 cards,
a 2x UART + GPIO M.2 module, various NVMe SSDs, display adapters (e.g., mDP to HDMI), and so on. It provides an even smaller alternative to DFI RPP051 Pico-ITX SBC, also offered with a range of 13th Gen Intel Core embedded SoCs, starting from the Intel Processor U300E.
Versalogic Raptor development samples will be available by the end of July 2026, and the company promises long-term availability with a 10+ year typical production lifecycle. However, pricing information has yet to be provided. Check out the product page and press release for more details.
